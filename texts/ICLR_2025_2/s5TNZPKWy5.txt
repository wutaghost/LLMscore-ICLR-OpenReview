009                                              A BSTRACT
010
011            Referring image segmentation (RIS) aims to precisely segment a target ob-
012            ject described by a linguistic expression. Recent RIS methods have introduced
013            Transformer-based networks that use vision features as query and linguistic ex-
014            pression features as key-value to find target regions by referring to the given lin-
015            guistic information. Since the Transformer-based network predicts based on the
016
               guidance information that guides the network on which regions to pay attention,
               the capacity of this guidance information has a significant impact on segmentation
017
               results in Transformer-based RIS. However, existing methods rely only on lin-
018
               guistic tokens as the guidance elements, which are limited in providing the visual
019            understanding of the fine-grained target regions. To address this issue, we present
020            a novel Multi-Expression guidance framework for Transformer-based Referring
021            Image Segmentation, M ETRIS, which allows the network to refer to the visual
022            expression tokens as the guidance information alongside the linguistic expression
023            tokens. The introduction of visual expression can complement the capability of
024            linguistic guidance by effectively providing the target-informative visual contexts.
025            To generate the visual expression from vision features, we introduce a visual ex-
026            pression extractor that is designed to endow with the target-oriented visual guid-
027
               ance ability and to acquire rich contextual information. This module strengthens
               the adaptability to the diverse image and language inputs, and improves visual un-
028
               derstanding of the fine-grained target regions. Extensive experiments demonstrate
029
               the effectiveness of our approach across the commonly used RIS settings and the
030            generalizability evaluation settings. Our method consistently shows strong perfor-
031            mance on three public RIS benchmarks.

034   1   I NTRODUCTION
035
036   Referring image segmentation (RIS) (Hu et al., 2016; Chen et al., 2022) is one of the challenging
037   vision-language tasks (Yan et al., 2023; Ghosh et al., 2024; Chen et al., 2024b; Hu et al., 2024),
038   and can be applied in various applications such as human-robot interaction and the object retrieval.
039
      Given an image and a natural language expression describing a target object within the image, one of
      the key points in this task is for the network to precisely segment the target object regions from the
040
      image by referring to the given expression. With the great success of Transformer-based networks
041
      (Vaswani, 2017; Dosovitskiy et al., 2020) in single modal segmentation tasks (Qian et al., 2023;
042   Zhou & Wang, 2024; Liu et al., 2024b), Transformer-based methods have been actively studied on
043   RIS task. To find specific regions by referring to the given information, RIS models use vision
044   features as query and the given information as key-value in the Transformer network, as shown in
045   Figure 1; the set of such information provided to the Transformer network as key-value is called
046   Guidance Set in this paper. Specifically, the role of the guidance set is to guide the network on
047   which regions to focus its attention, and the network predicts target regions based on the guidance
048   information. Motivated by this fact, we focus on that enhancing the capability of the guidance
049   set has a significant impact on segmentation performance in Transformer-based referring image
050
      segmentation.
051   Most previous works have approached this task by directly enhancing the language features to im-
052   prove the comprehension for the language expression. Some of these studies (Ding et al., 2022a; Hu
053   et al., 2023) obtain the enhanced linguistic features by allowing language features to refer to vision
      features via the language-vision cross-attention mechanism (Figure 1 (b)). More recent studies (Lai

071   et al., 2023; Liu et al., 2024a) to improve the understanding of the language expression via LLM’s
072   immense knowledge, and exploit the generated language token in the segmentation network (Figure

074   these enhanced linguistic features as key-values in Transformer-based segmentation networks.
075   Despite these advancements, all these methods rely on the linguistic-based tokens as elements of
076   the guidance set, as depicted in Figure 1. Since these tokens are insufficient to capture the visual
077   contexts, these linguistic-based tokens are limited in providing the target-informative visual under-
078   standing that helps guide the network to the target areas composed of the fine-grained regions with
079   different visual characteristics. For example, in Figure 2, the network guided by only linguistic-
080   based tokens segments only part of the target regions (i.e., 2a.A) or segments even non-target regions
081
      (i.e., 2a.B). To address this issue, we explore the introduction of the visual expression tokens that can
      complement the guidance capability of linguistic information by providing the target-informative vi-
082
      sual information.
083
084   In this paper, we propose a novel Multi-Expression guidance framework for Transformer-based Re-
085   ferring Image Segmentation, M ETRIS, which enables the network to refer to the extended guidance
086   set composed of the visual expression as well as the linguistic expression. The proposed frame-
087
      work is distinct from previous studies in that we produce the visual expression tokens equipped
      with target-informative visual guidance capability to enhance the capacity of the guidance set and
088
      to avoid relying only on the linguistic guidance, as illustrated in Figure 1. The visual expression
089
      tokens address the lack of the guidance capacity of language-based tokens by effectively providing
090   the visual contexts of the target regions, as shown in Figure 2a. To the best of our knowledge, our
091   approach is the first to generate the visual expression as a provider of the target guidance informa-
092   tion, deviating from the previous approach in that only language-based tokens can fulfill the role of
093   providing the target information to the network.
094
      Furthermore, we design a visual expression extractor from the terms of two points to generate the vi-
095
      sual expression from vision features. To qualify as an ‘expression’ in this task, the following points
096   are required: (1) It needs to concentrate more on the semantic information relevant to the target
097   regions from the image context, because the image context contains both target and non-target in-
098   formation and these distracting non-target information hinders the guidance capability (Chen et al.,
099   2024a). Thus, our module endows with the target-oriented visual guidance ability by selectively
100   exploiting the informative visual tokens and adaptively refining the curated visual information. (2)
101   It needs to capture rich visual contexts of the target regions. For this, our module considers both
102   comprehensive context and distinct attribute contexts by exploiting the global-local linguistic cues
103   (i.e., sentence-level and word-level cues), where each of linguistic cues has different contextual in-
104
      formation, and allows to acquire the relationship between each visual token. This design strengthens
      the model’s adaptability to diverse language and image inputs for robust segmentation, and improves
105
      the visual understanding of the fine-grained target regions.
106

136   In a nutshell, our contributions can be summarized in three-fold:
137
             • We propose M ETRIS, a novel Multi-Expression guidance framework for Transformer-
138
               based Referring Image Segmentation, which enables the introduction of visual expression
139            as elements of the guidance set alongside linguistic expression to enhance the robustness
140            of the guidance set. The visual expression addresses the lack of the guidance capacity of
141            linguistic information by effectively providing the target-informative visual contexts. Our
142            approach is the first to explore the potential of the visual expression as a provider of target
143            guidance information in Transformer-based referring image segmentation.

145            to endow with target-oriented visual guidance ability and to capture rich visual contexts of

147          • We extensively validate our approach across the commonly used RIS settings and the
148            generalizability evaluation settings, demonstrating the effectiveness of our framework for
149            Transformer-based referring image segmentation. Our method consistently shows strong
150            performance and surpasses the state-of-the-art methods on three public RIS benchmarks.

161   learnable tokens as guidance elements, which is fine-tuned based on the language expression, to
      extract diverse linguistic representations. Several methods (Kim et al., 2022; Ding et al., 2022a; Hu

      3     M ETHOD
195
196
      We propose a novel multi-expression guidance framework for Transformer-based referring image
197
      segmentation, M ETRIS, to avoid relying on linguistic guidance. Figure 3 shows the overall frame-
198   work. We first describe the vision and language feature extraction (Sec.3.1), and then introduce a
199   visual expression extractor (Sec.3.2). Finally, we explain a segmentation decoder (Sec.3.3).
200
201   3.1   V ISION AND L ANGUAGE F EATURE E XTRACTION
202
203   Given an image I and a linguistic expression Q that consists of L − 1 words, a vision encoder
204   extracts the vision features Fi ∈ RHi Wi ×Ci at each stage i ∈ {1, 2, 3, 4} and a language encoder
205   extracts the linguistic expression tokens QL = [qcls , q1 , ..., qL−1 ] ∈ RL×D . Note that Hi , Wi , Ci
206   and D denote the height, width, channel dimension of the feature maps at the ith vision stage, and
207   the channel dimension of linguistic features. The first token qcls of linguistic expression features
208   indicates a special [CLS] token, which is the global representation that understands the linguistic
209   expression at the sentence level. The word token qj indicates the local representation of j th word.
210

288   3.3     S EGMENTATION D ECODER
289
290   To segment the target region, the decoder leverages the guidance set G = {Q          b V } composed of
                                                                                      bL , Q
291
      the enhanced linguistic expression tokens and the visual expression tokens. The decoder can focus
      its attention on more precise target regions thanks to the target-informative visual guidance. At each
292
      decoder stage, the cross-attention layer, which uses the vision features as the query and the guidance
293
      tokens as the key-value, is employed to highlight the target regions. The vision decoder features
294   are then upsampled and concatenated with the corresponding vision encoder features to feed into
295   the next decoder stage. The final segmentation map is projected to a binary class mask by a linear
296   projection layer. The binary cross-entropy loss is used for the network training.

      Experimental settings. The vision encoder is Swin-B (Liu et al., 2021) initialized with the pre-

      trained weight on ImageNet-22K (Krizhevsky et al., 2012), and the language encoder is BERT-base
304
      (Devlin et al., 2018) initialized with the official pre-trained weight of the uncased version. The
305   decoder was randomly initialized. We trained models for 40 epochs with 16 batch size on 24G
306   RTX3090 GPUs. More details for settings are in Appendix A.
307
308
      Datasets. RefCOCO (Yu et al., 2016) and RefCOCO+ (Yu et al., 2016) are widely utilized datasets
      for referring image segmentation. RefCOCO contains 19,994 images with 142,209 language ex-
309
      pressions for 50,000 objects, and RefCOCO+ contains 19,992 images with 141,564 expressions
310
      for 49,856 objects. The expressions in RefCOCO+ do not include words about absolute locations,
311   which makes it more challenging than RefCOCO. For RefCOCO and RefCOCO+, the target object
312   category of the testA subset is mostly a person, and the target object of the testB subset consists
313   of all other object categories. G-Ref (Mao et al., 2016; Nagaraja et al., 2016) is also a commonly
314   used dataset, which contains 26,711 images with 104,560 language expressions for 54,822 objects.
315   G-Ref, which is the most challenging dataset, has more complex and longer expressions than Ref-
316   COCO and RefCOCO+.

318   (oIoU), mean intersection-over-union (mIoU), and precision at 0.5, 0.7 and 0.9 thresholds.

519   4.6   C ONCLUSION
520
521   We propose a novel Multi-Expression guidance framework for Transformer-based Referring Image
522   Segmentation, M ETRIS, which enables the introduction of the visual expression as elements of the
523   guidance set alongside the linguistic expression to enhance the robustness of the guidance capabil-
524   ity. Our approach explores the potential of the visual expression as a provider of target guidance
525
      information, beyond the previous approach in that only language-based tokens can fulfill the role of
      providing target-informative guidance information. The visual expression complements the capabil-
526
      ity of linguistic guidance by effectively providing the target-oriented visual guidance. To produce
527
      semantic visual expression, we present a visual expression extractor that is designed to endow with
528   the target-informative visual guidance ability and to acquire the rich contextual information of target
529   regions. This enhances the adaptability to diverse image and language inputs, and improves visual
530   understanding of the fine-grained target regions. Extensive comparisons and ablations demonstrated
531   the effectiveness of our approach for Transformer-based referring image segmentation.
532
      Limitation and Future Work. Despite M ETRIS’s stronger ability to understand the visual con-
533
      texts of the target regions than LLM-based models, our model showed lower performance on the
534   most challenging dataset (G-Ref), which consists of the difficult language samples. This means
535   that our model lacks the reasoning ability for the implicit and detailed descriptions in comparison
536   to the LLM-based models. This finding suggests that our performance bottleneck may still lie in
537   understanding the language expressions on this task, while our model has better performance than
538   the existing state-of-the-art Transformer-based RIS models in Table 1. Therefore, future work could
539   have a broader impact on this task via the exploration of combining our approach’s strength with the
      LLM’s strength, beyond relying on the LLM’s capability.

774   A      A DDITIONAL I MPLEMENTATION D ETAILS
775
776   Experimental Settings. Our method was implemented in PyTorch (Paszke et al., 2019). We used
777   the AdamW (Loshchilov & Hutter, 2017) optimizer with initial learning rate of 3e-5 and adopted
778   the polynomial learning rate decay scheduler. The input image resolution was 480×480. For gRef-
779   COCO that contains no-target samples, we used a no-target classifier (Liu et al., 2023a).
780
      Evaluation Metrics. Following previous works, we adopted the overall intersection-over-union
781   (oIoU), mean intersection-over-union (mIoU), and precision at 0.5, 0.7 and 0.9 thresholds. The oIoU
782   is the ratio between the total intersection regions and the total union regions of all test samples. The
783   mIoU is the average of IoUs between the predicted mask and the ground truth of all test samples.
784   The precision is the percentage of test samples that have an IoU score higher than a threshold.
785
786
      B      A DDITIONAL D ETAILS FOR G ENERALIZATION S ETTING
787
788
      To further validate the generalization ability of our model, we experimented on the generalization
789   setting introduced by (Tang et al., 2023). These setting splits the RIS datasets into the seen and
790   unseen categories on MSCOCO (Lin et al., 2014) of the open-vocabulary detection (Zareian et al.,
791   2021). The training set contains GT masks for only seen categories, and the test set consists of the
792   seen subset and the unseen subset. Following the previous work (Tang et al., 2023), we adopted the
793   text encoder of CLIP (Radford et al., 2021) as the language encoder for a fair comparison in this
794   experiment, and trained our model for 50 epochs.

796   C      A DDITIONAL D ETAILS FOR DATASETS
797
798   RefCOCO & RefCOCO+. These two datasets are distributed under the Apache-2.0 license, and
799   are collected from the two-player game (Yu et al., 2016). The evaluation sets of RefCOCO and
800   RefCOCO+ are splitted into the validation subset, the test A subset and the test B subset. The
801   images of the testA subset contain the multiple people, and the images of the testB subset contain
802   the multiple instances of all other objects. RefCOCO+, which forbids the words about the absolute
803   locations in the language expressions, is more challenging than RefCOCO.
804   G-Ref. This dataset is distributed under the CC-BY 4.0 license, and is collected on Amazon Me-
805   chanical Turk. We use both UMD (Nagaraja et al., 2016) and Google (Mao et al., 2016) partitions
806   for the evaluation. The UMD partition splits the evaluation set into the validation subset and the
807   test subset. The Google partition consists of only the validation set. The average length of the
808   language expressions is 8.4 words. This means that the G-Ref dataset contains longer and more
809   complex language expressions than the RefCOCO and RefCOCO+ datasets. Thus, G-Ref is the
      most challenging dataset.

833
      Supervision by the contrastive loss. In Table 8 (a), we experimented on supervising the relevance
      score map by the pixel contrastive loss (Eq.4). This result indicates that the contrastive loss helps
834
      to monitor the curation of the informative tokens associated with the correct target region and to
835
      prevent the high relevance scores between the linguistic features and incorrect regions.
836
837   Normalization with dynamic mask. We ablated on applying a softmax normalization with the
838   dynamic mask to the relevance scores (Eq.5). In Table 8 (b), normalizing without the dynamic
839
      mask showed a significant performance drop. This indicates that using the curated visual tokens is
      beneficial for robust segmentation than using all visual tokens including the distracting tokens.
840
841   The use of the meaningless words. we experimented the ablation on the use of the article tokens
842   such as “the”, “a” and “an”, which are meaningless words in the input sentence, in the process of
843   collecting informative visual regions. As shown in Table 9, compared to using all word tokens, ‘w/o
844   article’ resulted in 0.42%, 0.58% and 0.54% drops in oIoU on each dataset, respectively. These
845
      results indicate that the article tokens do not carry the noise information, and using all word tokens
      as linguistic cues are more effective at collecting the informative visual tokens. Since the relations of
846
      each word are considered during encoding the language input to capture the contextual information
847
      for the target object description, each language token is encoded with semantic representations to
848   guide to the target object.

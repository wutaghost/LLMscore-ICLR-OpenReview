010                                                A BSTRACT
011
012              For providing timely warnings and preventing potential damages, it is crucial to
013              detect anomalous actions that threaten public safety through surveillance cameras.
014              Compared to normal actions, anomalous actions often occupy only a small por-
015              tion of surveillance videos and exhibit more complex manifestations in terms of
016              time and space. Considering that normal action recognition methods fail to high-
017              light crucial information from small-sized patches, resulting in imprecise anomaly
018
                 modeling, we propose the Spatio-Temporal Key Patch Selection Network (SKPS-
                 Net). To tackle the challenge of detecting anomalous behaviors that manifest in
019
                 small and inconspicuous areas, we design a spatial adaptive key patch selection
020
                 module to select small but informative patches on input videos. Furthermore, the
021              long-short feature map spatio-temporal relation module is devised to make the key
022              patch effectively capture the continuous dynamic changes of anomalous actions.
023              Finally, we propose a spatio-temporal refined loss to reinforce fine-grained feature
024              learning. Experiments conducted on the HMDB51, Kinetics, and UCF-Crime v2
025              datasets demonstrate that our SKPS-Net achieves state-of-the-art performance in
026              few-shot action recognition, outperforming the most competitive methods by 1.2%
027              on the anomalous action dataset UCF-Crime v2.

030   1      I NTRODUCTION
031
032   Anomalous actions, such as fights, arson, robbery, etc., can pose significant threats to public safety.
033
      Surveillance videos often serve as the initial source for capturing anomalies. By recognizing the
      anomalous actions in the video, appropriate safety measures can be swiftly formulated based on
034
      the nature of the detected activity. In this paper, an autonomous method to make anomalous action
035
      recognition is proposed.
036
037   There are many difficulties in recognizing anomalous actions. To begin with, anomalous action typ-
038   ically manifests in a more intense and irregular way(Zhou et al., 2019), which requires more precise
039
      features for recognition. However, many existing models only consider the global feature(the fea-
      ture of the whole input frame)(Wang et al., 2021b; Perrett et al., 2021; Nguyen et al., 2022), and
040
      some of them are employed for anomalous action recognition. The objects in the anomalous ac-
041
      tion, including the person and the car, can be far from the surveillance camera and have a relatively
042   small size, which means the discriminative information only exists in small local patches(Xiao et al.,
043   2023b;a). These patches are called key patches due to the fact that they make more contributions
044   to the recognition task. As a result, it is hard to model the anomalous action only using the global
045   feature. Besides, the video contains complicated spatio-temporal changes, like continuous walking
046   and abrupt body movements, which are also critical details for anomalous action recognition. Con-
047   sequently, the selection of the key patch needs to rely on spatio-temporal information(Wang et al.,
048   2021a; 2022b).
049   Considering that vital information is concentrated in the local key patches, which makes the global
050   feature modeling of the anomalous action difficult, our work introduces spatial adaptive key patch
051   selection to enhance global feature representation. Using the object detection network to locate
052   the patch will highly raise the cost, so the lightweight module is proposed in this paper. Unlike
053   some methods that locate patches using the global feature vector (Wang et al., 2021a; 2022b), we
      suggest that the feature map, which retains two-dimensional spatial information, is more effective

073   Moreover, the patch selection also need to incorporate spatio-temporal information to capture the
074   detailed changes in the video. Whereas other methods focus on the patch-level information within
075   the individual frame(Thatipelli et al., 2022), we propose the long-short feature map spatio-temporal
076   relation module. Spatio-temporal modeling is usually considered at different scales(Wang et al.,
077   2021b; Jiang et al., 2019): long-range temporal aggregation and short-range motion. The existing
078   networks for spatio-temporal modeling, like 3D convolutional network (3D CNN) and Flownet, are
079   hard to integrate into the framework due to their large size. We use the lightweight module based on
080   2D convolution consisting of two submodules to make temporal and motion relations. With more
081
      than individual frame information, the related feature map can focus more on the patch where the
      action happens.
082
083   After the feature enhancement, the proper loss function is required to improve the generalization
084   ability. The simple loss function of few-shot learning mainly comes out of the final output features
085   of the network. We propose to calculate the loss with more refined spatial and temporal features.
086   The loss includes features from more levels and helps improve the feature robustness.
087   The spatio-temporal key patch selection network (SKPS-Net) for anomalous action recognition is
088   constructed, as can be seen in Fig.1. It uses few-shot learning to solve the lack of anomalous action
089   data. The main contributions of our paper can be summarized as follows:
090
      (1)The plug-and-play spatial adaptive key patch selection module is proposed to highlight the vital
091   but less obvious local information. It utilizes the spatial information within the feature map to
092   adaptively select the key patch, modeling the patch-level anomalous action.
093
094
      (2)The long-short feature map spatio-temporal relation module is proposed to enrich the information
      for selection in (1). It relates the long-range temporal information and short-range motion informa-
095
      tion of feature maps in the same video, enabling better localization of the key patch in videos with
096
      sophisticated spatio-temporal changes.
097
098   (3)The spatio-temporal refined loss function uses multi-level features for the recognition task, im-
099   proving the generalization capability. It conforms to the enhanced feature after (2) and (1), jointly
100
      learning features from different subspaces, and improves class-specific temporal discriminability
      using frame-level features.

106
107   We propose the SKPS-Net for anomalous action recognition. While the global feature vector is
      widely used, we try to exploit the potentiality of the global feature map to take advantage of features

      at multiple levels. In more detail, the network enriches the spatio-temporal context by connecting,
109   highlights the core information in complex scenarios, and makes feature extraction from high-value
110   and small-size input.
111
112
      Considering the difference in quantity, the anomalous action recognition can be seen as the few-shot
      action recognition task. The few-shot learning gives the model task-level knowledge from the large
113
      normal action dataset, so it can avoid the expense of expanding the existing datasets which contain
114
      mostly normal action data (Li et al., 2013; Luo et al., 2017) or have insufficient quantity and variety
115   of anomalous actions(Sultani et al., 2018; Öztürk & Can, 2021). The network learns the C-way K-
116   shot classification task in each episode. The input of the network includes the support set comprising
117   K labeled instances for each of the C classes and the query set of unknown samples. The goal is to
118   classify the unlabeled video in the query set according to the features and labels of the support set.
119   We follow the episodic training paradigm in prior works(Vinyals et al., 2016; Finn et al., 2017), and
120   randomly sample the tasks from the training set. A sequence of uniformly sampled frames serves as
121   the representation for each video.
122   As can be seen in Fig.1, we add a key patch branch on the commonly employed global feature
123   network. The feature map and feature vector extracted are separately used for local and global
124   representation. Then the long-short feature map spatio-temporal relation module can connect the
125   temporal and motion relations in the video. After connection, the feature maps are fed into the
126   spatial adaptive key patch selection module to get the small patch with distinctive objects out of
127   the original frame. The key patch feature extraction can be completed quickly because of the small
128   inputs. After feature enhancement, the improved features of the support set and query set calculate
129
      the spatio-temporal refined loss to match the action.

150   2.2      L ONG - SHORT F EATURE M AP S PATIO -T EMPORAL R ELATION M ODULE
151
152   The global feature map only reserves the spatial information from the individual frame and lacks the
153
      representation of the entire video information. In addition to the static spatial information, spatio-
      temporal changes are also crucial for effectively locating key patches in the video. More specifically,
154
      the motion information represents short-range changes between neighboring frames, highlighting ar-
155
      eas with sudden intense changes, while temporal information, derived from stacked frames, denotes
156   continuous long-range evolution. These two kinds of information are complementary and need col-
157   laborative utilization. Based on light-weight convolution modules(Jiang et al., 2019; Li et al., 2020),
158   the long-short feature map spatio-temporal relation module is proposed, as shown in Fig.2. It con-
159   sists of two submodules: the temporal relation module and the motion relation module. The final
160   feature map with spatio-temporal information is created by the element-wise addition of the out-
161   puts from two submodules. The module successfully establishes the temporal and motion relation
      between frames at the feature map level.

      2.2.1    T EMPORAL R ELATION M ODULE
163
164   Using temporal information can ensure that the key patches selected at different timestamps corre-
165   spond to the long-range changes. 3D convolutions can jointly learn the spatial and temporal features
166   from continuous frames. But the temporal information in the whole video is not mixed up at once
167   only using 3D convolution, because of the limited kernel size and the restricted computational cost.
168   So the 2D convolution is factorized to make a whole-range temporal relation. Then the 3D convolu-
169
      tion can process the features with information from a longer temporal range, thereby expanding the
      effective temporal receptive field of the 3D convolution.
170
171   As illustrated in Fig.2(a), given the input feature I ∈ RN ×T ×C×H×W , where N denotes the batch
172   size, T denotes the number of frames in temporal dimension, C denotes the channels, and H, W
173   denote the height and width. First, the input is averaged across channels to get F ∈ RN ×T ×1×H×W .
174   In order to fully fuse the spatial information along the temporal dimension, the feature is reshaped
175
      to F ′ ∈ RN ×1×T ×H×W and fed to the 2D convolution K1 with the kernel size of 1 × 1, as shown
      in formula (1).
176
177

179
180   K1 takes the time of F ′ as channel dimension, and each output channel of F ′′ comes from the con-
181   volution on all the input frames. Therefore, the relations between different times are established.
182   After the whole-range temporal relation, the effective temporal information excitation can be ac-
183
      complished by 3D convolution K2 , as shown in formula (2).
184

188   function to get the single-channel mask. Finally, the temporal relation feature map is produced by
189   the element-wise multiplication of the mask and the input global feature map.

191   2.2.2    M OTION R ELATION M ODULE
192
      Motion information, which depicts spatio-temporal changes between adjacent frames, can serves
193
      as a valuable source for selecting key patches in short range. For moving objects like people or
194
      vehicles, their state of motion helps determine the direction and extent of key patch movement. One
195   classical way to model motion information is by using optical flow(Liu & Ma, 2022; Koniusz et al.,
196   2021). However, given the large model size and heavy training costs of Flownet, we propose a
197   lightweight module that provides similar functionality. The motion relation module is built based
198   on the frame differencing method, and relates the motion information at the feature map level by
199   subtracting neighboring frames.
200
      As illustrated in Fig.2(b), given an input I ∈ RN ×T ×C×H×W , we first get M ∈ RN ×T ×1×H×W by
201   aggregating the information along the channel dimension with 1 × 1 2D convolution. To calculate
202   the changes between frames, the features of different frames are split in the temporal dimension.
203   The feature of the individual frame can be denoted as M ′ ∈ RN ×1×H×W and is fed to 3 × 3 2D
204   convolution K3 for spatial encoding. The encoded feature subtracts from the adjacent feature to
205   excite the motion information at the feature map level.
206

207
208   M ′′ t denotes the output motion feature at time t. The motion features calculated by formula (3)
209   at different times are concatenated to get the motion mask. The final motion feature map is then
210   generated similarly as in temporal relation.

      2.4     S PATIO - TEMPORAL R EFINED L OSS
283
284   After extracting the enhanced feature Fe , the proper loss function is required to make the model
285   learn the task-aware knowledge. The TRM in (Perrett et al., 2021) uses cross-Transformer attention
286   for action matching but struggles with jointly modeling multi-space features and the constructed
287   tuples may not guarantee the effective learning of frame-level features. To address these problems,
288   we propose the spatio-temporal refined loss, using more spatially and temporally refined feature,
289   as shown in Fig.4. It is the sum of two parts: multi-head attention crosstransformer and temporal
290   refined match loss.
291

      Datasets:(1)HMDB51(Kuehne et al., 2011) includes 51 actions and 6849 videos. Some actions
      (walk, run, sit, etc.) are common in real life, and some actions (shoot, kick, punch, etc.) are typically
343
      regarded as abnormal. We split this dataset as (Zhang et al., 2020) did, 31,10, and 10 actions are
344
      used for training, validation, and testing, respectively.
345
346   (2)Kinetics(Carreira & Zisserman, 2017) has 400 actions and 306245 videos in total. We select 100
347   actions, each of which contains 100 videos, and divide them into training, validation, and test subsets
348
      with 64, 12, and 24 actions, just as CMN(Zhu & Yang, 2018) and CMN-J(Zhu & Yang, 2020) do.
349   (3)UCF-Crime v2(Öztürk & Can, 2021). The UCF Crime v2 dataset is one of the few datasets that
350   includes a variety of anomalous behaviors. We can precisely get the anomaly clips by temporal
351   annotations. Due to the limited number and variety of anomalous behaviors, direct training is not
352   feasible. So we use large normal action data to help the training, and the model is trained on Kinetics
353   and evaluated on UCF-Crime v2.
354   Implementation Details: For a fair comparison with previous methods, we follow the same prepro-
355   cess steps on the video and uniformly sample 8 frames. These frames are resized to height 256 and
356   augmented with random horizontal flipping and crops.
357
      We trained the models under the 5-way 1-shot and 5-way 5-shot settings using 2×GeForce RTX
358   3090 Ti, while the model under the 5-way 10-shot setting was trained using 4×GeForce RTX 3090
359   Ti. To fit in the memory, the query set for each class included 3 videos under the 5-way 1-shot and
360   5-way 5-shot settings, and 2 videos under the 5-way 10-shot setting.
361
      We use the TRX(Perrett et al., 2021) as the baseline. For the selected patch, the size is fixed to
362
      128 × 128. The global and key patch feature extraction networks both use ResNet-50 initialized
363
      with Image-Net pretrained weights. The SGD is selected for training the model, and the initial
364   learning rate is set to 0.001. We randomly sample 20000 training episodes for HMDB51 and 30000
365   training episodes for Kinetics, and report the average accuracy over 10000 random test episodes.

507   4   C ONCLUSION
508
509   The SKPS-Net for anomalous action recognition is proposed in this paper. For more precise model-
510   ing of discriminative objects in the local area, the key patch features are extracted to make feature
511
      enhancement. In particular, the plug-and-play spatial adaptive key patch selection module locates
      the informative small-sized area with few extra training costs. Instead of merely adopting the spa-
512
      tial information to extract the key patch feature, the long-short feature map spatio-temporal relation
513
      module enriches the long-range temporal and short-range motion information in the selection. Also,
514   the spatio-temporal refined loss is proposed to reinforce effective feature learning on multiple levels.
515   Experiments are conducted on HMDB51, Kinetics, and the anomalous action dataset UCF-Crime
516   v2, showcasing the effectiveness of our SKPS-Net.

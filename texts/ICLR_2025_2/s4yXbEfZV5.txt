009                                                A BSTRACT
010
011            The lifting-based framework has dominated the field of monocular 3D human pose
012            estimation by leveraging the well-detected 2D pose as an intermediate represen-
013            tation. However, it neglects different initial states between 2D pose and per-joint
014            depth. The initial state of the 2D pose is well-detected, but the per-joint depth
015            is unknown and needs to be learned from scratch. The lifting-based framework
016
               encodes the well-detected 2D pose and unknown per-joint depth in an entangled
               feature space, explicitly introducing depth uncertainty to the well-detected 2D
017
               pose. To address this limitation, we present a progressive multi-task learning pose
018
               estimation framework named PrML. First, PrML introduces two task branches to
019            refine the well-detected 2D pose features and to learn the per-joint depth features.
020            This dual-branch design reduces the explicit influence of uncertain depth features
021            on 2D pose features. Second, PrML employs a task-aware decoder to indirectly
022            supplement the complementary information between the refined 2D pose features
023            and learned per-joint depth features. This step establishes the connection between
024            2D pose and per-joint depth, compensating for the lack of interaction caused by the
025            dual-branch design. We conduct theoretical analysis from the perspective of mutual
026            information and arrive at a loss to supervise this feature complementary process.
027
               Finally, we use two regression heads to regress the 2D pose and per-joint depth,
               respectively, and concatenate them to obtain the final 3D pose. Extensive experi-
028
               ments show that PrML outperforms the conventional lifting-based framework with
029
               fewer parameters on two widely used datasets: Human3.6M and MPI-INF-3DHP.
030            Code is available at https://anonymous.4open.science/r/PrML.

035   Monocular 3D human pose estimation has been a crucial problem in computer vision, which aims
036   to locate the 3D joint positions of a human body (Moon & Lee, 2020; Pavlakos et al., 2018; Chen
037   et al., 2021). Nowadays, monocular 3D human pose estimation finds widespread applications in
038   various scenarios, including motion prediction (Liu et al., 2021b; 2022b), action recognition (Zhang
039
      et al., 2022a), and human-robot interaction (Gong et al., 2022; Ye et al., 2021). Existing monocular
      3D human pose estimation methods can be categorized as the end-to-end manner and lifting-based
040
      manner. The end-to-end approaches (Kanazawa et al., 2018; Pavlakos et al., 2017; Sun et al., 2018)
041
      directly estimate the 3D pose from the input image without the intermediate 2D pose representation.
042   Different from the end-to-end manner, lifting-based methods (Martinez et al., 2017; Liu et al., 2020)
043   first obtain 2D pose using 2D pose detector (Newell et al., 2016; Chen et al., 2018) and then lift
044   the 2D pose in image coordinate to the 3D pose in camera coordinate. These lifting-based methods
045   usually outperform the end-to-end manner and dominate the monocular 3D human pose estimation.
046
      Recent lifting-based methods (Zheng et al., 2021; Li et al., 2022b; Zhang et al., 2022b; Yu et al., 2023;
047
      Zhu et al., 2023; Peng et al., 2024) for monocular 3D human pose estimation focus on designing
048   various spatio-temporal encoders. As shown in Figure 1 left, they project the 2D pose into an
049   entangled feature space and regress the 3D pose from it. This lifting process neglects the different
050   initial states of 2D pose and per-joint depth. It encodes the well-detected 2D pose and unknown
051   per-joint depth in an entangled feature space, which introduces a main limitation: the high
052   uncertainty of the per-joint depth may erode the 2D pose. It is well-known that the monocular 3D
053   human pose estimation task is an ill-posed problem and inherently suffers from depth ambiguity (Li
      et al., 2022b; Ma et al., 2021b; Wehrbein et al., 2021). One 2D pose possibly corresponds to multiple

150          Monocular 3D Human Pose Estimation. Existing methods for monocular 3D human pose estimation
151          can be categorized as end-to-end and lifting-based. End-to-end approaches (Kanazawa et al., 2018;
152          Pavlakos et al., 2017; Sun et al., 2018) directly estimate the 3D pose from the input image without the
153
             intermediate 2D pose representation. With the reliable achievement of 2D human pose detectors (Chen
             et al., 2018; Newell et al., 2016; Sun et al., 2019), lifting-based methods (Fang et al., 2018; Martinez
154
             et al., 2017; Zhao et al., 2019; Liu et al., 2020) first obtain 2D pose representations in the image
155
             and then lift the 2D joint coordinates to 3D space. Recently, Transformers (Vaswani et al., 2017)
156          have been applied to various visual tasks (Dosovitskiy et al., 2021; Carion et al., 2020). For the
157          monocular 3D human pose estimation task, PoseFormer (Zheng et al., 2021) introduces transformer
158          architecture to leverage spatial and temporal dependency. MHFormer (Li et al., 2022b) addresses
159          the depth ambiguity by learning multiple pose hypotheses and MixSTE (Zhang et al., 2022b)
160          constructs a mixed spatiotemporal transformer to capture the temporal motion of different body
161          joints. STCFormer (Tang et al., 2023) decomposed spatio-temporal attention and integrated the
             structure-enhanced positional embedding. On the other hand, MotionBERT (Zhu et al., 2023) trains


                                                                                                                                                                      3

162
      a unified model for multiple downstream tasks. In (Peng et al., 2024), KTPFormer uses two prior
163   attention modules to facilitate pose estimation. Moreover, MotionAGFormer (Soroush Mehraban,
164   2024) using two parallel transformer and GCNFormer streams to better learn the underlying 3D
165   structure. However, these methods are developed within the conventional lifting-based framework. In
166   contrast, we propose a progressive multi-task learning framework to estimate 3D human pose.
167
      Multi-Task Learning. Multi-Task Learning (MTL) (Caruana, 1997) is a learning paradigm in
168
      machine learning, and it aims to leverage useful information contained in multiple related tasks to
169   help improve the generalization performance of all the tasks (Zhang & Yang, 2021). Numerous models
170   have been explored (Vandenhende et al., 2020; Brüggemann et al., 2021) within the MTL framework.
171   Moreover, existing approaches analyze the optimization of multi-task learning by designing multi-task
172   loss (Liu et al., 2021a; Li et al., 2022a) or gradient manipulations (Yu et al., 2020; Wang et al., 2020).
173   MTL has been widely used in computer vision, such as image classification (Rebuffi et al., 2017),
174   semantic segmentation (Hoyer et al., 2021; Li et al., 2023), and dense prediction (Proesmans et al.,
175   2022; Hoyer et al., 2021). In (Iqbal et al., 2018), they introduce a novel scale and translation invariant
176   2.5D pose representation contain 2D pose and depth. Our approach is motivated by these former
177
      attempts but from the perspective of decomposing the single 3D human pose estimation task into two
      sub-tasks and learning them in a progressive manner, which is a novel and unexplored question.
178
179   Mutual Information. Mutual Information plays an important role in the representation learning.
180   As the pioneering work among mutual information methods, Linsker (Linsker, 1988) proposes to
181   maximize mutual information between the input and output. Designing optimization objectives based
182   on mutual information maximization has been extensively studied (Becker, 1992; Wiskott & Se-
183
      jnowski, 2002). For human pose estimation, CV-MIM (Zhao et al., 2021) introduces a representation
      learning method to disentangle pose-dependent and view-dependent factors from 2D human poses.
184
      FAMI-Pose (Liu et al., 2022a) designs a mutual information loss to maximize the complementary
185
      information between temporal frames. TDMI (Feng et al., 2023) proposes to minimize the mutual
186   information between useful and noisy constituents of the raw features. To the best of our knowledge,
187   we are the first to introduce mutual information loss to the monocular 3D human pose estimation task.

190   3    R ETHINKING L IFTING - BASED M ONOCULAR 3D H UMAN P OSE E STIMATION
191
192   Since SimpleBaseline (Martinez et al., 2017) proposes the 2D-to-3D lifting framework, numerous
193   methods (Pavllo et al., 2019; Zhang et al., 2022b; Li et al., 2022b; Shan et al., 2022; Zhao et al.,
194   2023b; Shan et al., 2023; Zhu et al., 2023; Tang et al., 2023; Peng et al., 2024; Mehraban et al., 2024)
195
      have been developed within this framework. These lifting-based methods usually outperform the
      end-to-end manner (Kanazawa et al., 2018; Pavlakos et al., 2017; Sun et al., 2018) and have been the
196
      dominant paradigm in monocular 3D human pose estimation for a long time.
197
198   The ensuing question is why lifting-based methods perform better than end-to-end approaches. We
199   argue that this is mainly attributed to leveraging the 2D pose as an intermediate representation. First,
200   there exists a high relevance between 2D pose and 3D pose. Regressing 3D pose directly from
201
      raw images is a highly nonlinear and challenging problem (Pavlakos et al., 2017). This difficulty
      also exists in 2D human pose estimation (Pfister et al., 2015; Tompson et al., 2014). In contrast,
202
      with the widespread usage of 2D human pose detectors (Chen et al., 2018; He et al., 2017; Newell
203
      et al., 2016; Sun et al., 2019), lifting-based methods could leverage the well-detected 2D pose,
204   which contributes to its 3D counterpart and make network training easy. Second, the 2D pose is
205   exceptionally lightweight regarding memory cost compared to raw image. This property enables
206   lifting-based methods to leverage long-term temporal clues to address the occlusion and achieve
207   advanced accuracy. (e.g., 243 frames for MixSTE (Zhang et al., 2022b), MotionBERT (Zhu et al.,
208   2023), and KTPFormer (Peng et al., 2024); large as 351 frames for MHFormer (Li et al., 2022b))
209
      Once we have a well-detected 2D pose, lifting it directly to 3D space is natural and simple. However,
210   these lifting-based methods neglect different initial states between 2D pose and per-joint depth and
211   encode the well-detected 2D pose features and unknown per-joint depth features in an entangled
212   feature space. This leads to the fact that despite these methods (Zhu et al., 2023; Peng et al., 2024; Li
213   et al., 2022b) striving to design various encoders to leverage the well-detected 2D pose, the 2D pose
214   itself is inevitably eroded by the uncertainty of depth features (see Figure 2). This paper presents a
215   progressive multi-task learning framework that addresses the different initial states between 2D pose
      and per-joint depth and provides a new choice for future monocular 3D human pose estimation.

244   4.1    P ROBLEM F ORMULATION
245
246   Given a 2D pose sequence X ∈ RT ×J×Cin , the goal of monocular 3D human pose estimation is to
247   estimate the 3D pose sequence Y ∈ RT ×J×Cout . Here, T refers to the number of input frames, and J
248
      refers to the number of joints. Cin and Cout denote the dimension of the input and output.
249
250   4.2    M ULTI -TASK L EARNING B RANCH
251
252   One of the widely used multi-task learning models is proposed by Caruana (Caruana, 1997; 1993),
253
      which has a shared-bottom model structure that substantially reduces the risk of overfitting (Ma et al.,
      2018). We first use a linear embedding layer to project the 2D pose sequence into high-dimensional
254
      features. Then, we employ the DSTFormer block proposed by MotionBERT (Zhu et al., 2023) as
255
      our shared bottom to extract general features F ∈ RT ×J×C . The DSTFormer block is composed of
256   spatial-temporal and temporal-spatial branches. The outputs of two branches are adaptively fused by
257   an attention regressor. Next, we add the learnable 2D pose position embedding and per-joint depth
258   position embedding to F to obtain the 2D pose features F2D ∈ RT ×J×C and the per-joint depth
259   features FD ∈ RT ×J×C respectively. C denotes the feature dimension. Subsequently, the 2D pose
260   branch and depth branch repeat the temporal transformer encoder (T FT ) and spatial transformer
261   encoder (T FS ) for N times to refine the 2D pose features and learn depth features separately as:
262

      Analysis on Effectiveness of Components. The results in Section 5.2 have demonstrated that our
433   framework achieves better results compared to the conventional lifting-based framework in terms
434   of accuracy and robustness with fewer parameters. In this section, we show how to construct our
435   proposed progressive multi-task learning pose estimation framework step by step.
436
      • Baseline: Most multi-task learning models have a shared-bottom model structure following (Caru-
437
        ana, 1997) which substantially reduces the risk of overfitting (Ma et al., 2018). We follow this
438
        design and use the DSTFormer block from MotionBERT (Zhu et al., 2023) as the shared bottom.
439     As shown in Table 4, this structure achieves 50.6mm MPJPE and serves as our baseline.
440   • Multi-Task Branch: To reduce the explicit influence of uncertain depth features on the well-
441     detected 2D pose features, we introduce the multi-task branch design to refine the 2D pose features
442     and learn the depth features separately. Based on the shared bottom, we incorporate the 2D pose
443     branch and depth branch respectively bringing 6.2mm and 9.0mm error reduction (see also Table 4).
444     With both 2D pose and depth branches, we achieve the MPJPE of 39.1mm.
445   • Task-Aware Decoder: As shown in Table 4, we achieve a reduction in MPJPE from 50.6mm to
446     46.2mm by adding task-aware decoder to the shared bottom. The performance is further improved
447
        to 38.6mm when task-aware decoder is introduced in conjunction with the multi-task branch.
      • PrML: By introducing the mutual information loss to explicitly supervise the feature complement
448
        and the learning of bias, our TAD module further resulted in a 2.2mm error reduction (from 46.2mm
449
        to 44.0mm). After incorporating all components, we obtain the complete version of our PrML,
450     which achieves the best performance with an MPJPE of 38.2mm.

      6      C ONCLUSION
527
528   This work presents a novel progressive multi-task learning framework named PrML for monocular
529   3D human pose estimation. Our framework addresses the limitation of the lifting-based framework
530   that neglects different initial states between a well-detected 2D pose and an unknown per-joint depth.
531   PrML first learns 2D pose features and per-joint depth features separately by multi-task branch design
532   and then employs a task-aware decoder to indirectly supplement information between the refined
533   2D pose features and the learned per-joint depth features. We also propose a mutual information
534   loss to supervise the feature complementary process. Extensive quantitative experimental results on
535   the Human3.6M and MPI-INF-3DHP datasets show that our PrML outperforms the conventional
536
      lifting-based framework in terms of accuracy and robustness with fewer parameters.
537   Future Work. The core contribution of our work is providing a new framework for monocular 3D
538   human pose estimation. To this end, we use the widely used spatial and temporal transformer as our
539   encoder to ensure a fair comparison with the lifting-based framework. It will be novel and interesting
      to design specific encoders for different tasks to extend our framework in future research.

827   B.1   DATASETS AND E VALUATION M ETRICS
828
829   Human3.6M (Ionescu et al., 2013) is the most popular benchmark for indoor 3D human pose
830
      estimation, which contains approximately 3.6 million frames captured by 4 cameras at different views.
      This dataset contains 11 subjects performing 15 typical actions (e.g., walking and sitting). To ensure
831
      a fair comparison, we follow previous methods (Zheng et al., 2021; Zhang et al., 2022b; Tang et al.,
832
      2023; Zhu et al., 2023) by using subjects 1, 5, 6, 7, and 8 for model training and subjects 9 and 11 for
833   evaluation.
834
835   MPI-INF-3DHP (Mehta et al., 2017) is a recently proposed large-scale challenging dataset with both
836
      indoor and outdoor scenes. The training set comprises 8 subjects, covering 8 activities, ranging from
      walking and sitting to complex exercise poses and dynamic actions. The test set covers 7 activities,
837
      containing three scenes: green screen, non-green screen, and outdoor environments. It complements
838
      existing test sets with more diverse motions (standing/walking, sitting/reclining, exercise, sports
839   (dynamic poses), on the floor, dancing/miscellaneous).
840
841   Evaluation Metrics. For the Human3.6M dataset, we use two common evaluation metrics: MPJPE
842
      and P-MPJPE. MPJPE (Mean Per Joint Position Error) is computed as the mean Euclidean distance
      between the estimated joints and the ground truth in millimeters after aligning their root joints (hip).
843
      P-MPJPE (Procrustes-MPJPE) is the MPJPE after the estimated joints align to the ground truth via a
844
      rigid transformation. For the MPI-INF-3DHP dataset, following previous works (Shan et al., 2022;
845   Tang et al., 2023; Chen et al., 2023; Zhu et al., 2023), we use ground truth 2D pose as input and
846   report MPJPE, Percentage of Correct Keypoint (PCK) with the threshold of 150mm, and Area Under
847   Curve (AUC) as the evaluation metrics.

850   B.2   I MPLEMENTATION DETAILS
851
852   Our model is implemented using PyTorch and executed on a server equipped with 2 NVIDIA 3090
853   GPUs. We apply horizontal flipping augmentation for both training and testing following (Tang
854   et al., 2023; Zhu et al., 2023; Foo et al., 2023; Zhao et al., 2023a). For model training, we set each
855
      mini-batch as 16 sequences. The network parameters are optimized using AdamW (Loshchilov &
      Hutter, 2017) optimizer over 90 epochs with a weight decay of 0.01. The initial learning rate is set to
856
      5e-4 with an exponential learning rate decay schedule and the decay factor is 0.99. In the experiments
857
      on Human3.6M, two kinds of input are utilized, including the 2D ground truth and the Stacked
858   Hourglass (Newell et al., 2016) 2D pose detection, following (Zhu et al., 2023; Ci et al., 2019). For
859   MPI-INF-3DHP, 2D ground truth is used following previous works (Zheng et al., 2021; Zhang et al.,
860   2022b; Shan et al., 2022; Zhu et al., 2023). While our proposed framework is capable of adapting to
861   input sequences of any length, to be fair, we choose specific input sequence lengths (denoted as T ) for
862   two datasets to compare our method with other approaches that have a certain 2D input length (Zheng
863   et al., 2021; Zhang et al., 2022b; Shan et al., 2022; Tang et al., 2023): Human3.6M (T = 81, 243),
      MPI-INF-3DHP (T = 9, 27, 81).

      C.6       ATTEMPT TO U SE I MAGE F EATURE .
965
966   As the inherent information entropy of the input 2D pose sequence, the performance of methods only
967   using 2D pose sequences as input gradually approaches the theoretical limit. Incorporating image
968   features is a potential solution to this general limitation. There are several single-frame lifting-based
969   methods that utilize image features, such as ContextPose (Ma et al., 2021a)and CA-PF (Zhao et al.,
970   2023a). Unfortunately, integrating these methods into existing multi-frame lifting methods or our
971   PrML is challenging due to their complex model architectures. Therefore, we employ one cross-
      attention layer to fuse image features, making a preliminary attempt to fuse image features into our


                                                                           18
       Under review as a conference paper at ICLR 2025


972
       framework. Due to the lightweight nature of 2D poses, traditional lifting-based frameworks typically
973    input a 2D pose sequence. However, inputting a long-term image sequence is clearly impractical.
974    How to effectively extend image input from a single frame to multiple frames remains an open
975    question. To tackle this problem, we employ VAE to compress the original images into latent features,
976    allowing for multi-frame input. We use the pre-trained VAE (Kingma, 2013) from SDXL (Podell
977    et al., 2023). Although we have compressed the images using VAE, processing 243 frames remains
978    computationally expensive. Therefore, we leverage VAE to incorporate 15 frames image feature,
979    making a preliminary attempt to fuse multi-frame image features into our framework.
980    As shown in the Table 13, despite using simple cross-attention to fuse image features, we still observe
981    performance improvement. When we incorporated image features into the 2D branch, we observed
982    a significant improvement. This is likely due to the fact that image features can easily capture 2D
983    spatial information. However, when we added image features to the depth branch, the improvement
984    was limited. This is because we used a simple cross-attention mechanism to process the images, and
985    extracting depth information from RGB images is a challenging task. Similarly, when we incorporated
986    image features into all branches, the improvement was modest. These experiments demonstrate the
987
       potential for further extending our framework.
988    Moreover, the insights from multi-task learning can be further extended. Using 2D pose detectors
989    to estimate 2D poses has been widely used, so why not similarly utilize powerful depth estimation
990    networks (like DepthAnything (Yang et al., 2024)) to estimate a relative depth from the image to
991    facilitate 3D human pose estimation as well? We can integrate depth features into existing single-
992    frame methods that utilize image features to validate this idea. As shown in Table 14, by incorporating
993
       depth features from DepthAnything (Yang et al., 2024), we can reduce the error of CA-PF by 1.9mm,
       which demonstrates the effectiveness of depth features.
994
995    In summary, how to leverage image features to facilitate multi-frame 3D human pose estimation is a
996    non-trivial question and our future research direction.

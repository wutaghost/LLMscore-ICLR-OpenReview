027                                               A BSTRACT
028
029            Image retargeting aims to change the aspect-ratio of an image while maintaining
030            its content and structure with less visual artifacts. Existing methods still gener-
031            ate many artifacts or lose a lot of original content or structure. To address this,
032            we introduce HALO, an end-to-end trainable solution for image retargeting. The
033            core idea of HALO is to warp the input image to target resolution. Since humans
034
               are more sensitive to distortions in salient areas than non-salient areas of an im-
               age, HALO decomposes the input image into salient/non-salient layers and applies
035
               different wrapping fields to different layers. To further minimize the structure dis-
036
               tortion in the output images, we propose perceptual structure similarity loss which
037            measures the structure similarity between input and output images and aligns with
038            human perception. Both quantitative results and a user study on the RetargetMe
039            dataset show that our algorithm achieves SOTA. Especially, our method increases
040            human preference by 13.21% compared with the second best method.

      1   I NTRODUCTION
044
045
      Images are displayed on a diverse set of platforms and devices, each with a different aspect-ratio.
046
      Content creators are often required to produce multiple versions of the same image in different
047
      aspect-ratios, a task that becomes increasingly burdensome with the growing number of platforms.
048   Resizing or cropping images are traditional approaches for it, but resizing can distort structures, and
049   cropping inevitably removes content. Image retargeting (Rubinstein et al., 2010; Tang et al., 2019)
050   seeks to address these problems and adjusts an image’s aspect-ratio while preserving its key content
051   and structure. As defined by (Rubinstein et al., 2010; Vaquero et al., 2010), a successful image
052   retargeting outcome is as follows: (a) The key content in the input image should be preserved in
053   the output image; (b) The inner structure of the input should be maintained in the output; (c) There
      should be no distortion or visual artifacts in the output image.

      Many image retargeting algorithms have been proposed, including traditional optimization ap-
055   proaches (Liu & Gleicher, 2005; Setlur et al., 2005; Wolf et al., 2007; Simakov et al., 2008; Ru-
056   binstein et al., 2009; Barnes et al., 2009; Pritch et al., 2009; Rubinstein et al., 2010; Chen et al.,
057   2010; Shi et al., 2010), weak- or self-supervised learning (Cho et al., 2017b; Tan et al., 2019), re-
058   inforcement learning (Kajiura et al., 2020), and generative modeling methods (Elnekave & Weiss,
059   2022; Granot et al., 2022). However, these methods still struggle to preserve both content and struc-
060   ture or generate less visual artifacts (e.g., out-of-boundary, or OOB, artifact) as shown in Figure 2.

075
076   To address these problems, we propose HALO (Human-Aligned Layered transfOrmations for image
077   retargeting), an end-to-end trainable model. The key idea behind HALO is to warp the input image
078   to a target resolution with layered transformations. Recognizing that humans are more sensitive to
079
      distortions in salient regions than in non-salient areas, HALO decomposes the image into salient and
      non-salient layers based on a saliency map and applies different transformations to each layer. This
080
      design enables HALO to preserve critical details in salient regions while handling non-salient areas,
081
      and it also avoids OOB issues.
082
083   To further reduce the structure and content loss in output images, we use perceptual loss function as
084   weak supervision to guide the algorithm to produce images close to the original image’s content and
085
      structure. DreamSim (Fu et al., 2023), which emphasizes mid-level structure and distortion, is well-
      suited as a perceptual loss for image retargeting. However, since DreamSim is trained on square
086
      images, it cannot be directly applied to image retargeting. To address this, we develop a layout
087
      augmentation technique that adapts DreamSim for image retargeting and we introduce a new loss
088   function, Perceptual Structure Similarity Loss (PSSL), which aligns closely with human perception.
089
090   Our contributions are as follows:
091         • A novel end-to-end trainable image retargeting algorithm based on layered transformations.
092         • A new Perceptual Structure Similarity Loss function for image retargeting tasks, aligning
093           well with human perception.
094         • Extensive quantitative results and a user study on the RetargetMe dataset demonstrate that
095           HALO achieves SOTA performance, with HALO significantly outperforming the second-
096           best approach in the user study.

098   2   R ELATED WORK
099
100
      Image Retargeting. Image retargeting is a task to generate images with arbitrary aspect-ratios
101
      given an input image. Over the years, various approaches have been proposed, including conven-
102   tional optimization-based methods (Rubinstein et al., 2008; 2009; Barnes et al., 2009; Simakov
103   et al., 2008; Wolf et al., 2007; Pritch et al., 2009; Wang et al., 2008; Karni et al., 2009), weakly-
104   supervised learning (Cho et al., 2017a; Tan et al., 2019), deep reinforcement learning (Kajiura et al.,
105   2020), GAN based models (Shaham et al., 2019; Shocher et al., 2019; Hinz et al., 2021; Zhang
106   et al., 2022), Patch Nearest Neighbor (PNN) (Granot et al., 2022; Elnekave & Weiss, 2022), and
107   diffusion models (Wang et al., 2022; Kulikov et al., 2023; Zhang et al., 2023; Nikankin et al., 2023).
      Compared to optimization-based methods, we train an end-to-end model and it has faster inference


                                                        2
      Under review as a conference paper at ICLR 2025


108
      speed. Compared to end-to-end methods, our method uses layered transformations and predicts
109   multiple warping flows, avoiding out-of-boundary issues and preserving salient contents better.
110
111   Layered representations. Layered representations (Lu et al., 2020; 2021; Yang et al., 2021) enable
112   more flexible manipulation for an image or a video on different layers. It has been widely used for
113   both images (He et al., 2009; Gandelsman et al., 2019) and videos (Lu et al., 2020; Liu et al., 2021;
114   Lu et al., 2021; Kasten et al., 2021; Lee et al., 2023). We adopt the idea of layered representations
115
      and use it in the image retargeting task. It avoids out-of-boundary issues in the previous methods.
116   Perceptual losses. With the revolution of deep-learning, many pretrained networks (Krizhevsky
117   et al., 2012; Simonyan & Zisserman, 2014; He et al., 2016a), can extract meaningful features from
118   the images. Defined by measuring the feature distances, learning-based metrics (Dosovitskiy &
119   Brox, 2016; Johnson et al., 2016; Zhang et al., 2018; Prashnani et al., 2018) show better alignment
120   with human perception than the classic ones. More recently, DreamSim (Fu et al., 2023) is proposed
121   to capture the mid-level similarities, such as structure and layout, between images. Perceptual losses
122   are also used in image retargeting (Cho et al., 2017b; Tan et al., 2019) in the absence of paired train-
123   ing data. We use DreamSim, a perceptual loss focusing on the mid-level features such as structures
124
      and layouts. We find previous perceptual loss functions (e.g., LPIPS (Zhang et al., 2018)) have dif-
      ficulties handling structure distortions. We further adapt DreamSim to the image retargeting task by
125
      proposing a layout augmentation.

133                0    0
      and width, H , W are the output height and width. The saliency map is a heatmap that measures
134   the importance of pixels in the input image. The saliency map could be generated by a saliency
135   detector (Gao et al., 2024), a segmentation network (e.g., SAM (Kirillov et al., 2023)), or a user-
136   defined mask. In this paper, we use saliency maps predicted by an off-the-shelf salient detector,
137   MDSAM (Gao et al., 2024). Given a saliency map M , we decompose the input I into a salient
138   layer as ISL = I M and a non-salient layer IN SL = I (1 − M ), where is the element-
139   wise multiplication. To fill in the holes of the non-salient layer, we inpaint it with an off-the-shelf
140   inpainting model (Suvorov et al., 2022): IN SLI = Inpaint(IN SL ).
141   The reason for decomposing an image into two layers is based on the observation that a single
142   transformation, as in (Cho et al., 2017b; Tan et al., 2019), cannot handle both salient and non-salient
143   contents simultaneously well and may result in out-of-boundary (OOB) issues as shown in Figure 2
144   and Figure 7. A single transformation is able to preserve the salient content to the new target size,
145   but may warp the non-salient pixels in an undesired way. Applying multiple transformations gives
146   the model more flexibility to achieve retargeting without suffering from the OOB issues (Figure 7).
147   We finally formulate the output image I 0 as
148                 I 0 = Warp(ISL , FSL ) M 0 + Warp(IN SLI , FN SL ) (1 − M 0 ) ,                       (1)
                                 0    0
149   where FSL , FN SL ∈ RH ×W ×2 are vector warping fields predicted by our Multi-Flow Network
150   (MFN), and the warped saliency map M 0 = Warp(M , FSL ).

      3.3      P ERCEPTUAL S TRUCTURE S IMILARITY L OSS
203
204   One of the challenges of training an image retargerting model is the absence of paired data for su-
205   pervision. Previous works, such as (Cho et al., 2017b; Tan et al., 2019; Mastan & Raman, 2020) use
206   a perceptual loss (e.g., VGG loss (Simonyan et al., 2014) or LPIPS (Zhang et al., 2018)) between the
207   input and the output as a weak supervision. These perceptual loss functions calculate the distance
208   between feature maps via a pretrained network, and do not enforce a strict supervision as pixelwise
209   `1 or `2 losses. However, popular perceptual losses like LPIPS are less sensitive to structural distor-
210   tions compared to DreamSim (Fu et al., 2023) in Figure 4. Therefore, we adopt DreamSim as our
211   perceptual quality metric.
212   Unfortunately, directly using DreamSim does not work for image retargeting, since DreamSim is
213   trained on square, undistorted images and preprocesses images by resizing them to a fixed square
214   size 224 × 224. As shown in Figure 5, the preprocessed IR (at 224 × 224) exhibits a very small
215   Dream loss with the input image I, despite IR having distortion at the target size. Consequently,
      supervising the training with DreamSim loss between the input I and the output leads to a similar,

294   4.1   S ETUP
295
296   Dataset. We train our model on the UHRSD dataset (Xie et al., 2022), which consists of 4, 932
297   training images and 988 test images. Each image comes with an annotated saliency map. It covers
298   diverse image categories including natural landscapes, street views, and animals. During training,
299   we resize the images so that their shorter side is scaled to 512. For example, if the height is greater
300   than the width, the image is rescaled to (512 × height
                                                          width , 512). We group the images by their aspect
301
      ratios and sample images from the same group into each batch. We test our model and compare
      with other baseline approaches on the common RetargetMe (Rubinstein et al., 2010) benchmark.
302
      RetargetMe contains 80 images with different scaling factors (0.50, 0.75 and 1.25) for the test.
303
304   Evaluation metrics. Previous evaluations (Cho et al., 2017b; Tan et al., 2019; Kajiura et al., 2020)
305   on image retargeting have relied heavily on user studies. Given the rapid advancements of the recent
306   visual representation learning, we propose to use pretrained networks to predict the image features
307   and assess the quality of the outputs based on these features. We use CLIP image embeddings (Rad-
308   ford et al., 2021) for the content evaluation. We compute the similarity between the input image
309   embedding and the output image embedding. To assess structure consistency, we use DreamSim
310
      similarity (Fu et al., 2023), which focuses on mid-level differences such as structure and layout. We
      use the original DreamSim since we do not wish introducing randomness from Eqn. 5 into evalua-
311
      tions. We use MUSIQ score (Ke et al., 2021) for aesthetics evaluation. To better align with other
312
      metrics, such as DreamSim and CLIP similarity, we re-normalize the MUSIQ score as a percentage.
313   Image retargeting also requires to minimize visual artifacts, such as object distortion, missing or
314   duplicated contents, or OOB artifacts. Since current assessment models struggle to reliably detect
315   these artifacts, we conduct a user study where participants select the output with the best image
316   quality. We use user preferences across different methods as a metric for visual artifact evaluation.
317   We include details about our user study in the Supplementary Material.

533
      We present HALO, an end-to-end framework for image retargeting that aligns with human percep-
534   tion. By using a layered representation for the input and applying distinct transformations to salient
535   and non-salient regions, our approach produces results with fewer visual artifacts, such as the OOB
536   issue. We also introduce a new Perceptual Structure Similarity Loss (PSSL) enabling training with-
537   out paired data for image retargeting and equips the model with distortion-awareness capabilities.
538   We conduct extensive evaluations across various methods, demonstrating that HALO outperforms
539   previous approaches. A user study further confirms that HALO aligns closely with human percep-
      tion, outperforming the SOTAs by a large margin.

765   resized version to obtain two feature maps. Two feature maps are fed into L Cross-Attention blocks.
766   Cross-Attention blocks. To condition the network on the target image size, we choose to naively
767   resize the input image to the target size. To better understand the rough layout at the target size,

769   2022), we choose to use cross-attention mechanism to share the information between the original
770   input and the resized input. We adopt the decoder block from CroCo-v2 (Weinzaepfel et al., 2023),
771   where it consists of LayerNorm, SelfAttention, CrossAttention and MLP. In practice, we use
772   L = 3 decoder blocks, and each block has 4 heads.

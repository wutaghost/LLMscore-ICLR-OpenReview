011                                                 A BSTRACT
012
013              Clustering algorithms are fundamental to data mining, serving dual roles as ex-
014              ploratory tools and preprocessing steps for advanced analytics. A persistent chal-
015              lenge in this domain is determining the optimal number of clusters, particularly
016
                 for time series data where prevalent algorithms like k-means and k-shape require
                 a priori knowledge of cluster quantity. This paper presents the first approach to
017
                 time series clustering that does not require prior specification of cluster numbers.
018
                 We introduce a novel extension of the Symbolic Pattern Forest (SPF) algorithm
019              that automatically optimizes the number of clusters for time series datasets. Our
020              method integrates SPF for cluster generation with the Silhouette Coefficient, com-
021              puted on a two-stage vector representation: first transforming time series into
022              Symbolic Aggregate approXimation (SAX) representations, then deriving both
023              bag-of-words and TF-IDF vectors. Rigorous evaluation on diverse datasets from
024              the UCR archive demonstrates that our approach significantly outperforms tradi-
025              tional baseline methods. This work contributes to the field of time series analysis
026              by providing a truly unsupervised, data-driven approach to clustering, with poten-
027
                 tial impacts across various temporal data mining applications where the underly-
                 ing number of clusters is unknown or variable.

169   3.2   Symbolic Pattern Forest
170
171   The Symbolic Pattern Forest (SPF) Li et al. (2019a) is a linear-time complexity algorithm for time
172   series clustering. It leverages the symbolic representation provided by SAX to create an ensemble
173   of weak clusterings, which are then combined to produce a final, robust clustering. Given a set of N
174   time series T = T1 , ..., TN , SPF operates as follows:
175   Each time series Ti is transformed into its SAX representation Wi .
176
177   A set of m random symbolic patterns P = P1 , ..., Pm is generated, where each Pj is a string of
178
      length l over the alphabet Σ.
179   For each pattern Pj , a binary partition πj of T is created:
180
181
                   1            if Pj is a substring of Wi 0
      πj (Ti ) =
182                otherwise
183   The set of partitions Π = π1 , ..., πm forms an ensemble of weak clusterings.
184
185
      The final clustering C is obtained by applying a consensus function f to the ensemble: C = f (Π)
186   The consensus function f can be implemented in various ways, such as majority voting or more
187   sophisticated methods like spectral clustering on the co-association matrix.
188
      The time complexity of SPF is O(N ), as the pattern matching step can be implemented efficiently
189   using finite automata. The space complexity is also linear, as the boolean arrays used for partitioning
190   are space-efficient.
191
192
      The effectiveness of SPF stems from its ability to capture diverse structural information through
      random symbolic patterns. The ensemble approach helps mitigate the impact of individual weak
193
      clusterings, leading to a more robust final clustering. Moreover, the use of symbolic patterns elim-
194
      inates the need for explicit distance calculations, further contributing to the algorithm’s efficiency.
195   Theoretical analysis has shown that the ensemble size m required for good performance does not
196   directly depend on the input size N , allowing for a fixed ensemble size to be used across differ-
197   ent datasets. This property ensures that the algorithm’s time complexity remains linear even as the
198   dataset size increases.

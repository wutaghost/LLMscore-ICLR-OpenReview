009                                             A BSTRACT
010
011            Density ratio estimation (DRE) is a fundamental machine learning technique
012            for capturing relationships between two probability distributions. State-of-the-
013            art DRE methods estimate the density ratio using neural networks trained with
014            loss functions derived from variational representations of f -divergence. However,
015            existing methods face optimization challenges, such as overﬁtting due to lower-
016            unbounded loss functions, biased mini-batch gradients, vanishing training loss
017            gradients, and high sample requirements for Kullback-Leibler (KL) divergence
018
               loss functions. To address these issues, we focus on α-divergence, which provides
               a suitable variational representation of f -divergence. Subsequently, a novel loss
019
               function for DRE, the α-divergence loss function (α-Div), is derived. α-Div is
020
               concise but offers stable and effective optimization for DRE. The boundedness
021            of α-divergence provides the potential for successful DRE with data exhibiting
022            high KL-divergence. Our numerical experiments demonstrate the effectiveness in
023            optimization using α-Div. However, the experiments also show that the proposed
024            loss function offers no signiﬁcant advantage over the KL-divergence loss function
025            in terms of RMSE for DRE. This indicates that the accuracy of DRE is primarily
026            determined by the amount of KL-divergence in the data and is less dependent on
027            α-divergence.

030   1   I NTRODUCTION
031
032   Density ratio estimation (DRE), a fundamental technique in various machine learning domains, es-
033
      timates the density ratio r∗ (x) = q(x)/p(x) between two probability densities using two sample
      sets drawn separately from p and q. Several machine learning methods, including generative mod-
034
      eling (Goodfellow et al., 2014; Nowozin et al., 2016; Uehara et al., 2016), mutual information esti-
035   mation and representation learning (Belghazi et al., 2018; Hjelm et al., 2018), energy-based model-
036   ing (Gutmann & Hyvärinen, 2010), and covariate shift and domain adaptation (Shimodaira, 2000;
037   Huang et al., 2006), involve problems where DRE is applicable. Given its potential to enhance a
038   wide range of machine learning methods, the development of effective DRE techniques has gar-
039   nered signiﬁcant attention.
040
      Recently, neural network-based methods for DRE have achieved state-of-the-art results. These
041
      methods train neural networks as density ratio functions using loss functions derived from varia-
042   tional representations of f -divergence (Nguyen et al., 2010), which are equivalent to density-ratio
043   matching under Bregman divergence (Sugiyama et al., 2012). The optimal function for a variational
044   representation of f -divergence, through the Legendre transform, corresponds to the density ratio.
045
      However, existing neural network methods suffer from several issues. First, an overﬁtting phe-
046
      nomenon, termed train-loss hacking by Kato & Teshima (2021), occurs during optimization when
047
      lower-unbounded loss functions are used. Second, the gradients of loss functions over mini-
048   batch samples provide biased estimates of the full gradient when using standard loss functions de-
049   rived directly from the variational representation of f -divergences (Belghazi et al., 2018). Third,
050   loss function gradients can vanish when the estimated probability ratios approach zero or inﬁnity
051   (Arjovsky & Bottou, 2017). Finally, optimization with a Kullback–Leibler (KL)-divergence loss
052   function often fails on high KL-divergence data because the sample requirement for optimization
053   increases exponentially with the true amount of KL-divergence (Poole et al., 2019; Song & Ermon,
      2019; McAllester & Stratos, 2020).

054
      To address these problems, this study focuses on α-divergence, a subgroup of f -divergence, which
055   has a sample complexity independent of its ground truth value. We then present a Gibbs density
056   representation for a variational form of the divergence to obtain unbiased mini-batch gradients, from
057   which we derive a novel loss function for DRE, referred to as the α-divergence loss function (α-Div).
058   Despite its conciseness, α-Div offers stable and effective optimization for DRE.
059
      Furthermore, this study provides technical justiﬁcations for the proposed loss function. α-Div has a
060
      sample complexity that is independent of the ground truth value of α-divergence and provides unbi-
061   ased mini-batch gradients of training losses. Additionally, α-Div is lower-bounded when α is within
062   a speciﬁc interval, thereby preventing train-loss hacking during optimization. By selecting α from
063   this interval, we also avoid vanishing gradients in neural networks when they reach extreme local
064   minima. We empirically validate our approach through numerical experiments using toy datasets,
065   which demonstrate the stability and efﬁciency of the proposed loss function during optimization.
066
      However, we observe that the root mean squared error (RMSE) of the estimated density ratios in-
067   creases signiﬁcantly for data with higher KL-divergence when using the proposed loss function. The
068   same phenomenon is observed with the KL-divergence loss function. These results suggest that the
069   accuracy of DRE is primarily determined by the amount of KL-divergence in the data and is less
070   inﬂuenced by the α-divergence, providing insights into the accuracy of downstream tasks in DRE
071   using f -divergence loss functions.
072
      The key contributions of this study are as follows: (1) A novel loss function for DRE, α-Div, is
073   proposed, offering a concise solution to the instability and biased gradient issues in existing f -
074   divergence loss functions. (2) Technical justiﬁcations for the proposed loss function are presented.
075   (3) We empirically conﬁrm the stability and efﬁciency of the proposed loss function in optimization.
076   (4) We ﬁnd that RMSE in DRE increases signiﬁcantly as the KL-divergence in the data rises when
077   using the α-divergence loss function, indicating that DRE and KL-divergence estimation accuracy
078   is primarily determined by the amount of KL-divergence in the data, rather than the α-divergence.

154   3.4   VANISHING GRADIENTS PROBLEM
155
156   The vanishing-gradient problem in optimizing divergences is a well-known issue in GANs
157   (Arjovsky & Bottou, 2017). To address the vanishing gradients of training losses, loss functions
158
      with penalty terms have been proposed (Gulrajani et al., 2017; Roth et al., 2017).
159   We believe this problem occurs when the following two conditions are satisﬁed: (i) The loss function
160   causes slight updates to the model parameters, and (ii) Updating the model parameters causes min-
161   imal changes in the model outputs. Therefore, the problem can arise when the following relations
      hold:

711   B     R ELATED WORK
712
713   Nguyen et al. (2010) proposed DRE using variational representations of f -divergences.
714   Sugiyama et al. (2012) introduced the density-ratio matching under the Bregman divergence, which
715
      is a general framework that uniﬁes various methods for DRE. As Sugiyama et al. (2012) mentioned,
      the density-ratio matching under the Bregman divergence is equivalent to DRE using variational rep-
716
      resentations of f -divergences. For estimation with high KL-divergence data, Rhodes et al. (2020)
717
      proposed a method that divides the high KL-divergence estimation into multiple smaller divergence
718   estimations. Choi et al. (2022) further developed a continuous decomposition approach by introduc-
719   ing an auxiliary variable for transforming the data distribution. DRE using variational representa-
720   tions of f -divergences has also been studied from the perspective of classiﬁcation-based modeling.
721   Menon & Ong (2016) demonstrated that DRE through f -divergence optimization can be represented
722   as a binary classiﬁcation problem. Kato et al. (2019) proposed using the risk functions in PU learn-
723   ing for DRE.
724   Finally we review prior studies on DRE regarding α-divergence loss functions. Birrell et al. (2021)
725   derived an α-divergence loss function from R’enyi’s α-divergence, while Cai et al. (2020) used a
726   standard variational representation of Amari’s α-divergence with α < 0 or α > 1. Kwon & Baek
727   (2024) presented the same α-divergence loss function as proposed in this study, using the Gibbs
728   density expression to measure entropy in thermodynamics. In contrast, we offer the proposed loss
729   function to address the biased gradient problem.

1607   applying the proposed method. Speciﬁcally, we conducted experiments on Domain Adaptation (DA)

1609   IW method builds a prediction model for a target domain using data from a source domain, while
1610

       employing the density ratio between the source and target domains as sample weights.

1615   Cross Validation (IWCV) method (Sugiyama et al., 2007a) on the source domain data.
1616   Through these experiments, we observed a decline in prediction accuracy on test data from the target
       domain as the data dimensionality increased. Speciﬁcally, there were instances where the accuracy
       worsened compared to models that did not use importance weighting―i.e., models trained solely on

       E.2.2    E XPERIMENTAL P ROCEDURE .
1638
1639   Step 1. Creation of feature tables. Many DA methods utilize feature embedding techniques to
1640   project high-dimensional data into a lower-dimensional feature space, facilitating the handling of
1641   distribution shifts between source and target domains (Ragab et al., 2023). However, our prelim-
1642   inary experiments revealed that model prediction accuracies were signiﬁcantly inﬂuenced by the
1643   embedding procedures. To address these effects on DA task accuracies, we explored an embedding
1644
       method with theoretical considerations detailed in the next section.
1645   Speciﬁcally, we selected an identical set of columns from the original data of both the source and
1646   target domains for each feature dimension, d = 8, 16, 32, 64, and 128, arranging the columns in
1647   ascending order of d. Let XdS and XdT denote the subsets of the original data for the source and
1648   target domains, respectively, determined by these selected columns for each d. We then generated
1649   a d × dim(XdS ) matrix Ad from a normal distribution. Finally, by multiplying XdS and Ad , and XdT
1650   and Ad , we obtained the feature tables ZdS and ZdT , embedding the original source and target domain
1651   data into a d-dimensional feature space. 3
1652
1653   Step 2. Estimation of importance weights. Using the proposed loss function with ZdS and ZdT
1654   obtained from the previous step, we estimated the probability density ratio r̂(ZdS ) for each feature
1655   dimension d, where r(ZdS ) = q(ZdT )/p(ZdS ). This ratio represents the density of the target domain
1656   relative to the source domain.
1657
1658   Step 3. Model construction. We constructed the target model using the IW method. Speciﬁcally,
1659   we built a classiﬁcation model using the training dataset (XdS , yS ), where the estimated density ratio
1660   r̂(ZdS ) served as the sample weights for the IW method. Additionally, we constructed a prediction
1661
       model using only the source data, i.e., a model built without importance weighting.
1662
1663   Step 4. Veriﬁcation of prediction accuracy To evaluate the prediction accuracy of the models,
1664
       we selected the ROC AUC score, as it measures accuracy independently of the thresholds used
       for label determination. For the classiﬁcation tasks in domain adaptation, we employed two clas-
1665
       siﬁcation methods, each representing a different algorithmic approach: LogisticRegression
1666
       from the scikit-learn library (Pedregosa et al., 2011) for linear classiﬁcation, and LightGBM
1667   (Ke et al., 2017) for nonlinear classiﬁcation.

009                                               A BSTRACT
010
011            Deep learning algorithms for object classification and 3D object pose estimation
012            lack robustness to out-of-distribution factors such as synthetic stimuli, changes
013            in weather conditions, and partial occlusion. Human vision, however, is typically
014            much more robust to all these factors. This is arguably because human vision
015            exploits 3D object representations which are invariant to most of these factors.
016
               Recently a class of 3D compositional models have been developed where objects
               are represented in terms of 3D meshes, with typically 1000 vertices associated
017
               with learnt vertex features. These models have shown robustness in small-scale
018
               settings, involving 10 or 12 objects, but it is unclear that they can be scaled up to
019            100s of object classes. The main problem is that their training involves supervised
020            contrastive learning on the mesh vertices representing the objects and requires
021            each vertex to be contrasted with all other vertices, which scales quadratically
022            with the vertex number. A newly available dataset with 3D annotations for 188
023            object classes allows us to address this scaling challenge. We present a strategy
024            which exploits the compositionality of the objects, i.e. the independence of the
025            feature vectors of the vertices, which greatly reduces the training time while also
026            improving the performance of the algorithms. We first refactor the per-vertex
027
               contrastive learning into contrasting within class and between classes. Then we
               propose a process that dynamically decouples the contrast between classes which
028
               are rarely confused, and enhances the contrast between the vertices of classes that
029
               are most confused. Our large-scale 3D compositional model not only achieves state-
030            of-the-art performance on object classification and 3D pose estimation in a unified
031            manner surpassing ViT and ResNet, but is also more robust to out-of-distribution
032            testing including occlusion, weather conditions, and synthetic data. This paves
033            the way for scalable 3D object understanding and opens exciting possibilities for
034            applications in robotics, autonomous systems, and augmented reality.

      1   I NTRODUCTION
038
039
      Recent progress in deep learning has yielded impressive results in different machine visual recognition
      tasks, such as object classification, detection, and pose estimation LeCun et al. (2015), with the help
040
      of large-scale training images and annotations. Cognitive scientists, however, suggest that human
041
      vision is more sophisticated and when classifying objects also recognizes their 3D structure including
042   their shape and pose in a unified way using compositional representations Biederman (1987); Leek
043   et al. (2005); Biederman (2000). We conjecture that endowing computer vision models with 3D
044   representations will improve their performance, particularly in challenging out-of-distribution (OOD)
045   scenarios, including domain shifts due to changes in weather, occlusions, and unfamiliar viewpoints,
046   for which humans show big robustness Zhu et al. (2019), but where standard deep network models
047   struggle Goodfellow et al. (2016); Koporec & Perš (2019); Zhu et al. (2019). The key insight is that
048   the 3D structure of objects rarely varies in most OOD settings while deep network features are much
049   more variable.
050   One promising avenue involves 3D Compositional Neural Networks (3D-CompNets) Wang et al.
051   (2024; 2021); Ma et al. (2022); Jesslen et al. (2024); Kaushik et al. (2024b). These models are
052   compositional in the sense that they represent objects by 3D meshes of vertices which are associated
053   to learned vertex features which are independent both during learning and inference. The feature
      vectors are computed by a DNN feature extractor, CNN or Transformer, which is trained to encourage

      them to be invariant to object viewpoint and instance. Using these compositional models, researchers
055   demonstrated superior performance in generalizing to OOD scenarios for tasks such as image
056   classificationJesslen et al. (2024), 3D pose estimation Wang et al. (2024); Kaushik et al. (2024b) and
057   6d pose estimation Ma et al. (2022). We note that the ability of these models to estimate object 3D
058   pose, as well as independently, makes them suitable for a range of tasks such as robotic manipulation
059   Jesslen et al. (2024)
060

061   as Pascal-3D+ Xiang et al. (2014) (12 object classes) and OOD-CVZhao et al. (2024) (10 object
062   classes), for two reasons. Firstly, because they require datasets with accurate 3D annotations for
063   learning. Secondly, because their learning algorithms scale badly. For example, for previous neural
064   mesh models Wang et al. (2024); Ma et al. (2022); Kaushik et al. (2024b); Jesslen et al. (2024), the
065   contrastive learning includes every vertex from every object class which scales quadratically. The
066   github code from Jesslen et al. (2024) will not run on more than 150 object classes due to memory
067   limitations. This raises the challenges we address in this work: (I) Can 3D compositional models
068   be scaled to a large number of object classes efficiently? (II) How will they perform compared to
069
      conventional neural networks in i.i.d. testing? (III) Most importantly, will they retain their important
      robustness properties (e.g., robustness to OOD, partial occlusion handling, multitasking) when scaled
070
      up?

118   Robust Image Classification and Pose Estimation Deep Networks have been shown to be non-
119
      robust Schneider et al. (2020); Rusak et al. (2021); Kortylewski et al. (2021) to simple nuisances in
      tasks like image classification ima (2021); Hendrycks & Dietterich (2019); Hendrycks et al. (2021)
120
      and 3D pose estimation Zhao et al. (2024). Nuisances like partial occlusion, weather, additive noise,
121
      etc. may not have much effect on human visual capabilities however can completely derail deep neural
122   networks outputs. A convincing theory attributes this fragility to lack of 3D compositional knowledge
123   in these models which humans possess Kaushik et al. (2024b). Methods like data augmentations, test
124   time adaptation, noise addition, input masking, etc. have been proposed to make neural models more
125   robust with varying but unsatisfactory levels of success with many arguing that we would need a
126   different architectural approach may be required Wang et al. (2017); Kortylewski et al. (2020) which
127   incorporates some 3D object knowledge in the models.
128   Robust Neural Compositional Models It refers to a family of 2D Kortylewski et al. (2020); Kaushik
129   et al. (2024a); Wang et al. (2017) and 3D models Wang et al. (2024); Jesslen et al. (2024); Kaushik
130   et al. (2024b) who have shown to be robust to out-of-distribution nuisances like partial occlusion Ko-
131   rtylewski et al. (2020); Wang et al. (2021), real and synthetic corruptions Jesslen et al. (2024); Kaushik
132   et al. (2024a;b) relative to conventional deep networks and have been utilized to perform robust image
133   classification Kortylewski et al. (2020); Jesslen et al. (2024), 3D and 6D pose estimation Wang et al.
134   (2024; 2021); Ma et al. (2022), amodal segmentation Sun et al. (2020) and unsupervised domain
135   adaptation Kaushik et al. (2024a;b). These models focus on learning object-centric, compositional
136
      neural representations and often employ the ideas of analysis-by-synthesis Yuille & Kersten (2006)
      in their applications. However, all of these previous works have only been shown to work on small-
137
      scale datasets often due to the computationally expensive nature of learning these compositional,
138
      object-centric representations. In this work, we build upon ideas introduced by this family of models
139   and scale them up efficiently to work with large datasets.
140
141   Contrastive Learning. Contrastive learning was originally developed for supervised learning Khosla
142
      et al. (2020); Misra & van der Maaten (2020); Wu et al. (2018); Chen et al. (2020); He et al. (2020)
      but has made its biggest impact when it was modified and applied to self-supervised learning Henaff
143
      (2020) giving state of the art results for many applications . Researchers have tried to adapt the idea
144
      of hard-negative mining Xia et al. (2022); Kalantidis et al. (2020); Wang et al. (2023) to improve
145   performance and to improve efficiency but for unsupervised or self-supervised contrastive learning the
146   lack of supervision makes it infeasible to adopt existing negative sampling strategies and motivates
147   the development of other strategies Robinson et al. (2021). Although these strategies can be effective
148   they are not always efficient because finding these hard samples takes time. Our approach differs in
149   two respects. Firstly, it is supervised so it is easier to define a hard negative (e.g., two objects that
150   are easily confused with each other). Secondly, we have a compositional structure of objects and
151   parts and so we can use contrastive learning on the parts can be driven by hard negative mining of the
152   objects.

      3.1   G ROUPED N EURAL O BJECT VOLUMES WITH DYNAMICALLY W EIGHTED
200         C OMPOSITIONAL C ONTRASTIVE L EARNING
201
202   Expanding on recent advances in Gaussian splatting Kerbl et al. (2023), we articulate the represen-
203   tation of a set of objects as a three-dimensional density field through a spatial arrangement of K
204   Gaussians, strategically placed on the surface geometries of distinct object categories. Each Gaussian
205   emanates a corresponding feature vector (which we will refer to as volume features), thereby delineat-
206   ing the volumetric representation termed as neural vertex feature. These features, representative of
207
      a single object category, is synthesized from feature maps derived from two-dimensional imagery,
      utilizing three-dimensional annotated poses of objects during training. For the task of classification
208
      and pose estimation, we find that cuboid geometries suffice Wang et al. (2021); Kaushik et al. (2024a);
209
      Jesslen et al. (2024) but more tightly defined geometries Wang et al. (2024) can also be used if
210   available.
211   In a departure from previous works, we train vertice features in a grouped manner, what we refer to
212   as Grouped Neural Vertex Contrasting using Dynamically Weighted Compositional Contrastive
213   Learning. We learn neural object volumes (and their volume features) using a contrastive loss where
214   the contrastive loss terms between similar object categories’ volume features have a higher weight and
215   dissimilar ones have a lower weight. This leads to a sparse and therefore more efficient contrastive
      loss calculation as most vertice feature pairs have the weight 0 associated with them i.e. we do not


                                                         4
      Under review as a conference paper at ICLR 2025


216
      calculate any corresponding loss terms. This contrastive loss formulation is termed compositional
217   since every vertice feature is composed of individual volume features which roughly correspond to
218   object parts. Additionally, we find that only a fraction of uniformly sampled volume features are
219   necessary for the grouped contrastive loss calculation making the training process even more efficient.
220
      Our grouped formulation helps us to ameliorate the drawbacks mentioned in the previous section 3.
221
      The advantages include
222
223         1. 95% reduction in the number of floating point operations for every contrastive loss calcula-
224            tion as we only calculate the distance between uniformly-sampled volume feature pairs of
225            categories with non-zero weights.
226
            2. Faster and easier contrastive loss optimization leading to better accuracy.

      Implementation The features extractor Φw of our architecture builds is a ViT-B/14 with DI-
391   NOv2 Oquab et al. (2023) pretraining. The input image size is 644 × 812 for ViT-B/14 backbone and
392   the output feature map F is 1/14th of the input size. Output features are projected to a dimension of
393   D = 128.
394
      A baseline model with ResNet50 He et al. (2016) feature extractor has two upsampling layers to
395
      integrate the output from the last three layers of ResNet50. The size of the feature map F is 1/8th of
396
      the input size. Output features are projected to a dimension of D = 128.
397
398   Our method is trained as described in section 3.1. For each class, the corresponding vertex feature is
399   composed of approximately K = 600 vertex features for each object class. To model the background,
400
      we use N = 2560 background features. We use momentum update for the vertex features using
      σ = 0.9 and sample T = 5 new background features from the image background to update B at each
401
      gradient step. Our model only takes 12 epochs to train fully.

410   Baselines We compare the performance of our approach to 2 competitive baseline architectures (that
411   is, Resnet50 and ViT-b-16) for the classification and 3D pose estimation tasks. During training,
412   these baselines are trained with a dual head: one for classification and one for pose estimation. This
413   approach allows to leverage of the 3D knowledge for classification and conversely class knowledge
414   for 3D pose estimation. For each baseline, the classification and pose estimation heads have an output
415

      the bin size (i.e., 3 · 40 = 120), respectively. We consider pose estimation as a classification task for
416

417
      classes of 9◦ each and predicting the corresponding bin. We finetuned each baseline for 100 epochs
418   using standard cross-entropy loss, and chose the best checkpoint with the highest test accuracy on
419   clean Imagenet3D+. In order to make baselines more robust, we apply standard data augmentation
420   (i.e., scale, rotation, and flipping) during training.

525   5     D ISCUSSION AND C ONCLUSION
526
527   In this work, we argue that endowing computer vision object models with 3D representations will
528   improve their performance, particularly in challenging out-of-distribution (OOD) scenarios. To
529   demonstrate this, we scaled up 3D-CompNets to 188 object categories taking advantage of a recent
530   dataset with 3D ground truth annotation. This scaling up required developing a modification of
531
      supervised contrastive learning, called Grouped neural Vertex with Dynamically weighted Composi-
      tional contrastive Learning(GVDComp). This algorithm exploited the compositional structure of
532
      3D-CompNets and can be used for other applications of contrastive learning of compositional models.
533
      GVDComp resulted in greatly increased speed and reduction in computation, for learning 3D-CGNs,
534   which increase significantly with the number of object classes. The resulting 3D-CompNets not only
535   outperformed conventional neural networks for object classification and 3D pose estimation when
536   tested IID (unlike 3D-CompNets tested previously) but also outperformed them even more signif-
537   icantly when tested in challenging OOD settings. We conclude that endowing object models with
538   explicit 3D representations has many advantages including improved performance and robustness,
539   as demonstrated in this paper, as well as applicability to a greater range of tasks including robotic
      manipulation.

009                                              A BSTRACT
010
011            Complex Query Answering (CQA) over Knowledge Graphs (KGs) is a fundamen-
012            tal yet challenging task. Given that KGs are usually incomplete, the CQA models
013            not only need to execute logical operators, but aslo need to leverage observed
014            knowledge to predict the missing one. Recently, a line of message-passing-based
015            research has been proposed to re-use pre-trained neural link predictors to solve
016
               CQA. However, they perform unsatisfactorily on negative queries and fail to ad-
               dress the unnecessary noisy messages between variable nodes in the query graph.
017
               Moreover, like most neural CQA models, these message passing models offer
018
               little interpretability and require complex query data and resource-intensive train-
019            ing. In this paper, we propose a Neural-Symbolic Message Passing framework
020            (NSMP) based on pre-trained neural link predictors. By introducing symbolic
021            reasoning and fuzzy logic, NSMP can generalize to arbitrary existential first order
022            logic queries without requiring training on any complex queries while providing
023            interpretable answers. Furthermore, we introduce an effective dynamic pruning
024            strategy to filter out noisy messages between variable nodes during message pass-
025            ing. Empirically, our model demonstrates strong performance and offers efficient
026            inference. Our code can be found at https://anonymous.4open.science/r/NSMP.

      1   I NTRODUCTION
029
030
      Knowledge graphs (KGs) store factual knowledge in the form of graph representations, which can
031   be applied to various intelligent application scenarios (Saxena et al., 2020; Wang et al., 2021a).
032   Complex Query Answering (CQA) over KGs is a fundamental and practical task, which requires
033   answering existential first order logic formula with logical operators including conjunction (∧), dis-
034   junction (∨), negation (¬), and existential quantifier (∃). A straightforward way is to traverse the
035   KG to identify the answers directly (Zou et al., 2011). However, given that modern KGs are usually
036   auto-generated (Toutanova & Chen, 2015) or built through crowd-sourcing (Vrandečić & Krötzsch,
037   2014), real-world KGs (Bollacker et al., 2008; Carlson et al., 2010) often suffer from incomplete-
038   ness, which is also known as the Open World Assumption (OWA) (Libkin & Sirangelo, 2009; Ji
039
      et al., 2021). This makes it impossible to answer a complex query with missing links using tradi-
      tional traversal methods. Therefore, the CQA models not only need to execute logical operators, but
040
      also need to utilize available knowledge to predict the unseen one (Ren & Leskovec, 2020).
041
042   Inspired by the success of neural link predictors (Bordes et al., 2013; Trouillon et al., 2016; Sun
043   et al., 2019; Li et al., 2022) on answering one-hop atomic queries on incomplete KGs, neural mod-
044   els (Hamilton et al., 2018; Ren & Leskovec, 2020; Zhang et al., 2021; Chen et al., 2022; Wang et al.,
045
      2023b; Zhang et al., 2024b) have been proposed to represent the entity sets by low-dimensional
      embeddings. Building on the foundation laid by these neural CQA models, message-passing-based
046
      research (Wang et al., 2023c; Zhang et al., 2024a) has demonstrated promising advancements in
047
      CQA. These message passing approaches represent logical formulas as query graphs, where each
048   edge represents an atomic formula containing a predicate with a (possible) negation operator, and
049   each node represents an input constant entity or a variable, as illustrated in Figure 1. By utiliz-
050   ing pre-trained neural link predictors, they perform one-hop inference on atomic formulas, thereby
051   inferring intermediate embeddings for variable nodes. An intermediate embedding is interpreted
052   as a logical message passed from the neighboring node on the corresponding edge. Following the
053   message passing paradigm (Gilmer et al., 2017), the embeddings of variable nodes are updated to
      retrieve answers. Due to the integration of pre-trained neural link predictors, these message pass-

      3.1   K NOWLEDGE G RAPHS
115
      Given a set of entities V and a set of relations R, a knowledge graph KG can be defined as a set
116   of triples E = {(ehi , ri , eti )} that encapsulates the factual knowledge, where each triple encodes
117   a relationship of type ri ∈ R between the head and tail entity ehi , eti ∈ V. According to the
118   definition in Wang et al. (2022b; 2023c), the KG is an LKG -structure defined by a language LKG .
119   Specifically, a first-order language L can be defined as a triple (F, R, C), where F, R, and C are sets
120   of function symbols, predicate symbols, and constants symbols, respectively. Under language LKG ,
121   the predicate symbols in R denote binary relations, the constant symbol set C is finite and comprises
122   all entities, and the set of function symbol satisfies F = ∅. In this case, each entity e ∈ V is also
123   a constant c ∈ C = V and each relation r ∈ R is a set r ⊆ V × V. In addition, only a part of the
124
      complete knowledge graph KG can be observed due to OWA. Let KG obs be the observed knowledge
      graph, we have KG obs ⊊ KG.

      3.2   E XISTENTIAL F IRST O RDER Q UERIES WITH A S INGLE F REE VARIABLE
128
      The complex queries that existing studies aim to address exclude universal quantifiers (Ren et al.,
129   2020; Ren & Leskovec, 2020). We follow Yin et al. (2024) and formally define such queries as
130   Existential First Order queries with a single free variable (EFO1 ), which are an important subset
131   of first-order queries, using existential quantifier, conjunction, disjunction, and atomic negation.
132   Following (Marker, 2006; Wang et al., 2022b; Yin et al., 2024), we give a set of definitions to
133   describe the EFO1 queries over knowledge graphs.
134   Definition 1. (Atomic Formula). An atomic formula is of the form ϕ = r(t1 , t2 ), where r ∈ R and
135   each ti is a term that is either a variable or a constant c ∈ C = V.
136   Definition 2. (Existential First Order Formula). The set of existential formulas is the smallest set Φ
137   that satisfies the following property:
138
139          • An atomic formula and its negation ϕ = r(t1 , t2 ), ¬ϕ = r(t1 , t2 ) ∈ Φ;
140
             • If ϕ, ψ ∈ Φ, then (ϕ ∧ ψ), (ϕ ∨ ψ) ∈ Φ;
141
142          • If ϕ ∈ Φ and xi is any variable, then ∃xi ϕ ∈ Φ.
143
144   A variable is a bound variable when associated with a quantifier. Otherwise, it is a free variable.
145   For an existential first order formula ϕ containing a free variable y, that is, an EFO1 formula, we
146   denote it as ϕ(y). Specifically, a formula with at least one free variable can be referred to as a query,
147   whereas one with no free variables can be called a sentence.
148   Definition 3. (Substitution). Given an EFO1 formula ϕ(y), for any constant entity e ∈ V, we denote
149   ϕ(e) as the result of replacing all free occurrences of y in ϕ with e.
150   Definition 4. (The Answer Set of EFO1 Query). For an EFO1 query ϕ(y), its answer set A[ϕ(y)] ⊆
151   V is a set of entities such that e ∈ A[ϕ(y)] iff ϕ(e) = T rue.
152
153   Most previous studies convert EFO1 queries into Disjunctive Normal Form (DNF) (Davey & Priest-
154   ley, 2002) to handle disjunction operators in a scalable manner (Ren et al., 2020; 2022; Wang et al.,
155
      2023b; Zhang et al., 2024a). Our work can also apply such DNF-based processing to answer com-
      plex queries. Therefore, we also give the definition of DNF here.
156
157   Definition 5. (Disjunctive Normal Form). The disjunctive normal form ϕDNF of an EFO1 formula
158
      is
159                                     ϕDNF (y) = CF1 (y) ∨ ... ∨ CFd (y),                                  (1)
160
161   where CFi = ∃x1 , ..., ∃xk .ai1 ∧...∧aini is a conjunctive formula, x1 , ..., xk are existential variables,
      y is the only free variable, and aij are atomic formulas or its negation.

      3.3   Q UERY G RAPH
168
169   As mentioned above, since a DNF query can be addressed by solving all its sub-conjunctive queries,
170   it is only necessary to define query graphs for conjunctive formulas. Following Wang et al. (2023c);
171   Zhang et al. (2024a); Yin et al. (2024), we represent the conjunctive formula as the query graph
172   where the terms are represented as nodes connected by the atomic formulas.
173   Definition 6. (Query Graph). For a conjunctive formula CF , its query graph G(CF ) is defined
174   as {(h, r, t, {0/1})}, where each quadruple represents an atomic formula and specifies whether it
175   is negated. The quadruple defines an edge with two endpoints h, t, and two attributes: r and 0/1.
176   Here, r denotes the relation, and 0/1 indicates whether the atomic formula is positive or negated.
177
      According to Definition 1, each node in the query graph is either a constant entity or a free or
178
      existential variable, as illustrated in the example in Figure 1.

      3.4   N EURAL L INK P REDICTORS
181
182   A neural link predictor is a differentiable model that embeds entities and relations into a low-
183   dimensional vector space to produce confidence scores for triples. Specifically, let h, r and t repre-
184   sent the embedding of the head entity, relation, and tail entity, respectively. The neural link predictor
185   uses its scoring function to compute the likelihood that a given triple exists. In particular, by apply-
186   ing a sigmoid function σ, it can output a continuous truth value φ(h, r, t) ∈ [0, 1] for the triple. For
187   example, we can get the truth value for a triple through ComplEx (Trouillon et al., 2016):
188

                                        φ(h, r, t) = σ(Re h ⊗ r, t ),                                       (2)
189
190   where ⊗ denotes element-wise complex number multiplication, ⟨·, ·⟩ represents the complex inner
191
      product, and Re refers to extracting the real part of a complex number. Similar to other message-
      passing-based models (Wang et al., 2023c; Zhang et al., 2024a), the input to the neural link predictor
192
      in our work includes not only embeddings of specific entities but also embeddings of variables.
193
      Following previous works (Arakelyan et al., 2021; 2023; Wang et al., 2023c), in our work, we use
194   ComplEx-N3 (Trouillon et al., 2016; Lacroix et al., 2018) as the neural link predictor of NSMP.

198   According to Definition 6, each edge in a query graph represents either an atomic formula or a
199   negated atomic formula. As such, an edge encodes information about the relation, logical negation,
200   and terms. Previous message passing CQA models (Wang et al., 2023c; Zhang et al., 2024a) leverage
201   this information to perform neural one-hop inference that maximizes the continuous truth value of
202   the (negated) atomic formulas, computing intermediate embeddings (i.e., logical messages) for the
203   variable nodes in the query graph. Specifically, a logical message encoding function ρ is proposed
204
      to define this neural one-hop inference on edges. On each edge, when a node is at the head position,

205

303   4.2     N EURAL -S YMBOLIC M ESSAGE PASSING
304
305   In this subsection, we propose a Neural-Symbolic Message Passing framework (NSMP). This frame-
306   work builds on the neural-symbolic one-hop inference proposed in Section 4.1 and incorporates a
307   dynamic pruning strategy. As a variation of the message passing neural network (Gilmer et al.,
308
      2017), each NSMP layer has two stages: (1) Passing neural-symbolic messages based on our dy-
      namic pruning strategy; (2) Updating the state of the variable node that has received messages.
309
310
311   4.2.1     M ESSAGE PASSING WITH DYNAMIC P RUNING
312
313   A query graph contains two types of nodes: constant and variable nodes. For constant nodes, we fol-
314   low Zhang et al. (2024a) and do not pass messages to constant nodes. However, the situation varies
315
      across different layers of message passing when passing messages to variable nodes. At the initial
      layers, messages passed from neighboring variable nodes, whose states have not yet been updated,
316
      can be regarded as noise, as these nodes do not carry any meaningful information at this stage. In
317
      contrast, at the later layers, messages from neighboring variable nodes with updated states provide
318   valuable information derived from constants, which is especially important for variable nodes not
319   directly connected to constant nodes. To dynamically filter out unnecessary noisy messages while
320   retaining valuable ones, we decide whether a variable node should pass messages to its neighboring
321   variable nodes based on whether its state has been updated. Specifically, a variable node is allowed
322   to pass messages to its neighboring variable nodes only when its state has been updated. This is
323   a dynamic pruning process, which can effectively avoid the computation of noise messages during
      message passing. Figure 2 illustrates an example of this pruning strategy at different layers.

      4.3    D ISCUSSION ON COMPUTATIONAL COMPLEXITY
379
380
      As a neural-symbolic CQA model based on message passing, NSMP is more efficient than the
381
      current state-of-the-art step-by-step precise symbolic search method FIT (Yin et al., 2024). Next, we
382   discuss the computational complexity of NSMP compared to FIT and reveal why message-passing-
383   based models can lead to more efficient inference than the step-by-step approach. Here, we focus
384   on time complexity, and the analyses of space complexity can be found in Appendix F.
385
      Since the computational bottleneck of NSMP lies in the symbolic-related parts, we focus on dis-
386
      cussing the time complexity of this aspect. Due to the sparsity of the adjacency matrix and fuzzy
387
      vector, both NSMP and FIT can utilize sparse techniques for efficient inference. But for simplicity,
388   we will not consider this sparsity in the following discussion.
389
390   According to Equations 7 - 10, the complexity of symbolic one-hop inference is O(|V|2 ), so the
391
      complexity of neural-symbolic message encoding function ϱ is approximately O(|V|2 ). This means
      that the complexity of message computation during message passing is linear to O(|V|2 ). For the
392
      node state update process, symbolic state update and neural state update are involved, corresponding
393
      to Equations 17 and 18, respectively. The complexity of the symbolic node state update is linear to
394   O(|V|), while the complexity of the neural node state update is O(|V|d), where d is the embedding
395   dimension. Since we have d ≪ |V|, the total computational complexity of NSMP is approximately
396   linear to O(|V|2 ). In particular, as a message-passing-based approach, the solving process of NSMP
397   is the same for both acyclic and cyclic queries, i.e., performing message passing on the query graph.
398   Consequently, the complexity of NSMP for any EFO1 formula is approximately linear to O(|V|2 ).
399
      For the previous step-by-step neural-symbolic method FIT, according to Yin et al. (2024), FIT solves
400   the acyclic query by continuously removing constants and leaf nodes, and the complexity of this
401   process is approximately linear to O(|V|2 ). However, for cyclic queries, FIT needs to enumerate one
402   variable within the cycle as a constant node, so the complexity is O(|V|n ), where n is the number of
403   variables in the query graph. In contrast, the complexity of NSMP on cyclic queries is approximately
404   linear to O(|V|2 ), which means that NSMP can provide more efficient inference on cyclic queries.
405   As we show in Section 5.2, NSMP achieves faster inference times when compared with FIT on cyclic
406   queries, with speedup ranging from 69× to over 150×. Moreover, while both NSMP and FIT exhibit
407   a complexity linear to O(|V|2 ) for acyclic queries, the computation of different messages in NSMP’s
408
      message-passing process is independent. This independence enables a parallelized computation of
      messages. In contrast, FIT requires removing constant and leaf nodes step by step, where the steps
409
      are interdependent, necessitating a serial process. As a result, even for acyclic queries, the message-
410
      passing-based NSMP can achieve more efficient inference. As demonstrated in Section 5.2, NSMP
411   achieves a speedup of at least 10× for acyclic queries in NELL995.

454
455   Datasets and Queries. We evaluate our model on two popular KGs: FB15k-237 (Toutanova &
456   Chen, 2015) and NELL995 (Xiong et al., 2017). We follow Chen et al. (2022); Xu et al. (2022) and
457   exclude FB15k (Bordes et al., 2013) since it suffers from major test leakage (Toutanova & Chen,
458   2015; Rossi et al., 2021). For a fair comparison with previous works, we evaluate our model using
459   both the datasets introduced by Ren & Leskovec (2020), which we refer to as the BetaE datasets,
460
      and the datasets proposed by Yin et al. (2024), which we refer to as the FIT datasets. In particular,
      according to Yin et al. (2024), the “pni” query type in the BetaE datasets is not a real EFO1 formula,
461
      we do not evaluate this query type in the BetaE datasets, but in the FIT datasets. For the graph
462
      representation of the query types and related statistics, please refer to Appendix B.
463
464   Evaluation Protocol. The evaluation scheme follows the previous works (Ren & Leskovec, 2020),
465   which divide the answers to each complex query into easy and hard parts. The difference between
466
      easy and hard answers lies in whether they can be obtained through direct graph traversal. Such hard
      answers cannot be found directly from the observed knowledge graph, which means that the CQA
467
      model needs to complete non-trivial reasoning. Specifically, for each hard answer of a query, we
468
      rank it against non-answer entities and compute the Mean Reciprocal Rank (MRR).
469
470   Baselines. We consider the state-of-the-art CQA models from recent years as our baselines, includ-
471   ing BetaE (Ren & Leskovec, 2020), CQD (Arakelyan et al., 2021), LogicE (Luus et al., 2021), ConE
472
      (Zhang et al., 2021), FuzzQE (Chen et al., 2022), GNN-QE (Zhu et al., 2022), ENeSy (Xu et al.,
473
      2022), CQDA (Arakelyan et al., 2023), LMPNN (Wang et al., 2023c), QTO (Bai et al., 2023b),
      CLMPT (Zhang et al., 2024a), and FIT (Yin et al., 2024), where LMPNN and CLMPT are based on
474
      message passing. We also compare more neural CQA models on the BetaE datasets in Appendix D.
475
      For details about the model, implementation and experiments, please refer to Appendix C.

532
533
      In this paper, we propose NSMP, a neural-symbolic message passing framework, to answer complex
534   queries over KGs. By integrating neural and symbolic reasoning, NSMP can utilize fuzzy logic
535   theory to answer arbitrary EFO1 queries without the need for training on complex query datasets,
536   while also offering interpretability through fuzzy sets. Moreover, we introduce a novel dynamic
537   pruning strategy to filter out unnecessary noisy messages during message passing. In our ablation
538   study, we validate the effectiveness of this strategy. Extensive experimental results demonstrate that
539   NSMP outperforms other message passing CQA models, achieving a strong performance, and can
      provide more efficient inference than previous state-of-the-art neural-symbolic CQA models.

784
785   A.1   N EURAL L INK P REDICTORS
786
787   Reasoning over KGs with missing knowledge is one of the long-standing topics in machine learning
788
      and has been widely explored. In the last few years, neural link predictors (Bordes et al., 2013;
      Yang et al., 2015; Trouillon et al., 2016; Dettmers et al., 2018; Sun et al., 2019; Chao et al., 2021)
789
      have been proposed to answer one-hop atomic queries on incomplete KGs. These latent feature
790
      models learn a low-dimensional vector for each entity and relation. By employing a well-defined
791   scoring function to assess link confidence, they can effectively predict unseen links. In addition to
792   these models, other research lines for one-hop KG reasoning include rule learning (Sadeghian et al.,
793   2019; Cheng et al., 2022; 2023), text representation learning (Yao et al., 2019; Wang et al., 2021b;
794   Saxena et al., 2022), reinforcement learning (Xiong et al., 2017; Das et al., 2018; Zhang et al., 2022),
795   and graph neural networks (Schlichtkrull et al., 2018; Vashishth et al., 2020; Teru et al., 2020).
796
797   A.2   N EURAL C OMPLEX Q UERY A NSWERING
798
799   Answering complex queries requires solving query formulas extended by atomic formulas combined
800   with existential first-order logical operators. Therefore, CQA is a more challenging task than one-
801
      hop KG reasoning. In recent years, neural models have been proposed to represent the set of entities
      by low-dimensional embeddings. Most of them conceptualize a complex query as an operator tree
802
      (Ren et al., 2020; Wang et al., 2022a), wherein first-order logical operators are replaced with cor-
803
      responding set operators. In particular, the existential quantifier induces a set projection operation,
804   which corresponds to the logic skolemization (Luus et al., 2021). Specifically, these CQA mod-
805   els embed the sets of entities in various forms, including geometric shapes (Hamilton et al., 2018;
806   Ren et al., 2020; Liu et al., 2021; Choudhary et al., 2021b; Zhang et al., 2021; Amayuelas et al.,
807   2022; Bai et al., 2022; Nguyen et al., 2023), probabilistic distributions (Ren & Leskovec, 2020;
808   Choudhary et al., 2021a; Yang et al., 2022; Long et al., 2022), fuzzy logic (Chen et al., 2022), and
809   bounded histograms (Wang et al., 2023b). Subsequently, neural set operations are applied to these
      set embeddings in accordance with the operator tree to derive the answer embedding. Other neural


                                                        15
      Under review as a conference paper at ICLR 2025


810
      models represent the complex queries as graphs and solve CQA with graph neural networks (Daza
811   & Cochez, 2020; Alivanistos et al., 2022) or transformers (Kotnis et al., 2021; Liu et al., 2022; Wang
812   et al., 2023a; Bai et al., 2023a; Xu et al., 2023). However, most of the aforementioned neural models
813   tend to be less effective than a simple neural link predictor on one-hop atomic queries.
814
      To this end, Wang et al. (2023c) proposed a logical message passing model, which is most related
815
      to our work. This model leverages the pre-trained neural link predictor to infer intermediate em-
816
      beddings for nodes in a query graph, interpreting these embeddings as logical messages. Through
817   message passing paradigm (Xu et al., 2019), the free variable node embedding is updated to re-
818   trieve the answers. While effective on both one-hop atomic and multi-hop complex queries, logical
819   message passing ignores the difference between constant and variable nodes, thus introducing noisy
820   messages. To mitigate this, recent work (Zhang et al., 2024a) proposed a conditional message pass-
821   ing mechanism, which can be viewed as a pruning strategy regardless of the messages passed by
822   variable nodes to constant nodes. However, this pruning strategy overlooks the noisy messages
823   between variable nodes at the initial layers of message passing. In contrast, the dynamic pruning
824   strategy employed in our model effectively eliminates these unnecessary noisy messages. Moreover,
825
      our proposed NSMP only needs to re-use a simple pre-trained neural link predictor without requiring
      training on complex query datasets, and offers interpretability through fuzzy sets.
826
827
828
      A.3    S YMBOLIC I NTEGRATION M ODELS
829   In addition to the models mentioned above, there are several neural models enhanced with symbolic
830   reasoning (Sun et al., 2020; Arakelyan et al., 2021; Zhu et al., 2022; Xu et al., 2022; Arakelyan
831   et al., 2023; Bai et al., 2023b) related to our work. These neural-symbolic models typically integrate
832   neural link predictors with fuzzy logic to address CQA. Most of them depend on the operator tree,
833   which, as noted in Yin et al. (2024), can only handle the existential first order logic formulas in an
834   approximate way. In contrast, our proposed neural-symbolic model, which utilizes the query graph,
835   enables a more natural and direct handling of these formulas. A recently proposed neural-symbolic
836   model (Yin et al., 2024) introduces an algorithm that cuts nodes and edges step by step to handle the
837
      query graph. Compared to this model, our message-passing-based approach can offer more efficient
      inference, as discussed in Section 4.3 and Section 5.2.

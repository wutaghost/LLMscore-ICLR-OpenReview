009                                               A BSTRACT
010
011            Fine-tuning large language models (LLMs) for downstream tasks has become
012            increasingly crucial due to their widespread use and the growing availability of
013            open-source models. However, the high memory costs associated with fine-tuning
014            remain a significant challenge, especially as models increase in size. To address
015            this, parameter efficient fine-tuning (PEFT) methods have been proposed to min-
016
               imize the number of parameters required for fine-tuning LLMs. However, these
               approaches often tie the number of optimizer states to dimensions of model param-
017
               eters, limiting flexibility and control during fine-tuning. In this paper, we propose
018
               sparse gradient compression (SGC), a training regime designed to address these
019            limitations. Our approach leverages inherent sparsity in gradients to compress
020            optimizer states by projecting them onto a low-dimensonal subspace, with dimen-
021            sionality independent of the original model’s parameters. By enabling optimizer
022            state updates in an arbitrary low-dimensional subspace, SGC offers a flexible
023            tradeoff between memory efficiency and performance. We demonstrate through
024            experiments that SGC can decrease memory usage in optimizer states more effec-
025            tively than exising PEFT methods. Furthermore, by fine-tuning LLaMA models
026            on various downstream tasks, we show that SGC can deliver superior performance
027
               while substantially lowering optimizer state memory requirements, particularly in
               both data-limited and memory-limited settings.

030   1   I NTRODUCTION
031
032   Large language models (LLMs) are increasingly being used across various disciplines, achieving
033   remarkable performance in a wide range of natural language processing tasks. With the release of
034   more open-source models, demand is growing to adapt them to downstream tasks (Touvron et al.,
035   2023; Dubey et al., 2024). This is typically achieved using full fine-tuning, where all the parameters
036   of a model are updated. However, as LLMs scale to billions of parameters, fine-tuning all the
037   parameters of a model becomes increasingly challenging, demanding substantial memory resources.
038   Full fine-tuning requires not only storing billions of model weights, but also maintaining the gradi-
039   ents and optimizer states needed during training, which can drastically increase memory consump-
040   tion (Chowdhery et al., 2022; Bai et al., 2023). For example, the Adam optimizer requires storing
041   both the first-and second-order moments of the gradients, doubling the memory needed compared to
042   storing the model’s trainable parameters (Kingma & Ba, 2017). These memory constraints limit the
043   practical ability to fine-tune LLMs, particularly in resource-constrained environments such as edge
044
      devices or personal computing platforms.
045   To address this problem, parameter efficient fine-tuning (PEFT) techniques have been introduced, to
046   train a model using a significantly smaller number of parameters (Ding et al., 2023; Han et al., 2024).
047   However, many existing methods lack the ability to provide both flexible and granular control over
048   the number of optimizer states used for fine-tuning. Flexibility refers to the capacity to accommodate
049   a broad range in the number of optimizer states, while granular control refers to the precision with
050
      which the number of optimizer states can be adjusted in small increments. This limitation may
      hinder the realization of a broader range of memory-performance tradeoffs, thereby restricting the
051
      potential of PEFT methods to achieve further efficiency gains.
052
053   On one end, we have approaches like BitFit (Zaken et al., 2022), which fine-tune on only the bias
      terms, using a minimal number of parameters, but is neither flexible nor offers granular control. On

069   the other hand, the popular low-rank adaptation (LoRA) is a more flexible approach that provides
070   some control over the number of trainable parameters (Hu et al., 2021). However, there still exists
071   limitations to both flexibility and granularity. LoRA reparameterizes the fine-tuned weight matrices
072   W (1) ∈ Rm×n into W (1) = W (0) + BA, where W (0) ∈ Rm×n is the frozen pre-trained weight
073   matrix, and A ∈ Rr×n and B ∈ Rm×r are two low-rank matrices of rank r (r ≪ min{m, n}) to
074   be trained. However, with LoRA, the number of optimizer states is a function of the dimensions
075   of A and B, which are dependent on n and m, respectively. The minimum number of trainable
076   parameters (achieved when r = 1) is equal to n + m, limited by the dimensions of W (0) . Therefore,
077
      there exists a bound dependent on n + m in which we cannot reduce the number of optimizer states
      during fine-tuning any further. Likewise, the granularity over parameters is also a function of n
078
      and m, and notice that both flexibility and granularity are impacted negatively with larger models.
079
      Although a slightly different formulation might be needed, a similar limitation exists with many
080   other approaches using prefix-tuning (Li & Liang, 2021) and gradient compression approaches,
081   such as GaLore (Zhao et al., 2024) (see Appendix A).
082
083
      To address the above limitation, we propose sparse gradient compression (SGC), a training regime
      that enables more flexible and granular control over the number of parameters to train during fine-
084
      tuning. SGC updates the optimizer states in a k-dimensional subspace, where k is independent
085
      of the original parameters dimension and represents the number of optimizer states. This allows
086   SGC to significantly reduce the number of optimizer states, irrespective of the pretrained model’s
087   size, with k providing flexibility to balance performance and memory efficiency (see Figure 1).
088   Importantly, this memory saving comes without sacrificing performance, as we will demonstrate in
089   our experimental results.
090
      The key idea behind SGC is leveraging the inherent sparsity of gradients during fine-tuning. By
091
      linearly projecting the optimizer states onto an arbitrarily lower-dimensional subspace, we can per-
092   form updates in this compressed space instead of the original space. A sparse recovery algorithm is
093   then used to project the result of the optimizer function back into the original space, estimating the
094   full-dimensional sparse vector from its lower dimensional representation, with sparsity originating
095   from the gradients. By fine-tuning LLaMA2-7B, LLaMA3-8B, and LLaMa2-13B (Touvron et al.,
096   2023; Dubey et al., 2024) on commonsense reasoning tasks, we show that SGC achieves comparable
097   or better results than other PEFT methods while using a significantly smaller number of optimizer
098   states. Additionally, we show that our approach yields improved fine-tuning performance in both
099   data-limited and memory-limited scenarios.

101   2   R ELATED W ORKS
102
103   Parameter Efficient Fine-tuning. PEFT methods are used to reduce the expensive memory re-
104   quirements for fine-tuning large models. Existing techniques can be split into several categories.
105   Adapter-based methods introduce additional trainable modules that are inserted into the original
106   frozen model (Houlsby et al., 2019; Pfeiffer et al., 2021; He et al., 2022; Mahabadi et al., 2021).
107   However, these approaches can introduce latency during inference. Prompt tuning, on the other
      hand, adapts a model by adding learnable prefix tokens to the input (Li & Liang, 2021; Lester et al.,


                                                        2
      Under review as a conference paper at ICLR 2025


108
      2021; Liu et al., 2022). Despite their simplicity, these methods have structural limitations since they
109   only train additional input tokens. LoRA is a widely used PEFT method that does not introduce
110   additional inference latency (Hu et al., 2021). LoRA employs low-rank matrices to approximate
111   the updates in the parameters during fine-tuning. Several variants of LoRA have been developed
112   to either improve performance or further reduce the number of trainable parameters (Zhang et al.,
113   2023; Xia et al., 2024; Liu et al., 2024; Kopiczko et al., 2024). Due to LoRA’s popularity, extensive
114   research has been conducted on both its theoretical foundations and empirical performance (Jang
115   et al., 2024; Hayou et al., 2024; Mao et al., 2024). Additionally, quantization-based methods have
116   been proposed to further reduce memory overhead Dettmers et al. (2023); Qin et al. (2024).
117   Gradient Compression. An area that has been relatively underexplored but is now gaining attention
118   is gradient compression (Zhao et al., 2024; Hao et al., 2024; Liang et al., 2024; Wu et al., 2024;
119   Song et al., 2024). These approaches selectively compress gradient information to reduce the size of
120   optimizer states during training. One category of methods uses projection matrices to obtain a lower-
121   rank gradients (Zhao et al., 2024; Hao et al., 2024; Liang et al., 2024). For instance, GaLore uses
122   singular value decomposition (SVD) to obtain projection matrices (Zhao et al., 2024), while FLoRA
123
      utilizes random projection matrices (Hao et al., 2024). Liang et al. (2024) propose a method that
      updates the projection matrix in an online fashion using principal component analysis. Alongside
124
      projection matrices, gradient sparsity is another emerging factor. SIFT shows that gradients are
125
      approximately sparse, and achieves efficient fine-tuning by selecting parameters corresponding to
126   the largest gradient magnitudes (Song et al., 2024). However, a significant limitation of this approach
127   is that the selected parameters remain static, failing to fully capture the dynamic nature of gradient
128   sparsity patterns during training.

208

209

210   fine-tuning. We highlight that Mt and Vt are functions of Gt ∈ Rd and G2t ∈ Rd , respectively.

215   sional subspace. This enables us to conduct fine-tuning with much greater efficiency and control
      over the memory usage.

245   Ideally, we would like to use Gt and G2t or their respective sparse versions G̃t and G̃2t for the

252
      to estimate a sparse vector from its compressed form, compressed through linear projection. CS is
253   a signal processing technique used to recover signals using fewer measurements than the Nyquist
254   rate, when the signals are sparse (Candes et al., 2004; Donoho, 2006). Consider an s-sparse signal
255   x ∈ Rd with s non-zero entries. We can reconstruct x from a set of linear measurements y = Ax, if
256   the measurement matrix A ∈ Rk×d satisfies the restricted isometry property (RIP) for some number
257   of measurements k ≤ d (Candes & Tao, 2005; Candes, 2008). The RIP conditions can be satisfied
258   with high probability if every element of A is independent√and identically distributed according to a
259   zero-mean normal distribution with standard deviation 1/ k, and k ≥ κs, where κ is an algorithm
260   dependent constant (Candes et al., 2004).

262   surements y (Marques et al., 2018). In this paper, we use a greedy algorithm named orthogonal

526   6      C ONCLUSION
527
528   In this work, we proposed a novel fine-tuning method, SGC, that enables flexible and granular con-
529   trol over the number of optimizer states. The key idea, leveraging the sparsity of the gradients, is
530   to compress them through a linear projection onto a subspace of an arbitrary dimension k, which
531   is independent of the original parameter dimensions. The updates are performed within this lower-
532   dimensional subspace, and the results are projected back into the original d-dimensional space,
533   effectively utilizing the gradient sparsity. This allows SGC to have significantly smaller and more
534
      granular number of parameters to train during fine-tuning compared to other PEFT approaches. We
      also provided two efficient implementations of SGC, MESGC and CESGC, and show through exper-
535
      iments that our approach can achieve comparable accuracy while being more memory efficient than
536
      other PEFT methods. Notably, we demonstrated that our approach achieves superior performance in
537   both data-limited and memory-limited settings, achieving higher accuracy than LoRA and GaLore.
538   Our approach is orthogonal to many gradient compression methods, opening opportunities for future
539   work to integrate them and explore SGC’s generalizability in domains like vision and audio.

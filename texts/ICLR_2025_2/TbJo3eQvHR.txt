009                                               A BSTRACT
010
011            Graph domain adaptation (GDA), which transfers knowledge from a labeled
012            source domain to an unlabeled target graph domain, attracts considerable atten-
013            tion in numerous fields. However, existing methods commonly employ message-
014            passing neural networks (MPNNs) to learn domain-invariant representations by
015            aligning the entire domain distribution, inadvertently neglecting category-level
016
               distribution alignment and potentially causing category confusion. To address
               the problem, we propose an effective framework named Coupling Category
017
               Alignment (CoCA) for GDA, which effectively addresses the category alignment
018
               issue with theoretical guarantees. CoCA incorporates a graph convolutional net-
019            work branch and a graph kernel network branch, which explore graph topology
020            in implicit and explicit manners. To mitigate category-level domain shifts, we
021            leverage knowledge from both branches, iteratively filtering highly reliable sam-
022            ples from the target domain using one branch and fine-tuning the other accord-
023            ingly. Furthermore, with these reliable target domain samples, we incorporate
024            the coupled branches into a holistic contrastive learning framework. This frame-
025            work includes multi-view contrastive learning to ensure consistent representations
026            across the dual branches, as well as cross-domain contrastive learning to achieve
027
               category-level domain consistency. Theoretically, we establish a sharper general-
               ization bound, which ensures the effectiveness of category alignment. Extensive
028
               experiments on benchmark datasets validate the superiority of the proposed CoCA
029
               compared with baselines.

050   Currently, various GDA methods have been proposed (Yin et al., 2022; 2023; You et al., 2022b;
051   Zhu et al., 2023) by combining domain adaptation techniques with graphs. They usually assume
052   the distribution invariance is limited (Garg et al., 2020; Verma & Zhang, 2019) and directly employ
053   adversarial training to align source and target distribution (Zhang et al., 2019b; Wu et al., 2020a).
      However, the classifier still tends to favor source domain features and makes incorrect predictions

054
      on the target domain due to category-agnostic feature alignment (Zhang et al., 2019a). To solve the
055   issue and efficiently design the GDA framework, we still need to address the following challenges:
056   (i) How to fully exploit the features of the source and target domain for representation learning.
057   Previous approaches typically employ the MPNNs to capture implicit topological semantics. How-
058   ever, the absence of labels for the target domain poses challenges in obtaining sufficient topological
059   semantics. (ii) How to effectively align category-level distribution. While there has been progress
060   in matching the marginal distributions between two domains, they may not efficiently align the cat-
061   egory distribution, leading to a degradation in classification performance. Certain methods have
062   attempted to acquire pseudo-labels for the target graphs in target domain training (Yin et al., 2023;
063
      Yehudai et al., 2021; Ding et al., 2021; Zhu et al., 2023), they are vulnerable to bias in cases of sig-
      nificant domain shift, leading to error accumulation in subsequent optimization. (iii) How to design
064
      the GDA framework with the grounded theoretical foundation. Theoretically, the generic domain
065
      adaptation (DA) bound is not specific to graph data and models (You et al., 2023). However, we can
066   still design a more precise model tailored for graphs with the theoretical guarantee.
067
068   To tackle these challenges, we propose a framework named Copling Category Alignment (CoCA)
069
      for unsupervised domain adaptive graph classification. Specifically, to fully exploit the features of
      both source and target graphs, CoCA incorporates an MPNN branch and a shortest path aggrega-
070
      tion branch. The MPNN branch leverages neighborhood aggregation to implicitly learn topological
071
      semantics, while the shortest path aggregation branch generates paths for each node and utilizes po-
072   sition encoding to extract informative graph-level semantics. This shortest path aggregation branch
073   provides explicit high-order structural semantics, serving as a complementary enhancement. To col-
074   laborate knowledge from two branches, we jointly train the branches by iteratively filtering highly
075   reliable samples from the target domain using one branch and fine-tuning the other branch accord-
076   ingly. Specifically, with the dual pre-trained branches, CoCA first fine-tunes the shortest path branch
077   with the highly reliable samples filtered from the target domain with the MPNN branch and then opti-
078   mizes the MPNN branch with the filtered target domain samples labeled by the shortest path branch.
079   Theoretically, the interactive optimization of one branch with the support of the other one would
080
      gradually mitigate the category-level distribution shift. Furthermore, we embed the iterative learn-
      ing process into a holistic contrastive learning framework, incorporating cross-domain contrastive
081
      learning to achieve category-level domain consistency, alongside multi-views contrastive learning
082
      to ensure consistent representations between branches. Overall, our approach emphasizes a unique
083   focus on achieving category-level domain alignment. Specifically, our methodology is centered on
084   Coupling Category Alignment (CoCA), which systematically iterates between branches to identify
085   and select reliable samples. This process facilitates cross-branch adjustments, effectively mitigating
086   potential domain shifts in an unsupervised manner. By iteratively refining the alignment process,
087   our approach enhances the model’s ability to achieve category-level alignment, supported by solid
088   theoretical foundations, distinguishing it from prior methods. Extensive experiments conducted on
089   various datasets with domain shifts for graph classification demonstrate that the proposed CoCA
090   outperforms state-of-the-art baselines.
091   In summary, the main contributions can be summarized as three-fold: (1) Problem Formulation:
092   We present a novel problem in graph domain adaptation, which highlights the discrepancy in the
093   distribution of graph categories between the source and target domains, posing significant chal-
094   lenges for accurate graph classification across domains. (2) Methodology: We propose a framework
095   named CoCA, which utilizes two branches to explore structural semantics and integrates them into
096   a category-level domain-invariant model. We provide theoretical proof demonstrating that CoCA
097
      is specifically designed to more accurately address the challenges of the graph domain. (3) Exper-
      iments: Extensive experiments conducted on various domain shift datasets for graph classification
098
      demonstrate the effectiveness of the proposed CoCA.

102
103   Graph Classification. GNNs (Kipf & Welling, 2017b) have shown exceptional performance across
104   a wide range of graph-based machine learning tasks, such as node classification (Rong et al., 2020;
105   Hu et al., 2019; Kipf & Welling, 2017b), graph classification (Fan et al., 2019; Song et al., 2016;
106   Liao et al., 2021; Wu et al., 2020b) and link prediction (Zhang & Chen, 2018; Cai et al., 2021).
107   The most prevalent GNNs follow the message-passing paradigm, which aggregates the neighbors
      for node update and applies graph pooling for graph representation. Nevertheless, MPNNs have


                                                         2
      Under review as a conference paper at ICLR 2025


108
      limited capacity to capture high-order topological structures, such as paths and motifs (Du et al.,
109   2019; Michel et al., 2023; Ju et al., 2022; Qiu et al., 2021). Therefore, numerous graph kernel
110   methods have emerged to overcome this flaw (Long et al., 2021; Cosmo et al., 2021; Wang et al.,
111   2022). However, these approaches typically require an ample supply of labeled annotations (Yin
112   et al., 2022; 2023) while this work delves into the realm of unsupervised graph domain adaptation
113   and introduces a novel approach CoCA to tackle this challenge.
114
      Unsupervised Domain Adaptation. Unsupervised domain adaptation is to learn domain-invariant
115   representations that enable the transfer of a model from a source domain with abundant labels to a
116   target domain with a scarcity of labels (Ma et al., 2021; Singh, 2021; Feng et al., 2023; Kouw &
117   Loog, 2019). The majority of technical routes can be broadly categorized into domain discrepancy-
118   based methods and adversarial approaches. The former methods typically incorporate different dis-
119   tribution metrics like maximal mean discrepancy (Saito et al., 2018) and Wasserstein distance (Shen
120   et al., 2018) to measure the discrepancy between different domains. Conversely, adversarial ap-
121   proaches involve a domain discriminator that is fused to implicitly reduce the domain discrepancy.
122   However, these methods typically concentrate on Euclidean data such as images and texts, while
123
      graph domain adaptation has not been extensively explored. In this work, we explores graph seman-
      tics by utilizing dual perturbation branches for effective graph domain adaptation.
124
125   Graph Domain Adaption. Due to the potential economic value, graph domain adaptation (Lin
126   et al., 2023; Wu et al., 2022b; Luo et al., 2023) is a crucial problem in the fields of social analysis
127   and molecular biology (You et al., 2022b; Zhu et al., 2023). Existing methods mainly focus on
128   how to transfer information from source graphs to unlabeled target graphs to learn effective node-
129
      level (Wu et al., 2022a; Zhu et al., 2021; Dai et al., 2022; Guo et al., 2022) and graph-level (Yin
      et al., 2023; Yehudai et al., 2021; Ding et al., 2021; Yang et al., 2020) representation. However,
130
      these approaches commonly merge GNNs with domain alignment (Luo et al., 2023) techniques,
131
      which overlook the alignment of category distributions in the presence of label scarcity and domain
132   shift, consequently leading to a deterioration in classification performance. Towards this end, CoCA
133   couples the dual branch in a variational optimization framework to address the issue.

      4.4     C OMPLEXITY A NALYSIS .
357
358   The algorithmic complexity of CoCA consists of two main components: dual-branch learning and
359   branch coupling. The complexity of the dual-branch learning process is given by O(LN (N + E +
360   Kd) + LN 2 d), where L denotes number of layers in the network, d is the dimension of features, N
361   is the number of nodes in the graph, E is the number of edges, and d is the maximum path length.
362   The complexity of branch coupling, which involves the interaction between branches, is O(B 2 d),
363   where B is the number of selected samples for coupling. Thus, the overall complexity of CoCA is
364   O((LN 2 + B 2 + LN K)d + LN E).
365   CoCA’s complexity is influenced by its hyperparameters d, L, B, and K. By appropriately control-
366   ling these values, the model’s complexity can be managed effectively to ensure scalability.For small
367   graphs, where N ≈ d and d ≈ E, the complexity simplifies to O((LN 2 + B 2 + LN K)d). For large
368   graphs, where N ≪ d, E ≪ d, N ≪ B with K customarily set to a small value, the complexity
369   reduces to O(LN 2 d + LN E).
370   For large graphs, CoCA’s complexity approximates that of the Graph Multiset Transformer (GMT),
371   which is O(LN 2 d). This demonstrates that CoCA maintains competitive scalability while incorpo-
372   rating additional capabilities for category-level alignment.

      5.5        S ENSITIVITY A NALYSIS
497
498   In this part, we investigate the influence of hyperparameters on the performance of the proposed
499   CoCA. We specifically examine the effects of two key hyperparameters, including the threshold
500   ζ in the branch coupling module for category alignment, and the shortest path length K in the
501   SP branch. We report the results of ζ and K in Figure 4. ζ determines the number of reliable
502   samples selected from each branch, and we vary ζ in the range from 0.5 to 0.9. The experimental
503   results presented in Figure 4 (a), (b) indicate an initial increase followed by stability or a decreasing
504   trend in performance as ζ increases. We attribute the reason to the fact that smaller values of ζ
505   introduce low-confidence samples, which would detriment the performance of CoCA. Conversely,
506
      larger values of ζ introduce high-confidence samples for training. However, excessively high values
      of ζ may lead to fewer filtered samples, potentially resulting in a decline in model performance.
507
      Therefore, we set ζ to 0.7 as default. Additionally, the parameter K controls the number of shortest
508
      paths extracted in the SP branch, and we vary K in the range of {2, 3, 4, 5, 6}. The results are
509   shown in Figure 4 (c) and (d). From the results, we observe that increasing K generally leads
510   to improved performance when the value is small. This suggests that incorporating more shortest
511   path aggregations can enhance the representation capability. However, when K becomes large, the
512   performance stays stable. Considering the significant increase in algorithmic complexity associated
513   with higher values of K, we set K = 5. Additionally, we examine the accuracy of filtered samples
514   from MP and SP branch within the branch coupling module, and the results are shown in Appendix I.

      In this paper, we study a practical problem of unsupervised graph classification and propose a novel
519   approach named CoCA. We utilize the dual branch, i.e., a message passing and a shortest path ag-
520   gregation branch, to explore the graph semantics from the implicit and explicit perspectives. The
521   introduction of the branch coupling module ensures effective category-level alignment, mitigating
522   the category-agnostic issues typically encountered in traditional GDA methods. Furthermore, the
523   framework incorporates cross-domain and multi-view contrastive learning, enhancing the consis-
524   tency of representations across domains and branches. Theoretical analysis demonstrates that CoCA
525   achieves a tighter empirical risk bound compared to existing GDA methods. Extensive experiments
526   across multiple datasets validate the superiority of CoCA in handling domain shifts. In future work,
527
      we aim to explore extending CoCA to more complex scenarios, such as domain generalization and
      source-free domain adaptation.

1005   E    I MPLEMENTATION D ETAILS
1006
1007   In our CoCA, we employ GMT (Baek et al., 2021) in the MP branch and the shortest path model (Ab-
1008   boud et al., 2022) in the SP branch. For the SP branch, the maximum path lengths K are set to 5 for
1009   all the datasets. For the adaptive perturbation, we set the T = 5 for perturbation learning. We warm
1010   up the dual branch for 50 epochs and update the branch coupling module 10 times. The pseudo-
1011   label filtering threshold ζ for the target dataset is set to 0.7. For all the methods, we use one of the
1012
       sub-datasets as source data and the remaining as the target data for performance comparison. We set
       the hidden size to 128 and the learning rate to 0.001 as default. All the experiments are conducted
1013
       on the same device, equipped with NVIDIA A6000 GPU.

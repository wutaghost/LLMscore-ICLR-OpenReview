009                                               A BSTRACT
010
011            Continual learning (CL) is a paradigm that adapts to and retains knowledge from
012            a stream of tasks. Despite the growing number of experimental methods in CL,
013            there is a lack of rigorous theoretical analysis, particularly in memory-based
014            continual learning (MCL), which remains an open research area. In this paper, we
015            theoretically analyze the impact of memory in CL and derive explicit expressions
016            for expected forgetting and generalization errors under overparameterized linear
017            models. We propose a detailed matrix decomposition of the data to distinguish
018
               between current and previous datasets, effectively decoupling the coupled data for
               different tasks. Additionally, we conduct a comprehensive mathematical analysis
019
               for scenarios with a small number of tasks and employ numerical analysis for
020
               larger task scenarios to evaluate the overall properties of expected forgetting and
021            generalization errors. Compared with CL, our theoretical analysis suggests that (1)
022            a larger memory buffer must be paired with a larger model to effectively reduce
023            forgetting; (2) training with a larger memory buffer generalizes better when tasks
024            are similar but may perform worse when tasks are dissimilar, while training with a
025            large model can help mitigate this negative effect. Ultimately, our findings here
026            sheds new light on how memory can assist CL in mitigating catastrophic forgetting
027            and improving generalization.

030   1    INTRODUCTION
031
032   Continual learning (CL) aims to dynamically learn from a sequence of tasks without forgetting
033
      previously acquired knowledge. Traditional neural networks are typically trained on fixed datasets
      and evaluated in static environments. However, in real-world scenarios, data is often presented in
034
      a non-stationary manner, requiring models to update their knowledge continuously. This dynamic
035
      learning faces a major challenge known as catastrophic forgetting (McCloskey & Cohen, 1989),
036   where acquiring new information results in the forgetting of previously learned knowledge.
037
038   Even though various empirical continual learning (CL) methods (Riemer et al., 2018; Buzzega
039
      et al., 2020; Bohao et al., 2024) have been proposed recently, rigorous theoretical analysis remains
      limited, particularly for memory buffer-based continual learning (MCL). As MCL is one of the
040
      most straightforward and widely used approaches to mitigate catastrophic forgetting, a thorough
041
      theoretical investigation into its mechanisms and limitations is essential. Recent works have attempted
042   to develop a theoretical understanding of memory in CL (Knoblauch et al., 2020; Chen et al., 2022;
043   Han et al., 2023). However, none of these studies provide explicit forms to characterize forgetting and
044   generalization for MCL. In this paper, we consider both limited and unlimited-size memory buffers
045   to provide a more comprehensive analysis of memory buffers. We derive explicit expressions under
046   overparameterized linear models to offer a clearer understanding of how memory can assist CL in
047   mitigating catastrophic forgetting and improving generalization.
048   To conduct these theoretical studies, there are two main challenges: (i) The training dataset contains
049   both the data from previous and current tasks, resulting in the coupling of the data matrix and label
050   vector. This coupling means that it is not possible to represent the labels for each task using a single
051   ground truth matrix. (ii) In the expressions for forgetting and generalization error, the increasing
052   and decreasing terms are coupled, making it difficult to analyze their monotonicity directly. To
053   address these challenges, we first perform a detailed matrix decomposition of the data matrix and
      the label vector, using the ground truth matrix from each previous task in combination to represent

092           ‚ÄĘ We consider the reservoir sampling-based memory buffer as our limited memory buffer and
093             the full rehearsal memory buffer as the unlimited memory buffer. For both buffer types,

      2     R ELATED W ORK
109
110
      Memory buffer-based CL. Memory buffer-based methods are mainly divided into experience replay
111
      and generative replay. Experience replay stores a subset of data from previous tasks in a replay
112   buffer, following specific sample selection principles, and uses it for training in the current task. One
113   common buffer strategy is reservoir sampling (Riemer et al., 2018; Buzzega et al., 2020), where every
114   previously seen example has an equal probability of being stored in the buffer at each training round.
115   More advanced strategies include gradient-based (Saha et al., 2021; Aljundi et al., 2019; Borsos
116   et al., 2020) and optimization-constrained approaches (Chaudhry et al., 2019; Yin et al., 2020; Yoon
117   et al., 2021). Generative replay typically requires an additional generative model to produce synthetic
118   samples for previous tasks. Earlier methods generally use GAN or VAE as the generator (Shin
119   et al., 2017; Chenshen et al., 2018; Zhai et al., 2019). Recently, leveraging the powerful generative
120
      capabilities of diffusion models, some methods have begun to explore using diffusion models as the
      generator (Gao & Liu, 2023; Kim et al., 2024). In addition, there are some methods combine memory
121
      with other techniques like knowledge distillation. For example, iCaRL (Rebuffi et al., 2017) uses a
122
      buffer to train a nearest-mean-of-exemplars classifier while preventing representation degradation in
123   later tasks through a self-distillation loss.
124
125   Theoretical analysis of CL. There have been only a few attempts to develop a theoretical understand-
126
      ing of memory in CL. For instance, (Knoblauch et al., 2020) uses set theory to show that achieving
      perfect memory for optimal CL is NP-hard. (Chen et al., 2022) applies the PAC framework to analyze
127
      the lower bound of memory requirements in CL. (Han et al., 2023) provides a convergence analysis
128
      of memory-based CL with stochastic gradient descent, framing it as a smooth nonconvex finite-sum
129   optimization problem. Several theoretical studies focus on linear regression tasks. (Evron et al.,
130   2022) proves an upper bound for catastrophic forgetting under specific task orderings. (Lin et al.,
131   2023) presents explicit expressions for forgetting and generalization error in overparameterized linear
132   models. (Goldfarb & Hand, 2023) analyze the impact of overparameterization for linear models in a
133   two-task scenario. (Ding et al., 2024) offers a general theoretical analysis of forgetting in the linear
134   regression model using stochastic gradient descent. (Zhao et al., 2024) provides a statistical analysis
135   of regularization-based continual learning across a sequence of linear regression tasks. However,
136   none of these works provide explicit forms of forgetting and generalization errors for memory-based
137
      continual learning (MCL). In contrast, we offer explicit results for MCL without making assumptions
      about the ground truth vectors and provide a comprehensive discussion on how memory buffer size
138
      and the number of parameters affect forgetting and generalization.
139
140   The most relevant study to our work is (Lin et al., 2023), which also investigated CL in overpa-
141   rameterized linear models. However, our work differs significantly from (Lin et al., 2023) in the
142   following aspects: (1) We focus on analyzing forgetting and generalization errors when training with
143
      memory buffers, whereas (Lin et al., 2023) considered training without memory buffers; (2) Our
      analysis explores the interplay between memory buffer size, overparameterization, and the number
144
      of tasks, emphasizing the conditions under which these factors can mitigate forgetting and enhance
145
      generalization. In contrast, (Lin et al., 2023) did not address the impact of the number of tasks;
146   (3) Our findings provide new insights into how memory can assist CL in alleviating catastrophic
147   forgetting and improving generalization.

      Multi-task Training For the t-th training round, the full rehearsal memory buffer retains all samples
      from the previous t ‚ąí 1 tasks. Consequently, at any given training round, training with a full rehearsal
438
      memory buffer effectively operates as a form of multi-task learning (MTL), utilizing samples from
439
      both the previous tasks and the current task simultaneously. Previous theoretical analyses of MTL
440   often rely on concepts such as VC dimension (Crammer & Mansour, 2012; Ben-David & Borbely,
441   2008), covering number (Baxter, 2000), and Rademacher complexity (Maurer, 2006; Pontil &
442   Maurer, 2013) to derive generalization bounds. However, these approaches do not account for the
443   overparameterized regime and fail to provide explicit forms for generalization error. In Theorem
444   2, we address these limitations by offering explicit expressions for forgetting and generalization
445   errors in MCL under the overparameterized linear regime. Additionally, while MTL learns tasks
446   simultaneously, MCL handles a sequential stream of tasks.
447   Training with a full rehearsal buffer may indeed exacerbate forgetting in certain cases, similar to
448   the reservoir sampling-based memory buffer discussed in Subsection 4.1. Moreover, training with
449   the full rehearsal buffer in these scenarios results in significantly higher computational and storage

496   5    CONCLUSION
497
498   In this paper, we studied MCL in overparameterized linear models. Specifically, we analyzed the
499   impact of limited-size memory and unlimited-size memory in CL. To achieve this, we selected the
500   reservoir sampling-based memory buffer and the full rehearsal memory buffer as the primary focus of
501   our analysis. We provided explicit expressions for the expected forgetting and generalization errors
502   for both memory strategies. Additionally, we investigated the impact of memory buffer size and the
503
      number of model parameters on both forgetting and generalization errors. To verify our findings, we
      conducted a comprehensive mathematical analysis for scenarios with a small number of tasks and
504
      employed numerical analysis for larger task scenarios.
505
506   Limitation: Our work focuses on overparameterized linear models, without considering deep neural
507   networks (DNNs) or nonlinear activation functions. This simplifies the theoretical analysis and allows
508   us to derive explicit expressions for expected forgetting and generalization errors. However, extending
509
      this analysis to DNNs and incorporating the effects of nonlinear activation functions remains an
      important direction for future work.

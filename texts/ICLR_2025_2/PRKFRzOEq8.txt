009                                                         A BSTRACT
010
011
               Conformal Prediction (CP) is a method to control prediction uncertainty by producing a small
012            prediction set, ensuring a predetermined probability that the true class lies within this set. This is
013            commonly done by defining a score, based on the model predictions, and setting a threshold on this
014            score using a validation set. In this study, we address the problem of CP calibration when we only have
015            access to a validation set with noisy labels. We show how we can estimate the noise-free conformal
016            threshold based on the noisy labeled data. Our solution is flexible and can accommodate various
017            modeling assumptions regarding the label contamination process, without needing any information
018            about the underlying data distribution or the internal mechanisms of the machine learning classifier.
019            We develop a coverage guarantee for uniform noise that is effective even in tasks with a large
020
               number of classes. We dub our approach Noise-Aware Conformal Prediction (NACP) and show on
               several natural and medical image classification datasets, including ImageNet, that it significantly
021
               outperforms current noisy label methods and achieves results comparable to those obtained with a
022
               clean validation set.

026   1    I NTRODUCTION
027
028   In machine learning for safety-critical applications, the model must only make predictions it is confident about. One
029   way to achieve this is by returning a (hopefully small) set of possible class candidates that contain the true class
030   with a predefined level of certainty. This is a natural approach for medical imaging, where safety is of the utmost
031   importance and a human makes the final decision. This allows us to aid the practitioner, by reducing the number
032   of possible diagnoses he needs to consider, with a controlled chance of mistake. The general approach to return a
033
      prediction set without any assumptions on the data distribution (besides i.i.d. samples) is called Conformal Prediction
      (CP) (Angelopoulos et al., 2023; Vovk et al., 2005). It creates a prediction set with the guarantee that the probability
034
      of the correct class being within this set meets or exceeds a specified confidence threshold. The goal is to return the
035
      smallest set possible while maintaining the confidence level guarantees. Recently, with the growing use of neural
036   network systems in safety-critical applications such as medical imaging, CP has become an important calibration tool
037   (Lu et al., 2022a;b; Olsson et al., 2022). We note that CP is a general framework rather than a specific algorithm. The
038   most common approach builds the prediction set using a conformity score, and different algorithms mostly vary in
039   terms of how the conformity score is defined.
040
      When dealing with conformal predictions, a critical challenge arises in applications such as medical imaging due to
041
      label noise. In these domains, datasets frequently contain noisy labels stemming from ambiguous data that can confuse
042   even clinical experts. Furthermore, physicians may disagree on the diagnosis for the same medical image, leading to
043   inconsistencies in the ground truth labeling. Noisy labels also occur when applying differential privacy techniques to
044   overcome privacy issues (Ghazi et al., 2021). While significant efforts have been devoted to the problem of noise-robust
045   network training (Song et al., 2022; Xue et al., 2022), the challenge of calibrating the models has only recently begun to
046   receive attention.
047
      In this study, we tackle the challenge of applying CP to classification networks using a validation set with noisy labels.
048   Einbinder et al. (2022) suggested ignoring label noise and simply applying the standard CP algorithm on the noisy
049   labeled validation set. This strategy results in large prediction sets especially when there are many classes. A recent
050   study suggests estimating the noise-free conformal score given its noisy version and then applying the standard CP
051   algorithm (Penso & Goldberger, 2024). The most related study to ours is (Sesia et al., 2023) which presents a noisy CP
052   algorithm using conservative coverage guarantee bounds which can result in large prediction sets. Here, we present a
053   novel algorithm for CP on noisy data that yields an effective coverage guarantee even in tasks with a large number of
      classes. We applied the algorithm to several standard medical and scenery imaging classification datasets and show that

      our method outperformed previous methods by a significant margin and achieved results comparable to those obtained
055   by using a clean validation set.

      2.1   C ONFORMAL P REDICTION
061
      Consider a setup involving a classification network that categorizes an input x into k predetermined classes. Given a
062   coverage level of 1 − α, we aim to identify the smallest possible prediction set (a subset of these classes) ensuring the
063   correct class is within the set with a probability of at least 1 − α. A straightforward strategy to achieve this objective
064   involves sequentially incorporating classes from the highest to the lowest probabilities until their cumulative sum
065   exceeds the threshold of 1 − α. Despite the network’s output adopting a mathematical distribution format, it does not
066   inherently reflect the actual class distribution. Typically, the network will not be calibrated and it tends to be overly
067   optimistic (Guo et al., 2017). Consequently, this straightforward approach doesn’t assure the inclusion of the correct
068   class with the desired probability.
069   The first step of the CP algorithm involves forming a conformity score S(x, y) that measures the network’s uncertainty
070   between x and its true label y (larger scores indicate worse agreement). The Homogeneous Prediction Sets (HPS) score
071   (Vovk et al., 2005) is SHP S (x, y) = 1 − p(y|x; θ), s.t. θ is the network parameter set. The Adaptive Prediction Score
072   (APS) (Romano et al., 2020) is the sum of all class probabilities that are not lower than the probability of the true class:

077   variant of APS, which is defined as follows:

088   sets (Angelopoulos et al., 2023). The CP prediction set of a data point x is defined as Cq (x) = {y|S(x, y) ≤ q}
089   where q is a threshold that is found using a labeled validation set (x1 , y1 ), ..., (xn , yn ). The CP theorem states

092   In the random case there is still a coverage guarantee, which is defined by marginalizing over all test points x and
093
      samplings u from the uniform distribution (Romano et al., 2020). Note that the coverage guarantee is for a marginal
      probability over all possible test points and coverage may be worse or better for different points. It can be proved that
094
      obtaining a conditional coverage guarantee is impossible (Foygel Barber et al., 2021).

315   NACP method for a noise matrix model is summarized in Algorithm 2.
316
317
      We can extend the finite sample term ∆ that was developed for a uniform noise to obtain a theoretical coverage guarantee
      for a noise matrix model (see Section A.2). However, this approach yields large prediction sets especially in tasks with
318
      many classes and thus is ineffective. In the experiment section we show that in practice, even without adding finite
319
      sample terms, we obtain the required coverage probability.

501   5   C ONCLUSIONS
502
503   We presented a procedure that applies the Conformal Prediction algorithm on a validation set with noisy labels. We first
504   presented our method in the simpler case of a uniform noise model and then extended it to a general noise matrix. We
505   showed that if the noise level is given, we can find the noise-free calibration threshold without access to clean data by
506   using the noisy-label data. We provided a finite-sample coverage guarantee for the case of uniform noise. We showed
507   that in case of uniform noise our method outperforms current noisy CP methods by a large margin, in terms of the
508   average size of the prediction set, while maintaining the required coverage. In all the experiments we conducted the
509   finite sample term was effective and yielded a small prediction set. We showed, however, that even without adding
510
      the finite term we obtained the required coverage. This indicates that the current coverage guarantee analysis is too
      conservative and there is room for future research to improve it. In this study, we focused on noise models that assume
511
      the noisy label and the input image are independent, given the true label. In a more general noise model, the label
512
      corruption process also depends on the input features.

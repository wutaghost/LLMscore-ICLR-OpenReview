                                                 A BSTRACT
008
009
               Defining the convolution on graphs has led to much progress in graph machine
010
               learning, particularly through approximations based on polynomials and, ulti-
011            mately, message-passing neural networks (MPNNs). However, this convolution
012            is defined for single-channel graph signals, i.e., a single feature is given at each
013            node, and a single new feature is assigned to each node. As multiple initial node
014            features are provided for many challenging tasks and convolutions are generally
015            defined for these multi-channel signals, we introduce multi-channel graph con-
016            volutions (MCGCs) by obtaining their form using the graph Fourier transform.
017            MCGCs highlight the critical importance of utilizing multiple edge relations to
018            amplify different signals for each feature channel. We further introduce localized
019
               multi-channel MPNNs and the multi-channel graph isomorphism network (MC-
               GINs), with which we can provably obtain linear mappings that are injective on
020
               multisets. Our experiments confirm the greatly improved capabilities of MCGCs
021
               and MC-GINs.

024   1   I NTRODUCTION
025
026   Many challenging tasks and applications are based on graph-structured data, e.g., property prediction
027   for molecules (Hu et al., 2021), fraud detection in transactions (Weber et al., 2019), and recommen-
028   dations (Monti et al., 2019). Developing expressive and well-performing methods to learn from such
029   data is thus an important challenge. As one such method, neural networks for graph-structured data
030   are based on defining a convolution on a graph. Spectral graph convolutions emerged by computing
031   the convolution exactly in the graph Fourier domain based on the convolution theorem (Hammond
032   et al., 2011; Bruna et al., 2014). Due to its high computational cost, various approximations based
033
      on polynomials of the graph Laplacian emerged (Levie et al., 2019; He et al., 2021; Koke & Cre-
      mers, 2024), e.g., using Chebyshev polynomials (Defferrard et al., 2016). This further led to the
034
      graph convolutional network (GCN) (Kipf & Welling, 2017) as a localized approximation. Most
035
      currently used message-passing neural networks (MPNNs) are derivations and improvements of this
036   spectral graph convolution and the GCN. This includes applying different aggregation functions like
037   the mean (Hamilton et al., 2017), the sum (Xu et al., 2019), utilizing attention coefficients (Velick-
038   ovic et al., 2018; Brody et al., 2022) or negative edge weights (Yan et al., 2022). More complex
039   methods like gating mechanisms (Li et al., 2016; Rusch et al., 2023), positional encodings (Kreuzer
040   et al., 2021; Rampásek et al., 2022; Huang et al., 2024), and normalization layers (Zhao & Akoglu,
041   2020) are typically combined with these approximated graph convolutions.
042   However, this convolution is defined for single-channel signals, i.e., each node has a single fea-
043   ture and a single feature per node is obtained. In most applications, each node has multiple initial
044   features assigned to it, e.g., a text embedding for documents and atom features for molecular data.
045   Similarly, the goal of these models is often to find rich node embeddings capturing both structural
046   properties and feature interactions, which are typically designed to have multiple feature channels.
047   The currently used definition of graph convolutions and, consequently, most MPNNs are ill-defined
048   for this task. As MPNNs amplify the same signal for each channel, issues like representational rank
049   collapse (Roth & Liebig, 2023) and over-smoothing (Oono & Suzuki, 2020) emerged.
050   The convolution that is mapping multi-channel signals to multi-channel signals is generally defined
051   differently. We introduce these multi-channel convolutions to graphs by obtaining their form based
052   on the convolution theorem and the graph Fourier transform. This multi-channel graph convolution
053   (MCGC) highlights the importance of utilizing multiple edge relations. Each edge relation corre-
      sponds to a different signal, and the feature transformation describes the amplification or damping

087   2     P RELIMINARIES
088
089   Let G = (V, E) be a connected and undirected graph consisting of a set of n nodes V and a set of
090   edges E. A graph signal is defined as a function x : V → Rd that assigns a vector of real values to
091   each node. For notational simplicity, we stack all node signals into a matrix X ∈ Rn×d based on
092   some node ordering. For d = 1, we refer to this as a single-channel signal, while for d > 1, we
093
      call it a multi-channel signal. These can either be initial features or expressive and informative node
      embeddings obtained by a suitable method. Let A ∈ {0, 1}n×n with Ai,j = 1 if (i, j) ∈ E and
094
      0 otherwise be the adjacency matrix corresponding to the same node ordering as X. The diagonal
095
      degree matrix is D ∈ Nn×n . The symmetrically normalized adjacency matrix is given by Ã =
096
      D −1/2 AD −1/2 and the graph Laplacian by L = In − Ã. Its eigendecomposition is L = U ΛU T
097
      where Λ ∈ Rn×n is a diagonal matrix containing its eigenvalues, and U ∈ Rn×n is an orthonormal
098
      matrix containing the corresponding eigenvectors as columns. In the graph domain, the Fourier base
099   is given by the eigenvectors U T of the graph Laplacian. Thus, the Fourier transformation F = U T
100   is performed by projecting a graph signal onto the eigenvectors, and its inverse transformation is
101   given by F −1 = U .

      3     M ULTI -C HANNEL G RAPH C ONVOLUTIONS

      We now consider a multi-channel graph signal X ∈ Rn×d with d channels for each node. By
204   applying a graph convolution, we want to obtain a convoluted multi-channel graph signal with dif-
205   ferent channel mixing for any input and output combination. In the last section, we have shown
206   that the currently used SCGC is not defined for this task. In general, the convolution is defined for
207   multi-channel signals X. To obtain a convoluted signal with c channels, the element-wise product
208   needs to map vectors with d channels to vectors with c channels. This requires elements of the
209   filter to be matrices Wi ∈ Rc×d and the full filter to be W ∈ Rn×c×d . This is commonly referred
210   to as the multi-channel convolution Burg (1964); Inouye & Sato (1999). It is commonly used in
211   signal processing (Burg, 1964; Inouye & Sato, 1999; Sainath et al., 2017), e.g., for multi-channel
212
      Wiener filtering (Brandstein & Ward, 2001) and finite impulse responses (Seltzer et al., 2004). Con-
      volutional neural networks (CNNs) (LeCun et al., 1989) are similarly based on the multi-channel
213
      convolution (Zhang et al., 2021).
214

      work (GIN) (Xu et al., 2019), and thus also to the Weisfeiler-Leman test (Weisfeiler & Lehman,
370
      1968). This is the maximal achievable expressivity for models following the message-passing
371
      scheme. Due to the universal approximation theorem (Hornik et al., 1989; Hornik, 1991), f can
372   be represented by a multi-layer perception (MLP). GIN similarly utilizes an injective function f that
373   is modeled by an MLP, which is applied instead of the linear feature transformation W1 with k = 1.
374   We will further to this l-MCGC as the multi-channel graph isomorphism network (MC-GIN). This is
375   a critical advantage of MC-GINs: As convolutions are linear operators, it is desired to approximate
376   multi-channel graph convolutions similarly by a linear transformation on X. While f may introduce
377   a complex transformation, it constructs a linear transformation that is applied to X. Allowing for
      learnable edge weights also lifts the constraint of requiring the sum as aggregation.

471   5     C ONCLUSION
472
473   In this work, we introduced multi-channel convolutions to graphs by obtaining their form based
474   on the convolution theorem and the graph Fourier transform. While the currently used single-
475   channel graph convolutions, their polynomial approximations, and most MPNNs are not defined
476   for nodes with multiple features, MCGCs are specifically defined for multi-channel signals. This
477
      allows MCGCs to obtain more informative embeddings, as different signals can be amplified for
      each feature channel. Corresponding multi-channel MPNNs need to utilize multiple aggregations
478
      and transformations, each amplifying a different signal. We introduce the multi-channel graph iso-
479
      morphism network (MC-GIN), which can obtain a linear mapping with equivalent expressive power
480   to the graph isomorphism network (Xu et al., 2019). Our experiments confirm the strongly im-
481   proved abilities to fit complex functions. While our experimental evaluation is limited, the benefits
482   of MCGCs compared to SCGCs are very clear. Having access to the mathematically correct con-
483   volution for multi-channel signals allows for the development of various approximations. As more
484   complex interactions between channels may not be required for all tasks, MCGCs increase the risk
485   of overfitting. Approximations of the MCGC will be particularly important as more complex graph-
      related tasks emerge and more powerful convolutions are needed.

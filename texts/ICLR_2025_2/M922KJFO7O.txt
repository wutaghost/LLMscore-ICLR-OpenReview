009                                              A BSTRACT
010
011            Despite the significant success of large language models (LLMs), their extensive
012            memory requirements pose challenges for deploying them in long-context token
013            generation. The substantial memory footprint of LLM decoders arises from the
014            necessity to store all previous tokens in the attention module, a requirement im-
015            posed by key-value (KV) caching. In this work, our focus is on developing an
016            efficient compression technique for the KV cache. Empirical evidence indicates
017            a significant clustering tendency within key embeddings in the attention module.
018            Building on this key insight, we have devised a novel caching method with sublin-
019
               ear complexity, employing online clustering on key tokens and online ℓ2 sampling
               on values. The result is a provably accurate and efficient attention decoding al-
020
               gorithm, termed C LUSTER G EN. Not only does this algorithm ensure a sublinear
021
               memory footprint and sublinear time complexity, but we also establish a tight
022            error bound for our approach. Empirical evaluations on long-context question-
023            answering tasks demonstrate that C LUSTER G EN significantly outperforms exist-
024            ing and state-of-the-art KV cache compression methods in terms of performance
025            and efficiency.

028   1   I NTRODUCTION
029
030   Large Language Models (LLMs) (Achiam et al., 2023; Touvron et al., 2023) play a crucial role
031   in various natural language processing applications, including dialog systems (Taori et al., 2023;
032   Chiang et al., 2023), coding assistance (Chen et al., 2021; Roziere et al., 2023), and image/video
033   generations from text (Radford et al., 2021; Ho et al., 2022). All of these models rely on the trans-
034   former architecture, with the attention mechanism serving as the key component.
035
      To fully harness the capabilities of LLMs, they must demonstrate both efficiency and accuracy in
036   generating long sequences. In practical applications, deploying LLMs to generate tokens in an
037   autoregressive manner involves a sequential decoding process, where attention is dynamically ap-
038   plied to each newly generated token. This process effectively constructs the output sequence in a
039   streaming manner, one token at a time. Therefore, as the sequence grows, the model has to produce
040   contextually relevant and coherent content.
041
      A common method for autoregressive attention decoding involves the use of key-value (KV)
042   caching, where key and value pairs from all preceding tokens are cached and reused to prevent
043   redundant computations. However, this approach faces memory constraints, particularly when han-
044   dling long sequences. In particular, the memory requirements and runtime for generating each new
045   token increase linearly with context size, posing a significant challenge for efficient processing of
046   extensive sequences. This linear scaling directly impedes practical applicability in real-world sce-
047   narios, such as chat systems, where large contexts are often encountered.
048   In this work, we delve into the primary computational and memory bottleneck of token generation.
049   We propose C LUSTER G EN, a novel approach designed to significantly reduce the memory and
050   runtime complexity of token generation, moving from conventional linear growth to sublinear scale.
051   To summarize, our goal is to answer the following question:
052
053                       Can we approximate the attention output in decoding phase
                            in sublinear space/time complexity in context length?

      1.1   R ELATED W ORK
055
056   Recent studies have underscored the need for efficient token generation, particularly with the rise of
057   long-range context datasets. Several recent works have developed efficient strategies for compress-
058   ing the KV cache. Zhang et al. (2023) proposed a greedy-type eviction algorithm that dynamically
059   keeps at most k ≪ n token embeddings based on the accumulated attention scores where they refer
060   to the Heavy Hitter Oracle (H2O). Liu et al. (2023a) empirically observed that tokens with initially
061   high attention scores tend to stay high during the future generation process. Motivated by this ob-
062
      servation, the authors proposed a strategy that only keeps the most recent and pivotal tokens whose
      attention scores are higher than a threshold. Ge et al. (2023) proposed an adaptive method of KV
063
      cache compression which identifies the intrinsic structures of attention heads and uses them to deter-
064
      mine the optimal compression policy. Xiao et al. (2023) observed that a simple eviction mechanism
065   that keeps only first few and last few tokens does not degrade much the decoding quality. They
066   additionally proposed a fine-tuning method to solve performance degradation from their method.
067   Liu et al. (2023b) developed an algorithm that reduces the generation latency by exploiting contex-
068   tual sparsity. In addition to algorithmic acceleration, there has also been a line of work optimizing
069   hardware resource configurations (Sheng et al., 2023; Hong et al., 2023). However, to the best of
070   our knowledge, none of these works have achieved an efficient method for KV cache with fully
071   sublinear-time memory space.
072   On the lower bound side, achieving subquadratic amortized runtime for producing output embed-
073   dings for n tokens in the worst-case instances is likely impossible without making assumptions about
074   the input tokens (Alman & Song, 2023; Sarlos et al., 2023). Therefore, to achieve fast runtime, it is
075   necessary to rely on certain assumptions about the input tokens.
076

478   5     C ONCLUSION
479
480   In this work, we develop C LUSTER G EN, an efficient KV cache compression algorithm via stream
481   clustering. Our motivation is that cached keys are well clusterable in their embedding space and
482   we apply a greedy-type clustering algorithm to find the most representative embeddings. Under
483   assumptions on bounded query norm and clusterability, we analyze that our algorithm can guarantee
484   a spectral error bound with sublinear time and memory. We further integrate keeping recent tokens
485   to the proposed clustering approach. For zero-shot line retrieval tasks, our algorithm outperforms
      other KV cache compression algorithms with the same memory budget.

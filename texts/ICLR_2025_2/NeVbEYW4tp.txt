009                                              A BSTRACT
010
011            Vision-language models have showcased impressive zero-shot classification capa-
012            bilities when equipped with suitable text prompts. Previous studies have shown
013            the effectiveness of test-time prompt tuning; however, these methods often require
014            per-image prompt adaptation during inference, which is computationally inten-
015            sive and limits scalability and deployment. To address this issue, we introduce a
016            novel framework: Self-supervised learning for efficient Test-time Prompt Tuning
017            (Self-TPT). The key feature of Self-TPT is its shift to efficient predefined class
018
               adaptation through self-supervised learning, thereby avoiding the computation-
               heavy per-image adaptation at inference. Self-TPT starts by co-training the
019
               self-supervised and supervised tasks using source data, then applies the self-
020
               supervision exclusively for new class understanding before making predictions.
021            Specifically, we propose Contrastive Prompt Learning (CPT) as the core task for
022            self-supervision. CPT is designed to minimize the intra-class distances while en-
023            hancing inter-class distinguishability via contrastive learning. Empirical evidence
024            suggests that CPT can partially mimic supervised learning in terms of gradients,
025            providing a plausible explanation for its effectiveness. Motivated by this find-
026            ing, we introduce a gradient matching loss to explicitly enhance gradient sim-
027            ilarity. We evaluated Self-TPT across three challenging zero-shot benchmarks.
028            The results consistently show that Self-TPT not only significantly reduces infer-
029
               ence costs but also achieves state-of-the-art performance, effectively balancing the
               efficiency-efficacy trade-off.

      1   I NTRODUCTION
033
034
      Open-set image classification is a fundamental yet challenging task in computer vision. Recently,
035
      Vision-Language Models (VLMs) (Jia et al., 2021; Li et al., 2022; Alayrac et al., 2022; Fang et al.,
036   2023) have shown promising capabilities in this field. A prominent model, CLIP (Radford et al.,
037   2021), encodes both images and language into a unified embedding space, allowing classification
038   by measuring similarities between image representations and textual class descriptions.
039
      Effective prompts for input classes are essential (Radford et al., 2021), but manually crafting them
040
      is time-consuming (Zhou et al., 2022b). Inspired by NLP advancements (Shin et al., 2020; Zhong
041
      et al., 2021), researchers have explored using continuous vectors as soft prompts, optimizing them
042   with a few labeled data (source data). These methods can automatically obtain task-specific prompts,
043   thereby improving performance (Zhou et al., 2022b; Zhu et al., 2024). However, the source data is
044   unlikely to encompass all possible classes, resulting in suboptimal open-set performance.
045
      Test-time adaptation (TTA) (Liang et al., 2020a; Wang et al., 2020; Niu et al., 2023) has recently
046
      gained attention for adapting models to new data distributions during the test phase. In this context,
047
      TPT (Shu et al., 2022) was proposed to tune prompts for new classes at test time, thereby improving
048   open-set generalization. As depicted in Fig. 1(a), TPT first learns prompts from source data (stage
049   1). It then adjusts these prompts for each test sample (stage 2) and uses the sample-tailored
050   prompts for predicting (stage 3). Despite its effectiveness, TPT requires a substantial compu-
051   tational budget. For each test sample, it takes multiple forward and backward passes through the
052   entire model and needs to retain the full computational graph, resulting in substantial latency and
053   memory usage. As shown in Fig. 1(c), TPT operates at ∼7 FPS while consuming ∼5GB of graphics
      memory. The latest TPT-based method, PromptAlign (Samadh et al., 2023), operates at ∼5 FPS

074   TPT to larger VLMs and deploying it on resource-limited platforms.
075   In this paper, we study the question: Can we harness TTA’s generalizability while bypassing its
076   huge computational overhead? We observed that, at inference, while images come sequentially, the
077   candidate class names are typically predetermined. Motivated by this, we introduce Self-TPT, an
078   efficient Test-time Prompt Tuning framework that employs text-oriented Self-supervised learning
079   (SSL). The idea is depicted in Fig. 1(b): the adaptation process (stage 2) of Self-TPT operates
080   solely on the predefined class names, allowing for directly predicting any image without the need
081   for prompt updates (stage 3), thereby effectively bypassing the substantial computational load.
082   For the SSL component, we propose using contrastive learning. The intuition is that effective clas-
083   sification requires embeddings of the same class to align closely, while those from different classes
084   remain distinct, ensuring clear inter-class separability. To achieve this, we introduce a novel ap-
085   proach called Contrastive Prompt Tuning (CPT). In CPT, we vary the insertion points of the class
086   token within prompt sequences as data augmentation to create positive pairs. Negative pairs are built
087   by contrasting a class against all others, thereby explicitly reinforcing the dissimilarity among class
088
      embeddings. Initially, we integrate CPT with supervised learning (stage 1) and subsequently rely
      exclusively on CPT for new class adaptation (stage 2).
089
090   Our empirical analysis shows that CPT and supervised learning consistently exhibit a positive gra-
091   dient correlation in nearly all cases. This suggests that both tasks drive the model’s optimization
092   in similar directions, allowing CPT to potentially serve as a proxy for supervised learning during
093   the adaptation phase. This evidence plausibly explains the effectiveness of CPT. Inspired by this,
094
      we introduce a gradient matching (GM) loss designed to enhance this positive correlation explicitly.
      The GM loss operates on the gradients derived from both supervised and CPT losses and aims to
095
      maximize their cosine similarities.
096
097   We evaluated Self-TPT’s performance on three challenging zero-shot benchmarks: cross-dataset
098   generalization, base-to-new generalization, and domain generalization. Our findings reveal that Self-
099   TPT consistently surpasses the prior state-of-the-art, PromptAlign, improving by 0.93%, 1.59%, and
100
      1.82% on these benchmarks, respectively. Notably, Self-TPT significantly enhances efficiency, as
      shown in Fig. 1(c), achieving a 25-fold increase in inference speed and reducing memory usage by
101
      30-fold compared to PromptAlign. Our experiments also verify that Self-TPT is data-efficient and
102
      generalizes well across different backbones, scales, and VLMs.

107   Vision-Language Models. Recent advances in computer vision and natural language processing
      have spurred significant interest in vision-language models (VLMs) (Radford et al., 2021; Jia et al.,

108
      2021; Li et al., 2022; Zhai et al., 2022; Alayrac et al., 2022; Fang et al., 2023). These models excel
109   in various multi-modal tasks by leveraging massive datasets of image-text pairs to develop robust
110   representations that span different modalities. Notably, CLIP (Radford et al., 2021) has demon-
111   strated exceptional open-vocabulary capabilities, enabling effective performance in image classifi-
112   cation (Zhou et al., 2022b; Zhu et al., 2024), video recognition (Weng et al., 2023; Huang et al.,
113   2024), and object detection (Du et al., 2022; Minderer et al., 2022), etc. A key aspect of deploy-
114   ing VLMs successfully involves crafting effective text prompts. In this paper, we introduce a novel
115   framework that optimizes prompt adaptation for better class comprehension during testing.
116   Prompt Learning. Recent advancements in NLP have inspired approaches like CoOp (Zhou et al.,
117   2022b; Chen et al., 2022b; Lu et al., 2022), which treats prompts for VLMs as continuous vectors.
118   However, CoCoOp (Zhou et al., 2022a) highlighted a significant flaw: these learned prompts often
119   overfit to seen classes, compromising performance on new ones. To mitigate this, recent studies
120   have introduced additional learnable components (Zhu et al., 2024; Zhou et al., 2022a; Khattak et al.,
121   2023a; Zang et al., 2022; Wang et al., 2023c; Singha et al., 2023) or specialized strategies (Lee et al.,
122   2023; Xu et al., 2023b; Long et al., 2023; Shi & Yang, 2023; Kan et al., 2023; Wang et al., 2023a) to
123
      enhance prompt generalization. Techniques like distillation and regularization (Yao et al., 2023; Zhu
      et al., 2022; Khattak et al., 2023b; Xu et al., 2023a; Bulat & Tzimiropoulos, 2023) are also employed
124
      to integrate task-specific knowledge and hand-crafted priors. Despite progress, achieving prompts
125
      that generalize across all possible classes remains challenging. Consequently, this paper shifts focus
126   to test-time adaptation strategies, dynamically adjusting prompts during testing to address open-
127   world application challenges.
128
129
      Test-Time Adaptation was developed to address shifts in data distribution between training and
      testing phases by dynamically adjusting the model at inference. Numerous methods have emerged,
130
      including entropy minimization (Liang et al., 2020a; Wang et al., 2020; Niu et al., 2023), batch-
131
      normalization activation (Wang et al., 2020; Zhao et al., 2023; Niu et al., 2023), pseudo label-
132   ing (Chen et al., 2022a; Liang et al., 2020b; Wang & Wibisono, 2022), feature alignment (Liu et al.,
133   2021; Wang et al., 2023b; Wang & Aitchison, 2022), and test-time training (Sun et al., 2020; Liu
134   et al., 2021; Huang et al., 2021; Chen et al., 2023; Gandelsman et al., 2022). Recently, test-time
135   prompt tuning (TPT) (Shu et al., 2022) for VLMs has gained attention. TPT optimizes prompts by
136   reducing prediction entropy and uses image augmentation to create prediction diversity. Techniques
137   like DiffTPT (Feng et al., 2023) employ Stable Diffusion (Rombach et al., 2022) to boost augmented
138   image diversity, while SwapPrompt (Ma et al., 2023) and PromptAlign (Samadh et al., 2023) focus
139   on maximizing prediction agreement and aligning token statistics, respectively. Despite their ef-
140
      fectiveness, these approaches often entail high computational costs, posing challenges for practical
      deployment. This study introduces a more efficient TPT framework, aiming to balance effectiveness
141
      with real-world applicability.

239   Within the Self-TPT framework, the core design is its self-supervised component. As indicated by
240   Sun et al. (2020), the auxiliary task (in this case, the SSL task) should have a strong correlation with
241   the main task (in this case, the classification task) to maintain effectiveness. In the realm of self-
242   supervised learning, contrastive learning methods (Chen et al., 2020; He et al., 2020) have yielded
243   impressive classification outcomes, even through the linear probing of frozen visual representations.
244   This underscores a positive connection between contrastive tasks and classification tasks, thereby
245   motivating our prioritization of contrastive learning in the SSL task design.
246   To construct the contrastive task, we adhere to the nature of classification that embeddings from the
247   same class should align closely, while those from different classes should remain distinct to ensure

249   ing (CPT). This task varies the insertion points of the class token “[CLS]” within prompt sequences
250   to generate positive pairs. As depicted in Fig. 3a, CPT strategically places the “[CLS]” token at the
251   end, front, and middle of the learnable prompts:

      4.3   A BLATION S TUDY
419
      Main components analysis. In developing Self-TPT based on CoOp, we made three key progress.
420   First, we integrated CPT during the source prompt learning phase. This integration aims to cultivate
421   more robust and generalizable feature representations. Second, we applied CPT during test-time
422   adaptation, improving the understanding of the new class prior to the prediction phase. Lastly, we
423   incorporated the GM loss in the source prompt learning stage to explicitly strengthen the gradient
424   correlation between CPT and the classification task. The effectiveness of each component is quan-
425   titatively assessed in Tab. 5a. The results demonstrate that each component of Self-TPT contributes
426   to consistent performance improvements across the board.
427   Study on different views in CPT. As shown in Fig. 3a, Self-TPT employs four distinct views per
428   class for contrastive learning. In Tab. 5b, we conduct an ablation study by sequentially removing
429   one view at a time. This results in a consistent decline in performance. Notably, removing the
430   handcrafted view causes the most significant drop, as adapting the CLIP text branch independently
431   without regularization may lead to misalignment with the visual branch. These findings suggest that
      incorporating multiple views enhances the effectiveness of the contrastive prompt tuning task.

477   5                      C ONCLUSION
478
479
      In this paper, we introduce Self-TPT, a novel framework for efficient test-time prompt tuning. Self-
480   TPT addresses the inefficiencies in inference found in existing TPT-based methods by using text-
481   oriented self-supervised learning for new class adaptation, transforming per-image adaptation into
482   a preprocessing step. We also introduce a novel contrastive prompt tuning (CPT) task for self-
483   supervision. Empirical results show that CPT has a positive gradient correlation with classification
484   tasks, highlighting its effectiveness. Based on this finding, we propose a gradient-matching loss to
485   further enhance this correlation. Extensive experiments confirm the efficiency, effectiveness, and
      versatility of our method.

832   Additional Implementation Details. The learnable prompts are initialized with pre-trained CLIP
833   word embeddings of “a photo of a” at the beginning of stage 1. To eliminate the need for an
834   additional validation set, we opt to select the model at the last step of both stage 1 and stage
835   2. Consistent with prior research (Zhou et al., 2022b;a; Zhu et al., 2022; Khattak et al., 2023a),

837   implement a warm-up strategy where the learning rate is initially set at 1e − 5 for the first epoch,
838
      and then it follows a cosine annealing schedule starting from 2e − 3. For the projection head, we

839
      an additional hidden layer with ReLU activation. By default, the projection dimension is set to 128.
840
      We initialize the weights of the projection head using the Xavier (Glorot & Bengio, 2010) and set

      C       L IMITATIONS AND F UTURE W ORK
910
911
      Although existing TTA techniques show promise and effectiveness, their computational costs hinder
912   their deployment in real-world scenarios. Self-TPT takes a stride forward by turning specific-sample
913   adaptation into a pre-processing step. However, the extra costs brought by the nature of test-time
914   adaptation still exist. There is a risk for Self-TPT degrading to vanilla TPT scenarios when the
915   samples come with different class sets. In the future, our objectives include refining the Self-TPT
916   framework to better align with practical applications. We also intend to investigate the potential of
917   test-time prompt tuning across various other fields, such as more intricate visual-language tasks (Liu
      et al., 2024; Zhu et al., 2023).

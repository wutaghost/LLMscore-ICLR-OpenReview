009                                               A BSTRACT
010
011            Prompt tuning has emerged as an effective way for parameter-efficient fine-tuning.
012            Conventional deep prompt tuning inserts continuous prompts of a fixed context
013            length into the input to each layer. When a pre-trained model is tailored to
014            a specific downstream task, different layers initialized with pre-trained weights
015            might have, depending on the distribution shift type, different levels of deviation
016            from the optimal weights. Inserted prompts with a fixed context length might
017            have redundant context tokens or insufficient context length. To address this
018
               issue, we propose a deep continuous prompting method dubbed Adapt that en-
               courages heterogeneous context lengths. Context lengths are automatically de-
019
               termined by iteratively pruning context tokens. We use the saliency criterion
020
               for the neural network pruning to compute the importance scores of context to-
021            kens in order to determine which tokens to prune. We examine the proposed
022            method on the pre-trained vision-language model CLIP. Extensive experiments
023            on 11 downstream datasets reveal the advantage of Adapt: the average test ac-
024            curacy increases from 79.83% to 81.70%. The highest performance gain on
025            individual datasets is 9.63%. At the same time, the computational overheads
026            are comparable to or smaller than baseline methods. We release the code in
027            https://anonymous.4open.science/r/Adapt-Release.

030   1   I NTRODUCTION
031
032   Large-scale models have gained significant attention in language (Brown, 2020; Wang et al., 2021;
033
      Touvron et al., 2023), vision (He et al., 2022; Zou et al., 2024; Esser et al., 2024) and multimodality
      (Radford et al., 2021; Lu et al., 2022; Lai et al., 2024; Liu et al., 2024). When applying pre-trained
034
      large-scale models to various downstream tasks, the zero-shot performance can be sub-optimal.
035
      Although fine-tuning remarkably elicits the potential of pre-trained models, the fine-tuning process
036   is computationally expensive. Parameter-efficient fine-tuning (PEFT) offers an efficient way to adapt
037   the pre-trained model to various downstream tasks at a low cost. PEFT enhances the performance
038   of pre-trained models by reparametrizing model weights (Hu et al., 2021; Dettmers et al., 2024;
039   Zhao et al., 2024), additive modules (Chen et al., 2022b; Zhang et al., 2023; Mou et al., 2024) and
040   selective weight updates (Ding et al., 2023; Lawton et al., 2023; Fu et al., 2023). Among PEFT
041   methods, prompting methods have the least effect on backbone models as they focus on the input
042   instead of model parameters.
043   Prompts can be categorized into discrete prompts and continuous prompts. Discrete prompts use
044   concrete word tokens to prompt pre-trained models. Compared to discrete prompts, continuous
045   prompts (also called soft prompts) relax the token embedding space to be continuous. Hence, con-
046   tinuous prompts are differentiable and parameterized by their weights. They can be automatically
047   tuned conditioning on downstream tasks.
048   Continuous prompts have shown competitive performance in language (Li & Liang, 2021; Gu et al.,
049   2021; Liu et al., 2021; 2023), vision (Jia et al., 2022; Bahng et al., 2022; Han et al., 2023) and
050   multimodality (Zhou et al., 2022b; Shu et al., 2022; Ju et al., 2022; Wang et al., 2022). Exist-
051   ing continuous prompting methods use the prompt depth and context length to design continuous
052   prompts. The underlying constraint is that the context length remains constant at different depths.
053   If different layers have different levels of deviation from the optimal weights for downstream tasks,
      the constraint might be detrimental to the performance.

054
      Recent works (Lee et al., 2022; Chiatti et al., 2023; Panigrahi et al., 2023) have found that some
055   layers of pre-trained models, depending on the distribution shift, are close to the optimal for down-
056   stream tasks. Fine-tuning layers that are far away from the optimal weights can achieve better
057   performance than training all the layers uniformly. For prompting methods, we postulate that the
058   layers far away from the optimal weights require longer context length while the layers close to the
059   optimal weights demand shorter context length or even no context token. Hence, we seek to remove
060   the constraint in the existing continuous prompting methods that require the same context length at
061   different depths.
062   To this end, we propose a method dubbed adaptive prompt tuing (Adapt) that automatically deter-
063   mines context lengths at various depths. Adapt uses a time-dependent binary mask to dynamically
064   control context lengths. The variation of the binary mask depends on the importance of context
065   tokens. The least important context token is constantly removed until the budget (a hyperparameter
066   to control the total context length) is reached. We test the performance of Adapt on various down-
067   stream task. Adapt outperforms baseline methods by a large margin as shown in Figure 1. To our
068   best knowledge, this is the first work to prune prompts for achieving heterogeneous context lengths.
069   The main contribution of adapt is summarized as follows:
070
071         • We propose a method that removes the constraint in the existing continuous prompting
072           methods that context lengths remain constant through the entire prompt depth. Adapt en-
073           courages a more flexible design for prompting methods.
074         • Context lengths are automatically determined in a non-parametric manner: prompts are
075           initialized with the maximum context length and then iteratively pruned based on the im-
076           portance score of context tokens. We use saliency criteria to characterize the importance
077           of inserted context tokens. Pruning can effectively reduce the computational overhead with
078
              the minimal performance drop.
079         • Context lengths can vary based on the downstream datasets. We use a hyperparameter of
080           total context lengths to ensure the complexity of Adapt on various datasets is approximately
081           the same.

105
106   Prompt Tuning Prompt tuning (PT) uses continuous prompts to improve the performance of pre-
107   trained models in diverse downstream tasks. CoOp (Zhou et al., 2022b) is the pioneering work to
      apply PT for vision-language models. PT has shown great potential in various areas including image


                                                                  2
      Under review as a conference paper at ICLR 2025


108
      classification (Zhou et al., 2022b;a; Hirohashi et al., 2024), out-of-distribution detection (Miyai
109   et al., 2024; Li et al., 2024), video understanding (Ju et al., 2022; Huang et al., 2023), object detection
110   (Du et al., 2022; He et al., 2023), etc. Due to the good alignment of text and image representations
111   of foundational vision-language models, there are emerging researches on applying those models
112   such as CLIP (Radford et al., 2021) to vision-language tasks. VPT (Jia et al., 2022) proposes a
113   paradigm of deep continuous prompting. PLOT (Chen et al., 2022a) applies the optimal transport
114   theory to improve the alignment between visual features and prompts. ProGrad (Zhu et al., 2023)
115   and KgCoOp (Yao et al., 2023) distill the prior knowledge from the pre-trained model to avoid
116   forgetting issues (Li & Hoiem, 2017; Gou et al., 2021). MaPLe (Khattak et al., 2023) uses linear
117
      transformation layers to enhance the coupling between the text and image branches. LAMM (Gao
      et al., 2024) uses dynamic category embedding and hierarchical loss to achieve an appropriate label
118
      distribution.
119
120
121   Network Pruning Over-parametrization is a well-known property of deep neural networks. Net-
122
      work pruning removes unimportant model parameters to improve efficiency. It can be categorized
      into structured pruning and unstructured pruning. Unstructured pruning such as (Han et al., 2015)
123
      removes individual parameters while structured pruning such as (Liu et al., 2018) prunes models
124
      at a higher level (e.g. neurons, filters, and layers). A fundamental question in network pruning is
125   to identify a saliency criterion to determine the importance of model parameters. Snip (Lee et al.,
126   2018) is a classic way to characterize the importance and can lead to a very sparse network without
127   sacrificing too much performance.
128
129
130
131   Sparse Training Sparse training decreases a portion of model parameters based on the pre-defined
132
      pruning strategy. SViTE (Chen et al., 2021) iteratively uses a prune-and-grow strategy to update the
      model sparsity. Specifically, the linear transformation layers to query, key, and value are pruned.
133
      Mask tuning Zheng et al. (2023) updates those layers in a differentiable manner, i.e., masks con-
134
      trolling sparsity is updated by learning instead of pruning. DRSformer Chen et al. (2023) utilizes
135   learnable selection operators for attention scores in lieu of linear transformation layers.
136
137   In both network pruning and sparse training, sparsity distribution is a key factor. Concentrated
138
      pruning on a neural layer can cause the disconnection issue in neural networks. Nevertheless, when
      pruning soft prompts, this issue is inherently solved. If the entire deep soft prompts for a layer
139
      of a pre-trained model are pruned (i.e. context length is 0), it indicates that the effective prompt
140
      depth decreases by 1. This scenario is equivalent to the case in manually designed deep prompting
141   methods where the depth of neural network layers is larger than the prompt depth.

146   We examine Adapt on the vision-language model CLIP (Radford et al., 2021). CLIP is pre-trained
147   over 400 million image-text pairs. The pre-training process is in a contrastive learning fashion to
148   promote the alignment between text and image representations. CLIP consists of an image encoder
149   and a text encoder. The prediction is done by matching the text and image representations.

275

      Datasets We examine the proposed Adapt method over 11 datasets: Caltech101 (Fei-Fei et al.,
317
      2004) and ImageNet (Deng et al., 2009) for the generic object recognition, DescribableTectures
318   (Cimpoi et al., 2014) for the texture recognition, EuroSAT (Helber et al., 2019) for the satellite im-
319   age recognition, FGVCAircraft (Maji et al., 2013), Food101 (Bossard et al., 2014), OxfordFlowers
320   (Nilsback & Zisserman, 2008), OxfordPets (Parkhi et al., 2012), and StanfordCars (Krause et al.,
321   2013) for the fine-grained image recognition, UCF101 (Soomro et al., 2012) for the action recog-
322   nition, and SUN397 (Xiao et al., 2010) for the scene recognition. We follow the few-shot learning
323   setting in CoOp (Zhou et al., 2022b). The number of shots is 16. For each dataset, the result is
      averaged over 3 runs. A detailed description of 11 datasets can be found in the Appendix.

      Baselines We compare the proposed method with CoCoOp (Zhou et al., 2022a), VPT (Jia et al.,
325   2022), PLOT (Chen et al., 2022a), MaPLe (Khattak et al., 2023), ProGrad (Zhu et al., 2023), Kg-
326   CoOp (Yao et al., 2023) and LAMM (Gao et al., 2024). The original implementation of PLOT uses
327   ResNet (He et al., 2016) in the image encoder. For a fair comparison, we replace ResNet with ViT
328   in the image encoder. CoCoOp, PLOT, ProGrad, KgCoOp and LAMM use shallow prompts while
329   VPT and MaPLe use deep prompts. CoCoOp adds continuous prompts conditioned on the input
330   Image. PLOT uses the optimal transport theory to align the vision and text modalities. ProGrad
331   uses gradient-aligned knowledge distillation to alleviate the forgetting issue in the fine-tuning pro-
332   cess. KgCoOp uses the prior knowledge from the hand-crafted prompt in the knowledge distillation.
333
      LAMM replaces the category tokens with trainable vectors and utilizes the hierarchical loss to pre-
      serve the generalization ability of the pre-trained model. VPT proposes a deep prompting method.
334
      MaPLe uses a coupling function to bridge the image branch and text branch.

      Implementation Details We use the pretrained ViT-B/16 CLIP model (Radford et al., 2021) in
337
      this work. The number of minibatches used for computing the score is nk = 80. In each pruning
338   step, the number of pruned context tokens is rp = 4. The batch size is 4. The learning rate is
339   2.5 × 10−3 . The total number of training epochs is 100. The test accuracy is obtained using the
340   model weights at the epoch of 100. We use the stochastic gradient descent (SGD) to optimize the
341   inserted prompts. Experiments are conducted using a single NVIDIA A40 GPU. Reported results
342   on 11 datasets are averaged over 3 runs.

      5     D ISCUSSION
471
472
      When tailoring a pre-trained model to various downstream tasks, the model can underperform due to
473
      the distribution shift (DS) (Taori et al., 2020; Fang et al., 2020; Wiles et al., 2021; Xiao et al., 2024).
474   When examining the model on a more granular level, a question arises “is the inferior performance
475   caused by the deviation from the optimal for all layers or a subset of layers”. Surgical fine-tuning
476   (Lee et al., 2022) answers this question by categorizing DS into four categories: input-level shift,
477   feature-level shift, output-level shift, and natural shift. Depending on the DS type, fine-tuning the
478   selective part of the pre-trained model achieves a performance comparable to or better than training
479   all layers. This result indicates that not all layers are at the same level of deviating from the optimal.
480   For example, when DS is the input-level shift, only the first few layers are deviating away from
481   the optimal. Training those layers while keeping the remaining layers frozen achieves favorable
482
      performance.
483   In the PT, the entire pre-trained model is frozen. Given the fact that some layers, depending on the
484   DS type, might already be close to the optimal, there is no need to insert continuous prompts for
485   those layers. Prompts can be inserted into layers that are deviating from the optimal. If we consider
      this strategy in a more granular way, context lengths for different layers can vary depending on


                                                             9
      Under review as a conference paper at ICLR 2025


486
      the level of deviating from the optimal. This leads to heterogeneous context lengths which are
487   challenging for the manually designed prompting methods.
488
489   The proposed Adapt method achieves the automatic design of heterogeneous prompts. There is
490
      no constraint for context lengths at various depths to be the same, nor for context lengths to be
      the same for different branches. The results on 11 datasets indicate that context lengths can be
491
      highly heterogeneous as shown in Appendix Figure 8. The automation is achieved by iteratively
492
      pruning unimportant context tokens. By setting Ttarget ≪ ℓ × ξ, the pruning greatly reduces the
493   computational overheads. We empirically find the performance of pruned prompts P ⊙ M(t) is
494   comparable to that of training prompts P without pruning from scratch as indicated in the Appendix
495   Table 4. At the same time, the total number of trainable parameters is decreased by 67%. In the
496   network pruning, pruning concentrated on one layer can cause the layer collapse issue (Lee et al.,
497   2019; Hayou et al., 2020). Pruning prompts, however, can have the minimal context length in one
498   layer without affecting the functionality of prompts for this layer.
499   By using M(t) conditioning on downstream datasets, Adapt adaptively changes for different
500   datasets. Compared to manually designed prompts, Adapt has a more flexible structure. It achieves
501   a pronounced performance gain compared to baseline methods. We use Ttarget to ensure the com-
502   plexity of Adapt is approximately the same over various datasets.
503
504
505
      Limitation While Adapt achieves the best average performance on the considered downstream
506
      tasks, we acknowledge that when the number of shots decreases (e.g., less than 4 shots), Adapt
507   may lose its advantages. However, most prompting methods assume at least 16-shots per category
508   (Hirohashi et al., 2024), which is the regime where Adapt outperforms the competitive methods.
509   We believe the limitation of Adapt in very few-shot settings may be addressed by considering some
510   specialized methods that cater to few-shot prompting, such as (Hirohashi et al., 2024) explicitly
511   designed for 1-shot setting.
512
513
      6   C ONCLUSION
514
515   We propose a continuous prompting method that adaptively changes during the fine-tuning process.
516   Different from existing prompting methods that require homogeneous context lengths for various
517   depths, our proposed method Adapt encourages heterogeneous context lengths. Adapt uses iterative
518   pruning to remove unimportant context tokens, which greatly reduces the computational costs with
519   nearly no performance drop. Extensive experiments over 11 datasets exhibit the strength of the
520   Adapt method.

      A.7    P RUNING P ROCESS OF B INARY M ASKS
966
967   Figure 8 shows the pruning process of binary masks in the text and image branches for 11 datasets.
968   The binary mask is initialized to be M(0) = 1ℓ×ξ . M(t) stays constant when Ttarget is reached. We
969   use Snip to compute the score for each context tokens. The binary masks are highly heterogeneous:
970   context lengths over prompt depth for different branches have a large variation. The heterogene-
971   ity feature exhibits the strength of the automatic design of context lengths compared to manually
      designed prompts which tend to be homogeneous.


                                                                   18
       Under review as a conference paper at ICLR 2025


972
       When Ttarget is reached, P ⊙ M(t) on some datasets have a context length of 1 at a certain depth.
973    Even though the pruning concentrated in the prompts for this layer, the performance is not negatively
974    affected. In the network pruning, however, the concentrated pruning can lead to the layer collapse
975    issue (Lee et al., 2019; Hayou et al., 2020). Considering an extreme case, if the weight of an entire
976    layer is pruned, the model will have a disconnection issue. When pruning the prompt, there is no
977    such issue. This indicates that pruning prompts can lead to highly heterogeneous prompt lengths.

1119   A.12      C OMPARISON WITH BASELINES A LIGNING T EXT AND I MAGE B RANCHES
1120
1121   We compare the Adapt method with prompting methods that have coupling functions between
1122   prompts inserted into the text branch and those for the image branch. MaPLe (Khattak et al., 2023)
1123   uses a linear transformation function to apply the alignment between text and image branches. UPT
1124
       (Zang et al., 2022) uses a transformer block to contextualize the prompts for the image branch and
       text branch.
1125
1126   The code is not released for the UPT method, so we implement the method on our own. Transformer
1127   block consisting of the self-attention operator, feed-forward network and layer normalization are
1128   used to contextualize the inserted embeddings. The contextualized embedding for each layer of
1129   the vision-language model is split for the text and image branches. A fully connected layer per
1130
       transformer layer of the pre-trained model is used to align the hidden dimension of the embedding
       to that for the image branch. Please refer UPT (Zang et al., 2022) for more details. The context
1131
       length is 4 and the prompt depth is 9.

       by a significant margin. At the same time, introducing the coupling functions inevitably introduces

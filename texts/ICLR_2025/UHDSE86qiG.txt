                                                   A BSTRACT
008
009
               Model owners often wish to introduce new capabilities into their trained models
010
               or remove undesired ones. Task Vectors (TVs) present a promising new approach
011            to editing models after training, allowing simple and controllable addition of new
012            capabilities to the model and the removal of undesired ones. But what happens
013            when the model owner wants to change multiple capabilities?
014            In this work, we study the interactions of task vectors in a multi-edit setting
015            for image classifiers and diffusion models. We start by quantifying the overall
016            model degradation induced by applying many specific TVs simultaneously. We
017            show that the overall model performance degrades rapidly as the quantity of TV
018            edits increases. Finally, we explore different ways to mitigate this degradation
019            and present an adaptive method to select the most relevant TVs to apply to a
020
               diffusion model during inference. Our technique achieves a 94.6% ROC AUC in
               identifying the correct TV, enabling the effective integration of multiple TV edits
021
               while significantly mitigating quality degradation.

024   1    I NTRODUCTION
025
026   As advances in machine learning increasingly rely on large foundation models trained by entities
027   with substantial resources, the need to adapt these models to various end-user preferences is growing
028   (Zhuang et al., 2020). One straightforward way to adapt a foundation model is to fine-tune it on a
029   relevant dataset directly. However, this approach may suffer from issues including privacy concerns
030   (Yu et al., 2021), stability (Wortsman et al., 2022b; Mitchell et al., 2021), and computational resources
031   (De Cao et al., 2021).
032   One popular alternative to fine-tuning are task vector edits (TV edits), which perform algebraic
033   operations or task arithmetic, such as addition or subtraction directly on the model weights. The task
034   vector (TV) is a learned set of weights representing the difference between the pre-trained model
035   weights θ0 and a fine-tuned model weights θft (Ilharco et al., 2022a). For example, to reduce the
036   likelihood of a model generating pictures in the style of Vincent Van Gogh, a model owner might
037   perform a TV edit, subtracting a task vector learned from Van Gogh’s images τ1 from the base model
038
      weights to yield a sanitized model θT V :
039
040
                                                 θT V = θ0 − α1 τ1                                          (1)
041
042   TV edits attain stability and robustness not achievable by other methods (Tsai et al., 2023; Pham
043   et al., 2023; 2024). In addition, TVs finetuned on a narrow task may allow for better control on
044   the generation of specific attributes (Gandikota et al., 2023). Even so, in the pursuit of better
045   target task performance on the new behavior, task vector edits can sometimes impair the quality of
046   unrelated generations, also known as control task performance. The trade-off between the target
047   task performance and control task performance has received considerable attention in prior work
048   on single TV applications (Gandikota et al., 2023; Pham et al., 2024). Additionally, a growing line
049   of research focuses on combining TVs representing broad fine-tunes of the model (also known as
050
      model merging), which should generally improve the overall performance and not degrade it (Yadav
      et al., 2024; Xu et al., 2024; Matena & Raffel, 2022). Yet, TVs aimed at a specific class or concept
051
      generally degrade the control task performance.
052
053   In this work, we raise a new, related question; what happens when the model owner wants to change
      multiple capabilities, requiring multiple TV edits togather? For example, erasing the ability to generate

054
      or recognize many different human identities (Zehavi & Shamir, 2023), removing copyrighted styles
055   (Pham et al., 2023), or precisely controlling generated attributes (Gandikota et al., 2023). We study the
056   interactions of task vectors in this multi-edit setting, and uncover an unusual phenomenon; multiple
057   TV edits interact not only with the model itself but also with one another in their effect on the control
058   task performance. We term this behavior multi-task interactions.
059
      We find that pairwise-task interactions of TVs can be modeled as a combination of two contrasting
060
      regimes: highly colinear TVs tend to have linear multi-task interactions, meaning the control task
061   accuracy decrease is a function of the total magnitude of the applied TVs1 . Less similar TVs (which
062   tend to be induced by unrelated tasks Ilharco et al. (2022a)) usually have non-linear multi-task
063   interactions: editing with two different vectors will be less harmful to the control task than using a
064   single TV with the total magnitude (see Fig.1). Going beyond pairwise interactions to very large
065   number of simultaneously applied TVs, we find that the linear interactions dominate; accuracy
066   degrades linearly with the total magnitude of the vectors being subtracted. We offer a simple
067   theoretical explanation for this observation.
068   Next, we identify and evaluate various natural approaches to mitigate this accumulated degradation in
069   model accuracy. We test the following methods: (i) merging Task Vectors with a non-linear merging
070   algorithm developed for model-merging (non concept-specific TVs), (ii) learning a per-TV magnitude
071   for a better erasure/control trade-off, (iii) training a single joint Task Vector for multiple concepts,
072   and (iv) using the Neural Tangent Vector TV method (Ortiz-Jimenez et al., 2024). We find that all
073   the above methods fall short in reducing the model degradation to acceptable levels when applying a
074   large number of edits.
075   Finally, we propose a technique to choose at inference time which TVs to use for a diffusion process.
076   As different TVs are tuned to edit different concepts, most generations do not require a large number
077   of concurrent subtractions. Motivated by this intuition, we investigate whether we can determine
078   which TVs to apply to a diffusion model with a given prompt only by analyzing the effect of the TV
079   edit on the generated image. We determine that applying a TV in the middle of the denoising process
080   allows us to quantify its relevance to a given prompt. We flesh this idea out into a technique that
081
      applies TVs only when they are relevant to avoid unnecessary model deterioration.
082
      Our Contributions.

085            performance of a model.
086

087            task-vector edits, and find that all methods fall short at sufficient scale.
088
             • We propose a novel inference-time solution to adaptively applying only the relevant Task

               Vectors to a given prompt.

092   2    BACKGROUND
093
094   Merging Model Edits. There is a growing research interest in methods that take a few models,
095
      trained or fine-tuned separately, and combine them post-training into a single model enjoying the
      strengths of all the individual fine-tunes. While some approaches individually run each model
096
      and combine the model outputs in one of several ways Dietterich (2000); Ovadia et al. (2019);
097
      Gontijo-Lopes et al. (2021), other methods combine the model parameters themselves Chung (1954);
098   Wortsman et al. (2022a); Yadav et al. (2024). The Task Vector method combines model parameters
099   by fine-tuning a few different models from the same checkpoint and averaging the differences in
100   parameters accumulated in each model along the fine-tuning process. While averaging models at
101   parameter space may sound unintuitive, it was shown to be semantically meaningfulIlharco et al.
102   (2022a), and a few techniques were suggested to better optimize it Ortiz-Jimenez et al. (2024); Yadav
103   et al. (2024); Goddard et al. (2024). Yet, these techniques were mostly focused on the setting where
104   each TV was trained to add to the model a relatively broad capability (e.g., better general generation
105   quality or robustness). Less focused was directed to TV for specific narrow concepts, and concept
106
      erasure.

      Diffusion Models. Diffusion models Sohl-Dickstein et al. (2015); Ho et al. (2020) are a class of
109   generative models that learn to sample from a distribution using a Markovian denoising process. In
110   the forward diffusion process, Gaussian noise is gradually added to an input image x0 for T timesteps
111   to yield a final noise latent xT . The model learns the reverse diffusion process, which, given a latent
112   xt at a timestep t, predicts the residual noise ϵt = xt − xt−1 . At inference time, a random Gaussian
113   noise tensor xT is sampled and passed through the reverse process for T timesteps to yield the final
114   data x0 . Latent Diffusion Models (LDMs) Rombach et al. (2022) reduce the memory footprint of
115   diffusion models by performing the denoising process in a latent space learned using an autoencoder.
116
117   Task Vector Edits to Diffusion Models. Among many methods suggested to address model editing
118   for diffusion models, we focus on Task Vectors, as they are most suited to study the interaction
119
      between tasks. Practically, Task Vectors have been used in diffusion models to achieve better
      controllability Gandikota et al. (2023) and concept erasure Pham et al. (2024); Liu et al. (2024).We
120
      acknowledge the vast literature covering other editing method for diffusion models Orgad et al.
121
      (2023); Bau et al. (2020); Croitoru et al. (2023); as well as on applying task vectors to edit other types
122   of models Hendel et al. (2023); Ramesh et al. (2024); Hojel et al. (2024). However, as our primary
123   focus in the paper is on conceptual questions, we adhere to Task Vector edits applied to Diffusion
124   Models and classifiers.

      3     TASK V ECTOR I NTERACTIONS
127
128
      Our study begins with a simple question:
129
130            How do multiple TV edits performed together affect the model performance?
131
132   Our practical motivation for studying multi-task interaction is applying multiple Task Vectors to
133   a single model simultaneously. Yet, there is also a deeper scientific motivation for exploring this
134   question. Ideally, Task Vectors aim to represent a single edit direction (e.g., happy vs. sad) of the
135   model while mostly keeping other edit directions unaffected (e.g., outdoor vs. indoor) (Ilharco et al.
136   (2022a)). Examining the interaction of multiple task vector edits allows us to better inspect the extent
137
      to which TV are non-interfering, and study the interference caused by the combination of different
      tasks.
138
139

      3.1   PAIRWISE TASK I NTERACTIONS
141   The simplest kind of multi-task interaction is a pairwise interaction; two task vectors τ1 , τ2 are
142   applied to one model θ0 with strengths (amplitudes) α1 , α2 , generating a model θT V :
143
144
                                           θT V = θ0 + α1 · τ1 + α2 · τ2                                    (2)
145   We follow previous works by evaluating a pre-trained CLIP-based classifier model as θ0 , our base
146   model; and extend it to examining the unet of a Stable diffusion model in App.C. We explore a large
147   variety of classification tasks for the CLIP-based model, and different artistic styles and objects for
148   the stable diffusion model. Finally, we plot the control task performance (classification or generation
149   of unrelated concepts) as a function of the edit strengths of the two vectors, τ1 , τ2 (the magnitudes
150
      are noted as α1 , α2 serve as the axes of the control task performance heatmap). While varying the
      control task reveals diverse interaction patterns, the pair-wise interaction effects on standard tasks
151
      mostly fall into two categories, see Fig. 1 (see Appendix for implementation details and more results).
152
      We note these categories as Linear interactions and non-linear interactions.
153
154
      Linear interactions. In this type of TV interaction the degradation effect of using one amplitude,
155   α1 , for one TV, and a second amplitude, α2 , for another, is similar to the effect of using the sum of
156   amplitudes (α1 + α2 ) with one of them (see upper panel of Fig. 1). One simple such case is the
157   interaction between a TV and itself (τ1 = τ2 ). In this case, Eq. 2 trivially becomes:
158
                                            θT V = θ0 + (α1 + α2 ) · τ1                                     (3)
159
160   Therefore, the performance degradtion trivially becomes a function of (α1 + α2 ). This kind of
161   interaction is also expected in highly correlated tasks that move the similar weights in the same
      direction.

283   In the last section, we saw that the control task performance impacts accumulate linearly at scale,
284   making it difficult to apply multiple TV edits. In this section, we investigate whether this degradation
285   can be mitigated. We survey four solutions using either existing methods or simple modifications
286   to the TV techniques, and conclude that none of them work well enough to allow the practical
287   application of TV-based concept erasure at scale.
288
289   Non-Linear TV Combination. The standard way to combine TVs is using simple algebric vector
290   addition in the weight space. It might be the case, though, that other notions of combination such as
291   non-linear TV combinations may better preserve the control task accuracy. In fact, such methods
292
      have already been proposed for model merging, where edits aim to represent global improvement
      rather than changes to a specific concept or class. In the model merging case, we aim to reap the
293
      benefits of all the fine-tuned instances together Wortsman et al. (2022a). Yet, we can also evaluate
294
      these techniques for narrow tasks vectors such as ones finetuned on a single class. We therefore
295   evaluate 4 alternatives for parameter-wise TV combination: (i) Linear - regular linear addition of
296   the model weights (Baseline). (ii) Sparse - we sparsify the TVs such that each vector contains only
297   the weights of the top p percentiles of TV parameters, sorted by magnitude, then add the sparse
298   TVs linearly as in the standard method. The precentile p is varied to inspect different points on the
299   control-target tradeoff. (iii) Median - Similar to Linear but taking the median rather than the sum of
300   each of the parameters. A global magnitude factor can be used to better explore the trade-off between
301   the control task performance and the target task performance. (iv) Tie merging Yadav et al. (2024) -
302   A leading method for combining positive TV edits. We find that all TV combination methods give a
303
      similar control-target trade-off (Fig. 3, App.D for implementation details).

      Task Arithmetic in the Tangent Space Ortiz-Jimenez et al. (2024). A recent work suggests that
342
      TV arithmetic works partly because of weight disentanglement. Namely, they claim that different TV
343
      mainly change different parameters in the model. The authors propose a method to encourage weight
344   disentanglement through learning TVs in the Neural Tangent Space Ortiz-Jimenez et al. (2024);

350   Joint TV Training. A possible simple modification to the TV setting is to train a single TV aimed
351   at jointly performing multiple target tasks together, instead of training N vectors individually and
352   combining them later. While this technique may convey some desired properties of the TV technique,
353
      like the the option to control and reverse the edit amplitude; it does not allow other benefits like
      combining TV from different sources. To evaluate this technique potential to better preserve a control
354
      tasks performance, we train a TV on many tasks together, and compare the control-target performance
355
      trade-off it provides to that of the TV baseline techniques. We can see in Fig. 6 (Co-Training)
356   that unlike the previous solutions explored in this section, this technique does provide a somewhat
357   better trade-off, and we recommend it as a practical solution for somewhat mitigating control task
358   degradation when possible. Yet, this solution is still not enough to allow the application of TV-based
359   erasure for large values of N .

      5    A DAPTIVE TASK V ECTOR S ELECTION
362
363   In the previous section, we found that existing solutions cannot sufficiently preserve control task
364   performance under multiple task edits. In this section we present a possible solution. Since combining
365   a large number of TVs significantly degrades the control task performance, we aim to decide during
366   inference time which TV should be applied for a given sample. Our main idea is that for a given
367   prompt, different TVs will differently affect the denoising process; and that this difference can be
368   tracked during inference time.
369
370   Adaptive test-time selection of Task Vector. Our technique relies on a simple assumption: a Task
371   Vector that is irrelevant to a given generation would tend to produce a smaller semantic difference in
372
      the output image compared to relevant one TV. Identifying irrelevant Task Vectors would enable us
      not to apply them to a given generation, and would prevent the degradation they cause the control task
373
      accuracy. Therefore, we first apply each Task Vector edit on its own to find which TVs are relevant.
374
375   Namely, we first generate an image using the original model Gθ0 and the text prompt p:

      7      C ONCLUSION
532
533
      We started this study by exploring the effects of multiple task vector interactions on a model’s
534   control task performance. Motivated by this, we turned to investigating how model degradation may
535   be mitigated when subtracting many different task vectors from the same model. We explored a
536   large variety of methods and found that simple or existing technique do not sufficiently mitigate
537   the degradation of the model on tasks unrelated to the applied TV edits. Therefore, we suggusted
538   an adaptive technique that finds the relevant TV to be applied to a diffusion model at the inference
539   time. Finally, we evaluate our suggested method and find it is effective in mitigating the degradation
      generations unrelated to the applied TV edits.

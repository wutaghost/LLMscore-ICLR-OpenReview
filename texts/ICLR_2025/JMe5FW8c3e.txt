009                                               A BSTRACT
010
011            In reinforcement learning, offline value function learning is the procedure of using
012            an offline dataset to estimate the expected discounted return from each state when
013            taking actions according to a fixed target policy. The stability of this procedure,
014            i.e., whether it converges to its fixed-point, critically depends on the represen-
015            tations of the state-action pairs. Poorly learned representations can make value
016            function learning unstable, or even divergent. Therefore, it is critical to stabilize
017            value function learning by explicitly shaping the state-action representations. Re-
018
               cently, the class of bisimulation-based algorithms have shown promise in shaping
               representations for control. However, it is still unclear if this class of methods can
019
               stabilize value function learning. In this work, we investigate this question and
020
               answer it affirmatively. We introduce a bisimulation-based algorithm called ker-
021            nel representations for offline policy evaluation (KROPE). KROPE uses a kernel to
022            shape state-action representations such that state-action pairs that have similar im-
023            mediate rewards and lead to similar next state-action pairs under the target policy
024            also have similar representations. We show that KROPE: 1) learns stable repre-
025            sentations and 2) leads to lower value error than baselines. Our analysis provides
026            new theoretical insight into the stability properties of bisimulation-based meth-
027            ods and suggests that practitioners can use these methods for stable and accurate
028            evaluation of offline reinforcement learning agents.

      1   I NTRODUCTION
032
033
      Learning the value function of a policy is a critical component of many reinforcement learning (RL)
      algorithms (Sutton & Barto, 2018). While value function learning algorithms such as temporal-
034
      difference learning (TD) have been successful, they can be unreliable. In particular, the deadly
035
      triad, i.e., the combination of off-policy updates, function approximation, and bootstrapping, can
036   make TD-based methods diverge (Sutton & Barto, 2018; Tsitsiklis & Van Roy, 1997; Baird, 1995).
037   Function approximation is a critical component of value function learning since it determines the
038   representations of state-action pairs, which in turn defines the space of expressible value functions.
039   Depending on how this value function space is represented, value function learning algorithms may
040   diverge (Ghosh & Bellemare, 2020). That is, the value function learning algorithm may not converge
041   to its fixed-point, or may even diverge away from it. In this work, we investigate how to explicitly
042   learn state-action representations to stabilize value function learning.
043   In seeking such representations, we turn to π-bisimulation algorithms. These algorithms define
044   a metric to capture behavioral similarity between state-action pairs such that similarity is based
045   on immediate rewards received and the similarity of next state-action pairs visited by π (Castro,
046   2019). The algorithms then use this metric to learn representations such that state-action pairs that
047   are similar under this metric have similar representations. Ultimately, the goal of π-bisimulation
048   methods is to learn representations such that state-actions pairs with similar values under π also have
049   similar representations (see Figure 1). While these algorithms have shown promise in improving the
050
      expected return of RL algorithms, it remains unclear whether they contribute to stability (Castro
      et al., 2023; Zhang et al., 2021; Castro et al., 2022). In this paper, we aim to understand whether
051
      π-bisimulation-based representations stabilize value function learning.
052
053   In this work, we focus on offline value function learning. Given a fixed, offline dataset generated
      by unknown and possibly multiple behavior policies, the goal is to estimate the value function of

      2.2   O FFLINE P OLICY E VALUATION AND VALUE F UNCTION L EARNING
128
129   In offline policy evaluation (OPE), the goal is to evaluate a fixed target policy, πe , using a fixed
130   dataset of m transition tuples D := {(si , ai , s′i , ri )}m
                                                                 i=1 . In this work, we evaluate πe by estimating
131   the action-value function q πe using D. Crucially, D may have been generated by a set of unknown
132   behavior policies that are different from πe , which means that simply averaging the discounted
133   returns in D will produce an inconsistent estimate of q πe . In our theoretical results, we make the
134   standard coverage assumption that ∀s ∈ S, ∀a ∈ A if πe (a|s) > 0, then the state-action pair (s, a)
135   has non-zero probability of appearing in D (Sutton & Barto, 2018; Precup et al., 2000).
136   We measure the accuracy of the value function estimate with the mean squared value error (MSVE).
137   Let q̂ πe be the estimate returned by a value function learning method using D. The MSVE of this
138   estimate is defined as MSVE[q̂ πe ] := E(S,A)∼D [(q̂ πe (S, A) − q πe (S, A))2 ]. In environments with
139   continuous state-action spaces, where it is impossible to compute q πe for all state-actions, we adopt
140   a common evaluation procedure from the OPE literature of measuring the MSE across only the ini-
141   tial state-action distribution, i.e., MSE[q̂ πe ] := ES0 ∼d0 ,A0 ∼πe [(q̂ πe (S0 , A0 ) − q πe (S0 , A0 ))2 ]. For
142   this procedure, we assume access to d0 (Voloshin et al., 2021; Fu et al., 2021). While in practice
143
      q πe is unknown, it is standard for the sake of empirical analysis to estimate q πe by executing unbi-
      ased Monte Carlo rollouts of πe or computing q πe exactly using dynamic programming in tabular
144
      environments (Voloshin et al., 2021; Fu et al., 2021).
145

167
168   When determining the stability of LSPE, we have following proposition from prior work:
169
      Proposition 1 (Asadi et al. (2024); Wang et al. (2021a)). LSPE is stable if and only if the spectral
170
      radius of (E[Φ⊤ Φ])−1 (γE[Φ⊤ P πe Φ]), i.e., its maximum absolute eigenvalue, is less than 1.
171
172   Therefore, the stability of LSPE largely depends on the representations Φ and the distribution shift
173   between the data distribution of D and πe . In this work, we study the stability of LSPE for a fixed
174   distribution of D and learn Φ. If a given Φ stabilizes LSPE, we say Φ is a stable representation.
175
      In addition to stability, we also care about the realizability and generalizability of Φ. We say Φ is a
176   realizable representation if q πe ∈ Span(Φ), where Span(Φ) is the subspace of all expressible action-
177   value functions with Φ. Note that even if Φ is a realizable and stable representation, LSPE may not
178   recover the q πe solution (Sutton & Barto, 2018). While generalization can have multiple interpreta-
179   tions, we say Φ generalizes well if the state-action features that are close in the representation space
180   also have similar q πe values (Lyle et al., 2022; Lan et al., 2021).

182   2.4   R ELATED W ORKS
183
184   In this section, we discuss the most relevant prior literature on OPE and representation learning.
185   Representations for Offline RL and OPE. There are several works that have shown shaping rep-
186   resentations can be effective for offline RL (Yang & Nachum, 2021; Islam et al., 2023; Nachum &
187   Yang, 2024; Zang et al., 2023a; Arora et al., 2020; Uehara et al., 2021; Chen & Jiang, 2019; Pavse
188   & Hanna, 2023b). Ghosh & Bellemare (2020) presented a theoretical understanding of how various
189   representations can stabilize TD learning. However, they did not discuss bisimulation-based repre-
190   sentations. Kumar et al. (2021); Ma et al. (2024); He et al. (2024) promote the stability of TD-based
191   methods by increasing the rank of the representations to prevent representation collapse. However,
192
      as we show in Section 4, these types of representations can still lead to inaccurate OPE. On the
      other hand, KROPE mitigates representation collapse and leads to accurate OPE. Chang et al. (2022)
193
      introduced BCRL to learn Bellman complete representations for stable OPE. While in theory, BC
194
      representations are desirable, we found that BCRL is sensitive to hyperparameter tuning. In contrast,
195   we show that KROPE is more robust to hyperparameter tuning. Pavse & Hanna (2023a) showed that
196   bisimulation-based representations mitigate the divergence of FQE; however, they did not provide
197   an explanation for divergence mitigation. Our work provides theoretical insight into the stability
198   properties of bisimulation-based algorithms.
199   Bisimulation-based Representation Learning. Recently, there has been lot of interest in π-
200   bisimulation algorithms for better generalization (Ferns et al., 2004; 2011; Ferns & Precup, 2014;
201   Castro, 2019; Zang et al., 2023b). These algorithms measure similarity between two state-action
202   pairs based on immediate rewards received and the similarity of next state-action pairs visited by
203   π. These algorithms first define a distance metric that captures this π-bisimilarity, and then use
204   this metric to learn representations such that π-bisimular states have similar representations (Castro
205   et al., 2022; Castro, 2019; Zhang et al., 2021; Castro et al., 2023; Chen & Pan, 2022; Kemertas &
206   Jepson, 2022). Zhang et al. (2021); Castro (2019) introduced a π-bisimulation learning algorithm
207
      but assume that the transition dynamics are either deterministic or Gaussian. Gelada et al. (2019)
      introduced a method closely related to bisimulation methods but required a reconstruction loss to
208
      work in practice. Castro et al. (2022) introduced MICO which allows for stochastic transition dynam-
209
      ics and no reconstruction loss, but was difficult to theoretically analyze. To overcome this difficulty,
210   Castro et al. (2023) took a kernel perspective of π-bisimulation methods, which made their algorithm
211   amenable to theoretical analysis. To the best of our knowledge, no works have studied the stability
212   properties of π-bisimulation algorithms. In our work, we address this gap in the literature. We first
213   extend Castro et al. (2023)’s kernel-based formulation from states to state-actions, and then show
214   that this formulation stabilizes offline value function learning. The proofs for KROPE’s basic theo-
215   retical properties (Section 3.1) follow those by Castro et al. (2023). Our stability-related theoretical
      results (Sections 3.2 and 3.3) and empirical analysis (Section 4) are novel to this work.

274   3.3     C ONNECTION TO B ELLMAN C OMPLETENESS
275
276   In this section, we draw a novel connection between KROPE representations and Bellman complete-
277   ness. We say a function class F is Bellman complete if it is complete under the Bellman operator:
278   T πe f ⊆ F, ∀f ∈ F. For instance, suppose F is the class of linear functions spanned by Φ,
279   F := {f ∈ RX : f := Φw}, w ∈ Rd . Then if T πe f, ∀f ∈ F is also a linear function within the
280   span of Φ, we say Φ is a Bellman complete representation. Bellman completeness is an alternative
281
      condition for stability and is typically assumed to ensure to data-efficient policy evaluation (Wang
      et al., 2021b; Szepesvári & Munos, 2005; Chang et al., 2022). We now present our second main
282
      result. It states that KROPE representations are Bellman complete:
283
284
285         Theorem 2. Let ϕ : X → X ϕ be the state-action abstraction induced by grouping state-
286
            actions x, y ∈ X such that if dKROPE (x, y) = 0, then ϕ(x) = ϕ(y), ∀x, y ∈ X . Then ϕ
            is Bellman complete if the abstract reward function rϕ : X ϕ ↣ (−1, 1) is injective (i.e.,
287
            distinct abstract rewards).

291
292         KROPE representations induce non-expansive value function updates and are Bellman com-

504   5   L IMITATIONS AND F UTURE W ORK
505
506
      In this section, we discuss limitations and future work. A shortcoming of our work is that KROPE’s
507
      learning algorithm is susceptible to instability since it is a semi-gradient method (Sutton & Barto,
508   2018). Moreover, its fixed-point optimization means it does not solve any objective function (Feng
509   et al., 2019). In our work, we employed commonly-used techniques such as layernorm and wide
510   neural networks to mitigate instability (Ota et al., 2021; Gallici et al., 2024). While these tech-
511   niques potentially side-step the issue, the consequences of a semi-gradient method may still exist.
512   In Appendix C.3.1, we present an empirical analysis to gauge when KROPE’s learning algorithm
513   may be unstable. We find that while individual off-policy transitions can determine the instability of
514   fixed-point and semi-gradient algorithms such as FQE, pairs of off-policy transitions can determine
515   KROPE ’s instability. Since we are unlikely to have control over the distribution over pairs of transi-
516
      tions in practice, we need to resort to fundamental changes to the algorithm. One potential change is
      based on that by Feng et al. (2019). Their key insight is to leverage the Legendre-Fenchel transfor-
517
      mation from optimization theory and replace the fixed-point loss function of semi-gradient methods
518
      with an equivalent expression that avoids semi-gradient learning (Rockafellar & Wets, 1998). How-
519   ever, a drawback with this approach is that the new learning objective is a minimax procedure,
520   which can be challenging to optimize in practice. In future work, we will explore the viability of
521   this approach to design a provably convergent version of KROPE.

524   6   C ONCLUSION
525
526   In this work, we tackled the problem of stabilizing offline value function learning in reinforcement
527   learning. We introduced a bisimulation-based representation learning algorithm, kernel represen-
528   tations for OPE (KROPE), that shapes the state-action representations to stabilize this procedure.
529   Theoretically, we showed that KROPE representations are stable from two perspectives: 1) non-
530   expansiveness, i.e., they lead to value function learning updates that enable convergence to a fixed-
531   point and 2) Bellman completeness, i.e., they satisfy a condition for data-efficient policy evaluation.
532
      Empirically, we showed that KROPE leads to more stable and accurate offline value function learn-
      ing than baselines. Our work showed that bisimulation-based representation learning effectively
533
      stabilizes long-term performance evaluations of offline reinforcement learning agents.

      A.1   B ISIMULATION M ETRICS
815
816   In this section, we present background information on bisimulations and its associated metrics.
817   Our proposed representation learning algorithm is a bisimulation-based algorithm. Bisimulation ab-
818   stractions are those under which two states with identical reward functions and that lead to identical
819   groups of next states under any action are classified as similar (Ferns et al., 2004; 2011; Ferns &
820   Precup, 2014). Bisimulations are the strictest forms of abstractions. In practice, the exact bisimula-
821   tion criterion is difficult to satisfy computationally and statistically. A more relaxed version of this
822   notion is the π-bisimulation metrics. These metrics capture the similarity between two states such
823
      that two states are considered similar if they have identical expected reward functions and expected
      transitions to identical groups of next states under a fixed policy π (Castro, 2019).
824
825   We first give the definition of bisimulation.
826   Definition 4. (Li et al., 2006) An abstraction ϕ : S → S ϕ over the state space S is a bisimulation
827   if for any action a and any abstract state sϕ ∈ S ϕP  , ϕ is such that for anyPtwo states s1 , s2 ∈ X ,
828   ϕ(s1 ) = ϕ(s2 ) implies that r(s1 , a) = r(s2 , a) and s′ ∈sϕ P (s′ |s1 , a) = s′ ∈sϕ P (s′ |s2 , a).
829
830   Below we define π-bisimulations for state-actions instead of states:
831   Definition 5. (Castro, 2019) An abstraction ϕ : X → X ϕ over the state-action space X is a π-
832   bisimulation for a fixed policy π if for any two state-actions x, y ∈ X and P

      Note that if the above inequality is strictly greater than zero whenever {c1 , . . . , cn } has at least one
865   nonzero, we say the kernel is positive definite. Given a kernel k on X with the reproducing property,
866   we can construct a space of functions Hk referred to as a reproducing kernel Hilbert space (RKHS)
867   with the following steps:
868
869         1. Construct a vector space of real-valued functions on X of the form {k(x, ·) : x ∈ X }.
870
871         2. Equip this space with an inner product given by ⟨k(x, ·), k(y, ·)⟩Hk = k(x, y).
872
873         3. Take the completion of the vector space with respect to the above inner product.
874
875   Our resulting vector space Hk is then an RKHS.
876   It is often convenient to write ψ(x) := k(x, ·) ∈ Hk , which is called the feature map and is an
877   embedding of x in Hk . One can also embed probability distributions into Hk . That is, Φ : P(X ) →
878   Hk , which maps probability distributions over X to Hk . We define Φ(µ) = EX∼µ [ψ(X)], which is
879   the mean embedding in Hk under µ.
880
      Given these embeddings in the Hilbert space, we can quantify the distances between elements in X
881   and P(X ) in terms of the embeddings.
882
883   Definition 6. Given a positive semidefinite kernel k, define ρk as its induced distance:
884
885                                          ρk := ∥ψ(x) − ψ(y)∥Hk .
886
887   By expanding the inner product, the squared distance can be written in terms of k:
888
889                                  ρ2k (x, y) = k(x, x) + k(y, y) − 2k(x, y).
890
891
      Similarly, we have distances on P(X ) using Φ:
892
893   Definition 7. (Gretton et al., 2012) Let k be a kernel on X and Φ : P(X ) → Hk be as defined
894   above. Then the Maximum Mean Discrepancy (MMD) is a pseudo metric on P(X ) defined by:
895
896                                    MMD(k)(µ, ν) = ∥Φ(µ) − Φ(ν)∥Hk .
897
898
899
      The core usage of the RKHS is to precisely characterize the nature of the KROPE kernel. In practice,
      we deal with neural network representations, which are embedded in Euclidean space. Therefore,
900
      our goal is to learn representations in Euclidean space that approximate the properties of representa-
901
      tions in the RKHS. For more details on the RKHS, we refer readers to Castro et al. (2023) and Gretton
902   et al. (2012).

1429   C.3.1     U NDERSTANDING THE S TABILITY OF THE KROPE L EARNING P ROCEDURE
1430
       In this section analyze how the deadly triad affects KROPE learning procedure. Since KROPE’s
1431
       learning algorithm is a semi-gradient method, it is susceptible to instability (Tsitsiklis & Van Roy,
1432   1997). As such, these experiments aim to understand when we might expect KROPE to diverge and
1433   when it might mitigate the divergence of FQE.
1434
1435
       We conduct our experiments on the Markov reward process in Figure 5(a), which was introduced
       by Feng et al. (2019). The MRP consists of 4 non-terminal states, 1 terminal state (the box), and
1436
       only 1 action. The value function estimate is linear in the weight w = [w1 , w2 , w3 ], so the native
1437
       features of the states are [1, 0, 0], [0, 1, 0] , [0, 0, 2], and [0, 0, 1] (clockwise starting from left-most).
1438   We set γ = 1, so the optimal weights are w∗ = [0.8, 1, 0]. In this setup, we say a transition is a
1439   bad transition if the bootstrapping target is a moving target for the current state (Asadi et al., 2024).
1440   For example, the transition from w3 to 2w3 is a bad transition since updates made to w3 may move
1441   2w3 further away. When this transition is sampled at a frequency that is different from the on-policy
1442   distribution, algorithms such as TD, LSPE, and FQE tend to diverge (Asadi et al., 2024).

       To better understand the stability of the learning procedures, we design the following experiment.
1459   We start with two datasets D1 and D2 . D1 consists of 2000 on-policy transitions and 5000 off-
1460   policy bad transitions from w3 to 2w3 , while D2 consists of only 2000 on-policy transitions. We
1461   then consider four different variations of D2 : D2w1 , D2w2 , D2w3 , and D22w3 , where each variation
1462   denotes a dataset where we add 5000 off-policy transitions from the specified state to D2 . With this
1463   setup, our goal is understand which dataset pairing with D1 : (D1 , D2w1 ), (D1 , D2w2 ), (D1 , D2w3 ), and
1464   (D1 , D22w3 ) increases the susceptibility of KROPE to divergence. Recall that at each training step,
1465   KROPE samples pairs of transitions. Thus, in this setup, KROPE will sample one transition from D1
1466   and the other transition from the other dataset.
1467
       In Figures 5(b) and 5(c), we show the training loss of KROPE only and FQE + KROPE with KROPE as
1468   an auxiliary task. In both cases, the sampled state-actions are fed into a linear network encoder with
1469   a bias component and no activation function which outputs a d = 3 representation. In case of FQE,
1470   this representation is then fed into a linear function to output the scalar value. From Figure 5(b),
1471   we find that even though D1 consists of mostly bad transitions, if KROPE also samples from D2w1
1472   or D2w2 , its divergence is mitigated. However, as expected, if KROPE uses D2w3 or D22w3 , KROPE
1473   diverges since the pairing of samples from (D1 , D2w3 ), and (D1 , D22w3 ) leads to KROPE chasing a
1474   moving bootstrapped target, resulting in divergence. Building upon this insight, we can then better
1475   understand when KROPE can mitigate FQE’s divergence. From Figure 5(c), we see the expected
1476   result that when FQE uses D1 only, it diverges since D1 consists of many bad transitions. However,
1477   if FQE uses KROPE as an auxiliary loss, KROPE may either reduce the divergence (D2w1 and D2w2 )
1478   or worsen it (D2w3 and D22w3 ) depending on which dataset is paired with D1 . While in practice, it is
1479   unclear to determine which situation is more likely, this result sheds light on the fact that the KROPE
1480   learning procedure can potentially mitigate the divergence of FQE’s learning procedure depending
1481
       on which pairs of transitions are sampled.

           Takeaway #3: Understanding KROPE Divergence
1483

1485       worsen the divergence of FQE.

1646
       In our main empirical section (Section 4), we used FQE as a representation learning algorithm on
1647
       our custom datasets. We adopted the linear evaluation protocol, i.e., an approach of analyzing the
1648   penultimate features of the action-value function network and applied LSPE on top of these features
1649   for OPE. This protocol enabled us to better understand the nature of the learned features.
1650
1651
       For the sake of completeness, we present results of FQE as an OPE algorithm where the action-value
       network is directly used to estimate the performance of πe . We present the results in Figures 10 and
1652
       11. As done in Section 4, we evaluate the performance of FQE and KROPE based on how they shape
1653
       the penultimate features of the action-value network. However, when conducting OPE, we evaluate
1654   two variants: 1) using LSPE (- L) and 2) using the same end-to-end FQE action-value network (- E 2 E).
1655
1656   From Figure 10, we find that there are hyperparameter configurations that can outperform the KROPE
1657
       variants. However, based on Figure 11, we find both KROPE variants are significantly more robust
       to hyperparameter tuning. This latter result suggests that KROPE does improve stability with respect
1658
       to the hyperparameter sensitivity metric as well.
1659
1660   Regardless of FQE’s hyperparameter sensitivity, it is still interesting to observe that when FQE is
1661   used as an OPE algorithm, it produces reasonably accurate OPE estimates. It even outperforms the
1662   FQE + KROPE combination. However, as shown in Section 4, the penultimate features of this same
1663
       network actually have poor property values such as high feature co-adaptation, high condition num-
       bers, and highly erratic OPE estimates with LSPE. Given that these features have weak properties,
1664
       it is unclear why FQE still can lead to accurate OPE. The primary difference between using FQE for
1665
       OPE vs. FQE features and LSPE for OPE is in how the last linear layer is trained. The former is trained
1666   by gradient descent while the latter is trained with the iterative LSPE algorithm on the fixed features.
1667   An interesting future direction will be to explore the learning dynamics of these two approaches.
1668
1669
       On a related note, we point out that the training dynamics of FQE are still not well-understood. For
       example, Fujimoto et al. (2022) show that the FQE loss function poorly correlates with value error.
1670
       That is, the FQE loss can be high but value error (and OPE error) can be low.

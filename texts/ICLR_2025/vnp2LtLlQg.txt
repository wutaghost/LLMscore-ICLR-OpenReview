                                                  A BSTRACT
008
009
               The attention mechanism is an important part of transformer architectures. It en-
010
               ables the network to compare samples within a sequence. Before the comparison
011            is performed, tokens are multiplied by trainable matrices. These matrices can
012            constitute a significant part of the total number of parameters. Their size creates
013            problems on systems with limited cache in the compute unit, especially if there
014            is limited bandwidth between compute unit and memory. In particular, GPUs on
015            mobile devices suffer from this double bottleneck.
016            Prior works mitigate this problem for instance by storing low-rank approxima-
017            tions, quantization or minimizing the amount of data that needs to be transferred.
018            In this paper, an alternative to the traditional attention mechanism is proposed
019            which does not require any trainable matrices to perform the attention. The idea
020            rests upon solving optimization problems, whereby memory is substituted for
021
               compute. It will be shown however, that the computational demand can be re-
               duced such that auto-differentiation becomes possible. An experimental evalua-
022
               tion shows that the proposed algorithm performs favorable compared with several
023
               baselines.

026   1   I NTRODUCTION
027
028   Transformers Vaswani et al. (2017) have been widely used in various applications, such as natural
029   language processing Devlin et al. (2019) or image recognition Dosovitskiy et al. (2021). More
030   recently, very large transformer models have been successfully applied to auto-regressive language
031   modeling, for instance GPT OpenAI et al. (2023), Llama Touvron et al. (2023), and Gemini Bao
032   et al. (2023). These powerful transformers require a vast amount of memory to run due to their large
033
      model sizes. While the parameters can be loaded layer by layer during inference, it still poses a
      significant challenge when running them on mobile devices, which typically have very limited cache
034
      in the neural processing unit – often, a few megabytes only. In addition, the attention operation
035
      incurs quadratic complexity w.r.t. number of input tokens, both in terms of memory usage and
036   computation.
037
038   Recently, researchers have proposed various techniques to reduce the computation and/or memory
039
      costs of attention. Efficient attention methods such as Letourneau et al. (2024) only require linear
      computation and memory costs while maintaining model accuracy. Other works optimize the imple-
040
      mentation of attention. For instance, FlashAttention Dao et al. (2022) uses tiling to decompose the
041
      softmax computation and other techniques to reduce the number of memory accesses. However, in
042   these approaches, large matrices mapping the input to query, key, and value tensors prior to attention
043   are still necessary. The severely limited cache at the compute units necessitates that only tiny pieces
044   of these matrices can be kept in cache at the same time, thus token data and matrix pieces need to
045   be retrieved from memory multiple times for each attention operation. This increases the amount of
046   data that needs to be transferred between slow main memory and fast cache via limited bandwidth,
047   which incurs high latency especially on resource-constrained devices.
048   In this work, we propose a novel, optimization-based approach to perform both self- and cross-
049   attention, in order to eliminate both the heavy weight matrices and the computationally expensive
050   soft-(arg)max operation during the attention computation. Instead of remixing the values based on
051   the similarity/attention matrix between queries and keys, we directly solve for the mixing coeffi-
052   cients for the value tensor. Specifically, we impose that the remixed values need to be similar to
053   the queries. We additionally impose a sparsity constraint, which leads to a sparse reconstruction
      problem.

      To reduce the high cost for iteratively solving these problems during any forward pass and to reduce
055   the memory demand to allow for backward passes, we propose to use several approximations to
056   increase efficiency, thereby enabling auto-differentiation with the proposed algorithm. To mitigate
057   the errors induced by the proposed techniques, we also propose to include random tokens in our
058   algorithm, similar to the idea proposed by Darcet et al. (2024), yet we offer a geometric intuition
059   regarding the effect these tokens have in the proposed algorithm. A scalar activation and a light-
060   weight normalization further promote sparsity and reduce overfitting.
061   A well-known property of the standard attention is that the attention coefficients, namely the soft-
062   (arg)max normalized scalar products between queries and keys, are very sparse, and thus focus on
063   a few key/value tokens. For images, it has been shown, e.g., by Dosovitskiy et al. (2021), that high
064   attention weights are concentrated on a few spots in the image from which the keys stem. Our
065   proposed approach achieves similar effects, but achieves this via sparse signal reconstruction. This
066   offers a new interpretation of the mechanism behind attention, as well as a new way to compute at-
067   tention by solving the sparse signal reconstruction problem. In other words, the proposed algorithm
068   optimizes over the given data to determine where to focus attention, ie where to place high attention
069
      scores.
070   An experimental evaluation shows that the proposed algorithm compares favorable with suitable
071   baselines. We provide qualitative example of semantically meaningful attention maps.
072
      This paper is structured as follows: In Sec. 2, we briefly summarize related works. We give a detailed
073   explanation of the standard attention mechanism in Sec. 3 and present our solution in Sec. 4. The
074   experimental evaluation follows in Sec. 5. The paper concludes with a discussion in Sec. 6.

078
079   Reducing Model Sizes Jacob et al. (2018) proposed to reduce model sizes by quantization. It is a
080   technique to reduce the memory requirements by converting the weights from some higher-precision
081   representation, such as 32-bit floating point numbers, to some lower precision, often integers. This
082
      process not only reduces the model sizes but may also increase the efficiency if the compute unit can
      process integer data faster.
083
084   Model sizes can also be reduced by so-called low-rank approximation (Guo et al., 2023) that is
085   to approximate matrices by low-rank approximations. Only the rank-reduced variants are being
086   stored. As with quantization, LoRA-approximation can also increase the time data needs to travel
087   through the network. Other techniques to reduce model sizes are pruning (Sun et al., 2024) or
088
      distillation (Oquab et al., 2023).
089
090   Computational Complexity of Attention Choromanski et al. (2020) used random orthogonal
091   features to estimate the a low-rank approximation of the attention matrix. Child et al. (2019) pro-
092
      posed to use attention masks with pre-defined sparsity patterns to reduce the computational demand
      of attention. Locality-sensitive hashing is used by Kitaev et al. (2019) in the attention, so that only
093
      those key tokens that are closest to each query are being used.
094
095   A transformer with linear complexity for computing self-attention was proposed by Wang et al.
096   (2020), albeit only for fixed sequence-lengths. Katharopoulos et al. (2020) proposed an attention
097   mechanism with linear complexity for variable sequence lengths. You et al. (2023) improved this
098
      idea by using an approximation to angular similarities as activation.
099   State-Space-Models (Poli et al., 2023), (Gu & Dao, 2023), (Patro & Agneeswaran, 2024) model
100   the interaction between tokens by first-order differential equations or approximations thereof. For
101   images, Hou et al. (2024) use a convolutional architecture that employs self-attention. To reduce
102   the computational complexity of this operation, the authors propose to approximate the attention
103   matrix by 2D-convolutions. Letourneau et al. (2024) generalize this idea by approximating the self-
104
      attention with polynomials. They show that their algorithm generalizes several others, for instance
      Conv2Former (Hou et al., 2024) or (Gu & Dao, 2023).
105
106   FlashAttention by (Dao et al., 2022) was proposed to minimize the number of accesses to the main
107   memory of the GPU since the bandwidth between the compute units to the memory is limited.
      Follow-up works Dao (2024); Shah et al. (2024) further optimize the idea. Since evaluating the

112   3    S TANDARD ATTENTION M ECHANISM
113
114   Attention performs exhaustive pairwise comparisons between the set of query tokens, q1 , . . . , qNq ,
115
      with the set of key tokens, k1 , . . . , kNk . The resulting similarity matrix, i.e., attention matrix, is
      used to linearly combine a set of value tokens, v1 , . . . , vNk .
116
117   In self-attention, the elements, qi , kj , vj , are created by multiplying the input by the weight matri-
118   ces, Wq , Wk , and Wv , which correspond to the queries, keys, and values, respectively. The weight
119   matrices have sizes D × Dq,k,v . Here, the three numbers Dq,k,v indicate the dimensions of the three
120   token sequences qi , kj , and vj .
121   In cross-attention, two different input sequences are used. The queries qi are created from one
122   sequence by multiplication with Wq , whereas the keys kj and values vj are created from another
123   sequence by multiplications with Wk and Wv , respectively. Some models such as the one proposed
124   by Carion et al. (2020) even use three different input sequences.
125   For multi-head   attention, the  vectors qi , kj and vj are split into parts of equal size, for instance,

      4    P ROPOSED A PPROACH : O PTIMIZING ATTENTION
136
137
      The original transformer Vaswani et al. (2017) can be formulated as a message-passing algorithm,
138   as shown by Veličković et al. (2018); Yun et al. (2019). We draw on this formulation and use a
139   simplified intuition in the following: At each attention, the nodes in the value set {vj } send update
140   messages vj to the nodes of the query set {qi }. The incoming messages into each node of the
141   query set are weighted by the normalized scalar products soft-argmax(ηqiT K T ). These messages
142   are sum-aggregated before the nodes in the query set are updated by an MLP. Normalization by
143   soft-argmax maintains the original numerical ranges for the sum-aggregated update messages.
144
      First, we formulate attention as a reconstruction problem, where we solve for the linear transforma-
145   tion that linearly combines the values so as to approximate the queries
146
147                                            min qiT − xTi · V                                              (2)
                                                 xi
148
149   for a suitable norm, where xi is an Nv -dimensional variable to be optimized. The scalars xi control
      the linear combination and subsume the role of the attention coefficients soft-argmax η QK T in

150
      Eq. equation 1.
151
152   In the case of cross-attention, the query and value tensors are the two inputs, respectively, without
153   being transformed by the Wq and Wv weight matrices. By solving the optimization, we project the
154   values to the subspace of the queries if the l2 -norm is being used; this resembles what the original
155   cross-attention does. In self-attention, both query and value tensors are from the same input, so we
156
      require the ith element of xi to be zero to avoid a trivial solution. In this way, the optimization
      reveals the correlation structure between every input element and the rest via this reconstruction.
157
158   Next, we impose the requirement that only few nodes in the value set should be allowed to send
159   messages to a particular node qi , which helps mitigate overfitting. Using an l1 regularization, we

      Efficient Updates. Naı̈vely inverting V V T + ρI in Eq. equation 4a is computationally expensive,
      since it is quadratic in the number of tokens in the value set and can be very large. Yet, we notice that
178
      V is extremely narrow; in fact, we usually have Nv ≫ Dv /NH . This implies that the eigenvalue
179
      decomposition V T V = XDX T can be efficiently computed. Since the non-zero eigenvalues of
180
      V V T equal those of V T V , we only need the left singular vectors Y ≈ V XD −1/2 of V =

188   the diagonal of V T V where the entries of r are drawn from a uniform distribution U (0, σ) with σ
189   being small.
190
      We notice that after some training, the matrix of values tokens V degenerates, i.e., many of the
191
      eigenvalues of V T V become very small whereby the minimal eigenvalue gap shrinks. This ne-
192
      cessitates more iterations, thus slowing down forward and backward passes through the network
193   and increasing memory demand. This problem does not appear in the original formulation of the
194   ADMM due to the term +ρI in Eq equation 4a which has the same effect as a Tikhonov-regularizer,
195   i.e., it prevents rank-deficiency. Furthermore, the ADMM might not reach a reasonable solution if
196   the query qi is far from the range space span(V T V ).
197
      Random Token as Regularization. Instead of reverting to the slow and memory-intensive formula-
198
      tion in Eq. equation 4a, we propose to create a set of tokens whose entries are drawn from a normal
199   distribution with mean zero and unit covariance. These random tokens are then appended to V .
200   This has two consequences. First, it increases the eigenvalues of V T V thereby acting similarly as a
201   Tikhonov-regularizer. Secondly, it endows the left singular vectors Y with components which span
202   parts of the nullspace of span(V T V ), hence the distance between between query and range space
203   reduces. In other words, the ADMM can better regress components of qi in the kernel space of the
204   non-augmented value matrix. We noticed that this reduces overfitting. The idea of using additional
205   tokens is similar to the idea proposed by Darcet et al. (2024) except that the tokens used here are not
206   trainable. Furthermore, they serve a specific purpose interpretable in terms of linear algebra.
207   Lastly, to further increase sparsity but avoid the computationally expensive soft-argmax operator,
208   we map the estimated xi by
209
                                                  x′i = ν(α(xi )),                                          (6)
210
211   where α(·) be the function that raises each entry of xi to its fifth power, and ν be a normalization
212   function that, for instance, divides by the sum of the absolute values of xi . We observed that this
213   reduces overfitting.

230   5     E XPERIMENTS
231
232   We evaluate the impact of the proposed optimization-based attention by comparing with transformer
233   models that use both self- and cross-attention. The ADMM parameters are set to λ = 0.1, ρ = 10
234   and the number of iterations is taken to be 5 for all experiments.
235   We compare with the model (DETR) proposed by Carion et al. (2020) which comprises six trans-
236   former layers with self-attention, followed by another six layers with both self- and cross-attention.
237   Specifically, we compare against several baselines that omit weight matrices in their attention mech-
238   anism, since our proposed approach also does not require weight matrices. We follow the procedure
239   in (Carion et al., 2020) and perform experiments on the COCO 2017 detection dataset (Lin et al.,
240   2014). As backbone, we use a ResNet50. We perform all experiments using 4 GPUs and a batch
241
      size of 8. We use the original learning rate, weight decay and learning rate scheduling.

      7   R EPRODUCIBILITY
495
496
      We integrated our algorithm in the excellent repository provided by the authors of DETR (Carion
      et al., 2020). After reshaping the token tensors into the heads, Eq. 4 needs to be iterated. Afterwards,
497
      the attention scores need to be multiplied with the mask tensor to account for padded images. Lastly,
498
      the scores are multiplied with the value tensor. For our efficient variants, Eq. 5 can be used. If
499   random tokens are to be used, they need to be appended to the value tensor.

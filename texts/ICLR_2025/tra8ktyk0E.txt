010
011
                   In the kernel density estimation (KDE) problem, we are given a set X of data
012                points in Rd , a kernel function k : Rd × Rd → R,                              d
                                                                      Pand a query point q ∈ R , and
013                the objective is to quickly output an estimate of x∈X k(q, x). In this paper, we
014                consider KDE in the dynamic setting, and introduce a data structure that efficiently
015                maintains the estimates for a set of query points as data points are added to X over
016                time. Based on this, we design a dynamic data structure that maintains a sparse
017                approximation of the fully connected similarity graph on X, and develop a fast
018                dynamic spectral clustering algorithm. We further evaluate the effectiveness of our
019                algorithms on both synthetic and real-world datasets.

022   1       I NTRODUCTION
023
024   Given a set X = {x1 , . . . , xn } of data points, a set Q = {q1 , . . . , qm } of query points, and a kernel
025   function k : Rd × Rd → R⩾0 , the KDE problem is to quickly approximate µq ≜ xi ∈X k(q, xi )
                                                                                                  P
026   for every q ∈ Q. As a basic question in computer science, this problem has been actively studied
027   since the 1990s (Greengard & Strain, 1991) and has comprehensive applications in machine learning
028   and statistics (Bakshi et al., 2023; Genovese et al., 2014; Scholkopf & Smola, 2018; Schubert et al.,
029   2014).
030   In this paper we first study the KDE problem in the dynamic setting, where both the sets of data and
031   query points change over time. The objective is to dynamically update the KDE estimates of µq as
032   data points are added to X. Building on the framework for static KDE developed by Charikar et al.
033   (2020), our algorithm processes: (i) insertions and deletions of query points, and (ii) insertions of
034   data points in ε−2 · n0.25+o(1) time for the Gaussian kernel1 . In particular, our algorithm maintains
035   (1 ± ε)-approximate estimates of the kernel densities for every query point q ∈ Q throughout the
036   sequence of data point insertions; see Theorem 3.1 for the formal statement. To the best of our
037   knowledge, this represents the first dynamic algorithm for the KDE problem that efficiently maintains
038   query estimates under data point insertions.
039   Among its many applications, an efficient algorithm for the KDE problem on X = Q ⊂ Rd can be
040   used to speed up the construction of a similarity graph for X, one of the key components used in many
041   graph-based clustering algorithms (e.g., spectral clustering). These clustering algorithms usually
042   have superior performance over traditional geometric clustering techniques (e.g., k-means) (Ng et al.,
043   2001; von Luxburg, 2007), but in general lack a dynamic implementation. Our second contribution
044   addresses this challenge, and designs a dynamic algorithm that maintains a similarity graph for the
045   dataset X with expected amortised update time n0.25+o(1) when new data points are added; see
046
      Theorem 4.1 for the formal statement. Our algorithm guarantees that, when the set Xt of data points
      present at any time t has a cluster structure, our dynamically maintained graph will have the same
047
      cluster structure as the fully connected graph on Xt ; hence a downstream graph clustering algorithm
048
      will perform well.
049
050   Our designed algorithms are experimentally compared against several baseline algorithms on 8
051   datasets from different domains, and these experiments confirm the sub-linear update time proven in
052
      theory. These experiments further demonstrate that

060   Related Work. Efficient algorithms for the kernel density estimation problem in low dimensions
061   have been known for over two decades (Gray & Moore, 2003; Greengard & Strain, 1991; Yang
062   et al., 2003). For the regime of d = Ω(log n), there has been some recent progress to develop
063   sub-linear query time algorithms (Charikar et al., 2020; Charikar & Siminelakis, 2017; 2019) based
064   on locality-sensitive hashing (Andoni & Indyk, 2008; Datar et al., 2004) and importance sampling
065   using algorithms for computing approximate nearest neighbors (Backurs et al., 2018; Karppa et al.,
066   2022). There has also been recent work studying the approximation of kernel similarity graphs in the
067   static setting (Macgregor & Sun, 2023; Quanrud, 2021).
068   Dynamic kernel density estimation has been studied in some restricted settings. Huang et al. (2024)
069   give a dynamic variant of the fast Gauss transform (Greengard & Strain, 1991) for low-dimensional
070   data. Given an initial dataset X, Liang et al. (2022) give an efficient algorithm for maintaining a
071   KDE data structure in which some data point xi is replaced with a new point z. In the same setting,
072   Deng et al. (2022) present a dynamic data structure that maintains a spectral sparsifier of the kernel
073   similarity graph, for smooth kernels.
074   Our work also relates to a number of works on incremental spectral clustering (Dhanjal et al., 2014;
075   Laenen & Sun, 2024; Martin et al., 2018; Ning et al., 2007). However, these works either assume a
076   fixed vertex set (Dhanjal et al., 2014; Martin et al., 2018; Ning et al., 2007), or are limited to handling
077   only single edge updates (Laenen & Sun, 2024).

      2.3     A PPROXIMATE S IMILARITY G RAPHS
174
175   Constructing a similarity graph from a set of data points is the first step of most modern clustering
176   algorithms. For any set X = {x1 , . . . , xn } of data points in Rd and kernel function k : Rd × Rd →
177   R⩾0 , a similarity graph F = (V, E, w) from X can be constructed as follows: each xi ∈ X is a vertex
178   in F , and every pair of vertices xi and xj is connected by an edge with weight w(xi , xj ) = k(xi , xj ).
179   While this graph F has Θ(n2 ) edges by definition, we can construct in O(n) e    time a sparse graph G
180   with O(n) edges such that (i) every cluster in F has low conductance in G, and (ii) the eigenvalue
             e
181   gaps of the normalised Laplacian matrices of F and G are approximately the same (Macgregor
182   & Sun, 2023); these two conditions ensure that a typical clustering algorithm on F and G returns
183   approximately the same result. We call such a sparse graph G an approximate similarity graph, and
184   refer the reader to Section A in the appendix for its formal definition.

333   cluster structure, an approximate similarity graph G for X can be dynamically maintained, and the
334   conductance of every cluster in G can be theoretically analysed, due to the formal definition of an
335   approximate similarity graph (Definition A.3). This is another difference between our work and many
336   heuristic clustering algorithms that lack a theoretical guarantee on their performance.

338   4.1   A NALYSIS FOR THE I NITIALISATION
339
340   The main component of the initialisation step is our designed C ONSTRUCT G RAPH(X) procedure,
341   which builds a (complete) binary tree T for X, such that each leaf corresponds to a data point
342   xi ∈ X, and each internal node of T corresponds to a dynamic KDE data structure (described in
343   Section 3) on the descendant leaves/data points. At a high level, C ONSTRUCT G RAPH(X) applies
344   the S AMPLE(X, T , ℓ) procedure to recursively traverse T and sample L = O(log |X|) neighbours
345
      for every vertex based on the KDEs maintained by the internal nodes. It also stores the paths
      Pxi ,ℓ (xi ∈ X, ℓ ⩽ L), each of which records the internal nodes that xi visits when sampling its ℓth
346
      neighbour; with these stored paths the algorithm can adaptively resample the tree as new data points
347
      arrive. In addition, the query points whose KDE estimates are dynamically maintained at any internal
348   node T ′ are the data points xi whose sample path Pxi ,ℓ visit T ′ .
349
350   Our proof follows Macgregor & Sun (2023) at a high level. However, one crucial difference between
351
      our analysis and theirs is that the weight of every edge (xi , xj ) added by our algorithm is set to
      be k(xi , xj )/ŵ(i, j). Here, ŵ(i, j) depends on min{T .kde.µ̂xi , T .kde.µ̂xj }, where T .kde is the
352
      dynamic KDE data structure maintained at the root of T . In particular, every sampled edge (xi , xj )
353
      is added with this weight independent of the edge being sampled from xi or xj . This modification
354   allows for correct reweighting and resampling after a sequence of data point insertions.

      We design the U PDATE G RAPH(G, T , z) procedure to dynamically update our constructed graph
359   such that the updated graph is an approximate similarity graph for X ∪ z. At a high level,

361
362         1. for every arriving data point z, U PDATE G RAPH(G, T , z) creates a new leaf node for z, and
363            places it appropriately in T ensuring that the new tree is a complete binary tree;
364

365
               tree, and for every such internal node it adds z as a new data point in the corresponding

367

369
               Pxi ,ℓ whether the KDE estimate of any xi has changed due to the insertion of z. If
               so, Pxi ,ℓ is added to the set A of paths that need to be updated. For every Pxi ,ℓ ∈ A,
370
               U PDATE G RAPH(G, T , z) finds the highest internal node T ′ where the KDE estimate of
371

686   We first introduce the notion of cluster-preserving sparsifiers.
687   Definition A.2 (Cluster-preserving sparsifier, Sun & Zanetti (2019)). Let F = (V, E, w) be any graph

689   subgraph G = (V, E ′ ⊂ E, wG ) a cluster-preserving sparsifier of F if (i) ΦG (Si ) = O(k · ΦF (Si ))
690   for 1 ⩽ i ⩽ k, and (ii) λk+1 (LG ) = Ω(λk+1 (LF )).
691
      Notice that graph F = (V, E, w) has exactly k clusters if (i) F has k disjoint subsets S1 , . . . , Sk
692
      of low conductance, and (ii) any (k + 1)-way partition of F would include some A ⊂ V of high
693   conductance, which would be implied by a lower bound on λk+1 (LF ) due to (A.1). Together

698   connected similarity graph F on X, we call a sparse graph G with O(n)    e     edges an approximate
699   similarity graph on X if G is a cluster-preserving sparsifier of F .

      Definition A.3.

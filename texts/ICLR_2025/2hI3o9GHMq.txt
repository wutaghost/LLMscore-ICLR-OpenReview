009                                               A BSTRACT
010
011            We focus on the problem of learning object representations from solely associ-
012            ation data, that is observed associations between objects of two different types,
013            e.g. movies rated by users. We aim to obtain embeddings encoding object at-
014            tributes that were not part of the learning process, e.g. movie genres. It has been
015            shown that meaningful representations can be obtained by constraining the learn-
016            ing with manually curated object similarities. We propose Self-Matrix Factor-
017            ization (SMF), a method that learns object representations and object similarities
018
               from observed associations, with the latter constraining the learned representa-
               tions.AA In our extensive evaluation across three real-world datasets, we com-
019
               pared SMF with SLIM, HCCF and NMF obtaining better performance at predict-
020
               ing missing associations as measured by RMSE and precision at top-K. We also
021            show that SMF outperforms the competitors at encoding object attributes as mea-
022            sured by the embedding distances between objects divided into attribute-driven
023            groups.

026   1   I NTRODUCTION
027
028   This paper focuses on the problem of learning object representations from observed associations
029   between objects of two different types. We assume that no data is available other than the associa-
030   tions between the objects, and our aim is to obtain representations that reflect object attributes and
031   properties that, in general, are unknown.
032   The observed associations between two groups of n and m objects respectively can be represented
033   by a data matrix X ∈ Rn×m , where the association between object i and object j is stored in Xi,j .
034   We consider the case in which this value can be either a binary or a positive number, and a value of
035   zero indicates no known association between objects i and j.
036   Learning meaningful object representations has been shown to be relevant for several tasks including
037   recovering missing associations and clustering objects into meaningful groups, and several methods
038   have been proposed. Matrix Factorization (MF) methods, for instance, assume that the association
039   matrix is low rank, allowing X to be decomposed into lower-dimensional matrix factors, containing
040   the representations of the objects. Techniques like principal component analysis (PCA) (Hotelling,
041   1933), singular value decomposition (SVD)(Eckart & Young, 1936) and non-negative matrix fac-
042   torization (NMF) (Lee & Seung, 1999) have been successfully applied to tasks such as recovering
043   associations (Sarwar et al., 2002; Vozalis & Margaritis, 2007; Luo et al., 2014) and clustering objects
044
      (Yang & Seoighe, 2016; Yeung & Ruzzo, 2001).
045   Deep Learning is another widely used technique for learning object embeddings, particularly
046   through graph neural networks (GNNs). An association data matrix can be thought of as a bipar-
047   tite graph where the nodes represent the objects in the data and the links represent the associations
048   between them. GNNs leverage this network structure to extract insights from the encoded graphs.
049   While deep learning methods have been shown to be particularly effective at incorporating prior
050
      known object properties (Wu et al., 2022), a number of techniques have also been developed that
      can use solely association data such as, for example, LightGCN (He et al., 2020), SEAL (Zhang
051
      et al., 2021) and HCCF (Xia et al., 2022).
052
053   Recently, learning strategies that rely on manually curated similarities between objects have been
      proposed to constrain embedding learning somehow. For example, Neo-GNNs (Yun et al., 2021)


                                                        1
      Under review as a conference paper at ICLR 2025


054
      and BUDDY (Chamberlain et al., 2023) are GNN methods relying on higher-order interactions
055   in the graph. These interactions function as additional node similarity features and were used to
056   enhance link-prediction performance. However, selecting such similarities is not trivial.
057
058
      In this paper, we argue that object similarities can be learned directly from the data matrix. We rely
      on the fact thatAA the objects lie on multiple linear low-dimensional manifolds embedded in a high-
059
      dimensional space (Elhamifar & Vidal, 2013). Our matrix decomposition approach, Self-Matrix
060
      Factorization (SMF), learns distributed representations while constraining them using learned object
061   similarities. These similarities depend on the manifold structures implicit in the association matrix
062   X and are learned together with the embeddings. In other words, the object similarities, determined
063   by their positions in the manifolds, naturally constrain the object embeddings during the learning.
064   Our method is the first to explore this idea in a matrix factorization modelAA . In our extensive
065   evaluation across three distinct benchmark datasets, we show that SMF consistently outperforms the
066   competitors at encoding object attributes as measured by the embedding distances between objects
067   divided into attribute-driven groups. We also performed experiments to recover missing values on
068   the different association matrices and show that SMF obtains comparable or better predictions than
069
      its competitors.

      MF and GNN techniques encompass numerous methods for learning object representations from
077
      association data (Koren et al., 2021; Wu et al., 2022). MF techniques decompose the association
078
      matrix X into two or more matrix factors, where the object representations are encoded as rows
079   or columns of these matrix factors, mapping objects to a shared latent space of lower dimension-
080   ality (Aggarwal et al., 2016). Several methods for link prediction have been proposed, including
081   SVD (Koren et al., 2009), SVD++ (Koren, 2008) and probabilistic matrix factorization (Yang et al.,
082   2014). NMF (Lee & Seung, 1999) and its variations have been used across fields ranging from
083   medicine to engineering (Hamamoto et al., 2022; Sturluson et al., 2021). Graph-regularized NMF
084   (Cai et al., 2010), symmetric NMF (Luo et al., 2021) and robust NMF (Peng et al., 2021) have
085   been successfully used for object clustering and community detection. Additionally, NMF with l1,
086   l2 or elastic net regularization has been applied successfully across diverse applications, including
087
      precision medicine (Hamamoto et al., 2022), gene-expression analysis (Sweeney et al., 2023) and
      recommender systems (Rendle et al., 2020), showing state-of-the-art performance.AA
088
089   GNNs have gained popularity for their strong capabilities in graph representation learning. These
090   methods can effectively learn node representations that are well-suited for link prediction tasks
091   (Zhang et al., 2021). One advantage of GNNs is their ability to incorporate external object fea-
092   tures, which can significantly enhance prediction performance (Wu et al., 2022). Some approaches,
093
      like graph-regularized NMF (Cai et al., 2010), BUDDY (Chamberlain et al., 2023), and Neo-GNNs
      (Yun et al., 2021), leverage similarity measures to improve object clustering and link prediction per-
094
      formance. HCCF, a specialized GNN technique, learns hyper-edges between objects, enabling it to
095
      simultaneously learn embeddings and refine object similarities for improved representation learn-
096   ing.AA
097
098   Manually curated similarities have proven useful for embedding learning, stemming from the fact
099
      that these similarities can themselves be used in recommender systems (Aggarwal et al., 2016).
      Sparse Linear Models (SLIM) (Ning & Karypis, 2011) are state-of-the-art recommender systems
100
      (Ferrari Dacrema et al., 2019) that rely on learning object similarities rather than embeddings. SLIM
101
      learns coefficients such that each object can be represented as a linear combination of other objects.
102   This means that a new link between objects i and j is predicted only if objects similar to i were
103   originally linked with j. The coefficients used to reconstruct objects depend on the linear manifolds
104   present in the data matrix X. In this way, new links are recommended to an object based on the
105   links other objects belonging to the same linear manifold have. Although these similarities have
106   demonstrated predictive power, they have not yet been used to inform embedding learning. In this
107   work, we address this gap by proposing a framework that jointly learns object embeddings and object
      similarities, where the latter constrains the embedding space, resulting in richer representations.AA

119   While this model is not new, its noveltyAA resides in the learning of the embeddings in W to encode
120   linear manifold information implicitly contained in the association data itself. Relying on the above
121   mentioned assumption that objects lie on multiple linear low-dimensional manifolds embedded in
122   high-dimensional space (Elhamifar & Vidal, 2013), let us consider the situation depicted in Figure
123
      1a in which we have points in the 3-D space that are approximately localized onto 3 distinct linear
      manifolds. Rows of X are represented as squares, triangles and circles, with triangles and squares
124
      lying on one-dimensional sub-space (red and brown lines) and circles lying on a two-dimensional
125
      sub-space (green plane). Let us focus on the three blue points of which i and p lie on the plane and q
126   on the red line.AA . We assume that objects that belong to the same subspace, are more similar to each
127   other than objects that reside in different subspaces. We would like these similarities to constrain
128   the learning of the embeddings – that is, we would like the embedding for two objects that belong
129   to the same subspace, to be more similar to each other than the embeddings of objects that reside in
130   different subspaces. Thus, in the embedding space (2-dimensional, in Figure 1b), object i should be
131   closer to object p than to object q, mimicking their behavior in the high-dimensional space. Figure
132   1b demonstrates the expected behavior of SMF-learned object embeddings. Points that belong to the
133   same linear manifold in the high-dimensional space are projected into a lower-dimensional space,
134
      where they closely approximate one another.AA

247   To test SMF, we compared its performance against three different models, namely NMF with elastic-
248   net regularization (Pauca et al., 2006), Hypergraph Contrastive Collaborative Filtering (HCCF) (Xia
249
      et al., 2022) and Sparse Linear models (SLIM) (Ning & Karypis, 2011). We compared the perfor-
      mance of the models at predicting associations and the quality of the embeddings through clustering
250
      analysis. NMF with elastic-net regularization and HCCF were chosen as representative of matrix de-
251
      composition techniques and GNN-based methods, respectively. NMF has been shown to be effective
252   at recovering missing associations and encoding object attributes. HCCF is a state-of-the-art GNN
253   model for link prediction. Like SMF, HCCF also learns similarities between objects of the same
254   type, avoiding dependence on manually curated similarities. However, it enhances embeddings by
255   learning hyper-edges (connections involving more than two objects) that contribute to the embed-

257   & Karypis, 2011), a state-of-the-art approach for predicting missing associations (Ferrari Dacrema
258   et al., 2019) that has been shown to be competitive with deep learning models. SLIM does not
259   learn object embeddings and for this reason, we could not perform any clustering analysis. Size

      embeddings for the models are given in the last three columns of Table 1, other algorithm details

388   4.2      E MBEDDING EVALUATION THROUGH CLUSTER ANALYSIS
389
390   To demonstrate that the SMF-derived embeddings offer a more meaningful encoding of previously
391   unseen latent object attributes, we analyzed these low-dimensional representations and compared
392   them with the ones learned by NMF and HCCF. Our aim is to assess their capacity for encapsulating
393   inherent data characteristics, which were not part of the training process but may play crucial roles
394   in establishing connections between objects. Our embedding analysis was conducted on two levels:
395
      first, to verify whether SMF effectively clusters objects into meaningful groups within the low-
      dimensional space; and second, to assess whether SMF achieves superior class separation of objects
396
      compared to NMF and HCCF.
397
398   We took advantage of this information and grouped the embeddings in W into disjoint sets based
399   on the classes to which their corresponding objects belong. Subsequently, we calculate a similarity
400   matrix, Wsim ∈ Rn×n , containing the cosine similarity between all the embeddings. Finally, we
401
      employ a two-sample t-test to assess whether the similarities between objects within the same class
      (intra-similarities) differ significantly from the similarities between objects in different classes (inter-
402
      similarities). An illustration of this procedure is provided in Figure 3a. This process was repeated
403
      30 times across different runs of the models.
404
405   For the Movielens dataset, we organized users based on their gender, and movies by their respective
406   genres. In our genre analysis, we refined the dataset to include only movies with a single genre,
407
      enabling the classification of movies into disjoint categories. In the case of the Drug-SE dataset, we
      categorized drugs according to their various levels within the ATC hierarchy1 . The lower levels of
408
      the ATC hierarchy provide more specific terms for drug classification. For this study, we compared
409
      the similarity of drug embeddings across three levels of the hierarchy: anatomical, therapeutical,
410   and chemical. Finally, the clothing items of the ModCloth dataset can be separated into different
411   groups depending on which type of clothes they are (dresses, jeans, blazers, etc).
412

      tions of intra- and inter-similarities for the clothing types included in the ModCloth dataset, all NMF,
414

415

445   5   C ONCLUSION AND D ISCUSSION
446
447   Many machine learning approaches rely on learning distributed representations able to reflect rel-
448   evant object attributes. A common strategy to enrich these embeddings is by directly constraining
449   them to follow similarities extracted from side information (Aggarwal et al., 2016). Similarly, one
450   can directly rely on the similarities in the association matrix to guide the embedding learning to
451   better uncover patterns in the data. In this work, we introduced Self-Matrix Decomposition (SMF),
452   a constrained matrix decomposition approach that learns low-dimensional representations by cons-
453
      training them to rely on object similarities. These similarities depend on linear manifolds implicit in
      the association data and are learned with the representations.
454
455   SMF can decompose a low-rank matrix while preserving its inherent similarities in W W ′ , lever-
456   aging the relationships between rows of X that are revealed thanks to the second term in Equation
457   2. This Self-Expressive term learns a coefficient matrix that allows X to be reconstructed by itself,
458   similarly to a Self-Expressive model (Elhamifar & Vidal, 2013). The loss function in Equation 5 fa-
459
      cilitates the embeddings to learn better representations and capture the latent attributes, as they allow
      the embeddings to glean information from objects residing within the same subspace (as depicted in
460
      Figure 1).
461
462   We conducted experiments to assess whether a set of known object properties could be effectively
463   encoded within the object embeddings. Prior research has also delved into similar investigations,
464   revealing, for instance, that various NMF variants can learn embeddings encoding movie genres
465
      (Gomez-Uribe & Hunt, 2015) and drug ATC categories (Galeano et al., 2020). We conducted an
      in-depth analysis of the similarities among embeddings generated by NMF, HCCF and SMF across
466
      multiple runs and diverse groupings. This analysis aimed to ascertain whether objects belonging to
467
      the same group consistently clustered together in the low-dimensional space. SMF offers signifi-
468   cantly higher stability in learning well-separated embeddings compared to NMF and HCCF. This
469   is evident from the fact that in multiple runs, SMF achieves statistical significance approximately
470   99% of the time, whereas NMF and HCCF accomplish this feat in only 41% and 87% of the runs,
471   respectively. Furthermore, the experimental results demonstrate that SMF consistently achieves su-
472   perior class separation in all conducted experiments (depicted in Figure 3). Consequently, SMF can
473   be used to cluster objects in meaningful groups, and an analysis of these groups may help reveal
474   hidden object attributes.
475   The experimental results in the supervised setting indicate that SMF attains overall better RMSE
476   values compared to NMF and SLIM suggesting that the subspaces encoded in the matrix coefficient
477   of W W ′ contribute to the learning of more descriptive embeddings for recommendations. While
478   AUROC and AUPRC are commonly used to evaluate classification tasks by showing how true pos-
479   itive samples are ranked when predicting associations, practical effectiveness is better measured by
480   the precision at top-K metric. This metric indicates how close the true positives are to the top of the
481   ranking, which is more relevant for suggesting new associations. In this regard, SMF consistently
482
      outperforms NMF, SLIM and HCCF, reinforcing the notion that SMF embeddings adeptly capture
      the distinctive patterns typically learned by reconstructing the data matrix X directly with W and H
483
      and indirectly by constraining W .
484
485   Although SMF aims at learning from association data, it can be applied to any data represented
      as a nonnegative matrix. Furthermore, we expect it to be able to integrate extra information. For

      We derived the computational time complexity of SMF optimization algorithm from equations 5, 6
828   and 7 that are used to check the convergence criterion and update W and H, respectively. As each
829   equation has a fixed number of matrix operations, it suffices to derive the asymptotic complexity of
830   the most expensive one, which involves matrix multiplication. Specifically, the computation of the
831   product between T ◦(W W ′ ) and X runs in time O(n2 ·m), where n and m are the dimensions of the
832   data matrix X ∈ Rn×m . Other multiplications, such as W H, run in time O(n · k · m), where k is the
833   dimension of the embedding space. Assuming that k << n, m, we have that the time complexity of
834   each iteration of the SMF algorithm is O(n2 · m).AA
835   SMF iterations have the same time complexity as SEM’s (O(n2 · m)), while NMF iterations are
836   less expensive (O(n · k · m)). To illustrate the empirical running time of the three approaches, we
837   obtained the mean iteration time (in seconds) in two datasets with different sizes (Movielens and
838   ModCloth). This is shown in the first two columns of Table 8. As SMF’s multiplicative update rules
839   involve more matrix multiplications than those of NMF and SEM, it is expected to have a higher
840   execution time per iteration. AA
841   A fundamental variable for analyzing the overall computational time of the SMF algorithm is the
842   total number of iterations until convergence. As it is not possible to derive it analytically, here we
843   show only the empirical total number of iterations. The last two columns of Table 8 show the number
844   of iterations that were necessary to achieve convergence for SMF, SMF, and SEM.AA

      The scalability of the SMF algorithm is inherently tied to that of NMF, as both share similar com-
846   putational structures. Like NMF, SMF faces known limitations when applied to large-scale datasets,
847   where the computational demands can hinder efficiency (Gan et al., 2021). This reflects a broader
848   challenge in scaling matrix factorization techniques to accommodate increasingly larger datasets.AA

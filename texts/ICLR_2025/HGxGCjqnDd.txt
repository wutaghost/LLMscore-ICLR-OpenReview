009                                               A BSTRACT
010
011            Low-Rank Adaptation (LoRA) has emerged as a prominent technique for fine-
012            tuning large foundation models, facilitating a reduction in trainable parameters
013            through the utilization of low-rank matrices to represent weight changes A and
014            B (i.e., ∆W = BA). Although LoRA has demonstrated considerable success,
015            its expressiveness is inherently limited by the constrained capacity of its low-
016            rank structure. To ameliorate this limitation, we introduce Fourier-based Flexi-
017
               ble Rank Adaptation (FoRA), which harnesses the robust expressiveness of the
               Fourier basis to re-parameterize A and B from a sparse spectral subspace. Uti-
018
               lizing FoRA, adaptation matrices can overcome conventional rank limitations,
019
               achieving up to a 15x reduction in the parameter budget. We illustrate that
020            FoRA achieves an optimal balance of efficiency and performance across var-
021            ious tasks, including natural language understanding, mathematical reasoning,
022            commonsense reasoning, and image classification. Our codes are available at
023            https://anonymous.4open.science/r/FoRA-0E9C.

026   1   I NTRODUCTION
027
028   In recent years, Large Foundation Models (LFMs), have showcased exceptional generalization capa-
029
      bilities, greatly improving performance in a wide array of tasks across natural language processing
      (NLP) (Brown et al., 2020; Touvron et al., 2023a), computer vision (CV) (Radford et al., 2021; Kir-
030
      illov et al., 2023), and other fields (Azad et al., 2023). Typically, adapting these general models for
031
      specific downstream tasks requires full fine-tuning, which involves retraining all model parameters
032   and can pose significant challenges, particularly in resource-limited environments. To address this
033   issue, Parameter-efficient fine-tuning (PEFT) techniques (Mangrulkar et al., 2022), have been devel-
034   oped, offering more feasible alternatives. Among these, Low-Rank Adaptation (LoRA) (Hu et al.,
035   2021), which decomposes the weight changes into the product of two low-rank matrices A and B,
036   has stood out for its effectiveness and simplicity.
037
      Despite its success, LoRA’s reliance on low-rank structures can limit its expressive potential. The-
038   oretically, the expressive capacity of LoRA is constrained by the ranks of A and B (Zeng & Lee,
039   2023). Therefore, more complex downstream tasks inherently necessitate higher ranks (Hu et al.,
040   2023; Biderman et al., 2024; Gao et al., 2024). To elucidate the significance of rank configurations
041   in practical applications, we delve into their effect on LoRA’s performance across various tasks
042   and present the corresponding observations in Figure 1. As depicted, while different tasks exhibit
043   varying sensitivities to rank, most demonstrate improved performance as the rank increases, with
044   performance peaking at higher ranks (i.e., no less than 26 ). This pattern aligns with the behavior
045   of LoRA when applied to the LLaMA family, where high ranks yield clear improvements (Bider-
046
      man et al., 2024). However, adapting LoRA to higher ranks inevitably engenders larger trainable
      parameter sizes, thereby imposing considerable overhead. Hence, a question is naturally raised:
047
048                        How can we unleash the rank-bounded potential of LoRA
049                             while still residing in the low-parameter jail?
050

052   to preserve expressive information while necessitating fewer learnable parameters. Despite the suc-
053   cess of its predominant pruning techniques (Han et al., 2015b; Frankle & Carbin, 2018), determining
      which modules to prune often requires complex strategies (Zhang et al., 2022). In contrast, classical

097   2.1            PARAMETER -E FFICIENT F INE -T UNING
098
099   Fine-tuning large pre-trained language models is crucial for improving NLP tasks. However, up-
100   dating all model parameters is computationally intensive and storage-demanding for models like
101   GPT-3 (Brown et al., 2020) and LLaMA (Touvron et al., 2023a). Parameter-efficient fine-tuning
102   (PEFT) methods address these issues by updating fewer parameters or adding lightweight modules.
103   One prominent approach in PEFT is the use of adapters — small bottleneck layers inserted within
104   each layer of a pre-trained model (Houlsby et al., 2019; Pfeiffer et al., 2020; Karimi Mahabadi et al.,
105   2021; He et al., 2021). Houlsby et al. (2019) introduced adapters that enable task-specific adaptation
106   while keeping the original model weights fixed. Building upon this, Pfeiffer et al. (2020) proposed
107   a modular adapter framework that facilitates multi-task transfer. To further optimize parameter ef-
      ficiency, Karimi Mahabadi et al. (2021) reduced the number of parameters by employing parameter


                                                                            2
      Under review as a conference paper at ICLR 2025


108
      sharing and low-rank approximations within adapters. Another line of research involves prompt
109   tuning, which modifies the input embeddings to guide the model toward specific tasks (Lester et al.,
110   2021; Liu et al., 2021; Li & Liang, 2021; Chen et al., 2023a). Lester et al. (2021) optimized con-
111   tinuous prompt embeddings while keeping the language model’s parameters fixed, demonstrating
112   the effectiveness of prompt tuning for task adaptation. Similarly, Prefix-Tuning (Li & Liang, 2021)
113   prepends trainable vectors to the input of each transformer layer without altering the model archi-
114   tecture, effectively steering the model toward desired behaviors with minimal parameter updates.
115   While these methods exhibit high efficiency and preserve the originality of the pre-trained model,
116   they inevitably introduce higher inference costs due to additional modules or modifications required
117   during deployment. In contrast, LoRA (Hu et al., 2021) and its variants (Zhang et al., 2023a; Bałazy
118   et al., 2024; Li et al., 2024; Liu et al., 2024; Nikdan et al., 2024; Gao et al., 2024) inject trainable
119   low-rank matrix decomposition into transformer layers. This approach not only reduces the number
120   of trainable parameters but also allows for merging these decompositions with the original model
121   weights, thereby avoiding increased inference burdens. However, the expressiveness of low-rank
122   adaptation methods like LoRA is often bounded by the chosen rank (Zeng & Lee, 2023). To ad-
123
      dress this limitation, Kopiczko et al. (2023) and Jiang et al. (2024) explored high-rank adaptations
      through projection matrices, aiming to enhance expressive capacity without significantly increasing
124
      parameter counts. Despite these advances, our empirical experiments indicate that while LoRA’s
125
      performance may peak at certain high-rank configurations, increasing the rank beyond this point
126   does not necessarily lead to better results.
127
128
      2.2   S PARSE L EARNING
129
130   Sparse neural networks exploit the fact that many weights in over-parameterized models can be
131   pruned with minimal impact on performance (Han et al., 2015b; Lee et al., 2018; Frankle & Carbin,
132   2018; Wang et al., 2020; Liu et al., 2022; Frantar & Alistarh, 2023). Techniques such as magnitude
133   pruning (Han et al., 2015a) remove weights with small magnitudes, effectively reducing model size.
134   Dynamic sparsity methods (Mocanu et al., 2018; Zhang et al., 2022; Chen et al., 2023b) adjust the
135
      sparsity patterns during training, allowing the network to discover efficient architectures on the fly.
136   Another innovative approach is learning in transformed domains like the sparse Fourier space. By
137   representing weight matrices in the frequency domain using Fourier transforms, neural networks can
138   exploit the sparsity inherent in the frequency representation of the data (Yang & Xie, 2016; Chen
139   et al., 2016). This allows for efficient compression by retaining only the significant frequency com-
140   ponents and discarding the less important ones. Learning in the sparse Fourier space facilitates the
141
      development of compact models that effectively capture essential patterns with fewer parameters.

146
147   Low-Rank Adaptation (LoRA) LoRA (Hu et al., 2021) proposes to use the product of two low-
148
      rank matrices to update the pre-trained weights incrementally. Let W′ ∈ Rd1 ×d2 deotes the fine-
      tuned weight, W0 ∈ Rd1 ×d2 the pre-trained weight, and ∆W ∈ Rd1 ×d2 the change in weight.
149
      LoRA models this change ∆W through a low-rank decomposition:

152   where W0 is kept unchanged during fine-tuing. The matrices A ∈ R                and B ∈ R        represents
153   the learnable low-rank matrices with the rank r ≪ {d1 , d2 }. Typically, A adopts Kaiming uniform
154   initialization (He et al., 2015) while B is initialized to zero at the start of the training process.
155   In the following parts, we present Fourier-based Flexible Rank Adaptation (FoRA), which re-
156   parameterizes adaptation matrices of LoRA by applying inverse Fast Fourier Transform on sparse
157   spectral coefficients. The overall framework is presented in Figure 2.

206   3.3        D ISCUSSION
207
208   Initialization strategies. Matrix initialization with consistent variance (Glorot & Bengio, 2010)
209   is crucial for maintaining numerical stability and accelerating convergence. However, unlike LoRA,
210   directly initializing the spectral space in FoRA can lead to suboptimal variance in spatial space due
211   to the involvement of the Fourier transform. To facilitate efficient training, for matrix Ã, we first
212   employ Xavier (Glorot & Bengio, 2010) or Kaiming initialization (He et al., 2015) to its spectral
213   coefficients s and a spatial auxiliary matrix A′ ∈ Rr×d2 . Next, we scale s by Var(A′ )/Var(Ã) to
214   approximate consistent variance. In contrast, matrix B̃ is initialized to zeros following the standard
215   practice of LoRA (Hu et al., 2021). We employ Kaiming initialization by default unless specially
      stated.

      Comparison to LoRA’s variants. Recent parameter-efficient variants of LoRA (Kopiczko et al.,
217   2023; Renduchintala et al., 2023; Li et al., 2024) have demonstrated competitive performance by
218   adapting at higher ranks through the use of simple linear projections. However, their strategies for
219   sparse learning, which essentially involve a collection of learnable scaling transformations, suffer
220   from limited expressiveness. To remedy this issue, FoRA leverages the more efficient and expressive
221   Fourier transform for matrix re-parameterization, striking a balance between performance and effi-
222   ciency. Compared with them, FoRA consistently provides enhanced representational expressiveness
223   while allowing flexible rank adaptation with fixed cost. Further details of the empirical analysis are
224   provided in Section 4.5.

227
228   In this section, we present a series of experiments to demonstrate the effectiveness of FoRA across
229   diverse tasks, including language and image domains. We begin by evaluating FoRA through fine-

      2021) where only bias vectors are fine-tuned. The second group is adapter-tuning, covering

240   4.1   GLUE B ENCHMARK
241
242   We evaluate FoRA on the General Language Understanding Evaluation (GLUE) benchmark (Wang,
243   2018), a sequence classification benchmark for natural language understanding (NLU) which covers

246   Our experimental setup closely follows (Hu et al., 2021), involving fine-tuning only the query

248   method, we randomly sample n = {250, 500} trainable spectral coefficients per low-rank matrix,
249   which we denote as FoRA† and FoRA, respectively. We adopt the baseline hyperparameters from
250   their original papers. For our approaches, we apply random search (Bergstra et al., 2013) to opti-
251   mize the learning rates and matrix rank. For comprehensiveness, we report the median performance
252
      across 5 random seed trials, selecting the best epoch for each run. Additionally, we report the num-
      ber of trainable parameters in the fine-tuned layers, excluding the classification head, as suggested
253
      by (Hu et al., 2021; Kopiczko et al., 2023). Further specifics are provided in Table 6 in the Appendix.
254
255
      Results. As highlighted in Table 1, FoRA generally delivers better or on-par performance com-
256   pared with baseline methods, while adapting at higher ranks with extremely lower budget. Notably,

      5          C ONCLUSION
477
478
      In this work, we aim to unlock the rank-bounded potential of LoRA while minimizing and control-
479
      ling parameter overhead. We present FoRA, a fine-tuning method that re-parameterizes adaptation
480   matrices from spectral subspace and is compatible with LoRA and its variants. With Fourier ba-
481   sis, FoRA allows for the representation of informative adaptation matrices from lower to potentially
482   unbounded ranks at fixed parameter cost. Empirically, FoRA consistently matches or surpasses
483   LoRA’s performance across various fine-tuning tasks and backbone models, requiring up to 15x
484   fewer trainable parameters. Moreover, a comprehensive analysis further substantiates FoRA as a
485   parameter-efficient alternative to LoRA. Our work demonstrates the potential for efficiently repli-
      cating LoRA’s capabilities, with opportunities for further exploration in future research.

      A.3     PARAMETER C OUNT OF S PARSE L EARNING S TRATEGIES
865
866   As the rank increases, the number of learnable parameters in LoRA grows linearly, leading to a
867   significant parameter overhead. While VeRA exhibits a minimal increase in parameters, its strong
868   dependence on the size of its adaptation matrices limits its flexibility in adapting to more complex
869   tasks. In contrast, both FoRA and random masking maintain a fixed number of learnable parameters
870   across different ranks, providing greater flexibility by allowing parameter adjustments based on task
871   complexity.

886   B     I MPLEMETATION
887
888   Algorithm 1 presents the PyTorch implementation of FoRA. Our approach allows for a straight-
889
      forward plug-in, with the only modification needed being the replacement of the PyTorch linear
      module with the FoRA linear module. Additionally, we create a cached empty spectral matrix that
890
      matches the size of the adaptation matrix. This caching strategy not only accelerates GPU compu-
891
      tations but also minimizes GPU overhead, as only the sparse spectral coefficients require backward
892   propagation.
893
894
895   Algorithm 1 FoRA PyTorch Implementation
896   class PEFT(nn.Module):
897       def __init__(self, ...):
              if LoRA:
898               A, B = torch.nn.Linear(d1, r), torch.nn.Linear(r, d2)
              elif FoRA:
899               A, B = FoRALinear(n, d1, r), FoRALinear(n, r, d2)
900
      class FoRALinear(nn.Module):
901       def __init__(self, n, din, dout):
              self.n = n # number of spectral coefficients
902           self.din = din # input dimension
903           self.dout = dout # output dimension
              # spectral location
904           self.L = torch.randperm(din * dout)[:n]
              self.L = torch.stack([self.L // din, self.L % dout], dim=0)
905           # spectral coefficients
906           self.s = nn.Parameter(torch.zeros(n))
              # cached spectral matrix
907           self.cached_F = torch.zeros(self.din, self.dout)
908         def weight(self):
909             # spectral assignment
                F = self.cached_F.clone()
910             F[self.L[0], self.L[1]] = self.s
                # spatial conversion
911             W = torch.fft.ifft2(F).real()
912             return W

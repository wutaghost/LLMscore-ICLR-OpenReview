009                                               A BSTRACT
010
011
               Follow-the-Regularized-Leader (FTRL) algorithms are a popular class of learning
012
               algorithms for online linear optimization (OLO) that guarantee sub-linear regret,
013
               but the choice of regularizer can significantly impact dimension-dependent fac-
014            tors in the regret bound. We present an algorithm that takes as input convex and
015            symmetric action sets and loss sets for a specific OLO instance, and outputs a reg-
016            ularizer such that running FTRL with this regularizer guarantees regret within a
017            universal constant factor of the best possible regret bound. In particular, for any
018            choice of (convex, symmetric) action set and loss set we prove that there exists an
019            instantiation of FTRL which achieves regret within a constant factor of the best
020            possible learning algorithm, strengthening the universality result of Srebro et al.,
021            2011.
022            Our algorithm requires preprocessing time and space exponential in the dimension
023            d of the OLO instance, but can be run efficiently online assuming a membership
024            and linear optimization oracle for the action and loss sets, respectively (and is
025            fully polynomial time for the case of constant dimension d). We complement
026
               this with a lower bound showing that even deciding whether a given regularizer is
               α-strongly-convex with respect to a given norm is NP-hard.

030   1   I NTRODUCTION
031
032
      Online Linear Optimization (OLO) is one of the most fundamental problems in the theory of online
033
      learning. Here, a learner must repeatedly (for T rounds) select an action xt from some bounded
034   convex action set X . Simultaneously, an adversary selects a linear loss function ℓt from a bounded
035   convex loss set L, and the learner receives loss ⟨xt , ℓt ⟩. The learner would like to minimize their
036   total loss, and more specifically minimize their regret: the gap between their total loss and the loss
037   of the best fixed action x∗ ∈ X in hindsight.
038
      By choosing the action set X and loss set L appropriately, online linear optimization captures
039
      many other learning-theoretic problems of interest. For example, when X = ∆d (distributions
040   over {1, 2, . . . , d}) and L = [0, 1]d , this captures the classical problem of learning with experts.
041   Similarly, when the loss set L is the ℓ2 unit ball, this variant of OLO is the core subproblem involved
042   in online convex optimization (specifically, of a Lipschitz function with domain X ). Even more gen-
043   erally, the works of Gordon et al. (2008) and Abernethy et al. (2011) demonstrate how to reduce the
044   problems of linear ϕ-regret minimization (including swap regret minimization) and Blackwell ap-
045   proachability to different instances of OLO. These problems in turn have many applications extend-
046   ing past learning theory, from designing algorithms for computing correlated equilibria in repeated
047   games, to producing calibrated forecasts, to constructing classifiers satisfying a variety of fairness
048
      criteria (Farina et al., 2021; Okoroafor et al., 2024; Chzhen et al., 2021).
049   For this reason, it is an extremely relevant problem to understand the best possible regret bounds
050   achievable for different instances of OLO. Here, the state-of-the-art leaves something to be desired.
051   It is well-known that learning
                                 √ algorithms such as Follow-The-Regularized-Leader (FTRL) achieve
052   regret that scales with O( T ), and that this dependence on T is tight. However, the dependence
053   of the optimal regret on the sets X and L (e.g., how the constant factor in the above regret bound
      depends on the dimension d of these sets) is in general poorly understood.

      Moreover, FTRL is not a single algorithm, but a family of algorithms parametrized by a convex
055   function f : X → R called the regularizer. The actual regret bounds achieved by FTRL can
056   vary greatly depending how the choice of regularizer interacts with the geometry  of X and L. For
                                                                               √

                                               √
059   results in an algorithm with a tight O( T log d) regret bound, with an exponential improvement in
060   dimension over the quadratic choice of regularizer. On the other hand, there exist other instances
061   (choices of X and L) where the quadratic regularizer is optimal. Understanding what the optimal
062   choice of regularizer is for a given instance of OLO is a major open problem.

087   The main technical ingredient in this algorithm is a new method for optimizing over the set of convex
088   functions that are α-strongly convex with respect to a given norm. This is important for the above
089   problem because one can show   √ that for any regularizer f , the regret of running FTRL with that
090   regularizer is bounded by O( DαT ) if the range of f over X (the maximum value of f minus the
091   minimum value of f ) is at most D and if f is α-strongly-convex with respect to the norm induced
092
      by the dual set of the loss set L. We can show that this regret-bound is constant-factor-optimal for
      the near-optimal variant of FTRL in Theorem 7, and hence it suffices to try to minimize Dα over all
093
      convex functions f .
094
095   To do this, we first show that we can approximate any smooth convex function f as a maximum of
096   several “quasi-quadratic” functions: quadratic functions gx0 centered at some point x0 with a small
097   cubic term which guarantee that that the contribution of gx0 to the Hessian of f decays far from
098
      x0 . Note that these are not just approximations of the values of f , but also also the gradients and
      Hessians of f ; in particular, if the original function was α-strongly-convex with respect to some
099
      norm, our approximation will be similarly strongly-convex.
100
101   By restricting our quasi-quadratic functions to be centered at points belonging to a (large but) finite
102   discretization of X , we demonstrate how to optimize over this set of approximations by solving a
103   large convex program with variables for the values, gradients, and Hessians of the quasi-quadratic
104
      functions at each point in the discretization. Solving this convex program involves implementing a
      separation oracle to verify whether a specific approximation is α-strongly-convex with respect to an
105
      arbitrary norm.

114   Applications of Online Linear Optimization. The problem of Online Linear Optimization (and
115   its generalization, Online Convex Optimization) are central problems in the field of online learning
116   – we refer the reader to Hazan et al. (2016) for a general-purpose introduction. Traditionally OLO
117   is studied in the case where the action sets and loss sets are unit balls in a standard norm (e.g. the
118   ℓ1 , ℓ2 , or ℓ∞ norms). However, there are many motivating settings where we wish to minimize
119
      regret with less standard sets. Several authors (Takimoto & Warmuth, 2003; Kalai & Vempala,
      2005; Koolen et al., 2010; Audibert et al., 2014) study variants of OLO where the action space has
120
      some combinatorial structure – for example, X could be the spanning tree polytope, or the polytope
121
      formed by all s-t paths in a graph. Minimizing external regret in extensive form games – one
122   standard method for computing coarse correlated equilibria Farina et al. (2020) – involves solving
123   an instance of OLO where X is the sequence form polytope. Finally, as mentioned earlier, the work
124   of Abernethy et al. (2011) and Gordon et al. (2008) allows us to translate any instance of Blackwell
125   approachability or ϕ-regret minimziation to a (usually non-standard) instance of OLO.
126
127   Follow-The-Regularized-Leader and Mirror Descent. The Follow-The-Regularized-Leader al-
128   gorithm can be thought of as a form of mirror descent, a family of first-order optimization algorithms
129   that generalize gradient descent by using arbitrary distance-generating functions. Originally, mir-
130   ror descent was proposed by Nemirovski & Yudin (1978) as an offline optimization algorithm with
131   ℓp norm constraints and ℓq Lipschitz assumptions, and was shown to have minimax optimal query
132   complexity. Sridharan & Tewari (2010) studied the optimality of mirror descent for online linear
133   optimization when the action and loss vectors are in the unit ball of two Banach spaces dual to each
134   other, proving the existence of a regularizer for mirror descent that almost achieves the minimax rate
135
      under an adaptive adversary. Later, Srebro et al. (2011) extended this approach to cases where the
      action and loss vectors come from independent convex balls in primal and dual Banach spaces. The
136
      existence of such strongly convex regularizers is also linked to the Burkholder method introduced
137
      by Foster et al. (2018) for more general online learning problems. In particular, the authors pro-
138   pose that given an online learning instance and a target regret bound, the existence of a Burkholder
139   function for that instance guarantees the existence of a prediction strategy that achieves the desired
140   regret. Notably, taking the dual of this Burkholder function for the online linear optimization (OLO)
141   problem results in a strongly convex regularizer that can be used effectively with FTRL Foster et al.
142   (2018).
143
      Many modern learning algorithms are actually variants of mirror descent / FTRL (Block, 1962;
144   Zinkevich, 2003; Kivinen & Warmuth, 1997; Littlestone, 1988; Kakade et al., 2010; Warmuth &
145   Kuzmin, 2007). Recently, Jin & Sidford (2020) used a variant of mirror descent to solve infinite-
146   horizon MDPs, achieving linear runtime in the number of samples. Aubin-Frankowski et al. (2022)
147   extended mirror descent to optimize convex functionals on an infinitesimal space, demonstrating
148   that the primal iterations of Sinkhorn’s algorithm for entropic optimal transport in a continuous
149   domain are an instance of mirror descent. Wibisono et al. (2022) studied alternating
                                                                                            mirror descent
150   for two-player bilinear zero-sum games, proving a regret bound of O T 1/3 . Mirror descent has
151   also been used in the context of stochastic optimization Nemirovski et al. (2009). Authors in Duchi
152   et al. (2010) study mirror descent for composite loss functions under both stochastic and online
153
      settings. Lei & Tang (2018) relaxed the subgradient boundedness condition from Duchi et al. (2010)
      and extended their analysis to examine the generalization performance of multi-pass SGD in non-
154
      parametric settings. Dani et al. (2008); Cesa-Bianchi & Lugosi (2011); Bubeck et al. (2012) applied
155
      mirror descent to address the problem of online linear optimization with bandit feedback. Allen-Zhu
156   & Orecchia (2014) introduced a novel interpretation of mirror descent as optimizing a dual-based
157   lower bound for the objective. Building on this perspective, they proposed a coupling between
158   mirror descent and gradient descent that achieves an accelerated convergence rate. (Yuan et al.,
159   2020; Shahrampour & Jadbabaie, 2017) applied mirror descent in distributed settings. Lobos et al.
160   (2021) utilized mirror descent for a constrained online revenue maximization problem with unknown
161   parameters. Authors in (Bansal & Coester, 2021; Lu et al., 2020; Balseiro et al., 2023) employ
      mirror descent for online resource allocation problems. Mirror descent has also been instrumental

      Here, we state the existence of an ideal regularizer such that running FTRL with this regularizer
      achieves the optimal rate up to a constant. This result is adapted from Srebro et al. (2011), except
707
      that they prove the same regularizer results in a regret bound which is off by a logarithmic factor
708
      of log(T ); this log factor is indeed not desirable for our purpose as we are interested in long time
709   horizon regimes when T can potentially be exponentially large in dimension. Our contribution here
710   is that we improve the result of Srebro et al. (2011) for p = 2 case and shave off this log factor. We
711   further show a type of continuity condition for this ideal regularizer that we use for our smoothing
712   arguments in Section B.

1061
1062   Definition 5 (Membership Oracle). For convex set D ∈ Rd , a membership oracle receives a vector
1063
       y ∈ Rd and real number δ > 0 and with probability 1 − δ asserts y ∈ B(D, δ), or it asserts y ∈      /
       B(D, −δ). We denote the computational cost of a query to our membership oracle by MEMX (δ).
1064
1065   Definition 6 (Set neighborhoods). For a subset D ⊆ Rd , let B(D, δ) be the set of points that are
1066
       within distance δ of D, and B(D, −δ) be the set of points that where a ball of radius δ around them
       is completely included in D.
1067
1068   Definition 7 (Separation Oracle). For a convex set L ⊆ Rd , a separation oracle receives a vector
1069   y ∈ Rd and real number δ > 0 and either asserts y ∈ B(L, δ), or it returns a unit vector c ∈ Rd
1070
       such that c⊤ y ≤ c⊤ x + δ for all x ∈ B(L, −δ). We denote the computation time of separation
       oracle by SEPL (δ).
1071
1072   Definition 8 (Linear Optimization Oracle). For a convex set L ⊂ Rd , a linear optimization oracle
1073
       receives a unit vector c ∈ Rd and real number δlin and returns a point y ∈ C such that ∀x ∈ C,
       c⊤ y ≤ c⊤ x + δlin . We denote the computational cost of calling the linear optimization oracle by
1074
       L IN OL (δlin ).

011                                               A BSTRACT
012
013            This paper develops the regularized conditional optimal transport for feature learn-
014            ing in an embedding space. Instead of using joint distributions of data, we intro-
015            duce conditional distributions to some reference conditional distributions in terms
016
               of the Kullback-Leibler (KL) divergence. Using conditional distributions provides
               the flexibility in controlling the transferring range of given data points. When the
017
               alternating optimization technique is employed to solve our model, it is interest-
018
               ing to find that conditional and marginal distributions have closed-form solutions.
019            Moreover, the use of conditional distributions facilitates the derivation of the gen-
020            eralization bound of our model via the Rademacher complexity, which character-
021            izes its convergence speed in terms of the number of samples. By optimizing the
022            anchors (centroids) defined in the model, we also employ optimal transport and
023            autoencoders to explore an embedding space of samples in the clustering prob-
024            lem. In the experimental part, we demonstrate that the proposed model achieves
025            promising performance on some learning tasks. Moreover, we construct a condi-
026            tional Wasserstein classifier to classify set-valued objects.

029   1   I NTRODUCTION
030
031   Achieving effective features of data (Boroujeni et al., 2018; Wang et al., 2019; Qian et al., 2019)
032   is a fundamental task in data analysis, and feature learning has been explored in some fields such
033
      as machine learning and computer vision. Feature learning aims at exploring a linear or nonlinear
      transformation to map the original features into an embedding space by optimizing the defined
034
      objective function. In the latent representation space, data can be explored, thereby providing some
035
      benefits from various learning tasks (Su & Hua, 2019).
036
037   Earlier feature learning algorithms focus on how to develop effective handcrafted extractors for vi-
038   sualizing high-dimensional data and reducing the effect of the curse of dimensionality. Marginal
039
      Fisher analysis (Yan et al., 2007) adopts a graph embedding framework to provide an intrinsic graph
      with intra-class compactness and a penalty graph with inter-class separability. Max-min distance
040
      analysis (Bian & Tao, 2011) achieves the low-dimensional data by maximizing the minimum pair-
041
      wise distance. A robust linear discriminant analysis method based on the L2,1 norm (Nie et al., 2021)
042   is developed to obtain robust projection features, and an effective iterative optimization algorithm
043   is derived to solve a general ratio minimization problem. In Flamary et al. (2018), Wasserstein dis-
044   criminant analysis from optimal transport (Li et al., 2021; Serrurier et al., 2021) is implemented by
045   employing the regularized Wasserstein distance to capture the global and local interactions between
046   classes.
047
      Kernel-based methods that capture the nonlinear features of data have been developed to search for
048   an effective feature space by selecting proper kernel functions. Unlike classical dimensionality re-
049   duction methods, the embedding space of data may be an infinite-dimensional feature space since
050   data may be well separated in high-dimensional spaces. Kernel principal component analysis (PCA)
051   and kernel linear discriminant analysis(LDA) are two effective methods for achieving effective fea-
052   tures of data. To address the outliers of data, L1 norm kernel LDA (Zheng et al., 2014) is developed
053   to achieve the nonlinear discriminant features of data. In unsupervised learning, finding effective
      features contributes to the improvements in the performance of clustering. The classical k-means

469   3.4    C LASSIFICATION OF SET- VALUED OBJECTS
470
471   Here we modify our model to make it capable of handling set-valued classification problems. For set-
472   valued classification problems, each object contains many examples. Unlike previous experiments,
473   we assume that the set {x1 , · · · , xn } is a set-valued object containing n examples in the validation
474   set or test set. For the data point xi , we can obtain its k neighbours y1|i , · · · , yk|i and these k
475   neighbours are from the training set. Since we know the labels of y1|i , · · · , yk|i in the training set,
476   we assign the label of xi to the label of yj|i with the largest pj|i (j = 1, · · · , k). Thus, we obtain
477
      the label of each example in a set-valued object. Finally, the majority voting strategy is employed to
      achieve the label of the set-valued object. We refer to our model as the deep conditional Wasserstein
478
      classifier (DCWC). Our model will degenerate into the deep nearest-neighbor classifier if each object
479
      only contains an example and the parameter λ1 approaches the positive infinity. Here we need to use
480   the validation set to learn the embedding space of data and hyperparameters. In the test stage, we
481   fix the parameters of autoencoders and optimize pj|i . We test DCWC on two medical image sets in
482   binary classification problems (Yang et al., 2021). We use 780 images from the breast image set and
483   4708 images from the pneumonia image set. To evaluate the performance of DCWC, we compare
484   it with several set-valued data classification methods such as the second-order cone programming
485   (SOCP) approach (Shivaswamy et al., 2006), the sparse approximated nearest point (SANP) method
      (Hu et al., 2011), regularized collaborative representation classification (RCRC) (Zhu et al., 2014),


                                                                 9
      Under review as a conference paper at ICLR 2025


486
      support measure machines(SMMs) (Muandet et al., 2012), and support function machines (SFMs)
487   (Chen et al., 2017). Figure 2 shows experimental results on two medical image sets.
488
489   As can be seen from Figure 2, SFMs are not superior to DCWC since SFMs generally give sparse
490
      support vectors. It is found that DCWC yields the best performance on these data sets since DCWC
      explores the weight of each example in the set-valued objects. Among these methods, SFMs are
491
      sampling-based methods. SANP, RCRC and SMMs explore all possible representations of images.
492
      If the representations of images contain distorted features, these distorted features will affect the
493   performance of classifiers. Our DCWC makes use of the conditional optimal transport and recon-
494   struction errors of data to achieve effective features. The experimental results indicate that it is
495   reasonable to employ the optimal transport theory to classify set-valued data.

511   4   R ELATED WORK
512
513   Many feature learning methods based on the deep architectures of neural networks have been devel-
514   oped. Multi-layer learning models (Yuan et al., 2015) have been proposed to deal with the scene
515   recognition problem, and they are available in an unsupervised way. The deep semi-nonnegative
516   matrix factorization (Trigeorgis et al., 2014) can find the latent representation of data in a low-
517
      dimensional space, and the new description can improve the clustering performance of data. Deep
      Fisher discriminant analysis (Diaz Vico & Dorronsoro, 2020) takes advantage of deep neural net-
518
      works to capture the nonlinear features of data. To deal with sequence data, deep order-preserving
519
      Wasserstein discriminant analysis (Su et al., 2022) achieves a nonlinear transformation by maximiz-
520   ing the inter-class distance and minimizing the intra-class distance. The Wasserstein autoencoder
521   (Tolstikhin et al., 2018) was proposed to achieve a generative model of data distributions. However,
522   these feature learning methods do not explore their generalization bounds.
523
      Kernel k-means clustering methods can deal with the nonlinear structure of data in unsupervised
524
      learning. For bounded random vectors, the expected excess clustering risk was studied in the work
525
      (Maurer & Pontil, 2010). An upper generalization bound of the kernel k-means method in a reduced
526   space (Yin et al., 2022) is derived in terms of the Rademacher complexity. The deep clustering model
527   via the t distribution (DEC) (Xie et al., 2016) has been proposed. The improved DEC (IDEC) (Guo
528   et al., 2017) used autoencoders to enhance the performance of DEC. However, the generalization
529   bounds of DEC and IDEC are not explored.
530
531
      5    C ONCLUSIONS AND FURTHER WORK
532
533
      In this paper we have introduced a feature learning framework relying on optimal transport and
534   autoencoders. The use of the conditional probability in the proposed model is to make each data
535   point adapt to its neighborhood, and this may well be suitable for the characteristics of data. The
536   experimental results on real data sets demonstrate the feasibility of the proposed model on some
537   classification and clustering tasks. Since the performance of the proposed model is affected by
538   autoencoders, how to select proper autoendcoders for data sets is worth exploring. In the future, we
539   will focus on the problem of how to employ advanced autoencoders to improve our model to deal
      with complex data sets in the real world.

       All experiments are conducted on a PC with an Intel Core i7- CPU and a RTX 3080 GPU. The
1048
       structure of encoders we use in this paper is a fully-connected network with the form of [m, 500,
1049
       500, 2000, d] and the decoder is a mirror of the encoder, where m is the dimension of input data
1050   and d is the dimension of the latent space. The popular ReLU functions are employed in each layer.
1051   We employ Adam (Kingma & Ba, 2015) as the backpropagation optimizer. In the experiments, ρ
1052   takes the L2 norm, r̄ = 2, and λ2 = 1000. We let λ4 = 0 due to the use of the same encoders. The
1053   parameters λ1 and λ3 are selected from the set {10i , i = −3, −2, · · · , 2, 3}. In the classification
1054   experiments, we need to determine k in (6) in our paper. Note that x1 , · · · , xn consist of the training
1055   set and yj|i (j = 1, · · · , k) are taken from the samples that have the same label as xi . We think
1056   that the samples in the same class are neighbors. Thus, k will be determined by the number of
1057   samples in each class. As a result, k will vary since the number of samples in each class is different
1058
       in the training set. In the clustering experiments, k in the model of (15) is set to be the number of
       clusters. We find that good performance can be obtained by this setting since the encoder has strong
1059
       representations of features.
1060
1061   The outer loop is 10 iterations and the inner loop for autoencoders is run with an Adam optimizer
1062   for 100 epochs, with an initial learning rate of 0.001. In the data sets except for two large-scale data
1063   sets , all data sets are handled with a full-batch mode. For the large-scale data sets, the batch size is
1064
       2000.

1066   (16/10/7494), Satimage (36/6/4435), Iris(4/3/150), and WaveForm (21/3/5000). In addition, we also
1067   explore four face image data sets and an object data set. The ORL face database contains 40 distinct

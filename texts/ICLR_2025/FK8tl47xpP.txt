009                                               A BSTRACT
010
011            Learning to optimize is an approach that leverages training data to accelerate the
012            solution of optimization problems. Many approaches use unrolling to parametrize
013            the update step and learn optimal parameters. Although L2O has shown empiri-
014            cal advantages over classical optimization algorithms, memory restrictions often
015            greatly limit the unroll length and learned algorithms usually do not provide con-
016            vergence guarantees. In contrast, we introduce a novel method employing a greedy
017            strategy that learns iteration-specific parameters by minimizing the function value
018
               at the next iteration. This enables training over significantly more iterations while
               maintaining constant GPU memory usage. We parameterize the update such that
019
               parameter learning corresponds to solving a convex optimization problem at each
020
               iteration. In particular, we explore preconditioned gradient descent with multiple
021            parametrizations including a novel convolutional preconditioner. With our learned
022            algorithm, convergence in the training set is proved even when the preconditioner
023            is neither symmetric nor positive definite. Convergence on a class of unseen func-
024            tions is also obtained, ensuring robust performance and generalization beyond
025            the training data. We test our learned algorithms on two inverse problems, image
026            deblurring and Computed Tomography, on which learned convolutional precondi-
027            tioners demonstrate improved empirical performance over classical optimization
028            algorithms such as Nesterov’s Accelerated Gradient Method and the quasi-Newton
029
               method L-BFGS.

058   Learned optimization algorithms often lack convergence guarantees, including many that use RNNs

060   et al. (2023) consider methods of the form xt+1 = xt − Gt ∇f (xt ) + bt , for f : Rn → R, a diagonal
061   matrix Gt ∈ Rn×n , and a vector bt ∈ Rn . The Gt and bt are constructed using the outputs of neural
062

063   Other approaches achieve provable convergence, which can be enforced with safeguarding (Heaton
064   et al., 2023), or constructing convergent algorithms by learning parameters within a provably con-
065   vergent set (Banert et al., 2020; 2024). Tan et al. (2023a;b) learn mirror maps using input-convex
066   neural networks within the mirror descent optimization algorithm such that the algorithm is provably
067   convergent. Lastly, Sucker et al. (2024) and Sambharya & Stellato (2024) consider applying the
068

069   Unlike NAG, Newton’s method accelerates convergence by applying the inverse Hessian to the
070   gradient, which can be costly in practice. Quasi-Newton methods like BFGS (Nocedal & Wright,
071   2006b) approximate the Hessian, and L-BFGS (Liu & Nocedal, 1989) is used when BFGS is too
072   memory-intensive. Similarly, we aim to accelerate the optimization by learning a preconditioner Gt
073   in the update xt+1 = xt − Gt ∇f (xt ).
074   Adaptive algorithms improve optimization during use. For example, Armijo line-search (Armijo,
075   1966) seeks to find a good step size at each iteration, while methods like AdaGrad (Duchi et al., 2011)
076   and optimal diagonal preconditioners (Qu et al., 2024) adapt preconditioners. Online optimization
077   (Hazan et al., 2016), with methods such as Coin Betting (Orabona & Pál, 2016) and Adaptive Bound
078   Optimization (McMahan & Streeter, 2010), offers a game-theoretic perspective to optimization.

080   1.1    C ONTRIBUTIONS
081
082
      Our paper contributes in the following ways:
083           • A novel approach to L2O that learns parameters at each iteration sequentially, using a greedy
084             approach by minimizing the function value at the next iteration. This enables training over
085             significantly more iterations while maintaining constant GPU memory usage: Section 3.
086
              • Convergence in the training set is proved even when the preconditioner is neither symmetric
087             nor positive definite: Section 4. Furthermore, convergence is proved on a class of unseen
088             functions under certain conditions using soft constraints for parameter learning.
089
              • Learning parameters is a convex optimization problem for ’linear parametrizations’ of Gt ,
090
                enabling training that is significantly faster, with closed-form solutions for least-squares
091             functions: Section 5.
092
              • A novel parametrization of Gt as a convolution operator. At iteration t we learn a convo-
093
                lutional kernel κt such that Gt x = κt ∗ x. This parametrization is shown to outperform
094
                Nesterov’s Accelerated Gradient and L-BFGS on test data: Section 6.
095
096   In Section 6, we validate our learned algorithms on two inverse problems: image deblurring and
097   Computed Tomography (CT). Inverse problems represent a crucial class of optimization problems
098   that appear in important fields such as medical imaging and machine learning. Many such problems
099   have an associated forward operator which is highly ill-conditioned, making them an ideal test for
100   optimization algorithms.

104   Let X be a Hilbert space with corresponding field R and norm ∥ · ∥. A function f : X → R is
105   convex if for all x, y ∈ X and for all α ∈ [0, 1] f (αx + (1 − α)y) ≤ αf (x) + (1 − α)f (y). A
106   function f : X → R is L-smooth with parameter L > 0 if its gradient is Lipschitz continuous, i.e.,
107   if for all x, y ∈ X , ∥∇f (x) − ∇f (y)∥ ≤ L∥x − y∥. A function f : X → R is bounded below if
      there exists some M ∈ R such that f (x) ≥ M for all x ∈ X . We say that f ∈ FL if f is convex,

109   therefore, admits a finite orthonormal basis {e1 , · · · , en }. For x ∈ X and j ∈ {1, · · · , n}, define

529   7    C ONCLUSIONS
530
531   Our contribution is a novel L2O approach for minimizing unconstrained convex problems with
532   differentiable objective functions. Our method employs a greedy strategy to learn a linear operator at
533   each iteration of an optimization algorithm, meaning that GPU memory requirements are constant
534   with the number of training iterations. Parameter learning in our framework corresponds to solving
535   convex optimization problems, enabling the use of fast algorithms. Both factors allow training over a
536   large number of training iterations, which would otherwise be prohibitively expensive. Furthermore,
537   we obtain convergence results on the training set even when the preconditioner is neither symmetric
538
      nor positive definite, and for a class of unseen functions under certain conditions. The numerical
      results on imaging inverse problems demonstrate that our approach with a novel convolutional
539
      parametrization outperforms NAG and L-BFGS.

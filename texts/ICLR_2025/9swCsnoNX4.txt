                                                 Abstract
010

                Many object counting methods rely on density map estimation (DME) us-
011
012
                ing convolutional neural networks (CNNs) on discrete grid image represen-
013             tations. However, these methods struggle with large variations in object
014             size or input image resolution, typically due to different imaging conditions
015             and perspective effects. Worse yet, discrete grid representations of density
016             maps result in information loss with blurred or vanished details for low-
017             resolution inputs. To overcome these limitations, we design Scale-Invariant
018             Implicit neural representations for counting (SI-INR) to map arbitrary-scale
019             input signals into a continuous function space, where each function produces
020             density values over continuous spatial coordinates. SI-INR achieves robust
021
                counting performances with respect to changing object sizes, extensive ex-
                periments on commonly used diverse datasets have validated the proposed
                method.

025   1     Introduction
026
027   Understanding the distribution and abundance of people as well as geographic entities such
028   as buildings and cars within these environments becomes crucial for various “smart city”
029   applications, such as urban planning, traffic management and beyond. Object counting
030   holds promising potential for such tasks, which have also been studied in other domains
031   too, including crowd counting for security (Ma et al., 2019; Li et al., 2018), animal crowd
032   estimations (Ma et al., 2015), and cell counting for biomedicine (Paul Cohen et al., 2017).
033
      Successful counting methods have been developed by introducing deep learning (Fu et al.,
      2015; Wen et al., 2021) and self-attention (Gao et al., 2020; Lin et al., 2022). In recent years,
      the best-performing methods are mostly based on density map estimation (DME) (Ma et al.,
034
035
      2019; Gao et al., 2022; Lin et al., 2022; Wan et al., 2021), training convolutional neural
036   networks (CNNs) to generate discrete density maps.
037
038   However, several challenges persist in applying current deep learning methods for reliable
039
      counting: 1) Scale-Dependence: CNNs (LeCun et al., 1998) lack intrinsic scale equivari-
      ance, leading to degraded performance when object sizes deviate from those seen during
      training. This issue is particularly pronounced for inputs at resolutions differing from the
040
041
      training set, as CNNs rely on fixed receptive fields that cannot dynamically adapt to scale
042   variations; 2) Expressiveness-Bottleneck: Traditional grid-based density maps approximate
043   object distributions with Gaussian kernels, imposing a fixed spatial structure that misaligns
044   with irregular object arrangements (Wan & Chan, 2019). Gaussian smoothing reduces noise
045   but blurs local details, degrading fidelity in dense and sparse regions, and limiting counting
046   accuracy in complex scenes.
047
      To address these issues, we design a new object counting framework named Scale-Invariant
048   Implicit Neural Representations (SI-INR) mapping arbitrary-scale discrete images into 2D
049   continuous functions which are invariant to the object or structure scales. This allows the
050   model to preserve the fine details and reduce potential information loss for better counting
051   accuracy and generalizability. Moreover, the scale-invariance, an important property for
052   the mapping between input images and output density maps, is explicitly introduced as the
053   inductive bias of model itself to potentially improve data efficiency and model robustness.
      Our main contributions can be summarized as follows:

065   2   Related Work
066
067
      Object counting: Object counting, e.g. well-studied crowd counting (Lin et al., 2001), has
068   been mostly developed by detecting or segmenting individual objects in the scene. End-to-
069   end learning to directly map image features to object counts has been the most success-
070   ful counting strategy with rapid advancements in deep learning Krizhevsky et al. (2012);
071   Wang et al. (2015), especially for object counting in densely populated scenes (Chan et al.,
072   2008). Counting based on Density Map Estimation (DME) using convolutional neural net-
073   works (CNNs) (Lempitsky & Zisserman, 2010; Fu et al., 2015; Ranjan et al., 2018; Sindagi
074   & Patel, 2019) to preserve translation-invariant multi-scale image features has shown supe-
075   rior performance over conventional object counting techniques. More recent ASPDNet (Gao
076
      et al., 2020) and PSGCNet (Gao et al., 2022) have integrated attention, deformable convo-
      lution, pyramidal scale modules (PSM) to address challenges in counting such as complex
      cluttered backgrounds, viewing perspective, object appearance, and size variability. Besides,
077
078
      Huang et al. (2023) propose an optimized global regression model EfreeNet which is more
079   more annotation-efficient. Yi et al. (2023) introduce a lightweight multiscale context fusion
080   module (LMCFM) and a lightweight counting scale pooling module (LCSPM) to reduce the
081   numbers of network parameters and computing cost. These models have achieved state-
082   of-the-art (SOTA) counting performance on the RSOC (Remote Sensing Object Counting)
083   dataset (Gao et al., 2020).
084
      Scale-equivariance and invariance: The concept of scale-equivariance and invariance was
085   first proposed in image processing and computer vision (Lowe, 1999; 2004). To handle vari-
086   ations in scale effectively, multi-scale features can be learned by applying the convolutions
087   to several rescaled versions of the images or feature maps in every layer (Kanazawa et al.,
088   2014; Marcos et al., 2018) or by rescaling trainable filters (Xu et al., 2014). Cai et al.
089   (2016) proposed a pyramidal structure to learn scale-dependent features, which is widely
090   used in the object detection field nowadays. Later, Gaussian scale-space theory (Lindeberg,
091   1994) and group theory (Cohen & Welling, 2016) have been widely used for achieving scale-
092   equivariance and invariance. Yang et al. (2023); Lindeberg (2022) parameterized convolu-
093
      tional filters as a linear combination of Gaussian derivative filters with different scales, and
      achieved scale-equivariance in image classification and segmentation tasks. Unlike models
      rooted in Gaussian scale-space theory, Sosnovik et al. (2019) proposed a Scale-Equivariant
094
095
      Steerable Network (SESN), which utilizes steerable filters parameterized by a trainable lin-
096   ear combination of pre-calculated Hermite basis functions. These models all first build
097   a scale-equivariant model, and use a simple pooling layer or rescale the outputs to con-
098   vert the model into a scale-invariant one if needed (Sosnovik et al., 2019). However, such
099   methods have significant demands on memory and computational resources and can lose
100   information when doing the equivariance to invariance conversion. In SI-INR, we adopt
101   a scale-equivariant model to learn a deep representation that is equivariant to input scale
102   variations, and introduce a scale-invariant model to convert the equivariant mapping to an
103   invariant one.
104   Implicit neural representations: Implicit Neural Representations (INRs) allow for continuous
105   flexible representations of complex objects and scenes in computer vision (Mescheder et al.,
106   2019; Oechsle et al., 2019; Barrowclough et al., 2021; Wang et al., 2022). Together with
107   recently developed positional encoding strategies (Tancik et al., 2020; Sitzmann et al., 2020)
      and end-to-end hypernetwork-based learning (Dupont et al., 2022; Lee et al., 2024; Kim


                                                     2
      Under review as a conference paper at ICLR 2025


108
      et al., 2023), efficient training to capture high-frequency details has been achieved to better
109   model complex natural signals.
110
111   However, hypernetwork-based INR models are not translation-equivariant, making them
112
      unsuitable for handling scale variations in input signals. As a result, these models typ-
      ically require the input to be of a fixed size. More recent Hierarchical Neural Operator
113
      Transformer (HiNOTE) (Luo et al., 2024) integrates neural operators in implicit neural rep-
114
      resentation, which can preserve more local information and improve the generalizability of
115   INR models.
116
117   In SI-INR, we adopt a lightweight INR implementation and replace the traditional coor-
118
      dinate input by a continuous latent. In this case, our INR model transforms continuous
      latent to the continuous representation of targets, which can be seen as a deep neural oper-
119
      ator (Kovachki et al., 2023). This approach improves stability, accelerates model training,
120
      and offers greater flexibility when incorporating images of varying sizes during training.

123   3     Method
124
125
      We propose a novel object counting framework, Scale-Invariant Implicit Neural Represen-
126
      tation (SI-INR), that explicitly models scale-invariance in the adopted continuous INR for
127   robust object counting. We start the discussion by first presenting the problems of existing
128   methods in Section 3.1. Next, we describe our solution and concept of SI-INR with the
129   corresponding analysis in Section 3.2.1. We then describe the detailed construct of SI-INR
130   using SESN and a deep neural operator based INR architectures in Section 3.2.2. Lastly in
131   Section 3.3, we provide our sampling based model training for SI-INR on object counting.

      3.1   Problem statement
134
135
      In many real-world computer vision applications, input images can vary widely in dimension
136   and size, which requires the corresponding object detection and counting models to be able
137   to robustly identify objects ranging from a few pixels to thousands of pixels in scale. While
138   existing off-the-shelf CNNs often take the input images of the fixed size, the flexibility
139   to handle arbitrary resolution inputs and generate arbitrary resolution outputs is highly
140   desirable to take the best advantage of heterogeneous data available.
141
      For a given image I, existing methods aim to establish a mapping f : RdI ⇒ RdDgt from
142   input image I ∈ RdI to corresponding outputs Dgt ∈ RdDgt , where both input images
143   and outputs are in typical discrete-grid representations and the outputs’ resolution dDgt is
144   typically lower than inputs’ resolution reflected by the input dimension dI .
145
      However, since traditional convolution models are not scale-invariant and always have a fixed
      downsampling ratio, these methods usually struggle with handling varying resolution inputs

185   Following the above formulation, to achieve a scale-invariant mapping from the image space
186   I to the function space F, we propose the SI-INR modular framework consisting of three
187   components: a Scale-Equivariant backbone (SE-Backbone), a Hybrid Pyramidal Scale Mod-
188   ule (HPSM), and an INR decoder. The SE backbone is designed to extract deterministic
189   scale-equivariant features resilient to different resolutions of inputs and sizes of objects; then

191
      converts extracted features into a scale-invariant output, which is a continuous representa-

      3.2.1        Model Components
194
195   Traditional computer vision backbones struggle with scale variations of objects where dif-
196   ferent sizes of the same objects can have different appearances in the feature maps. This
197   results in inconsistent counting estimates and requires the model to have higher capacity to
198   memorize the same objects in different scales (Zhan et al., 2022).
199
      To construct a model which is data efficient and capable of handling unseen resolution in-
200   puts, we want our model to be scale-invariant so that consistent outputs can be generated
201   with respect to varying scales of objects in input images. As the combination of scale-
202   equivariant and scale-invariant models is scale-invariant (Sosnovik et al., 2019), we can
203   construct the model by ensuring that either each component in the framework is invariant,
204   or the framework appropriately combines scale-equivariant and scale-invariant components.
205   It is more reasonable to implement a scale-equivariant mapping first to preserve fine de-
206   tails (Sosnovik et al., 2019), and then convert the equivariant mapping into a scale-invariant
207   one for consequent prediction tasks.
208   SI-INR adopts this strategy and takes three steps to achieve scale-invariant mapping for
209   arbitrary-scale signals as shown in Figure 1: first, a scale-equivariant model B(·) is adopted
210   to extract deterministic features so that scale changes in objects will only affect the scale of
211   feature maps while keeping the appearance:
212
                                                                   B(p1 (h)(Ia )) = pB (h)(B)(Ia ),                                                                                                                                         (2)
213
214   where pB (·) denotes the corresponding group actions of h acting on the feature domain. We
215   emphasize that any scale-equivariant method can be adopted into SI-INR to handle different
      tasks.

224

      we introduce a scale-invariant continuous representation mapping H. For any query image
225

226

      4.1   Experimental Setup
348
      Datasets. We evaluate the model’s performance on three challenging datasets: (1) the
349   Remote Sensing Object Counting (RSOC) dataset (Gao et al., 2020); (2) the Car Parking
350   Lot Dataset (CARPK) (Hsieh et al., 2017); and (3) the Pontifical Catholic University of
351   Parana+ Dataset (PUCPR+) (Hsieh et al., 2017). Details on the datasets and the train-test
352   split are provided in Appendix B.1.
      Baselines. We compare our SI-INR with three baseline methods: ASPDNet (Gao et al.,
353
354
      2020), an attention-based network with scale pyramid and deformable convolutions; PSGC-
355   Net (Gao et al., 2022), which integrates pyramidal scale and global context modules; and
356   eFreeNet (Huang et al., 2023), an ensemble of first-rank-then-estimate networks. Network
357   architecture details are available in Appendix B.2.
358
      Implementation. We implement our scale-equivariant backbone by stacking four SESC
      layers (Sosnovik et al., 2019) with residual connections where the kernel size of SESC is set to
359
360
      3 to reduce computational costs. Our scale-invariant encoder E(·) consists of two parts. The
361   first part involves a scaling operator, the second part involves a VGG-19 network (Simonyan
362   & Zisserman, 2014) to learn deep features, following a pyramidal scale module that has
363   increasing kernel sizes from 3 × 3 to 11 × 11, and applies SESC layers with maximum scale
364   projection. Our INR-based decoder network consists of four fully connected layers with
365   residual connections, and one fully connected layer with learnable parameters to output raw
366   density maps. We use the Adam (Kingma & Ba, 2014) optimizer for both our SI-INR and
367   baseline models, and set the learning rate to be 1e − 4. We initialize parameters in SI-INR
368   by random sampling from a Gaussian distribution N (0, 0.012 ). We set σ = 8 in our loss
369
      function and SIN R = 64 when generating density maps for different inputs unless specified.
      We evaluate our SI-INR with baseline models on the RSOC dataset, CARPK dataset, and

563   5   Conclusions
564
565   In this paper, we introduce SI-INR, a novel scale-invariant INR implementation that maps
566   discrete grid image signals into continuous 2D function space, maintaining invariance to scal-
567   ing variation of the input signals. For object counting, SI-INR achieves SOTA performance,
568   our experiments demonstrate that SI-INR is exceptionally robust and flexible compared to
569   existing methods, capable of processing images of unseen resolutions during testing and ef-
570
      fectively handling images of various scales during training. This flexibility allows SI-INR to
      learn and capture more detailed features from different input training images. SI-INR can
571
      be easily applied to other image analysis tasks to achieve arbitrary-scale SOTA performance
572
      robustly with respect to input image size/resolution. Future work will focus on applying
573   SI-INR to multi-task scenarios, integrating object detection, image segmentation, and depth
574   estimation alongside counting.

790   A.1   Scale equivariance and invariance
791
      Consider a Scale-Translation Group consisting of a Scaling Group GS and a Translation
792
      Group GT , H = {h = (s, t) | s ∈ GS , t ∈ GT }, where h denotes an element of H and rep-
793   resents one scale-translation operator, GS denotes the Scaling Group, which accounts for
794   transformations that scale an object or function, and GT denotes the Translation Group,
795   which handles shifting the object or function within its domain. Besides, s is the scaling pa-
796   rameter, indicating how the input is stretched or compressed; t is the translation parameter,
797   specifying the shifting in the domain.
798
      From the group theory, given an image Ia ∈ V1 , a mapping Φ : V1 → V2 is scale-equivariant
799   if:
800

803   where V2 denotes the output domain, p1 (·) and p2 (·) denote the corresponding group actions
804   of h acting on V1 , V2 . If p2 (h) is the identity mapping, the mapping Φ is scale-invariant.

1049   B.6   Effect of different methods for handling multi-scale challenges
1050
       For efficiency and adaptability, scale-equivariant methods adjust to different scales without
1051
       having separate filters for each scale, unlike traditional methods that may rely on resizing
1052   inputs or using multiple filters for different scales. Many traditional multi-scale methods,
1053   such as image pyramids or multi-resolution networks, may struggle with high computational
1054   costs because they process the same image at multiple resolutions, leading to increased
1055   complexity especially with large images or when handling many scales. Besides, traditional
1056   multi-scale approaches do not focus on deriving scale-invariant outputs compared with our
1057   SI-INR.
1058   In our experiment, we have compared our SI-INR with four SOTA methods (PSGCNet (Gao
1059   et al., 2022), MMNet (Dong et al., 2020), MFANet (Zhu et al., 2021), MSFFA (Li et al.,
1060   2023)) that mainly focus on handling multi-scale challenges. PSGCNet applies a pyramidal
1061   network to handle multi-scale challenges, MMNet leverages multi-level density-based spatial
1062   information, MFANet introduces multi-level feature aggregation, and MSFFA integrates
1063   multi-scale feature fusion and attention mechanisms. As we report in Table 6, our SI-INR
1064   outperforms these methods on the UCF-QNRF crowd-counting dataset, which demonstrates
1065   its superior performance in handling objects of different sizes.

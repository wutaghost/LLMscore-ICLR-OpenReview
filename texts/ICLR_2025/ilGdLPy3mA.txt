011                                              A BSTRACT
012
013            Although a 3D Convolutional Neural Network (CNN) has been applied to explain-
014            able medical imaging in recent years, understanding the relationships among in-
015
               put 2D image patches, input 3D image blocks, extracted feature maps, top-ranked
               features, heatmaps, and final diagnosis remains a significant challenge. To help
016
               address this important challenge, firstly, we create a new 2D Grad-CAM-based
017
               method using feature selection to produce explainable 2D heatmaps with a small
018            number of highlighted image patches corresponding to top-ranked features. Sec-
019            ondly, we design a new 2D image patch ranking algorithm that leverages the newly
020            defined feature matrices and relevant statistical data from numerous heatmaps to
021            reliably rank axial patches, coronal patches, and sagittal patches. Thirdly, we cre-
022            ate a novel 3D image block ranking algorithm to generate a “Block Ranking Map
023            (BRM)” by using the axial patch ranking scores, coronal patch ranking scores, and
024            sagittal patch ranking scores. Lastly, we develop a hybrid 3D image block rank-
025            ing algorithm to generate a reliable hybrid BRM by using different block ranking
026
               scores generated by the 3D image block ranking algorithm using different top fea-
               ture sets. The associations between brain areas and a brain disease are reliably
027
               generated by using hybrid information from ChatGPT and relevant publications.
028
               The simulation results using two different 3D data sets indicate that the novel hy-
029            brid 3D image block ranking algorithm can identify top-ranked blocks associated
030            with important brain areas related to AD diagnosis and autism diagnosis. A doc-
031            tor may conveniently use the hybrid BRM with axial, coronal, and sagittal views
032            to better understand the relationship between the top-ranked blocks and medical
033            diagnosis, and then can efficiently and effectively make a rational and explainable
034            medical diagnosis.

      1   I NTRODUCTION
038
039
      In recent years, explainable deep learning techniques have been used in 3D medical imaging appli-
      cations such as explainable 3D brain imaging. The explainable 3D CNN model with the gradient-
040
      weighted class activation mapping is made to predict prognosis using whole-body diffusion-
041
      weighted MRI data (Morita et al., 2024). The efficient methods are developed to generate visual
042   explanations from 3D CNNs for Alzheimer’s disease (AD) classification (Yang et al., 2018). In
043   addition to the above mentioned methods using 3D information, another approach to increasing ex-
044   plainability of a 3D CNN is using relevant 2D information to interpret decisions of the 3D CNN.
045   For instance, A 2D transformer-based medical image model with different transformer attention en-
046   coders is made to diagnose AD in 3D MRI images (Wang et al., 2024). However, two remaining
047   significant challenges include (1) the fundamental research problem that is how to rationally inter-
048   pret the relationship among 3D image blocks, extracted feature maps, important features, and final
049   decisions of a 3D deep learning model, and (2) the fundamental application problem that is how to
050
      let a doctor conveniently understand such a multi-domain relationship and then efficiently make an
      explainable and correct medical diagnosis.
051
052   Currently, increasing explainability of a 3D deep learning model is important for high-quality medi-
053   cal imaging applications. For example, it is helpful and convenient for a medical doctor to easily see
      top-ranked visualized 3D brain image blocks that are closely associated with a brain disease such as


                                                        1
      Under review as a conference paper at ICLR 2025


054
      AD, and then make a correct and explainable brain disease diagnosis. In addition, it is useful to iden-
055   tify the most important blocks related to the brain disease in large 3D brain regions. For instance, it
056   is important to identify top-ranked 3D blocks within the large Hippocampus. Thus, a fundamental
057   problem is how to effectively select top-ranked 3D image blocks, and then better understand the
058   relationship between the top image blocks and decisions of a 3D CNN.
059
      Recently, various methods using 2D images have been developed to improve explainability of deep
060
      learning models. Relevant intelligent 2D techniques are developed to explain the decisions of a 2D
061   CNN (Zhou et al., 2016a; Selvaraju et al., 2017; Zhang et al., 2018; Schöttl, 2022; Wang et al.,
062   2022). For example, the Grad-CAM applies gradients to generate heatmaps for visual explanations
063   of a deep neural network (Selvaraju et al., 2017). An efficient patch-based deep learning network
064   with explainable 2D patch localization and selection is created for AD diagnosis (Zhang et al.,
065   2023a). The feature selection (FS) is useful in not only improving the model performance but
066   also in interpreting a deep neural network. For example, MediMLP with FS using Grad-CAM was
067   developed for lung cancer postoperative complication prediction (He et al., 2019). Based on current
068   explainable machine learning methods, we propose a novel method using axial, coronal, and sagittal
069
      2D images extracted from 3D images to rank 3D image blocks.
070   For the two remaining significant challenges, we have new works described as follows. In section 2,
071   new feature matrices with the top-ranked features’ properties are defined, and then the new FS-Grad-
072   CAM method is developed to generate explainable heatmaps with a smaller number of highlighted
073   areas associated with top-ranked features. Also, a new 2D image patch ranking algorithm using both
074   top-ranked features’ properties and relevant statistical information in a large number of heatmaps is
075
      developed to reliably rank image patches. In section 3, a novel hybrid 3D image block ranking
      algorithm using the axial, coronal, and sagittal patch ranking scores is created to robustly rank 3D
076
      image blocks in order to allow a user to more easily understand the relationship between the top-
077
      ranked blocks and a decision. In section 4, simulation results are analyzed. In section 5, conclusions
078   are given. In section 6, future works are discussed.

086   The last Maxpooling layer of a CNN generates n H ×W feature maps F l with the shape H ×W ×n
087   for l = 0, 1, . . . , n − 1. A H̄ × W̄ input image has P (H̄/H) × (W̄ /W ) patches for P = HW
088   (assuming H̄ is divisible by H and W̄ is divisible by W ). The feature maps F l have features fij     l

089   that are associated with a patch at (i, j) for i = 0, 1, . . . , H − 1, j = 0, 1, . . . , W − 1, and for
090   l = 0, 1, . . . , n − 1. The n feature maps are converted to m flattened features for m = n × H × W .
091   The m features have m feature index numbers (0, 1, . . . , m − 1).
092   An extracted feature map has m flatten features that are associated with the relevant m input image
093   patches, and the m flatten features are used by a classifier as inputs to make decisions. Since all m
094   flatten features in all extracted feature maps are used by the classifier, all associated patches have the
095   same number of associated features. Because all patches are equally important for final decisions,
096   the patches associated with all features cannot be ranked based on the number of associated features.
097   We developed a new CNN with the FS layer that applies a FS method to generate a small number
098   of top-ranked features that are associated with a small number of patches. Other patches with 0
099   associated features are eliminated since they are not useful for decision-making, so they are not used
100   for further patch ranking. Importantly, the FS method not only selects top-ranked features, but also
101   identifies important associated patches. Thus, the identified patches associated with the top-ranked
102   features can be ranked based on the number of associated features.
103   In addition, other feature properties, such as feature ranking scores, can be used to rank patches. For
104   example, if Patch A with an average feature ranking score 4.6 and the best feature ranking score 4
105   of 14 associated features, and Patch B with an average feature ranking score 7.9 and the best feature
106   ranking score 6 of 9 associated features, then Patch A is more important than Patch B for decision-
107   making. To reliably rank patches, we defined different informative feature structures with useful
      feature properties as follows.

313   axial images, 19, 640 coronal images, and 19, 640 sagittal images are extracted from the 982 3D
314   images (i.e., 20 consecutive slices with indices from 22 to 41 are extracted from the middle of each
315   3D brain image). 13, 748 training images (i.e., 70% of the 19, 640 images) and 2, 636 testing images
316   (i.e., 30% of the 19, 640 images) are used for simulations.

508   6       C ONCLUSIONS
509
510   Both informative feature matrices and heatmap matrices generated by using top-ranked features are
511   useful to reliably rank patches. The new FS-Grad-CAM method using top-ranked features, the new
512   2D image patch ranking algorithm using different top feature sets, and the novel 3D image block
513
      ranking algorithm using the axial, coronal, and sagittal patch ranking scores are able to generate
      relevant and useful information for robustly ranking 3D image block. The simulation results using
514
      the two different 3D data sets for AD diagnosis and autism diagnosis indicate that the novel hybrid
515
      3D image block ranking algorithm can identify top-ranked blocks associated with important brain
516   areas related to AD diagnosis and autism diagnosis. Thus, it is feasible and effective to robustly rank
517   3D image blocks by using the axial, coronal, and sagittal patch ranking scores together.
518
519
      It is useful and efficient to use both ChatGPT and relevant publications together to reliably verify
      if a brain area is associated with a disease diagnosis. The hybrid BRM with axial, coronal, and
520
      sagittal 2D views of top-ranked 3D blocks is informative and convenient for a user to understand
521
      the relationship among 3D blocks, 2D patches, extracted feature maps, selected features, and final
522   decisions. For example, a doctor may use the hybrid BRM with axial, coronal, and sagittal 2D views
523   to conveniently, efficiently, and effectively make an explainable and rational medical diagnosis.

526   7       F UTURE W ORKS
527
528   Firstly, we will develop more powerful FS methods to select top-ranked features that are used to
529   generate highly informative feature matrices and heatmap matrices. Secondly, it is important to
530   build an accurate deep learning model using the top-ranked features. Thirdly, the FS-Grad-CAM
531
      method, the 2D image patch ranking algorithm, and the hybrid 3D image block ranking algorithm
      will be improved by using other intelligent methods and optimized top feature sets. Fourthly, BRMs
532
      for correct decisions and BRMs for incorrect decisions will be generated to analyze the relationship
533
      among the top-ranked blocks, top-ranked features, relevant active elements in heatmaps, relevant
534   brain areas, and final decisions. Fifthly, it is critical to find a precise function mapping indices
535   (i, j, k) of a 3D image block to corresponding world coordinates (I, J, K) of a 3D image block
536   of the standard 3D brain of the “ebrains” software tool. Sixthly, a more effective 2D image patch
537   ranking algorithm using more factors better than the 5-factor 2D image patch ranking algorithm will
538   be developed. Seventhly, the hybrid 3D image block ranking algorithm will be evaluated by using
539   other 3D data sets, such as lung cancer data. Finally, a new block ranking algorithm directly using
      3D images, 3D deep learning with FS, and a 3D CAM-based method will be developed.

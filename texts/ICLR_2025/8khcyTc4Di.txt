                                                  A BSTRACT
008
009
               The goal of few-shot learning is to generalize and achieve high performance on
010
               new unseen learning tasks, where each task has only a limited number of examples
011            available. Gradient-based meta-learning attempts to address this challenging task
012            by learning how to learn new tasks by embedding inductive biases informed by
013            prior learning experiences into the components of the learning algorithm. In this
014            work, we build upon prior research and propose Neural Procedural Bias Meta-
015            Learning (NPBML), a novel framework designed to meta-learn task-adaptive pro-
016            cedural biases. Our approach aims to consolidate recent advancements in meta-
017            learned initializations, optimizers, and loss functions by learning them simultane-
018            ously and making them adapt to each individual task to maximize the strength of
019
               the learned inductive biases. This imbues each learning task with a unique set of
               procedural biases which is specifically designed and selected to attain strong learn-
020
               ing performance in only a few gradient steps. The experimental results show that
021
               by meta-learning the procedural biases of a neural network, we can induce strong
022            inductive biases towards a distribution of learning tasks, enabling robust learning
023            performance across many well-established few-shot learning benchmarks.

026   1   I NTRODUCTION
027
028   Humans have an exceptional ability to learn new tasks from only a few example instances. We can
029   often quickly adapt to new domains effectively by building upon and utilizing past experiences of
030   related tasks, leveraging only a small amount of information about the target domain. The field
031   of meta-learning (Schmidhuber, 1987; Vanschoren, 2018; Peng, 2020; Hospedales et al., 2022) ex-
032   plores how deep learning techniques, which often require thousands or even millions of observations
033
      to achieve competitive performance, can acquire such a capability. In meta-learning, the learn-
      ing process is often framed as a bilevel optimization problem (Bard, 2013; Maclaurin et al., 2015;
034
      Grefenstette et al., 2019; Lorraine et al., 2020). The outer optimization aims to learn the underlying
035
      regularities across a distribution of related tasks and embed them into the inductive biases of a learn-
036   ing algorithm. Consequently, in the inner optimization, the learning algorithm is utilized to quickly
037   adapt to new learning tasks using only a few example instances.
038
039
      Model-Agnostic Meta-Learning (MAML) (Finn et al., 2017) and its variants (Nichol & Schulman,
      2018; Rajeswaran et al., 2019; Song et al., 2019; Triantafillou et al., 2020), are a popular approach
040
      to meta-learning. In MAML, the outer optimization aims to learn the underlying regularities across
041
      a set of related tasks and embed them into a shared parameter initialization. This initialization is
042   then used in the inner optimization’s learning algorithm to encourage fast adaptation to new tasks.
043   While successful, these methods resort to simple gradient descent using the cross-entropy loss for
044   classification or squared loss for regression for the inner learning algorithm. Consequently, subse-
045   quent research has extended MAML to meta-learn additional components, such as the learning rate
046   (Behl et al., 2019; Baik et al., 2020), gradient-based optimizer (Li et al., 2017; Lee & Choi, 2018;
047   Simon et al., 2020; Flennerhag et al., 2020; Kang et al., 2023), loss function (Antoniou & Storkey,
048   2019), and more (Antoniou et al., 2019; Baik et al., 2023). This enables the meta-learning algorithm
049   to induce stronger inductive biases on the learning algorithm, further enhancing performance.
050   In this paper, we propose Neural Procedural Bias Meta-Learning (NPBML), a novel gradient-based
051   framework for meta-learning task-adaptive procedural biases for deep neural networks. Procedural
052   biases are the subset of inductive biases that determine the order of traversal over the search space
053   (Gordon & Desjardins, 1995), they play a central determining role in the convergence, sample ef-
      ficiency, and generalization of a learning algorithm. As we will show, the procedural biases are

085   2     BACKGROUND
086
087   In few-shot meta-learning, we are given access to a collection of tasks {T1 , T2 , . . . }, otherwise
088   known as a meta-dataset, where each task is assumed to be drawn from a task distribution p(T ).
089   Each task Ti contains a support set DS , and a query set DQ (i.e. a training set and a testing set),
090   where DS ∩DQ = ∅. Each of these sets contains a set of input-output pairs {(x1 , y1 ), (x2 , y2 ), . . . }.
091
      Let x ∈ X and y ∈ Y denote the inputs and outputs, respectively. In few-shot meta-learning, the
      goal is to learn a model of the form fθ (x) : X → Y , where θ are the model parameters. The
092
      primary challenge of few-shot learning is that fθ must be able to quickly adapt to any new task
093
      Ti ∼ p(T ) given only a very limited number of instances. For example, in an N -way K-shot
094   few-shot classification task, fθ is only given access to K labeled examples of N distinct classes.
095

124   3     N EURAL P ROCEDURAL B IAS M ETA -L EARNING
125
126   In this work, we propose Neural Procedural Bias Meta-Learning (NPBML), a novel framework that
127   replaces the fixed inner update rule in MAML, i.e., Equation (3), with a meta-learned task-adaptive
128   learning rule. This modifies the inner update rule in three key ways:
129
            1. An optimizer is meta-learned by leveraging the paradigm of preconditioned gradient de-
130
               scent. This involves meta-learning a parameterized preconditioning matrix Pω with meta-
131
               parameters ω to warp the gradients of SGD.
132
            2. The conventional loss function Lbase (i.e. the standard cross-entropy or squared loss) used
133
               in the inner optimization is replaced with a meta-learned loss function Mϕ , where ϕ are the
134
               meta-parameters.
135
136
            3. The meta-learned initialization, optimizer, and loss function are adapted to each new task
               using Feature-Wise Linear Modulation F iLMψ , a general-purpose preconditioning method
137
               that has learnable meta-parameters ψ.
138
139   For clarity, we provide a high-level overview of NPBML and contrast it to MAML, before expanding
140   on each of the new components in Sections 3.2, 3.3, and 3.4, respectively. Additionally, pseudocode
141   for the outer and inner optimizations is provided in Algorithm 1 and 2, respectively, in Appendix A.

206   3.3   M ETA -L EARNED L OSS F UNCTION
207
208   Unlike MAML, which defines the inner loss function Lbase to be the cross-entropy or squared loss
209   for all tasks, NPBML uses a meta-learned loss function M(ϕ,ψ) that is learned in the outer opti-

211
      label y and the model predictions f(θ,ω,ψ) (x), meta-learned loss functions can, in principle, be con-

212

250   3.4   TASK -A DAPTIVE M ODULATION
251
252   Although all tasks in few-shot learning are assumed to be sampled from the same task distribution
253
      p(T ), the optimal parameter initialization, optimizer, and loss function may differ between tasks.
      Therefore, in NPBML the meta-learned components Φ are modulated for each new task, providing
254
      each task with a unique set of task-adaptive procedural biases. To achieve this, Feature-wise Linear
255
      Modulation (FiLM) layers (Perez et al., 2018; Dumoulin et al., 2018) are inserted into both the
256   encoder z(θ,ω,ψ) and the meta-learned loss function M(ϕ,ψ) as shown in Figures 3 and 4. F iLMψ
257   is defined as follows, where γ and β are the scaling and shifting vectors, respectively, and ψ are the
258   meta-learnable FiLM parameters:
259
260
                                   F iLMψ (x) = (γψ (x) + 1) ⊙ x + βψ (x).                              (12)
261   Affine transformations conditioned on some input have become increasingly popular, being used
262   by several few-shot learning works to make the learned component adaptive (Oreshkin et al., 2018;
263   Jiang et al., 2018; Vuorio et al., 2019; Zintgraf et al., 2019; Baik et al., 2021; 2023). Furthermore,
264   FiLM layers can help alleviate issues related to batch normalization (Ioffe & Szegedy, 2015), which
265   have been empirically observed to cause training instability due to different distributions of features
266   being passed through the same model in few-shot learning (De Vries et al., 2017; Antoniou et al.,
267
      2019). In our work, we have found that conditioning the FiLM on the output activations of the
      previous layers is an effective way to achieve task adaptability. This form of conditioning is in
268
      essence a simplified version of that used in CNAPs (Requeima et al., 2019); however, we have
269
      omitted the use of global embeddings as we found it was not necessary for our method.

      3.5   I NITIALIZATION
271
272   Due to the large number of learnable meta-parameters, initialization becomes an important and
273   necessary aspect to consider. Here we detail how to initialize each of the meta-learned components,
274   i.e., Φ0 = {θ0 , ω0 , ϕ0 , ψ0 }, in NPBML. Firstly, we pre-train the encoder weights θ0 prior to meta-
275   learning, following many recent methods in few-shot learning (Rusu et al., 2019; Qiao et al., 2018;
276   Requeima et al., 2019; Ye et al., 2020; Ye & Chao, 2021), see Appendix A.3 for more details. For
277   the linear projection layers ω, we leverage the fact that in PGD, setting Pω to the identity I recovers
278
      SGD. Therefore, we set ∀l ∈ {1, . . . , L} : ω (l) = I; note for convolutional layers this corresponds
      to Dirac initialization. Regarding the meta-learned loss function M(ϕ,ψ) , the weights ϕ at the
279
      start of meta-training are randomly initialized ϕ0 ∼ N (0, 1e−2 ); therefore, the E[M(ϕ0 ,ψ0 ) ] =
280
      0, assuming an identity output activation. Consequently, the definition of the meta-learned loss
281
      function in Equation (11) can be modified to

293   4     I MPLICIT M ETA -L EARNING
294
295   In NPBML, the parameter initialization, gradient-based optimizer, and loss function are explicitly
296   meta-learned in the outer loop. Critically, we make a novel observation that many other key pro-
297
      cedural biases are also implicitly learned by meta-learning these three fundamental components
      (hence the name given to our algorithm). For example, consider the scalar learning rate α, which is
298
      implicitly meta-learned since the following equality holds:
299
300                             ∃α∃ϕ : θi,j − α∇θi,j Lbase ≈ θi,j − ∇θi,j Mϕ ,                           (15)
301
      and if ϕ is made to adapt on each inner step as done in (Baik et al., 2021; Raymond et al., 2023b),
302
      then by extension NPBML also learns a learning rate schedule. Another straightforward related
303
      observation is that NPBML implicitly learns a layer-wise learning rate {α(1) , . . . , α(L) }, since for
304   each block {ω (1) , . . . , ω (L) } in the block diagonal preconditioning matrix Pω the following holds:
305
306                        ∀l(∃ω (l) ∃α(l) ) : (ω (l) (ω (l) )T )∇θ(l) Mϕ ≈ α(l) ∇θ(l) Mϕ .              (16)
307
      There are also less obvious connections that can be drawn. For example, through a transitive re-
308   lationship, NPBML implicitly learns early stopping when the implicitly learned learning rate ap-
309   proaches zero, as discussed in (Baydin et al., 2018). Furthermore, since there is a linear scaling rule
310   between the batch size and the learning rate (Smith et al., 2017; Smith & Le, 2017; Goyal et al.,
311   2017), NPBML implicitly learns the regularization behavior of the batch size hyperparameter. An-
312   other non-trivial example is label smoothing regularization (Müller et al., 2019), which, as proven
313   in (Gonzalez & Miikkulainen, 2020), can be implicitly induced when meta-learning a loss function.

315   5     R ELATED W ORK
316
317   Meta-learning approaches to few-shot learning aim to equip models with the ability to quickly adapt
318   to new tasks given only a limited number of examples by leveraging prior learning experiences
319   across a distribution of related tasks. These approaches are commonly partitioned into three cate-
320   gories: (1) metric-based methods, which aim to learn a similarity metric for efficient class differenti-
321   ation (Koch et al., 2015; Vinyals et al., 2016; Sung et al., 2018; Snell et al., 2017); (2) memory-based
322   methods, which utilize architectures that store training examples in memory or directly encode fast
323   adaptation algorithms in the model weights (Santoro et al., 2016; Ravi & Larochelle, 2017); and (3)
      optimization-based methods, which aim to learn an optimization algorithm specifically designed for


                                                          6
      Under review as a conference paper at ICLR 2025


324
      fast adaptation and few-shot learning (Finn et al., 2017). This work explored the latter approach by
325   meta-learning a gradient update rule.
326
327   MAML (Finn et al., 2017), a highly flexible task and model-agnostic method for meta-learning a
328
      parameter initialization from which fast adaptation can occur. Many follow-up works have sought
      to enhance MAML’s performance by addressing limitations in the outer-optimization algorithm,
329
      such as the memory and compute efficiency (Nichol & Schulman, 2018; Rajeswaran et al., 2019;
330
      Raghu et al., 2019; Oh et al., 2020), or the meta-level overfitting (Rusu et al., 2019; Flennerhag et al.,
331   2018). Relatively fewer works have focused on enhancing the inner-optimization update rule, as is
332   done in our work. Most MAML variants continue to use an inner update rule consisting of SGD
333   with a fixed learning rate minimizing a loss function such as the cross-entropy or squared loss.
334
      Of the works that have explored improving the inner update rule, the vast majority focus on improv-
335
      ing the optimizer. For example, early MAML-based methods such as (Behl et al., 2019; Antoniou
336
      et al., 2019; Li et al., 2017) explored meta-learning the scalar, layer-wise, and parameter-wise learn-
337   ing rates, respectively. More recent methods have explored more powerful parameterization for the
338   meta-learned optimizer through the utilization of preconditioned gradient descent methods (Lee &
339   Choi, 2018; Park & Oliva, 2019; Flennerhag et al., 2020; Simon et al., 2020; Kang et al., 2023),
340   which rescale the geometry of the parameter space by modifying the update rule with a learned
341   preconditioning matrix. While these methods have advanced MAML-based few-shot learning, they
342   often lack a task-adaptive property, falsely assuming that all tasks should use the same optimizer.
343   A small number of recent works have also investigated replacing the inner loss function (e.g., cross-
344   entropy loss) with a meta-learned loss function. In (Antoniou & Storkey, 2019), a fully transductive
345   loss function represented as a dilated convolutional neural network is meta-learned. Meanwhile, in
346   (Baik et al., 2021), a set of loss functions and task adapters are meta-learned for each step taken in
347   the inner optimization. Although these approaches have shown a lot of promise, their potential has
348   yet to be fully realized, as they have not yet been meta-learned in tandem with the optimizer as we
349   have done in this work.

512   7     C ONCLUSION
513
514   In this work, we propose a novel meta-learning framework for learning the procedural biases of a
515   deep neural network. The proposed technique, Neural Procedural Bias Meta-Learning (NPBML),
516   consolidates recent advancements in MAML-based few-shot learning methods by replacing the fixed
517   inner update rule with a fully meta-learned update rule. This is achieved by meta-learning a task-
518   adaptive loss function, optimizer, and parameter initialization. The experimental results confirm
519   the effectiveness and scalability of the proposed approach, demonstrating strong few-shot learn-
520   ing performance across a range of popular benchmarks. We believe NPBML provides a principled
521   framework for advancing general-purpose meta-learning in deep neural networks. Looking ahead,
522
      numerous compelling future research directions exist, such as developing more powerful parame-
      terizations for the meta-learned optimizer or loss function. We expect that further investigation of
523
      this topic will result in more expressive inner update rules, resulting in increased robustness and
524
      efficiency within the context of optimization-based meta-learning. Finally, broadening the scope of
525   the proposed framework to encompass the related domains of cross-domain few-shot learning and
526   continual learning may be a promising avenue for future exploration.

800   A.1    DATASET C ONFIGURATIONS
801
      The few-shot learning experiments follow a prototypical episodic learning setup (Ravi & Larochelle,
802
      2017), where each dataset contains a set of non-overlapping meta-training, meta-validation, and
803
      meta-testing tasks. Each task Ti has a support DS and query DQ set (i.e., training and testing set,
804   respectively). For an N -way K-shot classification task, the support set contains N × K instances,
805   and the query set contains N × 15 instances, where N refers to the number of randomly sampled
806   classes, and K refers to the number of instances (i.e., shots) available from each of those classes.
807   For example, in a 5-way 5-shot classification task, the support set contains 5 × 5 = |DS | instances
808   to train on, and the query set contains 5 × 15 = |DQ | instances to validate the performance on.

841   A.2     N ETWORK A RCHITECTURES
842
      4-CONV: Following the encoder architecture settings in (Zintgraf et al., 2019; Flennerhag et al.,
843
      2020; Kang et al., 2023), the network consists of four modules. Each module contains a 3 × 3
844   convolutional layer with 128 filters, followed by a batch normalization layer, a ReLU non-linearity,
845   and a 2×2 max-pooling downsampling layer. Following the four modules, we apply average pooling
846   and flatten the embedding to a size of 128.
847
      ResNet-12: The encoder zθ follows the standard network architecture settings used in (Baik et al.,
848
      2020; 2021). The network, similar to 4-CONV, consists of four modules. Each module contains
849
      a stack of three 3 × 3 convolutional layers, where each layer is followed by batch normalization
850   and a leaky ReLU non-linearity. A skip convolutional over the convolutional stack is used in each
851   module. Each skip connection contains a 1 × 1 convolutional layer followed by batch normalization.
852   Following the convolutional stack and skip connection, a leaky ReLU non-linearity and 2 × 2 max-
853   pooling downsampling layer are placed at the end of each module. The number of filters used in each
854   module is [64, 128, 256, 512], respectively. Following the four modules, we apply average pooling
855   and flatten the embedding to a size of 512.
856   Classifier: As MAML and its variants are sensitive to label permutation during meta-testing, we opt
857   to use the permutation invariant classification head proposed by (Ye & Chao, 2021). This replaces
858   the typical classification head hθ ∈ Rin×N with a single-weight vector θhead ∈ Rin×1 which is
859   meta-learned at meta-training time. During meta-testing the weight vector is duplicated into each
860   output, i.e., hθ = {θc = θhead }N  c=1 , and adapted in an identical fashion to traditional MAML.
861   This reduces the number of parameters in the classification head and makes NPBML permutation
862   invariant similar to MetaOptNet (Lee et al., 2019), CNAPs (Requeima et al., 2019), and ProtoMAML
863   (Triantafillou et al., 2020).

864
      Loss Network: The meta-learned loss function M is composed of three separate networks LS , LQ ,
865   and R whose scalar outputs are summed to produce the final output as shown in Equation (11).
866   The network architecture for all three networks is identical except for the input dimension of the
867   first layer as discussed in Section 3.3. The network architecture is taken from (Bechtle et al., 2021;
868   Psaros et al., 2022; Raymond et al., 2023b) and is a small feedforward rectified linear neural network
869   with two hidden layers and 40 hidden units in each layer. Note LS and LQ are applied instance-wise
870   and reduced using a mean reduction, such that they are both invariant to the number of instances
871   made available at meta-testing time in the support and query set.
872   Relation Network: The transductive loss LQ takes an embedding from a pre-trained relation net-
873   work (Sung et al., 2018) as one of its inputs, similar to (Rusu et al., 2019; Antoniou & Storkey,
874   2019). The relation network used in our experiments employs the previously mentioned ResNet-12
875   encoder to generate a set of embeddings for each instance in DS ∪ DQ . These embeddings are
876   then concatenated and processed using a relation module. The relation module in our experiments
877   consists of two ResNet blocks with 2 × 512 and 512 filters, respectively. The output is then flattened
878   and averaged pooled to an embedding size of 512, which is passed through a final linear layer of
879
      size 512 × 1 which outputs each relation score. In regard to training, the relation network is trained
      using the settings described in (Sung et al., 2018).

883
      A.3.1    P RE - TRAINING S ETTINGS
884
885   In NPBML, the encoder portion of the network zθ is pre-trained prior to meta-learning, following
886   many recent methods in few-shot learning (Rusu et al., 2019; Qiao et al., 2018; Requeima et al.,
887   2019; Ye et al., 2020; Ye & Chao, 2021). For both the 4-CONV and ResNet-12 models, we fol-
888   low the pre-training pipeline used in (Ye et al., 2020; Ye & Chao, 2021). During pre-training, we
889
      append the feature encoder zθ with a temporary classification head fθ and train it to classify all
      classes in the Dtrain (e.g., over all 64 classes in the mini-ImageNet) using the cross-entropy loss.
890
      During pre-training, the model is trained over 200000 gradient steps using SGD with Nesterov mo-
891
      mentum and weight decay with a batch size of 128. The learning rate, momentum coefficient, and
892   weight decay penalty are set to 0.01, 0.9, and 0.0005, respectively. Additionally, we use a multistep
893   learning rate schedule, decaying the learning rate by a factor of 0.1 at the following gradient steps
894   {100000, 150000, 175000, 190000}. Note that in our ResNet-12 experiments on mini-ImageNet,
895   CIFAR-FS, and FC-100, we observed some meta-level overfitting, which we hypothesize is due
896   to the expressive network architecture coupled with the relatively small datasets. We found that
897   increasing the pre-training weight decay to 0.01 in these experiments resolved the issue.
898
899   A.3.2    M ETA -L EARNING S ETTINGS
900
      In both the meta-training and meta-testing phases, we adhere to the standard hyperparameter values
901
      from the literature (Finn et al., 2017). In the outer loop, our algorithm is trained over S meta =
902   30, 000 meta-gradient steps using Adam with a meta learning rate of η = 0.00001. As with previous
903   works, our models are trained with a meta-batch size of 2 for 5-shot classification and 4 for 1-shot
904   classification. In the inner loop, the models are adapted using S base = 5 base-gradient steps using
905   SGD with Nesterov momentum and weight decay. The learning rate, momentum coefficient, and
906   weight decay penalty are set to 0.01, 0.9, and 0.0005 respectively. During meta-testing, the same
907   inner loop hyperparameter settings are employed, and the final model is evaluated over 600 tasks
908   sampled from Dtesting . Notably, unlike prior work (Antoniou et al., 2019; Baik et al., 2020; 2021;
909   2023), we do not ensemble the top 3 or 5 performing models from the same run, as this significantly
910
      increases the number of parameters and expressive power of the final models.

      A.3.3    NPBML S ETTINGS
912
913   In our experiments, the backbone encoders zθ (i.e., 4-CONV and ResNet-12) are modified in two key
914   ways during meta-learning: (1) with linear projection (warp) layers ω for preconditioning gradients,
915   and (2) feature-wise linear modulation layers F iLMψ for task adaptation. In all our experiments,
916   we modify only the last (i.e., fourth) module with warp and FiLM layers based on the findings from
917   (Raghu et al., 2019), which showed that features near the start of the network are primarily reused
      and do not require adaptation. Additionally, we freeze all preceding modules in both the inner and

      outer loops, which significantly reduces the storage and memory footprint of the proposed method,
919   with no noticeable effect on the performance.

      convolutional stack and before the residual connection ends in the ResNet modules. Regarding
      the FiLM layers, they are inserted after each batch normalization layer in a manner similar to the

      B    F URTHER D ISCUSSION
931
932
      The proposed NPBML framework is a highly flexible and general approach to gradient-based meta-
      learning. As a result of this generality, many existing meta-learning algorithms are closely related
933
      to NPBML, with some potentially arising as special cases of NPBML. In what follows, we discuss
934
      some salient examples:
935
936        • MAML++: In (Antoniou et al., 2019), the inner optimization of MAML is modified by
937          learning per-step batch normalization running statistics, weights and biases, and layer-wise
938          learning rates. This is closely related to the FiLM layers used in NPBML, as they also
939          output meta-learned scale and shift parameters, i.e., γψ and βψ , which improve robustness
940          to varying feature distributions. Furthermore, as shown by Equation (16), NPBML implicitly
941          learns a layer-wise learning rate through Pω .
942        • SCA: In (Antoniou & Storkey, 2019), the authors propose replacing Lbase with a meta-
943          learned loss function Mϕ similar to NPBML. The primary difference between their param-
944          eterization and ours is that their meta-learned loss function is represented as a 1D dilated
945          convolutional neural network with DenseNet-style residual connectivity. In our experiments,
946          we found that a simple feedforward ReLU network is sufficiently expressive to obtain high
947
             performance as shown in the ablation study in Table 3.
948        • MeTAL: In (Baik et al., 2021), a set of task-adaptive loss functions, represented as small
949          feedforward networks are meta-learned for each step in the inner optimization. Task-
950          adaptivity is achieved by using a separate set of networks to generate FiLM scale and shift pa-
951
             rameters which are applied directly to ϕ instead of the network’s activations. Unlike MeTAL,
             NPBML does not maintain separate networks for each step; thus, it can be applied to more
952
             gradient steps at meta-testing time than it was initially trained to do.
953
954        • ALFA: In (Baik et al., 2023), MAML’s inner optimization is modified with meta-learned
955
             layer-wise learning rate values and weight decay coefficients. As shown in Equation (16),
             NPBML implicitly learns layer-wise learning rate values. Additionally, since the learned
956
             loss function Mϕ in NPBML is conditioned on θi,j , our method also implicitly learns a
957
             scalar weight decay values, since ∃λ∃ϕ : θi,j − ∇θi,j (Lbase + λ||θ||2 ) ≈ θi,j − ∇θi,j Mϕ .
958
959
           • MetaSGD: In (Li et al., 2017), a parameter-wise learning rate is meta-learned in tandem with
             the parameter initialization. This corresponds to using an identical inner update rule to Equa-
960
             tion (6) where M(ϕ,ψ) = Lbase and P(ω,ψ) = diag(ω), i.e., no meta-learned loss function,
961
             and a diagonal matrix used instead of a block diagonal matrix for gradient preconditioning.
962
963
           • WarpGrad: In (Flennerhag et al., 2020), a full gradient preconditioning matrix is induced by
             modifying the meta-learned linear projection layers employed in T-Nets (Lee & Choi, 2018)
964
             with non-linear activations. Furthermore, a trajectory-agnostic meta-objective is used. As
965
             NPBML utilizes T-Net style gradient preconditioning, simply introducing non-linear activa-
966          tion functions and replacing the outer objective Lmeta in Equation (4) would be sufficient to
967          recover WarpGrad’s gradient preconditioning.
968
           • ModGrad: In (Simon et al., 2020), a low-rank gradient preconditioning matrix is meta-
969
             learned. This is, in essence, an identical inner update rule to Equation (6) where
970          M(ϕ,ψ) = Lbase and P(ω,ψ) = ω1 ⊗ ω2 , where ω1 ∈ Rin×r , ω2 ∈ Rr×out , and ⊗ is the
971          outer product between the two low-rank matrices with rank r.

011                                               A BSTRACT
012
013            The Segment Anything Model 2 (SAM2) has emerged as a powerful foundation
014            model for object segmentation in both images and videos, paving the way for
015
               various downstream video applications. The crucial design of SAM2 for video
               segmentation is its memory module, which prompts object-aware memories from
016
               previous frames for current frame prediction. However, its greedy-selection mem-
017
               ory design suffers from the “error accumulation” problem, where an errored or
018            missed mask will cascade and influence the segmentation of the subsequent frames,
019            which limits the performance of SAM2 toward complex long-term videos. To this
020            end, we introduce SAM2Long, an improved training-free video object segmen-
021            tation strategy, which considers the segmentation uncertainty within each frame
022            and chooses the video-level optimal results from multiple segmentation pathways
023            in a constrained tree search manner. In practice, we maintain a fixed number of
024            segmentation pathways throughout the video. For each frame, multiple masks are
025            proposed based on the existing pathways, creating various candidate branches. We
026
               then select the same fixed number of branches with higher cumulative scores as the
               new pathways for the next frame. After processing the final frame, the pathway
027
               with the highest cumulative score is chosen as the final segmentation result. Ben-
028
               efiting from its heuristic search design, SAM2Long is robust toward occlusions
029            and object reappearances, and can effectively segment and track objects for com-
030            plex long-term videos. Without introducing any additional parameters or further
031            training, SAM2Long significantly outperforms SAM2 on six VOS benchmarks.
032            Notably, it achieves an average improvement of 3.8 points across all model sizes
033            and, in some cases, up to 5 points in J &F on long-term video object segmentation
034            benchmarks SA-V and LVOS.

      1   I NTRODUCTION

039
      The Segment Anything Model 2 (SAM2) has gained significant attention as a unified foundational
      model for promptable object segmentation in both images and videos. Notably, SAM2 (Ravi et al.,
040
      2024) has achieved state-of-the-art performance across various video object segmentation tasks,
041
      significantly surpassing previous methods. Building upon the original SAM (Kirillov et al., 2023),
042   SAM2 incorporates a memory module that enables it to generate masklet predictions using stored
043   memory contexts from previously observed frames. This module allows SAM2 to seamlessly extend
044   SAM into the video domain, processing video frames sequentially, attending to the prior memories of
045   the target object, and maintaining object coherence over time.
046
      While SAM2 demonstrates strong performance in video segmentation, its greedy segmentation
047
      strategy struggles to handle complex video scenarios with frequent occlusions and object reappearance.
048   In detail, SAM2 confidently and accurately segments frames when clear visual cues are present.
049   However, in scenarios with occlusions or reappearing objects, it can produce mask proposals that
050   are highly variable and uncertain. Regardless of the frame’s complexity, a uniform greedy selection
051   strategy is applied to both scenarios: the mask with the highest predicted IoU is selected. Such greedy
052   choice works well for the easy cases but raises the error potential for the challenging frames. Once an
053   incorrect mask is selected into memory, it is uncorrectable and will mislead the segmentation of the
      subsequent frames. We show such an “error accumulation” problem in Figure 1 both qualitatively and

077   To this end, we redesign the memory module of SAM2 to enhance its long-term compatibility
078   and robustness against occlusions and error propagation. Our improvement is completely free of
079   additional training and does not introduce any external parameters, but simply unleashes the potential
080   of SAM2 itself. Our approach is motivated by the observation that the SAM2 mask decoder generates
081
      multiple diverse masks, accompanied by predicted IoU scores and an occlusion score when handling
      challenging and ambiguous cases. However, SAM2 only selects a single mask as memory, sometimes
082
      disregarding the correct one. To address this, we aim to equip SAM2 with multiple memory pathways,
083
      allowing various masks to be stored as memory at each time step, thereby improving predictions for
084   subsequent frames.
085
086   In particular, we introduce a novel constrained tree memory structure, which maintains a fixed number
087
      of memory pathways over time to explore multiple segmentation hypotheses with efficiently managed
      computational resources. At each time step, based on a set of memory pathways, each with its
088
      own memory bank and cumulative score (accumulated logarithm of the predicted IoU scores across
089
      the pathway), we produce multiple candidate branches for the current frame. Then, among all the
090   branches, we select the same fixed number of branches with higher cumulative scores and prune
091   other branches, thereby constraining the tree’s growth. After processing the final frame, the pathway
092   with the highest cumulative score is selected as the final segmentation result. Moreover, to prevent
093   premature convergence on incorrect predictions, we select hypotheses with distinct predicted masks
094   when their occlusion scores indicate uncertainty, in order to maintain diversity in the tree branches.
095   This tree-like memory structure augments SAM2’s ability to effectively overcome error accumulation.
096   Within each pathway, we construct an object-aware memory bank that selectively includes frames
097   with confidently detected objects and high-quality segmentation masks, based on the predicted
098   occlusion scores and IoU scores. Instead of simply storing the nearest frames as SAM2 does, we filter
099   out frames where the object may be occluded or poorly segmented. This ensures that the memory
100   bank provides effective object cues for the current frame’s segmentation. Additionally, we modulate
101   the memory attention calculation by weighting memory entries according to their occlusion scores,
102   emphasizing more reliable entries during cross-attention. These strategies help SAM2 focus on
103   reliable object clues and improve segmentation accuracy with negligible computational overhead. As
104
      evidenced in Figure 1(a), our approach successfully resolves occlusions and re-tracks the recurring
      balloon, where SAM2 fails.
105
106   We provide a comprehensive evaluation demonstrating that SAM2Long consistently outperforms
107   SAM2 across six VOS benchmarks, particularly excelling in long-term and occlusion-heavy scenarios.
      For instance, on the challenging SA-V validation set, SAM2Long-Large improves the J &F score

      2.1   V IDEO O BJECT S EGMENTATION
122
      Perceiving the environment in terms of objects is a fundamental cognitive ability of humans. In
123
      computer vision, Video Object Segmentation (VOS) tasks aim to replicate this capability by requiring
124   models to segment and track specified objects within video sequences. A substantial amount of
125   research has been conducted on video object segmentation in recent decades (Fan et al., 2019; Oh
126   et al., 2019; Hu et al., 2018a; Oh et al., 2018; Perazzi et al., 2017; Wang et al., 2019; Hu et al., 2018b;
127   Li & Loy, 2018; Bao et al., 2018; Zhang et al., 2019; Li et al., 2020; Johnander et al., 2019; Zhang
128   et al., 2023; Ventura et al., 2019; Li et al., 2022; Wu et al., 2023; Wang et al., 2023).
129
      There are two main protocols for evaluating VOS models (Pont-Tuset et al., 2017; Perazzi et al.,
130   2016): semi-supervised and unsupervised video object segmentation. In semi-supervised VOS,
131   the first-frame mask of the objects of interest is provided, and the model tracks these objects in
132   subsequent frames. In unsupervised VOS, the model directly segments the most salient objects from
133   the background without any reference. It is important to note that these protocols are defined in the
134   inference phase, and VOS methods can leverage ground truth annotations during the training stage.
135
      In this paper, we explore SAM2 (Ravi et al., 2024), for its application in semi-supervised VOS.
136   We enhance the memory design of SAM2, significantly improving mask propagation performance
137   without requiring any additional training.

      2.2   M EMORY-BASED VOS
141
      Video object segmentation remains an unsolved challenge due to the inherent complexity of video
142
      scenes. Objects in videos can undergo deformation (Tokmakov et al., 2023), exhibit dynamic
143
      motion (Brox & Malik, 2010), reappear over long durations (Hong et al., 2024; 2023), and experience
144   occlusion (Ding et al., 2023), among other challenges. To address the above challenges, adopting a
145   memory architecture to store the object information from past frames is indispensable for accurately
146   tracking objects in video. Previous methods (Bhat et al., 2020; Caelles et al., 2017; Maninis et al.,
147   2018; Robinson et al., 2020; Voigtlaender & Leibe, 2017) treat VOS as an online learning task,
148   where networks are test-time tuned on the first-frame annotation. However, this approach was
149   time-consuming due to test-time fine-tuning. Other techniques (Chen et al., 2018; Hu et al., 2018b;
150   Voigtlaender et al., 2019; Yang et al., 2018; 2020; 2021b) use template matching, but they lack the
151   capability of tracking under occlusion.
152   More recent approaches have introduced efficient memory reading mechanisms, utilizing either
153   pixel-level attention (Cheng et al., 2023; Zhou et al., 2024; Duke et al., 2021; Liang et al., 2020; Oh
154   et al., 2018; Seong et al., 2020; Cheng & Schwing, 2022; Xie et al., 2021; Yang & Yang, 2022; Yang
155   et al., 2021a) or object-level attention (Athar et al., 2023; 2022; Cheng et al., 2024). A prominent
156   example is XMem (Cheng & Schwing, 2022), which leverages a hierarchical memory structure for
157   pixel-level memory reading combined. Building on XMem’s framework, Cutie (Cheng et al., 2024)
158
      further improves segmentation accuracy by processing pixel features at the object level to better
      handle complex scenarios.
159
160   The latest SAM2 (Ravi et al., 2024) incorporates a simple memory module on top of the image-based
161   SAM (Kirillov et al., 2023), enabling it to function for VOS tasks. However, by selecting only the
      temporally nearest frames as memory, SAM2 struggles with challenging cases involving long-term

162
      reappearing objects and confusingly similar objects. we redesign SAM2’s memory to maintain
163   multiple potential correct masks, making the model more object-aware and robust.

      2.3   S EGMENT A NYTHING M ODEL
166
167   Segment Anything Model (SAM) (Kirillov et al., 2023) is recognized as a milestone vision foundation
168   model that can segment any object in an image using interactive prompts. Its impressive zero-shot
169   transfer performance has shown great versatility in various vision tasks, including segmentation
170   applications (Li et al., 2023; Ma et al., 2024; Xu et al., 2024), image editing (Gao et al., 2023) and
171   object reconstruction (Lin et al., 2024).
172   Building on SAM, SAM 2 (Ravi et al., 2024) extends its functionality to video segmentation through
173   a memory-based transformer architecture for real-time video processing. SAM 2’s memory stores
174   information about objects and past interactions, enabling it to generate segmentation masks across
175   video frames more accurately and efficiently than previous methods. To further enhance SAM 2, we
176   introduce a constrained memory tree structure. This training-free design leverages the SAM2’s ability
177   to generate multiple candidate mask proposals with predicted IoU and occlusion score, mitigating
178   error accumulation during segmentation.

185   dings. In contrast to SAM, where frame embeddings are fed directly into the mask decoder, SAM2
186
      incorporates a memory module that conditions the current frame’s features on both previous and
      prompted frames. Specifically, for semi-supervised video object segmentation tasks, SAM2 maintains
187
      a memory bank at each time step t ≥ 1:

      where K is the number of memory tokens per frame, C is the channel dimension, and I is the set of
      frame indices included in the memory. In SAM2, memory set I stores up to N of the most recent
192
      frames, along with the initial mask, using a First-In-First-Out (FIFO) queue mechanism.
193
194   Each memory entry consists of two components: (1) the spatial embedding fused with the predicted
195   mask (generated by the memory encoder), and (2) the object-level pointer (generated by the mask
196   decoder). After cross-attending to the memory, the current frame’s features integrate both fine-grained

198   The mask decoder, which is lightweight and retains the efficiency of SAM, then generates three
199   predicted masks for the current frame. Each mask is accompanied by a predicted Intersection
200   over Union (IoU) score IoUt ≥ 0 and an output mask token. Additionally, the mask decoder
201   predicts a single occlusion score ot for the frame, where ot > 0 indicates object presence, ot < 0
202   indicates absence, and the absolute value |ot | depicts the model’s confidence. The mask with the
203
      highest predicted IoU score is selected as the final prediction, and its corresponding output token is
      transformed into the object pointer for use as the memory.

272   In each memory pathway, we devise object-aware memory selection to retrieve frames with discrimi-
273   native objects. Meanwhile, we modulate the memory attention calculation to further strengthen the
274   model’s focus on the target objects.
275

276   selectively choose frames from previous time steps based on the predicted object presence and
277   segmentation quality. Starting from the frame immediately before the current frame t, we iterate
278   backward through the prior frames i = {t − 1, t − 2, . . . , 1} in sequence. For each frame i, we
279   retrieve its predicted occlusion score oi and IoU score IoUi as reference. We include frame i in the
280   memory bank if it satisfies the following criteria:
281
                                          IoUi > δIoU    and    oi > 0,
282
283   where δIoU is a predefined IoU threshold. This ensures that only frames with confidently detected
284   objects and reasonable segmentation masks contribute to the memory. We continue this process
285

      may be occluded, absent, or poorly segmented, thereby providing more robust object cues for the

314   4.1   DATASETS
315
316   To evaluate our method, we select 6 standard VOS benchmarks and report the following metrics: J
317   (region similarity), F (contour accuracy), and the combined J &F. All evaluations are conducted in
318   a semi-supervised setting, where the first-frame mask is provided. The datasets used for testing are
319   detailed as follows:
320   SA-V (Ravi et al., 2024) is a large-scale video segmentation dataset designed for promptable visual
321   segmentation across diverse scenarios. It encompasses 50.9K video clips, aggregating to 642.6K
322   masklets with 35.5M meticulously annotated masks. The dataset presents a challenge with its
323   inclusion of small, occluded, and reappearing objects throughout the videos. The dataset is divided
      into training, validation, and testing sets, with most videos allocated to the training set for robust

      Under review as a conference paper at ICLR 2025


324
      model training. The validation set has 293 masklets across 155 videos for model tuning, while the
325   testing set includes 278 masklets across 150 videos for comprehensive evaluation.
326
327   LVOS v1 (Hong et al., 2023) is a VOS benchmark for long-term video object segmentation in realistic
328
      scenarios. It comprises 720 video clips with 296,401 frames and 407,945 annotations, with an average
      video duration of over 60 seconds. LVOS introduces challenging elements such as long-term object
329
      reappearance and cross-temporal similar objects. In LVOS v1, the dataset includes 120 videos for
330
      training, 50 for validation, and 50 for testing.
331
332   LVOS v2 (Hong et al., 2024) expends LVOS v1 and provides 420 videos for training, 140 for
333   validation, and 160 for testing. This paper primarily utilizes v2, as it already includes the sequences
334
      present in v1. The dataset spans 44 categories, capturing typical everyday scenarios, with 12 of these
      categories deliberately left unseen to evaluate and better assess the generalization capabilities of VOS
335
      models.
336
337   Long Videos Dataset (Liang et al., 2020) contains 3 long-form video sequences, each averaging over
338   2,000 frames, designed to evaluate VOS performance in real-world scenarios. For evaluation, 20
339   frames from each video are uniformly annotated.
340   VOST (Tokmakov et al., 2023) is a semi-supervised video object segmentation benchmark that
341   emphasizes complex object transformations. Unlike other datasets, VOST includes objects that
342   are broken, torn, or reshaped, significantly altering their appearance. It comprises more than 700
343   high-resolution videos, captured in diverse settings, with an average duration of 21 seconds, all
344   densely labeled with instance masks.
345   DAVIS2017 (Pont-Tuset et al., 2017) is a well-known benchmark dataset comprising 60 training
346   videos and 30 validation videos, with a total of 6,298 frames. It offers high-quality, pixel-level
347   annotations for every frame, making it a standard resource for evaluating different VOS methods.

447   Number of Memory Pathways P . We ablate the number of memory pathways to assess their impact
448   on SAM2Long in Table 5. Note that setting P = 1 reverts to the SAM2 baseline. Increasing the
449   number of memory pathways to P = 2 yields a notable improvement, raising the J &F score to
450   80.9. This result demonstrates that the proposed memory tree effectively boosts the model’s ability
451
      to track the correct object while reducing the impact of occlusion. Further increasing the number
      of memory pathways to P = 3 achieves the best performance. However, using P = 4 shows no
452
      additional gains, suggesting that three pathways strike the optimal balance between accuracy and
453
      computational efficiency for the SAM2 model.
454
455   In terms of speed, since pruning is performed at every step, the speed is effectively maintained. Using
456   three memory pathways slows down the model by only 18%, while yielding nearly a 5-point increase
457
      in performance.
458   Iou Threshold δiou . The choice of the IoU threshold δiou is crucial for selecting frames with reliable
459   object cues. As shown in Table 6, setting δiou = 0.3 yields the highest J &F, indicating an effective
460   trade-off between filtering out poor-quality frames and retaining valuable segmentation information.
461   In contrast, having no requirement on mask quality and feeding all masks containing objects into
462   memory (δiou = 0) decreases the score to 80.5, showing that unreliable frames with poor segmentation
463
      harm the SAM2 model. Meanwhile, an overly strict selection (δiou = 0.9) degrades performance even
      more severely to 78.0, as it excludes too many potentially important neighboring frames, causing the
464
      model to rely on frames that are too far away from the current frame as memory.
465
466   Uncertainty Threshold δconf . The uncertainty threshold δconf controls the selection of hypotheses
467   under uncertain conditions. Our results in Table 7 indicate that setting δconf to 2 provides the highest
468   J &F score, indicating an optimal level for uncertainty handling. Lower values (e.g., 0.5) result in
469
      suboptimal performance, as they may prematurely commit to incorrect segmentation hypotheses,
      leading to significant performance drops due to error propagation. On the other hand, higher values
470
      (e.g., 5) do not further improve performance, suggesting that beyond a certain threshold, the model
471
      does not benefit from additional mask diversity and can efficiently rely on the top-scoring masks
472   when the segmentation is confident.
473
474   Memory Attention Modulation [wlow , whigh ]. We explore the effect of modulating the attention
475
      weights for memory entries using different ranges in Table 8. The configuration [1, 1] means no
      modulation is applied. We find that the configuration of [0.95, 1.05] achieves the best performance
476
      while increasing the modulation range to ([0.9, 1.1]) slightly decreases performance. This result
477
      indicates that slight modulation sufficiently emphasizes reliable memory entries.

533
      In this paper, we introduce SAM2Long, a training-free enhancement to SAM2 that alleviates its
      limitations in long-term video object segmentation. By employing a constrained tree memory structure
534
      with object-aware memory modulation, SAM2Long effectively mitigates error accumulation and
535
      improves robustness against occlusions, resulting in a more reliable segmentation process over
536   extended periods. Extensive evaluations on six VOS benchmarks demonstrate that SAM2Long
537   consistently outperforms SAM2, especially in complex video scenarios. Notably, SAM2Long
538   achieves up to a 5-point improvement in J&F scores on challenging long-term video benchmarks
539   such SA-V and LVOS without requiring additional training or external parameters.

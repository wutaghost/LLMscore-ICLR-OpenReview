009                                                A BSTRACT
010
011            Visual feature extraction is fundamental to many vision tasks. Most existing meth-
012            ods extract visual features by encoding an image into a generic feature vector.
013            However, an image naturally contains rich information, and there may be multiple
014            perspectives to describe it. For each application, we might be interested in dif-
015            ferent aspects of an image and want to prioritize those features over others. For
016            instance, in an image of a dog carrying a toy, if we are primarily interested in the
017            dog, we would expect the extracted features to emphasize the dog over the toy. In
018
               this work, we introduce FocalLens, a conditional visual feature extraction method
               that produces different representations for the same image based on the context
019
               of interest, expressed flexibly through natural language. We leverage vision in-
020
               struction tuning data and contrastively tune a pretrained vision encoder to take
021            natural language instructions as additional inputs and produce conditional image
022            representations. Extensive experiments validate that conditional image represen-
023            tation from FocalLens better pronounce the visual features of interest compared to
024            generic features produced by standard vision encoders like CLIP. In addition, we
025            show FocalLens further leads to performance improvements on a range of down-
026            stream tasks including image-image retrieval, image classification, and image-text
027            retrieval, with an average gain of 5 and 10 points on the challenging SugarCrepe
028            and MMVP-VLM benchmarks, respectively.

      1   I NTRODUCTION
032
033
      Visual feature extraction is a crucial component that underlies many modern vision and multi-modal
      systems (Ramesh et al., 2021; Li et al., 2022; Liu et al., 2024a; Reid et al., 2024; McKinzie et al.,
034
      2024). In recent years, vision foundation models that are pretrained with large-scale datasets (Doso-
035
      vitskiy, 2020; Chen et al., 2022; Radford et al., 2021; Schuhmann et al., 2022) have become the
036   cornerstone for visual feature extraction, powering downstream applications ranging from classifi-
037   cation (Dosovitskiy, 2020), segmentation (Caron et al., 2021), retrieval (Radford et al., 2021), to
038   embodied applications (Driess et al., 2023). Despite the variety of pretraining schemes (Radford
039   et al., 2021; Caron et al., 2021; He et al., 2022; Oquab et al., 2023; El-Nouby et al., 2024), most
040   commonly used vision foundation models, such as CLIP (Radford et al., 2021), are designed to en-
041   code the rich information contained in (a patch of) an image into a single feature vector, wherein
042   this general feature representation is expected to encapsulate all information that may be leveraged
043   by various potential downstream tasks.
044   However, by aiming to extract general-purpose features that can serve as many downstream tasks
045   as possible, image representations obtained from these task-agnostic vision foundation models may
046   inevitably compromise relevant information that is specific to the downstream task of interest. For
047   instance, CLIP models are known to produce image representations that capture the high-level se-
048   mantics well (Radford et al., 2021; Ramesh et al., 2021), but often struggle with understanding the
049   finer-grained details and intrinsics of the image, such as attribute associations, spatial relationships,
050
      camera perspective, and so on (Vaze et al., 2023; Hsieh et al., 2024; Tong et al., 2024b).
051   In this work, instead of aiming to learn a model that produces a fixed image representation in fulfill-
052   ing different goals, we consider learning an adaptive vision foundation model that encodes an image
053   differently conditioned on the downstream task of interest, allowing the resultant image represen-
      tations to prioritize information relevant to the specified condition over other available semantics.

070   Furthermore, as opposed to pre-defining the downstream tasks in a priori (Salehi et al., 2024; Wu
071   et al., 2021), our goal is an adaptive generalist model that is able to adapt to broad potential use
072   cases in a zero-shot fashion. Specifically, we consider utilizing free-form natural language texts as
073   a rich and flexible interface to condition 1 the model given different downstream purposes, inspired
074   by recent literature (Wei et al., 2021; Su et al., 2022; Liu et al., 2024a). For instance, given a task
075   of retrieving images of similar background scene to a given query image, by specifying through the
076
      text condition: “What is the background in the image?”, we expect to guide the model in focusing
      more on the background features of the image, as illustrated in Fig. 1.
077
078   To develop an adaptive vision foundation model, we start out by first identifying two fundamen-
079   tal characteristics required by the model: (1) the ability to comprehend text conditions, and (2)
080   the ability to produce a corresponding image representation with the focus on the given condition.
081   Recent advancement in multi-modal large language models (MLLMs) (Li et al., 2022; 2023; Liu
082
      et al., 2024a; McKinzie et al., 2024; Beyer et al., 2024) have shown that these models possess strong
      capability in understanding both text conditions and images, as demonstrated through their impres-
083
      sive performances on vision question answering tasks. Nonetheless, as these models are inherently
084
      trained with the goal of producing next- (text) token predictions, it is unclear whether they could
085   be used to produce image representations that generally requires encoding not only the semantics
086   but also other low-level dense features. On the other hand, while existing vision encoders (Radford
087   et al., 2021; Oquab et al., 2023), can produce versatile image representations for various downstream
088   tasks, their representations are fixed and can omit information relevant to specific contexts.
089
      In this work, we propose FocalLens, which aligns representations of a pretrained vision-language
090
      model (VLM) leveraging instruction tuning data of MLLMs in a contrastive learning manner, to
091   achieve a text-conditional feature extractor to better “focus” on information relevant to the instruc-
092   tions. We take representative pretrained MLLM and vision encoder, LLaVA (Liu et al., 2024a) and
093   CLIP (Radford et al., 2021), as examples, and turn them into conditional feature extractors via Focal-
094   Lens. We call the resulting models FocalLens-MLLM and FocalLens-CLIP, respectively, as shown
095   in Fig. 2. In both setups, we use a relatively small vision instruction tuning dataset in the form of
096   (instruction, image, output), as considered in prior MLLM studies (Dai et al., 2023; Liu
097   et al., 2024a). We learn to align the visual representations to respect the specified condition.
098   Through extensive evaluations on over 60 tasks, we observe that FocalLens models demonstrate a
099   strong ability to condition representations based on the given text instructions, significantly outper-
100   forming existing baselines like CLIP. On average, FocalLens achieves up to 9 points higher perfor-
101   mance, with even greater improvements on specific tasks, for image-image retrieval tasks. In addi-
102   tion, when used in downstream applications, FocalLens’s conditional image representations further
103   lead to clear gains compared to existing baselines. For instance, on image-text retrieval bench-
104   marks, we show an average improvements of 5 and 10 points respectively on SugarCrepe (Hsieh
105
      et al., 2024) and MMVP-VLM (Tong et al., 2024b), comparing favorably to other CLIP models
      that are much larger (up to 2.5×) in size. On image classification, FocalLens also shows superior

112   2    R ELATED WORK
113
114
      Foundation models for vision encoding. Recently, the computer vision community has witnessed
      a transformative revolution wherein foundation models pretrained with web-scale datasets (Doso-
115
      vitskiy, 2020; Jia et al., 2021; Schuhmann et al., 2022; Oquab et al., 2023) are used as the common
116
      underlying visual feature extractor to produce versatile image representations that drive various
117   downstream applications (Dosovitskiy, 2020; Radford et al., 2021; Ramesh et al., 2021; Kirillov
118   et al., 2023; Zhou et al., 2022). While there are many pretraining objectives (Oquab et al., 2023;
119   He et al., 2022; El-Nouby et al., 2024; Radford et al., 2021), existing schemes typically train the
120   vision models to produce a single “general” image representation that hopefully captures all rele-
121   vant information contained in the given image, or utilize information derived from diverse captions
122   to help learning more discriminative image features (Lavoie et al., 2024). Nonetheless, as an image
123   naturally contains rich and dense information, a fixed and general-purpose representation may not
124   sufficiently pronounce information relevant to specific downstream contexts of interest (Kar et al.,
125
      2024; Wang et al., 2024; Tong et al., 2024b; Hsieh et al., 2024). These observations motivate our
      design of a foundation vision encoder that is capable of extracting different representations from
126
      a single image conditioned on downstream use cases at test-time, different from universal image
127
      embedding approaches that aim to learn a universal model for different domains without explicit
128   conditioning (Google Research, 2023; Ypsilantis et al., 2023).
129
130   Conditional vision representations. Implicit and task-specific conditioning of visual features
131   have been studied in the literature (Liu et al., 2024a; Dai et al., 2023; Tong et al., 2024a; Eftekhar
132   et al., 2023; Vani et al., 2024; Chameleon Team, 2024). For instance, the hidden representations
133   in MLLMs may be interpreted as a type of conditional image representation, where the visual fea-
134   tures are fused with text instructions for producing different output responses (Chameleon Team,
135   2024). Along this line, InstructBLIP (Dai et al., 2023) further designed a mechanism to extract
136   instruction-aware visual features that are shown to better guide MLLMs to focus on relevant visual
137
      features. In the context of embodied AI, conditional (or typically called selective) visual represen-
      tation has also been demonstrated to much improve downstream agent’s performances in navigation
138
      and displacement tasks (Eftekhar et al., 2023). Nonetheless, conditional visual representations con-
139
      sidered in these work are designed and tied in specifically to their model and respective applications.
140   In this work, we are interested in conditional visual representations that may be adopted in broad
141   downstream use cases.
142
143   Vision-language joint representation learning. There is a rich literature in vision-language
144   (joint) representation learning (Lu et al., 2019; Li et al., 2019; Kim et al., 2021; Radford et al., 2021;
145   Jiang et al., 2024). Our work is related as we aim for a model that can comprehend both images and
146   natural language conditions. Concurrent to our work, E5-V (Jiang et al., 2024) considers MLLM’s
147   output space as a universal representation space for both the vision and language inputs. Nonethe-
148   less, in addition to the MLLM-based approach, we investigate an alternative promising CLIP-based
149   approach with comprehensive analysis. Relatedly, composed image retrieval (Wu et al., 2021; Saito
150
      et al., 2023; Zhang et al., 2024) considers developing models of underlying similar capabilities that
      generate image embeddings given both image and text. However, different from our goal to use text
151
      conditioning to extract downstream-specific intrinsic visual features, their goal is to extrinsically
152
      “compose” semantics from both texts and images, largely towards image-retrieval purposes.

159
      able to generalize to broad instructions.

185   utilize contrastive learning (Radford et al., 2021) on the triplet instruction tuning data. Specifically,

187   embedding of output. We call the proposed method as FocalLens, which leverage instruction tuning
188   data to contrastively tune the pretrained image encoder, such that it can better focus on desired infor-
189   mation and generalize to diverse downstream tasks. We explore tuning two different representative
190   vision-language models with FocalLens: MLLMs (Section 3.1) and CLIP (Section 3.2).

192   3.1     F OCAL L ENS WITH MLLM S
193
194   MLLMs (Liu et al., 2024a; Dai et al., 2023) generate textual responses regarding an image based on
195   the given input text instructions. Given (instruction, image), the goal is to generate output.
196
      However, as the original model objective is text generation rather than producing explicit represen-
      tation for downstream tasks, the conditional visual information may be dispersed throughout the
197
      model, and there is no direct access to them by design.
198
199   In FocalLens, instead of training the MLLM to generate output given (image, instruction)
200   as in the original auto-regressive objective, we append a special indicator token <eos token>
201   to MLLM’s input sequence, and consequently train the indicator’s output token to align with the
202
      CLIP text embedding of the targeted output in a constrative manner. Here, we use an off-the-shelf
      frozen CLIP text encoder to obtain the target output embedding. With the contrastive objective, we
203
      encourage the model to condense information relevant to the image, with the specified instructions,
204
      into a single output representation. We show the overall model architecture in Fig. 2a.
205

      3.2     F OCAL L ENS WITH CLIP
208
      Unlike MLLMs, CLIP models by design generate image representations (Radford et al., 2021),
209
      where these image embeddings are already widely utilized in a variety of downstream tasks (Ramesh
210   et al., 2021; Liu et al., 2024a). However, CLIP models are inherently limited to producing a fixed
211   representation for each image, regardless of the downstream task of interest. Although strong in
212   capturing high-level semantics, these general visual features are shown to lack various aspects of
213   fine-grained image details that can be critical for downstream tasks (Hsieh et al., 2024; Tong et al.,
214   2024b). To tackle this, we propose to make CLIP’s vision encoder task-aware, such that it is able
215   to adapt its representations based on specific requirements, thereby capturing specific aspects of the
      image essential for different applications.

      To incorporate natural language instructions into CLIP’s vision encoder, we consider first convert-
217   ing instruction into a “condition text embedding”, which is then treated as an additional token
218   that is fed into the image encoder alongside the standard image tokens and the CLS token. Af-
219   terwards, the model is trained as in standard CLIP using a contrastive loss, aligning the resultant
220   text-conditioned image representations with their corresponding textual outputs. By instruction tun-
221   ing, we aim to allow the vision encoder to generalize to a broad range of scenarios of interest that
222   can be described via natural language at test-time (Wei et al., 2021; Su et al., 2022). We illustrate
223   the FocalLens-CLIP training setup in Fig. 2b.

226
227
      In this section, we first demonstrate the benefits of conditional image representations (Section 4.1)
228
      over the generic representations produced by CLIP, using a toy dataset. We then extensively eval-
229   uate FocalLens models’ capability in characterizing downstream conditions on a variety of tasks,
230   compared to existing baselines (Section 4.2). By zooming in on FocalLens-CLIP, we demonstrate
231   that its conditional image representations improve performance across a range of downstream tasks,
232   including image-text retrieval, image classification, and image-image retrieval (Section 4.4).
233
234   Setup. We train FocalLens models with the visual instruction tuning data used in LLaVA (Liu
235   et al., 2024a). The dataset contains around 150k examples, wherein 60k examples are multi-turn
236   conversations and thus can be treated as multiple triplets of (image, instruction, output),
237   where the image remains the same. During training, we expand conversation data within batches to
238   encourage models to output different representations given the same image but different instructions.
239   For FocalLens-MLLM, we follow the training recipe of LLaVA (Liu et al., 2024a) to obtain a base
240   MLLM before further training with the proposed contrastive loss. For FocalLens-CLIP, we initialize
241
      the base CLIP model with OpenAI’s CLIP-ViT-L-14-336 (Radford et al., 2021), which is also the
      underlying vision encoder used in LLaVA. We initialize the additional text encoder for instructions
242
      to have the same weight as the original text encoder.

      5      C ONCLUSION
526
527
      In this work, we introduced FocalLens, a zero-shot conditional visual embedding model that focuses
528   the representation on specific aspects of the image described in the given text. FocalLens is trained
529   using existing visual instruction tuning datasets to align the conditional image representation with
530   the textual description. Experiments on a comprehensive set of tasks, including image-to-image
531   retrieval, image classification, and image-to-text retrieval, demonstrate that FocalLens matches or
532   exceeds the performance of state-of-the-art models.
533
534   Limitations. Although experiments demonstrate that FocalLens can be effectively trained using
535   visual instruction tuning datasets, model performance could be enhanced by designing customized
536   datasets for this task, which we leave for future study. Moreover, the relatively small scale of the
537   visual instruction tuning datasets may hinder alignment accuracy for highly specialized concepts
538
      that are entirely absent from the dataset.

777   B      BASELINES
778
779   CLIP. We consider CLIP as a task-agnostic vision encoder baseline. In all experiments, we use
780   OpenAI’s CLIP-ViT-L-patch14-336 released checkpoint (Radford et al., 2021). The model size is
781   428M including both vision and text encoder. We consider the same model checkpoint in FocalLens-
782
      MLLM and FocalLens-CLIP.
783
784
      InstructBLIP. InstructBLIP (Dai et al., 2023) is a MLLM that connects a frozen vision encoder,
      CLIP (Fang et al., 2023), to a large language model (LLM) decoder to enable multi-modal capa-
785
      bilities. Specifically, it adopts an instruction-aware Q-former architecture (Li et al., 2023) as the
786
      connector. The Q-former takes in as input the image embedding extracted from the underlying vi-
787   sion encoder, along with tokenized text instructions. Through cross-attention design, the Q-former
788   outputs multiple instruction-aware image tokens to be fed into the LLM decoder. In our experiments,
789   we average over all image tokens to obtain the image representation used in our evaluations. We use
790   the same instructions as in FocalLens for conditioning InstructBLIP.
791
792   MagicLens. MagicLens (Zhang et al., 2024) is a model trained specifically for composed image
793   retrieval with a web-scale 36M-sized dataset. The model takes in both a reference image and natural
794   language text to produce image representations that composes the semantics from both the input
795   image and text. In our experiments, we condition MagicLens model using the same text instructions
796   used for FocalLens.

011                                                 A BSTRACT
012
013            Heterogeneous graphs are composed of multiple types of edges and nodes. The
014            existing heterogeneous graph neural network can be understood as a node feature
015            smoothing process guided by the graph structure, which can accurately simulate
016
               complex relationships in the real world. However, due to real-world privacy and
               data scarcity, some node features are inevitably missing. Furthermore, as model
017
               depth increases and multiple types of meta-paths are aggregated, node embeddings
018
               tend to be consistent, leading to semantic confusion and overfitting problems. To
019            improve the quality of node embeddings, we propose topology-driven residual
020            boosting network (TDR-HGN). It introduces one-hot encoding and node type en-
021            coding to generate initial features, uses topological structure features to guide
022            feature completion, combines residual networks to deal with semantic confusion
023            and over-fitting problems, and builds neighbor-based high-order graph networks
024            through meta-paths to achieve feature enhancement. We conduct extensive experi-
025            ments on three heterogeneous graph datasets, and the results show that TDR-HGN
026            can significantly improve the performance compared to other methods.

029   1    I NTRODUCTION
030
031   Many real-world objects and phenomena can be accurately abstracted into network models, such
032   as traffic networks (Xu et al., 2023) and social networks (Peng et al., 2022; Kumar et al., 2023).
033
      Network representation learning (Zhou et al., 2022) serves as the basis for downstream analysis
      tasks, such as node classification (Yang et al., 2022), node clustering (Wu et al., 2023), and visual-
034
      ization (Zhao et al., 2023a). Its goal is to learn accurate low-dimensional representations of nodes
035
      in the network. Among them, graph neural networks (GNNs) (Ju et al., 2024) are one of the most
036   competitive network representation learning technologies, which have received extensive attention
037   and in-depth research in academia and industry. Heterogeneous graphs contain rich semantics and
038   can model various types of nodes and relationships. Current heterogeneous graph neural networks
039   (HGNNs) follow a message passing framework to learn neighbor features. Although they can ef-
040   fectively extract multiple content and structure features, there are still two challenges that cannot be
041   ignored.
042   Challenge 1: There are certain deficiencies in feature completion (Chen et al., 2020). At present,
043   mainstream HGNN mainly uses manual methods to process nodes with missing features. When
044   processing missing features, there are problems such as insufficient accuracy, inability to capture
045   complex relationships, and poor versatility, resulting in the lack of depth and adaptability of the
046   generated node representation. Recent studies have proposed using pre-trained topological learning
047   (Jiang et al., 2021a) to guide feature completion strategies, but there are still two limitations: over-
048   reliance on pre-training information and ignoring contextual semantic information (Dong et al.,
049   2017; Grover & Leskovec, 2016). The pre-training information obtained based on self-supervision
050
      methods (Jiang et al., 2021b) is difficult to reflect the characteristics and task requirements of specific
      application scenarios, and the generated node representation has negative transfer problems and low
051
      generalization problems.
052
053   Challenge 2: Semantic confusion (Ji et al., 2021). Similar to oversmoothing in GNNs (Wang et al.,
      2022b), semantic confusion means that HGNN injects the semantics of multiple neighbors into

083   node embeddings. Based on this idea, we propose a topology-driven residual enhancement network.
084   First, by combining node category encoding and onehot encoding, we use first-order neighbors to

      learn topological structure features that contain more comprehensive information, use transformer-
      based multi-head attention mechanism and topological structure features for feature completion,

092
             • proposes a residual-enhanced heterogeneous graph network (TDR-HGNN), which uses a

096

101   2.1   H ETEROGENEOUS G RAPH N EURAL N ETWORK
102
103   The HGNN method learns heterogeneous graph embedding from graph structure and node fea-
104
      tures through neural networks.According to the technology adopted in the learning mode, the ex-
      isting HGNN modeling methods are mainly divided into three types: convolution-based methods,
105
      autoencoder-based methods and adversarial-based methods (Bing et al., 2023).
106
107   Convolution-based HGNN processes graph data through multiple layers of stacked heterogeneous
      convolutional layers. Meta-path-based models capture high-order neighborhood features by aggre-


                                                        2
      Under review as a conference paper at ICLR 2025


108
      gating nodes at both ends of meta-paths. Wang et al. (2019) designs hierarchical attention model
109   HAN calculates the importance scores of nodes and meta-paths through node-level and semantic-
110   level attention, and learns node embeddings in heterogeneous graphs based on the scores. MAGNN
111   (Fu et al., 2020) uses linear mapping to solve the dimension mismatch problem and adopts a single-
112   layer GCN to aggregate all nodes in the meta-path. The relation-based model selects the most useful
113   meta-path for downstream tasks by comparing the importance between nodes of different types,
114   providing support for the interpretability of the model. A model with a hierarchical aggregation
115   architecture (Yang et al., 2021) is proposed to dynamically assign edge weights through improved
116   graph convolution kernels and attention mechanisms to accurately aggregate neighbor feature. To
117
      enhance the robustness of node embedding, HGSL (Zhao et al., 2021) uses graph attention network
      and multi-view learning to capture the latent relations in the graph structure. AC-HEN (Wang et al.,
118
      2022a) generates multi-view embeddings through feature aggregation and structure aggregation,
119
      and fills in missing features with the embedding fusion module in the weakly supervised learning
120   paradigm. Meta-path-based methods ignore first-order neighbors and edge types, resulting in in-
121   sufficient learning feature in HG. Relation-based methods require more model parameters to handle
122   complex interactions between multiple types of nodes and edges, increasing space consumption.
123
      Autoencoder-based methods reduce the dimension of input feature through encoders and restore
124
      data to high-dimensional space through decoders, and learn effective embeddings by minimizing
125
      the reconstruction error between input data and restored data. Wang et al. (2021) proposes a het-
126   erogeneous graph attention autoencoder HGATE, which reconstructs node features and edges of
127   heterogeneous graphs through stacked encoder and decoder layers, and captures semantic infor-
128   mation by combining node-level and semantic-level attention. Liu et al. (2023) proposes a new
129   bidirectional encoding HGBER unsupervised framework, which discovers the optimal node distri-
130   bution by introducing the minimization of the clustering constraint objective function L3, making
131   the representation of nodes of different categories more dense and compact. Adversarial methods
132   (Lan et al., 2020; Zhao et al., 2020) utilize the competition mechanism between the generator and the
133   discriminator of the generative adversarial network to improve the quality and diversity of node and
134
      edge generation. Autoencoder-based methods require a lot of computing resources and are overly
      dependent on the original graph structure. Adversarial methods focus on utilizing graph structure
135
      feature and ignore content features. Therefore, comprehensive considerations are needed to design
136
      efficient and accurate models.
137
138
      2.2   FEATURE COMPLETION IN GRAPH NEURAL NETWORKS
139
140   In the real world, due to privacy or other reasons, relationships are partially observed, resulting in
141   incomplete graph structures and missing node features. To learn feature completion in incomplete
142   heterogeneous graphs, HGNN-AC (Jin et al., 2021) uses existing heterogeneous network embedding
143   methods to obtain the topological structure embedding of nodes, guides the weighted aggregation
144   of neighbor node features based on the topological structure feature, and uses the MAGNN (Fu
145   et al., 2020) model to further enhance node embeddings. AC-HEN (Wang et al., 2022a) adopts
146   a multi-view fusion strategy to capture richer feature representations through three views. It uses
147
      the k-nearest neighbor method to select similar nodes in the feature space, uses GCN to aggregate
      neighbor nodes in the structure space, and combines random walk sampling to aggregate high-order
148
      neighbors.
149
150   Li et al. (2023) proposes a heterogeneous residual graph attention network HetReGAT-FC, and de-
151   signed the HetReGAT module through onehot encoding and multi-head attention mechanism. Simi-
152   lar to the idea of HGNN-AC, it uses the HetReGAT module to learn the topological structure feature
153
      of the heterogeneous graph, and uses the attention coefficient obtained from the topological fea-
      ture as a guide for feature completion. HOAE (Li et al., 2024) uses a self-attention mechanism
154
      based on high-level transformers to fill in missing features and uses first-order neighbors to enhance
155
      node embeddings. RA-HGNN (Zhao et al., 2023b) introduces a type conversion matrix to optimize
156   the embedding of heterogeneous network graphs and uses a residual attention network for feature
157   completion. At the same time, Xia Yong’s random dropout method reversely optimizes the feature
158   completion performance. However, HGNN-AC, RA-HGNN, and HetReGAT-FC do not consider
159   high-order neighbors of the same type resulting in suboptimal node embeddings. AC-HEN uses
160   stacked GCN layers for multi-neighbor feature aggregation, which easily leads to over-smoothing
161   problems. HOAE performs feature aggregation based on multiple GAT layers, ignoring the infor-
      mation in the feature space.

350   5.1   I MPLEMENTATION DETAILS
351
      This paper uses three different heterogeneous graph datasets for experiments to comprehensively
352
      evaluate the performance of the model in processing heterogeneous graphs. We compare TDR-
353
      HGNN with ten competitors, among which HAN (Wang et al., 2019), GTN (Yun et al., 2019),
354   MAGNN (Fu et al., 2020), HGSL (Zhao et al., 2021), RoHE (Zhang et al., 2022) and HetReGAT-FC
355   (Li et al., 2023) are convolution-based methods, ie-HG (Yang et al., 2021), AC-HEN (Wang et al.,
356   2022a), RA-HGNN (Zhao et al., 2023b) and HOAE (Li et al., 2024) are encoder-based methods.
357
      We use the same dataset partitioning ratio and meta-path for all models, and take the average results
358
      of five experiments as the final test results. For different models, we select the model layer with
359
      the best performance on the validation set as the baseline to ensure that each model can maximize
360   the advantages of its structure to achieve the best performance. In the experiment, the hidden layer
361   dimension is set to 64, the output layer dimension is set to 16, the dropout rate is set to 0.5, the weight
362   decay is set to 0.001, and the number of attention heads K is set to 8, because 8 attention heads can
363   produce more stable results. For AC-HEN and RA-HGNN, we use MAGNN as the downstream
364   model.

366   5.2   N ODE CLASSIFICATION
367
368   We use SVM for node classification with training rates ranging from 20% to 80%. ACM and IMDB
369
      are node classification datasets with original features, while DBLP is a node classification dataset
      without original features. According to Table1, TDR-HGNN, HOOE, HRG-FC, and RA-HGNN
370
      generally outperform HAN, GTN, MAG, IEHG, and ROHE. This is because the feature completion
371
      module can help the model obtain richer node representations and use structure feature to allevi-
372   ate the sparsity problem of node features. Compared with RA-HGNN, TDR-HGNN has an over-
373   all improvement of 0.6%-3.0%, which may be because RA-HGNN only relies on original feature
374   for feature completion, and the expression ability of nodes is limited. Compared with HRG-FC,
375   TDR-HGNN has an overall improvement of 0.2%-1.0%, which may be because HRG-FC adopts
376   a relationship-based approach, which leads to certain limitations in capturing high-order neighbor
377   functions. Compared with HOAE, TDR-HGNN has an overall improvement of 0.6%-1.0%, which
      may be due to the overfitting and semantic confusion caused by the multi-layer stacking of HOAE.

457   5.4   C OMPATIBILITY
458
459   In order to evaluate the compatibility of different feature completion methods with heterogeneous
460
      graph models, we selected two mainstream heterogeneous graph models, MAGNN and ie-HGCN,
      for node classification tasks. Four different feature completion methods, avg, HetReGAT-FC,
461
      HOAE, and TDR-HGNN, were used in the experiments on the ACM dataset.
462
463   The results show that the average interpolation strategy weakens the quality of embedding and per-
464   forms poorly in downstream tasks. Compared with MAGHRG-FC and MAGHOAE , the performance of
465   our feature completion model is improved by nearly 0.5% to 1.5%. Compared with ie-HGHRG-FC and
466
      ie-HGHOAE , ie-HGTDR-HGNN also has an improvement of nearly 0.5%. This is because the HRG-FC
      model uses topological feature to complete feature completion, focusing on the connection relation-
467
      ship between nodes, while ignoring the subtle differences between node features. The HOAE model
468
      takes into account the local contextual feature of nodes, but cannot fully capture the complex patterns
469   of graph structures. The results show that the feature completion module in the TDR-HGNN model
470   can adapt well to most heterogeneous graph neural networks and show significant compatibility.

      A.1   P RELIMINARY
651
652   Problem 1: Heterogeneous Graph Embedding. For a given heterogeneous graph G, the goal of
653   node embedding is to learn a mapping function f : V → Rd : v → h, where d ≪ |V |. The function
654   f aims to accurately reflect the connection relationship between nodes.
655
      Problem 2: Feature Completion. Given a heterogeneous graph G = (V, E, T, R) as input, the goal
656
      of feature completion is to learn a mapping function f1 (G, A, h) : h → h+ . h denotes the node with
657   missing features, and h+ denotes the node with completed features. The function f1 aims to make
658   h+ close to real features.
659
660
      Problem 3: feature Enhancement. Given a heterogeneous graph G = (V, E, T, R) as input, feature
      enhancement aims to learn a mapping function f2 (G, A, h) : h → z, where z denotes the enhanced
661
      node feature. Through function f2 , the node feature z can learn the neighbor features and global
662
      structural feature.

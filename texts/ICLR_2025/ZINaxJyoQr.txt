011                                              A BSTRACT
012
013            Barlow Twins is a feature-contrastive self-supervised learning framework built on
014            the principle of redundancy reduction. The idea is to train a network by maximiz-
015            ing the correlation between corresponding features and minimizing the correlation
016            between non-corresponding features in distorted views of the same image, through
017            this facilitating effective pretraining of a backbone network for a subsequent clas-
018
               sification head. This is achieved by diagonalizing the cross-correlation matrix of
               the network’s representations and scaling it towards the identity matrix. We show
019
               that the cross-correlation matrix of distorted images is inherently symmetric, in-
020
               dependent of the backbone network’s weights, which leads to two key insights:
021            (i) the cross-correlation matrix can always be diagonalized using a linear trans-
022            formation (layer), and (ii) the core idea of maximizing correlations between cor-
023            responding features while minimizing them for non-corresponding features alone
024            is insufficient for effective backbone network pretraining. Nevertheless, Barlow
025            Twins provide highly effective pretraining. We show that this is due to the nor-
026            malization of the cross-correlation matrix in the Barlow Twins cost function. This
027            normalization leads to minima of the cost function which are equivalent to the
028            minima of sample contrastive approaches to enforce invariance.

031   1   I NTRODUCTION
032
033   In self-supervised learning, the goal is to learn meaningful representations without relying on la-
034   bels, which can be costly to obtain, especially for large datasets. Approaches like SimCLR (Chen
035   et al., 2020), SwAV (Caron et al., 2020), or SimSiam (Chen & Kaiming He, 2021) show that self-
036   supervised methods can produce strong representations which achieve competitive results compared
037   to supervised approaches. These approaches are often called sample-contrastive, as they make sam-
038   plewise comparisons between different instances. Another class of self-supervised learning ap-
039
      proaches is called feature-contrastive, which works by comparing different instances at the feature
      level rather than the sample level. Examples of these methods are the Barlow Twins (Zbontar et al.,
040
      2021), VICReg (Bardes et al., 2022), VICRegL (Bardes et al., 2024), or W-MSE (Ermolov et al.,
041
      2021). A significant advantage of all of these methods lies in their ability to harness the inherent
042   structure of unannotated data.
043
044   The Barlow Twins framework (Zbontar et al., 2021) introduced feature-contrastive learning,
045
      grounded in the principle of redundancy reduction in neural representations, initially proposed by
      H. Barlow (Barlow et al., 1961). The Barlow Twins approach minimizes the distance between a
046
      modified cross-correlation matrix and the identity matrix in order to extract representations with
047
      decorrelated feature dimensions. In Zbontar et al. (2021), a ResNet-50 f (He et al., 2016) is adapted
048   by deleting the fully connected layer and applying a projector network, which is a large multilayer
049   perceptron (MLP) p. In Zbontar et al. (2021), this MLP has three linear layers, each with 8192 output
050   dimensions and no biases. The first two layers are followed by a batch normalization layer as well
051   as a rectified linear unit (ReLU). Figure 1 shows a schematic overview of this architecture. Note that
052   the projector network p has approximately 2048 · 8192 + 2 · 81922 = 150,994,944 ≈ 151 million
053   parameters (the two batch normalization layers have negligible numbers of learnable parameters)
      whereas the standard ResNet-50 architecture utilizes only 25.6 million parameters.

094   An intuitive explanation for the loss function is an invariance term for the first sum and a redundancy
095   reduction term for the second sum. The invariance term produces embeddings that are ”invariant”
096   to distortions T A , T B in the sense that an image feature encoded by an embedding dimension shall
097   highly correlate in different distortions of the same image. Additionally, it avoids the trivial solu-
098
      tion of all features being zero. The redundancy reduction term decorrelates different embedding
      dimensions (features) in order to avoid encoding similar image properties in multiple dimensions.
099
      Additionally, it avoids the trivial solution of all features being constant.
100
101   After the pretraining phase is concluded, a simple linear classification head is trained via supervised
102   learning. In this phase, the ResNet-50 backbone’s parameters are frozen and not further updated, and
103   the projector network is discarded. The classification head is a linear layer that takes in the feature
104
      vectors that were extracted by the backbone and assigns a class label. A different interpretation is
      that the ResNet-50 backbone transforms the images into an embedding space where, in the best case,
105
      the features are linearly seperable such that a linear layer can easily classify the respective image.
106
107   The Barlow Twins framework has been successfully applied in different scenarios. Bielak et al.
      (2022) presented the application of the framework for graph neural networks, where the encoder

121   The mentioned approaches consistently yield results comparable to or surpassing the state-of-the-
122   art, all achieved with a relatively small labeled dataset. In the following, we will demonstrate that
123
      this effectiveness of the Barlow Twins approach, in fact, is not achieved by Barlow’s redundancy
      principle but rather by enforcing invariance through the normalization in equation 1. In Garrido et al.
124
      (2023) the authors analysed the the connection between sample- and feature-contrastive learning by
125
      focussing on the non-diagonal elements of the cross-correlation matrix. Here we focus on the role
126   of the diagonal elements.

230   4   C ONCLUSION
231
232   We demonstrated that the core idea behind Barlow Twins — reducing redundancy by maximiz-
233   ing the correlation between corresponding features and minimizing the correlation between non-
234
      corresponding features in distorted images — is insufficient on its own for pretraining a backbone
      network. The cross-correlation matrix of arbitrary outputs, even from untrained backbones, is sym-
235
      metric and, hence, can be diagonalized already by a linear projection head. A multilayer perceptron
236
      (MLP) as a projection head is even more expressive at achieving diagonalization than a simple linear
237   layer. However, since non-zero eigenvalues of the cross-correlation matrix are enforced, it is ensured
238   that the backbone output spans the entire space, preventing degenerate solutions such as constant or
239   zero output. This is one of the conditions for which usually explicit terms in the cost function are
240   used. For example, VICReg (Bardes et al., 2022) employs a term in the cost function which enforces
241   non-zero variance.
242
      The primary objective of self-supervised learning is to achieve invariance in the backbone output
243   for perturbed images. Approaches like VICReg and other contrastive methods explicitly incorporate
244   terms like 8 in the cost function. We showed that the Barlow Twins method achieves the same
245   invariance by using the Pearson correlation matrix, which inherently includes normalization. This
246   normalization is crucial, as it enforces the same minima in the cost function as the explicit term 8.
247   Without this normalization, the Barlow Twins loss would not yield meaningful solutions.
248
      In conclusion, Barlow Twins offer an elegant approach to achieving the key objectives of self-
249   supervised learning: They ensure invariance in the network output to image perturbations like ex-
250   plicit terms used in contrastive approaches, while simultaneously avoiding trivial solutions like con-
251   stant or zero output, by simply diagonalizing the Pearson correlation matrix with non-zero diagonal
252   elements.

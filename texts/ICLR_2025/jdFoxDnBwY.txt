009                                               A BSTRACT
010
011            Recognizing object parts from images plays a pivotal role in various real-world
012            applications. However, existing work mostly learn models from large-scale 2D
013            part annotations. In this paper, we propose a part recognition model that can rec-
014            ognize 3D parts from a 2D image with only annotations of parts on one 3D mesh
015            model for each object category. Specifically, we build a category-level 3D fea-
016            ture bank for meshes that could overcome geometric variance among objects and
017            precisely align with diverse 2D images of this object category. To achieve this,
018            we propose to learn two types of correspondence. Firstly, we learn mesh-to-mesh
019
               correspondence between distinct 3D mesh models by matching geometry-aware
               features, which allows us to create a shared 3D feature bank for this object cate-
020
               gory. Secondly, we establish mesh-to-image correspondence by aligning features
021
               in the 3D feature bank with features extracted from 2D images. During inference,
022            given a single image, our method recognizes 3D object parts via a Render-and-
023            Compare approach. It predicts object parts by gradient-based optimizing each
024            part’s 3D configuration, minimizing a feature-level reconstruction loss between
025            the projected 3D features and the image features while ensuring geometric con-
026            sistency between object parts. The position, rotation, and shape of each part are
027            optimized to match the cues from the image, thus recognizing the 3D parts from a
028            2D image. Experiments on VehiclePart3D, PartImageNet, and UDA Part dataset
029            show our method outperforms baselines significantly for 2D part segmentation
030
               and pioneering 3D part recognition from a single image.

033   1   I NTRODUCTION
034
035   Object part recognition from images plays an important role in real-world applications, e.g., au-
036   tonomous driving and embodied AI. Existing work (Liu et al., 2021; Tritrong et al., 2021) achieves
037   object part recognition by learning segmentation models from large-scale image sets with parts an-
038   notated on each 2D image pixel. Unfortunately, annotating pixel-level object parts on large-scale
039
      training sets is tedious and requires huge efforts of human work. Moreover, current part recogni-
      tion approaches are specialized to identify the exact part definition from the training labels whereas
040
      different vision tasks require different object part definitions, e.g., robotics needs to detect inter-
041
      actable parts like handles (Ainetter & Fraundorfer, 2021), and traffic cameras need to detect parts
042   like car doors (Morris & Trivedi, 2008). The various demands of part definition further narrow the
043   potential applications of current approaches. While defining object parts at a more fine-grained level
044   and merging them differently in different applications is a possible solution, it makes the annotation
045   process almost impractical.
046
      Cognitive psychology studies suggest that humans recognize objects as a composition of simple
047
      geometric components in 3D space (Biederman, 1987). Therefore, annotating parts and building
048   a part recognition model in 3D space may be a natural solution to the data annotating challenge.
049   Annotated parts in 3D space are more informative since all parts are visible and avoid the ambiguity
050   of the part annotation on 2D images. In this paper, we aim to recognize 3D object parts from a single
051   RGB image by inferring the 3D configuration of each object part. Moreover, our model learns from
052   a single 3D annotation for each object category, which makes the annotation process much faster
053   compared with 2D approaches, and addresses the part definition issue by just alternating the part
      definition mesh under different applications without training again.

070
      Conducting part recognition from only a single 3D mesh annotation will introduce two challenges:
071
      (1) The geometry of different objects in one object category can vary a lot, e.g., different shapes of
072   airplanes. Thus, finding a commonly shared representation of diverse shapes is essential. (2) We
073   need an inference algorithm, which bridges the parts on 3D meshes with the pixels on 2D images.
074
075
      To solve these challenges, our idea is to form a part compositional neural mesh for the annotated 3D
      object, a representation in latent space that attaches features to each vertex of the mesh. The features
076
      are shared across different objects in the object category and are aligned with pixels in 2D. We
077
      further make the neural mesh deformable to overcome the shape variance among different objects.
078   The feature sharing and deformation make our part recognition method a category-level paradigm
079   and the 3D-to-2D alignment facilitates the render-and-compare inference algorithm on images.
080
081
      To build such a neural mesh, we propose to learn two types of correspondence. First, we learn
      mesh-to-mesh correspondence among 3D meshes, which means for any vertex on one mesh, we
082
      can find the most geometrically similar vertex on any other mesh within the object category, thus
083
      these vertices could share the same feature in the category-level neural mesh. The correspondence
084   is obtained by matching geometry descriptors learned in a self-supervised manner. Based on the
085   correspondence, we further train a deformation network that could reshape the neural mesh into
086   diverse geometries given different shape latent. Second, we establish a mesh-to-image correspon-
087   dence, which aligns the 3D features on the neural mesh with the 2D features extracted from images.
088   We first use a diffusion model prompt by meshes and camera configurations (Ma et al., 2023) to
089   create a training set of semi-realistic images. Note that this process provides accurate ground truth
090   poses and 3D meshes of objects in the generated images. Using the generated images, we learn the
091   mesh-to-image correspondence by training an image feature extractor to precisely match the features
092
      on neural mesh vertices with features extracted from the pixels that the vertices are projected on.
093   For inference, as shown in Figure 1, given a testing image, we extract image features using the
094   trained feature extractor and conduct feature level render-and-compare to optimize the 3D configu-
095   ration of each part in the neural mesh to minimize a feature space reconstruction loss. Specifically,
096   the 3D configuration of parts includes location, rotation, scale, and shape. For the location, rotation,
097   and scale, we apply gradient descent on them to search for the optimal 3D pose and size. As for the
098
      shape, we optimize the shape latent, given that the trained deformation network will deform each
      part mesh to fit the geometries in the image. Also, the optimization involves a geometry consistency
099
      loss between object parts as additional constraints. Note this optimization process allows each part
100
      to have its own freedom for optimization while keeping the holistic spatial relations.
101
102   We evaluate our approach on both 2D and 3D part detection tasks. Our learned model can be
103   evaluated by standard 2D object part segmentation tasks by simply projecting the reconstructed 3D
104
      parts on the image plane. We compare our approach with advanced approaches for one-shot 2D part
      segmentation on real-world datasets. Besides, we conduct experiments to infer 3D object parts from
105
      images and measure the accuracy. Both quantitative and qualitative results show the effectiveness of
106
      our framework in recognizing 3D parts from 2D images with only one annotation.
107
      Our contributions can be summarized as follows:

122   2   R ELATED W ORK
123
124   Learning Object Parts in 3D. Learning 3D parts from 3D inputs, e.g., pointclouds, has been widely
125   explored. The 3D semantic segmentation focuses on grouping 3D points into parts. Previous works
126   have explored many effective network architectures (Qi et al., 2017a;b; Yu et al., 2019; Shi et al.,
127   2020; Zhang et al., 2022) and training methods (Landrieu & Simonovsky, 2018; Afham et al., 2022;
128   Liu et al., 2023; Zhang et al., 2023b) to improve the ability of models to recognize 3D parts. Another
129   important area is the 3D part discovery, which involves decomposing 3D objects into self-defined
130
      parts. This task is critical for applications in reconstruction, assembling, and canonicalization. (Xu
      et al., 2019; Luo et al., 2020; Sun et al., 2021; Koo et al., 2022). However, a common limitation of
131
      these approaches is their reliance on 3D observations. As the field progresses, there is an increasing
132
      trend towards 3D object inference from single images (Wang et al., 2022). Recognizing this trend,
133   our work intends to explore the task of detecting 3D parts from a single image.
134
135   2D Object Part Segmentation. Semantic object part segmentation is a long-standing problem in
136
      computer vision. The pioneering work Pictorial Structure (Fischler & Elschlager, 1973) along with
      following works (Weber et al., 2000; Felzenszwalb & Huttenlocher, 2005; Fei-Fei et al., 2006; Zhu
137
      & Mumford, 2007; Girshick et al., 2011) explicitly model parts and their spatial relations. These
138
      methods share a common topic that the object-part models provide rich representations of objects
139   and help interpretability. However, in the era of deep learning with data-driven models, research
140   on part segmentation gets hindered due to the lack of large-scale datasets. As a result, most recent
141   works (Hung et al., 2019; Choudhury et al., 2021; Liu et al., 2021; Tritrong et al., 2021; Gadre et al.,
142   2021; Ziegler & Asano, 2022; Saha et al., 2022) focus on unsupervised or self-supervised co-part
143   segmentation. There are also some works (Liu et al., 2022; Peng et al., 2023) leveraging synthetic
144   data and domain adaptations on part segmentation. In this paper, we explore a novel direction to
145   solve 2D part segmentation by detecting the object parts in 3D space and projecting the 3D parts to
146   generate 2D segmentations, significantly improving the data efficiency and performance.
147
148   3   M ETHOD
149
150   We formulate the object part recognition problem as establishing a part compositional neural
151   mesh (Section 3.1) that represents the annotated object as a combination of deformable object parts
152   with features on mesh vertices.
153
      To achieve this, as shown in Figure 2, we build mesh-to-mesh correspondence and mesh-to-image
154
      correspondence. The mesh-to-mesh correspondence (Section 3.2) is defined as: for each vertex on
155
      one mesh, which vertices of other meshes have the highest geometric similarities. This correspon-
156   dence allows us to establish the category-level shared features, which means corresponding vertices
157   could share the same feature in the neural mesh. We also train a deformation network based on
158   the correspondence. The mesh-to-image correspondence (Section 3.3) learns the alignment of mesh
159   vertices with image pixels, measured by the similarities between features on the neural meshes and
160   the features extracted from the images. We use contrastive learning to match the 3D features on
161   vertices with their projected pixels. Such a formulation makes detecting 3D parts from 2D images
      approachable by differentiating the process of rendering the neural mesh into 2D feature maps.

476   5     C ONCLUSIONS
477
      We present Part321 to address the challenge of recognizing 3D object parts from a 2D image using
478
      one-shot annotation. Our method establishes a part compositional neural mesh by introducing two
479
      correspondences: mesh-to-mesh correspondence, enables sharing features in the same category;
480   and mesh-to-image correspondence, utilizing features on mesh vertices to align 3D parts with
481   2D images. We also propose a deformation network to reshape the parts into diverse geometries.
482   Building upon the learned part compositional neural mesh, we propose an inference pipeline capable
483   of predicting the 3D configuration of object parts from a single image with Render-and-Compare
484   method. Our experiments on 2D segmentation show that our method outperforms state-of-the-art
485   2D segmentation approaches with pseudo-labeling. The experiments on 3D recognition demonstrate
      the effectiveness of our framework on pioneering one-shot 3D part detection from a single image.

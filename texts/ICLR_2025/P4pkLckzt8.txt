009                                              A BSTRACT
010
011            Graph Neural Networks often struggle with long-range information propagation
012            and local heterophilous neighborhood aggregation. Inspired by the observation
013            that cluster patterns manifest at global and local levels, we propose to tackle both
014            challenges with a unified framework that incorporates a clustering inductive bias
015            into the message passing mechanism, using additional cluster-nodes. Central to
016            our approach is the formulation of an optimal transport based clustering objective.
017            However, optimizing this objective in a differentiable way is non-trivial. To
018
               navigate this, we adopt an iterative process, alternating between solving for the
               cluster assignments and updating the node/cluster-node embeddings. Notably,
019
               our derived optimization steps are themselves simple yet elegant message passing
020
               steps operating seamlessly on a bipartite graph of nodes and cluster-nodes. Our
021            clustering-based approach can effectively capture both local and global information,
022            demonstrated by extensive experiments on heterophilous and homophilous datasets.

025   1   I NTRODUCTION
026
027   Graph Neural Networks (GNNs) have emerged as prominent models for learning node representations
028   on graph-structured data. Their architectures predominantly adhere to the message passing paradigm,
029   where node embeddings are iteratively refined using features from its adjacent neighbors (Kipf and
030   Welling, 2016; Defferrard et al., 2016; Gilmer et al., 2017). While this message passing paradigm
031   has proven effective in numerous applications (Ying et al., 2018; Zhou et al., 2020), two prominent
032   challenges have been observed. First, long-range information propagation over a sparse graph can
033
      be challenging (Li et al., 2018; Zhou et al., 2021; Rusch et al., 2023). Expanding the network’s
      reach by increasing the number of layers is often suboptimal as it could encounter issues such as
034
      over-squashing (Alon and Yahav, 2020; Topping et al., 2021), where valuable long-range information
035
      gets diluted as it passes through the graph’s bottlenecks, diminishing its impact on the target nodes.
036   Second, some graphs exhibit heterophily, where connected nodes are likely to be dissimilar. In such
037   cases, aggregating information from the dissimilar neighbors might introduce noise and hinder the
038   graph representation learning performance (Zhu et al., 2020b; 2021).
039
      In this paper, we focus on the task of supervised node classification using GNN and explore clustering
040
      as an inductive bias to address both challenges. Our approach is motivated by the observation
041
      that cluster patterns can be utilized at both global and local levels in graph data. Globally, cluster
042   patterns appear when nodes that are far apart in the graph exhibit similar features (see Fig. 1).
043   These patterns can be leveraged to enable efficient long-range information transfer, by clustering
044   nodes by their latent space representations, rather than structural proximity. Locally, particularly
045   for heterophilic neighbourhoods, it would be desirable to disconnect edges across dissimilar nodes
046   while maintaining connections across similar nodes. Clustering nodes within local neighbourhoods
047   provides a mechanism for this.
048   To embed the clustering inductive bias explicitly into the network architecture, we propose Dif-
049   ferentiable Cluster Graph Neural Network (DC-GNN), an efficient end-to-end learning framework
050   designed to address both over-squashing and heterophily. We first formulate the problem of inferring
051   cluster-aware node representations as an optimization task with a novel clustering-based objective
052   function. This objective function is composed of a weighted sum of global and local clustering terms,
053   which promote long-range information propagation and effective aggregation in heterophilic neighbor-
      hoods, respectively. A key challenge in integrating clustering into an end-to-end learning framework

067
      define the objective function that seeks to move node representations to their cluster-centroids through
      the lens of an Optimal Transport (OT) problem (Villani et al., 2009). Unfortunately, optimizing
068
      this clustering-based objective function as a conventional training loss is inherently difficult due to
069
      the latent nature of cluster assignments. To overcome this, we propose an alternating optimization
070   approach based on block coordinate descent which iterates between (1) solving for a soft cluster
071   assignment matrix that probabilistically assigns a node to a cluster, and (2) updating the node/cluster
072   embeddings given the cluster assignment matrix.
073
      Intriguingly, this iterative alternating optimization algorithm for minimizing the clustering-based
074
      objective function can be interpreted as an iterative message passing procedure. It operates on a
075
      bipartite graph consisting of original nodes and cluster nodes representing centroids. Unlike previous
076   approaches that treat clustering as a separate component, our method directly embeds the clustering
077   process into the message passing network architecture. This ensures the clustering-based objective
078   function is optimized as part of the message passing network itself, enforcing the clustering inductive
079   bias during both training and inference. The resulting cluster-aware node embeddings are then fed into
080   a task-specific loss for supervised node classification, allowing DC-GNN to be trained end-to-end.
081
      DC-GNN is efficient and has a linear complexity with respect to the graph size. Additionally, our
082   framework can be viewed as a form of graph rewiring, where the introduction of cluster nodes
083   creates new pathways between original nodes. This rewiring reduces the overall graph’s effective
084   resistance, helping to mitigate oversquashing (Black et al., 2023), as shown in our experiments. To
085   assess the effectiveness of DC-GNN, we conduct extensive evaluations on 14 datasets, spanning both
086   heterophilous and homophilous graphs. Our results demonstrate that DC-GNN consistently achieves
087   superior or competitive performance compared to state-of-the-arts.

      2    R ELATED W ORK
091
092   Prominent GNN models typically follow a message passing paradigm that iteratively aggregates
093
      information in a node’s neighborhood (Kipf and Welling, 2016; Bruna et al., 2013; Defferrard et al.,
      2016; Gilmer et al., 2017; Veličković et al., 2017; Xu et al., 2018). This local message passing,
094
      however, requires the stacking of multiple layers to pursue long-range information and can encounter
095
      issues such as over-smoothing (Li et al., 2018; Cai and Wang, 2020; Rusch et al., 2023) and over-
096   squashing (Alon and Yahav, 2020; Topping et al., 2021; Banerjee et al., 2022; Karhadkar et al., 2022).
097   To tackle oversquashing, most existing works design graph rewiring techniques that change graph
098   topology (Topping et al., 2021; Nguyen et al., 2023; Arnaiz-Rodríguez et al., 2022; Karhadkar et al.,
099   2022). Other works like Chen et al. (2024) leverages random walk sequences and Kosmala et al.
100   (2023) leverages Fourier basis representation. Similar to some existing methods (Black et al., 2023),
101   our approach of adding cluster-nodes also changes graph topology and is shown to reduce effective
102   resistance, thereby helpful in mitigating oversquashing (Black et al., 2023).
103   Additionally, some graphs contain heterophilous neighborhoods, in which traditional aggregation
104   promoting similarity among neighbors is suboptimal. (Zhu et al., 2020b; 2021). There are three types
105   of approaches to address this, including design of high-pass filters in message passing (Chien et al.,
106   2020; Fu et al., 2022; Dong et al., 2021), exploring global neighborhoods (Xu et al., 2022; Jin et al.,
107   2021; Li et al., 2022; Abu-El-Haija et al., 2019), and use of auxiliary graph structures (Pei et al.,
      2020; Zhu et al., 2020a; Lim et al., 2021; Yan et al., 2021). Many of these approaches are often


                                                               2
      Under review as a conference paper at ICLR 2025


108
      computationally expensive and may struggle on homophilous graphs. Our method seeks to explore
109   global neighborhoods by introducing cluster-nodes as auxiliary structures and conduct clustered
110   aggregation in local neighborhoods, with linear complexity.
111
112
      Our work focuses on using a clustering inductive bias to enhance the supervised node classification
      task. This differs from tasks like graph clustering (Tsitsulin et al., 2023; Tian et al., 2014) and graph
113
      pooling (Bianchi et al., 2020; Duval and Malliaros, 2022), which are constrained by graph typology
114
      and aim to partition a graph into substructures. Instead, our approach primarily utilizes feature
115   information for global clustering. Unlike previous methods, we also explore clustering within local
116   neighborhoods, which has not been explored before.
117
118
      Clustering within a differentiable pipeline has been explored particularly in unsupervised and self-
      supervised settings (Feng et al., 2022; Saha et al., 2023; Caron et al., 2018; Stewart et al., 2024).
119
      However, most of these approaches do not extend to graph-structured data. Few methods, such as
120
      DCAT (Zhou et al., 2024), have applied differentiable clustering in graph-based models, typically
121   employing clustering as an auxiliary loss function. In contrast, our approach directly integrates clus-
122   tering into the message-passing mechanism. By deriving the message-passing steps as optimization
123   steps toward a clustering objective, we effectively embed a clustering algorithm into the model’s
124   architecture. This design enables clustering to be performed both during training and inference.
125
      Our approach, which integrates clustering into message passing, is realized through an Optimal
126
      Transport-based clustering objective. OT has been recently applied to graph learning for tasks like
127   graph classification (Titouan et al., 2019; Bécigneul et al., 2020; Vincent-Cuaz et al., 2022; Ma et al.,
128   2024), regularizing node representations (Yang et al., 2020) and finetuning (Li et al., 2020). However,
129   most of these methods leverage OT as a separate component and do not integrate it within message
130   passing. Distinct from these approaches, we implicitly optimize an OT-based clustering objective
131   function via message passing.

135
136
      In this section, we introduce our unified clustering-based GNN message passing method to address
      both long-range interactions and heterophilous neighborhood aggregation. This involves transforming
137
      the input graph into a bipartite graph by introducing cluster-nodes, defining an optimal transport
138
      (OT) based clustering objective function, and optimizing it through our derived message passing
139   steps within a differentiable coordinate descent framework. Notations are introduced where needed
140   throughout the paper. A complete list of these notations is available in Appendix A.

      3.1   DC-GNN FORMULATION
143
144   We begin by constructing a bipartite graph, denoted as G = (V, C, E). This bipartite graph is derived
145   from the original graph G = (V, E) and comprises two distinct sets of nodes. The first set, V, is
146   a direct copy of the nodes V from the original graph. The second set, C, consists of cluster-nodes
147   divided into two categories: global clusters (Ω) and local clusters (Γ). E and E represent the set of
148   edges in the bipartitie graph and original graph respectively.
149
      In this bipartite graph, each global cluster-node from Ω connects to all nodes in V, thereby facilitating
150   long-range interactions across distant nodes. Meanwhile, each local cluster-node from Γ is associated
151   with a specific node i in V, and connected to its ego-neighborhood. The ego-neighborhood of node
152   i, denoted as Ni+ , includes i itself and its one-hop neighbors, formally defined as Ni+ = Ni ∪ {i},
153   where Ni = {j : (i, j) ∈ E}. Here, Γi represents the set of local clusters associated with node
154   i in V. The P total number of nodes in C equals the nodes in Ω plus those in all local clusters, i.e.,
155   |C| = |Ω| + i∈V |Γi |. An illustration of the bipartite graph construction is provided in Appendix C.
156   Following the bipartite graph construction, we propose our Differentiable Cluster Graph Neural
157   Network (DC-GNN) to learn the node embeddings. An illustration of DC-GNN architecture is
158   presented in Fig. 2. Given Xinput as input node features, we first transform it with an MLP to
159   produce X. The transformed features together with the bipartite graph G are then put through an
160   iterative DC-MsgPassing algorithm to produce the embeddings of nodes Z and cluster-nodes C. A
161   learnable readout function such as a simple MLP, is then used on Z to produce the class probabilities Y
      in our node classification task. Then, DC-GNN is trained end-to-end with a task-specific loss function

445   4.3    B ENEFIT OF CAPTURING LONG RANGE INFORMATION IN SPARSE LABEL SETTINGS
446
447   Our method introduces shortcuts between distant nodes via the cluster-nodes. To empirically evaluate
448   the effects of shortcut construction, we consider a generalized scenario on homophilous graphs where
449   information from labeled nodes needs to propagate over a long distance to reach most unlabeled
450   nodes. Specifically, we evaluate our method with only a small number of training labels per class.
451   Our hypothesis is that information propagation becomes more challenging when useful training
452   information is more scarce, making the effects of shortcut construction more pronounced in sparsely-
453   annotated graph datasets. The hypothesis is supported by experiment results in Tab. 3, where our
454   method outperforms other methods on Cora and Pubmed by substantial margins. We have also
455   conducted low label rate experiments on heterophilous datasets, with just 5% of training labels. As
456   shown in Tab. 4, DC-GNN continues to outperform the baselines across four heterophilous datasets,
457   with a notable 4% improvement on Penn94.

      prompting the development of various graph rewiring strategies to diminish Rtot within the underlying

474   cluster-nodes. This effectively creates new pathways among the original nodes, thereby reducing the
475   graph’s Rtot and aiding in mitigating the oversquashing issue (Black et al., 2023). To validate this, we
476   conduct an empirical analysis of the total pairwise effective resistance among the original nodes in
477   our bipartite graph, with varying number of global and local clusters. Fig. 3a and Fig. 3b display a
478   heatmap of Rtot , with darker shades representing higher Rtot values. The results indicate that Rtot
479   decreases sharply as we increase the number of both global and local clusters. In addition, we also
480   perform experiments on synthetic random graphs with varying degrees of sparsity and show that the
481   bipartite graph construction reduces Rtot on these graphs. Details can be found in Appendix E.1.
482   To further validate the oversquashing mitigation capability, we conduct experiments on Tree-

519   5     C ONCLUSION AND FUTURE WORK
520
521   This paper tackles the dual challenges in Graph Neural Networks: capturing global long-range
522   information and preserving performance in heterophilous local neighborhoods. We proposed a novel
523   differentiable framework that seamlessly embeds a clustering inductive bias into the message passing
524   mechanism, facilitated by the introduction of cluster-nodes. At the heart of our approach is an optimal
525   transport based implicit clustering objective function whose optimization presents a considerable
526   challenge. We addressed this through an iterative optimization strategy, alternating between the
527
      computation of cluster assignments and the refinement of node/cluster-node embeddings. Importantly,
      the derived optimization steps effectively function as message passing steps on the bipartite graph.
528
      The message passing algorithm is efficient and we show that it is guaranteed to converge. The efficacy
529
      of our clustering-centric method in capturing both local nuances and global structures within graphs
530   is supported by extensive experiments on both heterophilous and homophilous datasets.
531
532   Finally, we identify some limitations and future directions of our approach. Firstly, our work is
533
      motivated by the node classification task. The alignment of our proposed clustering inductive bias
      with graph-level tasks is somewhat unclear. We conduct some preliminary experiments with mixed
534
      results (Appendix E.4). To better align with graph-level tasks, one possible direction is to share
535
      clusters across different graphs. Secondly, the idea of embedding clustering inductive bias could
536   potentially be extended to hierarchical clustering, which might be beneficial on very large graphs.

       E.5   W HEN WILL GLOBAL CLUSTERING HELP ?
1246
1247   we posit that global clustering is particularly beneficial when intra-class nodes exhibit strong clustering
1248   tendencies in feature space. One way of quantifying this is graph conductance. Graph conductance is
1249   a measure of how well-connected a subset of nodes is to the rest of the graph relative to its internal
1250   connectivity. Specifically, it measures the ratio of the number of edges that cross the boundary of a
1251   set to the minimum of the number of edges in the set or its complement. Formally, the formula for
1252   conductance Φ(S) of a subset S of nodes is given by:
1253
1254                                                     |cut(S, S)|
1255                                       Φ(S) =                        ,
                                                     min(vol(S), vol(S))
1256
1257   where |cut(S, S)| is the number of edges between the set S and its complement S (the rest of the
1258   graph), vol(S) is the sum of the degrees of the nodes in S, and vol(S) is the sum of the degrees of
1259   the nodes in S. Lower conductance values indicate that the set S is well-clustered, meaning it has
1260   relatively few connections to the rest of the graph, suggesting a strong internal cohesion within the
1261   cluster.
1262   To measure conductance in the feature space, we construct a k-nearest neighbor (k-NN) graph based
1263   on the node features. In this graph, each node is connected to its k nearest neighbors according to
1264   feature similarity, rather than graph topology. We then measure conductance on this k-NN graph,
1265   using the same formula as above. For our experiments, we set k = 5 and report the average
1266   conductance across classes when there are more than two classes.

1300   The description of each dataset is as follows.
1301   Roman-empire is based on the Roman Empire article from English Wikipedia. Each node in the
1302   graph represents one word in the text, and each edge between two words represents either one word
1303   following another word or if the two words are connected in the dependency tree. Node features is its
1304   FastText word embeddings. The task is to predict a node’s syntactic role.
1305
       Amazon-ratings is based on the Amazon product co-purchasing network metadata. Nodes represent
1306   products and edges connect products frequently purchased together. Node features are the mean of
1307   FastText embeddings for words in product description. The task is to predict the class of products’
1308   ratings.
1309
       Minesweeper is a synthetic dataset inspired by the Minesweeper game. The graph is a regular
1310
       100x100 grid where each node is connected its eight neighboring nodes. 20% of the nodes are
1311
       randomly assgined as mines. The node features are one-hot-encoded numbers of the neighboring
1312   mines. The task is to predict if the nodes are mines.
1313
1314   Tolokers is based on data from the Toloka crowdsourcing platform. Nodes represent workers who
1315
       have participated in the selected projects, while edges connect two workers who work on the same
       task. Node features are based on worker’s profile information and task performance. The task is to
1316
       predict which workers have been banned.
1317
1318   Questions is based on question-answering data from website Yandex Q. Nodes represent users and
1319   edges connect an answer provider to a question provider. Node features are the mean of FastText
1320   word embeddings of user profile description, with an additional binary feature indicating users with
1321
       no descriptions. The task is to predict if the users remain active on the website.
1322   Penn94, Cornell5 and Amherst41 (Lim et al., 2021) are friendship network datasets extracted from
1323   Facebook of students from selected universities from 2005. Each node in the datasets represent a
1324   student, while node label represents the reported gender of the student. Node features include major,
1325   second major/minor, dorm/house, year, and high school.
1326   Wisconsin (Pei et al., 2020) is a web page dataset collected from the computer science department of
1327   Wisconsin Madison. In this dataset, nodes represent web pages and edges are hyperlinks between
1328   them. Feature vectors of nodes are bag-of-words representations. The task is to classify the web
1329   pages into one of the five categories including student, project, course, staff and faculty.
1330
       Genius (Lim et al., 2021) is a sub-network from website genius.com, a crowd-sourced website of
1331   song lyrics annotations. Nodes represent users while edges connect users that follow each other.
1332   Node features include expertise scores expertise scores, counts of contributions and roles held by
1333   users. Around 20% of the users are marked with a "gone" label, indicating that they are more likely
1334   to be spam users. The task is to predict which users are marked.
1335
       US-election (Jia and Benson, 2020) is a geographical dataset extracted from statistics of Unite States
1336
       election of year 2012. Nodes represent US counties, while edges connect bordering counties. Node
1337   features include income, education, population etc. The task is a binary classification to predict
1338   election outcome.
1339
1340
       Cora, Citeseer and Pubmed (Pei et al., 2020) are citation graphs, where each node represents a
       scientific paper and two papers are connected when a paper cites the other. Each node is labeled with
1341
       the research field and the task is to predict which field the paper belongs to. All three datasets are
1342
       homophilous.

009                                               A BSTRACT
010
011            Causal discovery from observational data is an important problem in many applied
012            sciences. Incorporating a recently proposed smooth characterization of acyclicity,
013            gradient-based causal discovery approaches search for a Directed Acyclic Graph
014            (DAG) by optimizing various neural models. Although they show some inspir-
015            ing results given certain assumptions satisfied, their capability of modeling com-
016            plex nonlinear causal generative functions is still unsatisfactory. Motivated by
017            recent advances in deep generative models, we propose to use diffusion mod-
018
               els for causal discovery, and search for the DAG under continuous optimization
               frameworks. The underlying nonlinear causal generative process is modeled with
019
               diffusion process, and with flexible parameter configurations, it has the ability to
020
               represent various functions, and the proposed causal discovery approach are able
021            to generate graphs with satisfactory accuracy on observational data generated by
022            either linear or nonlinear causal models. This is evidenced by empirical results on
023            both synthetic and real data.

026   1   I NTRODUCTION
027
028   Causal discovery from observational data is a research field that has received widespread attention in
029   recent years (Pearl, 2009; Spirtes et al., 2001; Peters et al., 2017) which aims to automatically learn
030   the causal relationship (directed acyclic graph, DAG) between variables from non-experimental data.
031   To tackle the NP-hard structure learning problem(Chickering et al., 2004), constraint-based meth-
032   ods uses conditional independence tests to locate causal edges between variable pairs admissible to
033
      the given dependence measurements (Spirtes et al., 2001; Zhang, 2008). On the other hand, score-
      based methods deploy a score to measure the fitness of a causal graph over data, with a suitable
034
      search procedure such as greedy search (Chickering, 2002) and dynamical programming (Silander
035
      & Myllymaki, 2012). Bayesian score criteria is usually adopted, including Bayesian information
036   criteria (BIC, (Maxwell Chickering & Heckerman, 1997)), Bayesian Gaussian equivalent (BGe)
037   score (Kuipers et al., 2014) and minimum description length (MDL, (Bouckaert, 1993)) with con-
038   sistency and equivalence property maintained. A common challenge is that the complexity of the
039   search space is superexponential with the number of variables, which poses a big challenge to the
040   optimization procedure.
041
      Recently, an acyclic constraint of the adjacency matrix that is a continuous function is proposed
042   (Zheng et al., 2018). Transforming the traditional combinatorial nature of the acyclicity to a contin-
043   uous optimizable one, it triggers a series of differentiable causal discovery approaches, considering
044   various model settings (Zheng et al., 2018; Lee et al., 2019). Yu et al. (2019) incorporates the
045   acyclicity constriants with the framework of variational autoencoders (Kingma, 2013) for causal
046   discovery, and Zhu et al. (2019) uses reinforcement learning (Kaelbling et al., 1996) as the search
047   algorithm. Although the nonlinearity of the optimization problem often makes the solutions only
048   local optimal, empirical results show certain closeness to that obtained by performing exhaustive
049   combinatorial search (Zhu et al., 2019).
050   Nonetheless, most deep learning (generative) model based approaches tie to a specific functional
051   model class, the generalized linear causal models. The model assumptions already set limitations on
052   the representation capability, and may fail to model observational data distributions when they follow
053   a complicated underlying generative process. An example is the post-nonlieanr causal model (Zhang
      & Hyvärinen, 2009), where a post-nonlinear transformation directly follows an additive noise model

055   models is thus helpful for wider applicability of differentiable algorithms. Meanwhile, learning a
056   DAG via optimizing the evidence lower bound of variational autoencoders (Yu et al., 2019), reward

058   DAG constraint (Zheng et al., 2020) is known to suffer from certain unstability due to the richness

061   (Croitoru et al., 2023; Sohl-Dickstein et al., 2015) for causal discovery. The proposed approach has

      2   R ELATED W ORK
075
076
      Constraint-based causal discovery methods initiate their process by leveraging conditional indepen-
      dence tests to outline a causal skeleton. Edge orientation approaches are then applied to determine
077
      DAGs up to Markov equivalence class. Examples include PC algorithm and SGS algorithm (Spirtes
078
      et al., 2001) and several extensions when kernel based conditional independence measurements are
079   used (Zhang et al., 2011; Sun et al., 2007). Nonetheless, the accuracy of such methods heavily relies
080   on the reliability of the independence tests, and resolving possible conflicting constraints caused
081   by multiple independently-conducted tests remains substantially challenging (Tsamardinos et al.,
082   2012). The DAG identification framework with experimental data available is also considered (Hyt-
083   tinen et al., 2013).
084
      Score-based causal discovery mainly uses a given score to measure the quality of the causal graph,
085   coupled with algorithms to search for the optimal graph (Rolland et al., 2022). Scores include BIC
086   (Maxwell Chickering & Heckerman, 1997) and MDL (Bouckaert, 1993), and the generalized kernel-
087   based scoring function (Huang et al., 2018). By incorporating additional assumptions about the data
088   distribution and/or underlying generative functions, scores based on well-defined functional causal
089   models are proposed. These methods differ from constraint-based approaches, which often pre-
090   sume faithfulness and are limited to identifying the Markov equivalence class. Given assumptions
091   satisfied, these functional causal model-based methods can effectively distinguish between distinct
092   directed acyclic graphs (DAGs) within the same equivalence class, with applications covering lin-
093
      ear non-Gaussian model (Shimizu et al., 2006), the nonlinear causal models with additive noise
      (Hoyer et al., 2008; Peters et al., 2014), and the post-nonlinear causal model (Zhang & Hyvärinen,
094
      2009) which accommodates a post-nonlinear transformations after the nonlinear causal functions.
095
      Recently, diffusion model based estimation of the Hessian matrix of the additive noise model has
096   also been investigated (Sanchez et al., 2022).
097
098   Differentiable causal discovery is based on a recent smooth characterization of acyclicity constraint
099
      (Zheng et al., 2018), which transforms the combinatorial DAG search problem to be continuously
      optimizable. This enables searching for a DAG with gradient-based optimizations in an end-to-end
100
      way (Geffner et al., 2022), using various black-box solvers. Different searching approaches have
101
      been proposed, considering linear models (NOTEARS, (Zheng et al., 2018)), nonlinear variational
102   autoencoder-based models (DAG-GNN (Yu et al., 2019)) and reinforcement learning-based mod-
103   els (RL-BIC, (Zhu et al., 2019)). Wei et al. (2020) presents a theoretical perspective that the edge
104   missing in linear model equals to a non-satisfaction of KKT condition in optimization, motivating a
105   local search algorithm. Lee et al. (2019) and Lachapelle et al. (2019) utilize polynomial regression
106   and neural networks to deal with nonlinear causal relationships with flexibility to represent condi-
107   tional distributions, respectively. Zheng et al. (2020) reconsiders the algebraic smooth constraint
      of acyclicity in the perspective of nonparametric structural equation models, via leveraging sparsity

108
      characterized by partial derivatives. Causal discovery via graph autoencoder (Ng et al., 2019) and at
109   latent conceptual level (Yang et al., 2021) are also studied.
110
111   Diffusion models are recently shown to be with superior performance on various application do-
112
      mains upon content generations (Cao et al., 2024). Unlike generative adversarial networks (Good-
      fellow et al., 2020) or variational authencoders (Kingma, 2013), it essentially consists of two in-
113
      tricately linked processes: a predefined forward process and a corresponding reverse process. The
114
      forward process transforms the data into a simple prior distribution, typically a Gaussian, while the
115   reverse process leverages a trained neural network to progressively reverse the effects of the forward
116   process following stochastic differential equations. The neural network for the denoising process
117   (usually Markovian) is trained under the denoising score-matching function (Song et al., 2020b).
118   The capability of the model on capturing complex nonlinear generative process is from the flexi-
119   bility of configuring the denoising variance sequences, with remarkable success on bio-informatics
120   (Xu et al., 2022), natural language processing (Li et al., 2022) and computer vision (Croitoru et al.,
121   2023). The property is possibly beneficial for representation learning (Abstreiter et al., 2021).

123   3     S TRUCTURE L EARNING VIA D IFFUSION M ODEL
124
125   In this section, we propose our diffusion model based causal discovery method. The basic idea
126   is to use the deep diffusion process to model nonlinear data generative mechanisms, incorporat-
127   ing the designed causal layer with an adjacency matrix being an learnable element. To start with,
128   we introduce the denoised diffusion probabilistic model (DDPM), which is a discrete approxima-
129   tion (Markov process) of the continuous diffusion process represented by stochastic differentiation
130
      equation (Ho et al., 2020).
131

      3.1   D ENOISED D IFFUSION P ROBABILISTIC M ODEL
133
      In diffusion models, the generative process unfolds over a direction of time. It represents a sequence
134   of data distributions that characterize the model’s gradual transformation from the initial data distri-
135   bution to a target noise distribution. Denote the initial state as Y0 , sampled from the data distribution.
136   Noise is incrementally added at each subsequent time step to push the distribution to gradually con-
137   verge towards a known prior state Yt (normally following a Gaussian distribution). The evolution of
138   the data distribution is through the fine-gained sequence of intermediate states, each corresponding
139   to a specific time point within the process. By varying the setting of noise addition, the diffusion
140   model is able to capture a variety of underlying generative mechanisms. To be mathematically pre-
141   cise (Ho et al., 2020), we denote the forward process F and the reverse process R, and the diffusion

      3.3   O PTIMIZATIONS
244
245   In this section, we discuss the core elements of the model based optimization for graph search, ba-
246   sically consisting of objective functions, acyclicity constraints and the acceleration strategies when
247   efficiency is also of consideration.
248
      Acyclicity constraints. The acyclicity of the adjacency matrix A should be enforced in the graph
249
      searching procedure. Approaches such as greedy equivalence search (Chickering, 2002) check ex-
250   plicitly for the satisfaction of acylic constraint when each edge operation is conducted, e.g., edge
251   addition, which is of high complexity when the number of node is large. In this work, we optimize
252   the matrix A as a whole under a continuous framework, admitting simultaneous change of multiple
253   entries in a single round, and we thus adopt the recently proposed smooth characterization of the
254   acyclicity (Zheng et al., 2018). A directed graph associated weighted adjacency matrix A is acyclic
255   if and only if
256                                       h(A) = trace(eA⊙A ) − n = 0,                                (16)
257
258   where ⊙ is the Hadamard product and eA is the matrix exponential which can be interpreted as a
259   Taylor expansion of power series that ties to the reweighed number of closed walks in the matrix.
260   Objective function. Recall the modified diffusion process with the graph inserted, FG (X, A, Γ)
261   and RG (Z, A, Γ) in (14). The objective function is the evidence lower bound described in (5), with
262   single-step loss estimation approach as (7). Given an set of observational data of some variable X,
263   denote the loss function over the samples as L(A, Θ) where Θ is a set of all editable parameters
264   needed to determine the diffusion process. Incorporating an acyclicity constraint, the optimization
265   problem becomes

      4        EXPERIMENTS
343
344
      In this section, we report empirical results on synthetic and real datasets, comparing performance of
345   our proposed method to both score-based and constraint-based methods, including PC (with Fisher
346   independence test upon threshold on p-value 0.01) (Spirtes et al., 2001), Causal Additive Model
347   (Bühlmann et al., 2014), ICA-LiNGAM (Shimizu et al., 2006), DAG-GNN (Yu et al., 2019) and
348   GraN-DAG (Lachapelle et al., 2019), NOTEARS-MLP (Zheng et al., 2020), GAE (Ng et al., 2019)
349   and DiffAN (Sanchez et al., 2022). The implementations of all these algorithms are from available
350   online source and more details are provided in Appendix C. Unless stated, default settings of the al-
351   gorithms are used in the experiments. The thresholding method for graph pruning (e.g., abandoning
352   an edge if the estimated (absolute) value of the corresponding entry in the adjacency matrix is less
353
      than a threshold) is adopted for ICA-LiNGAM, NOTEARS and DAG-GNN, NOTEARS-MLP and
      GAE. For CAM, DiffAN and GraN-DAG, the significance testing of covariates is applied and the
354
      threshold of p-values for declaration of significance is 0.001.
355
356   The proposed diffusion based algorithm named DAG-Diffusion is implemented based on PyTorch
357   implementations of denoised diffusion probabilistic models (Ho et al., 2020). The forward and
358   reverse process are then reformed as described by (14). The reverse process is accelerated by imple-
359
      menting DDIM modules for efficient sampling. More details are reported in Appendix B.
360   The evaluation metrics include False Discovery Rate (FDR), True Positive Rate (TPR), F1 score
361   and the Structural Hamming Distance (SHD) which is the minimum number of edge operations
362   (including additions, deletions and reversals) needed to convert the estimated graph to the given
363   true DAG. The SHD is a comprehensive measurement considering both the false positives and false
364   negatives, and it is widely used for quantifying the accuracy of causal discovery algorithms. A lower
365
      SHD indicates a better estimated graph.

525   5      C ONCLUDING R EMARKS
526
527
      In this paper, we propose a novel approach for causal discovery utilizing diffusion models within a
528   continuous optimization framework. We integrate a smooth characterization of the acyclicity con-
529   straint into our objective function to generate acyclic adjacency matrices. The diffusion process’s
530   prowess in nonlinear modeling provides some theoretical justifications for our model’s capacity to
531   uncover causal graphs from observational data stemming from possibly intricate generative mech-
532   anisms. Extensive experiments on linear and nonlinear models with various types of noise, as well
533   as real world datasets, are conducted. The empirical results show that our proposed model achieves
534   satisfactory accuracy in causal discovery.
535   For future work, it is remarkable that training diffusion models, particularly those with deep layers,
536   remains a computational challenge. Improving the training efficiency of diffusion causal models is
537   consequently a promising direction for future research. Furthermore, determination of the config-
538   uration parameters of diffusion process, such as its depth, remains largely heuristic. Research on
539   techniques for automatic parameter tuning is thus important for its performance on various applica-
      tion scenarios.

      The equivalence condition between unconstrained optimization problem and its transformed con-
707
      strained optimization problem has been discussed in some literature (Boyd & Vandenberghe, 2004).
708   We follow a similar logic of (Zhu et al., 2019) to complete the proof. Suppose that A is a solution
709   to the problem (17). Then let L∗ = L(A, Θ∗ ) with Θ∗ being some coupled parameters for the min-
710   imization problem. Clearly, A is acyclic due to the hard constraint in (17). If A is not a solution to
711   the problem (18), then this indicates that there exist an adjacency matrix A′ such that
712                                      L∗ > min L(A′ , Θ) + λh(A′ ).                                  (24)
713                                             Θ
                        ′
714   By assumption, A cannot be acyclic, otherwise we have a directed acyclic graph that corresponds
715   to a score lower than the assumed minimum L∗ . Then it follows that
716                               min L(A′ , Θ) + λh(A′ ) ≥ Ll + λh∗ ≥ Lu ,                      (25)
                                   Θ
717
      which violates the assumption that Lu upper-bounds the loss function.
718
719   Then we consider the other direction. Suppose there exists a solution A that satisfies problem (18)
720   but is not a solution to problem (17). Then there exist two cases: i) A is not acyclic; ii) A is
721
      acyclic but yields a higher loss value than the assumed minimum. The second case makes a clear
      contradiction of the definition of minimum loss. For the first case, assume that there is a acyclic
722
      matrix A′ that achieves the minimum score. Then one can easily get the inequality of (24) since
723
      h(A′ ) = 0 in this case, which leads to a contradiction that A minimizes the loss function.

777   C    A LGORITHM I MPLEMENTATIONS
778
779   We use the exiting implementations of causal discovery algorithms for comparisons, as below:
780
            • NOTEARS (Zheng et al., 2018) is a causal discovery algorithm targeting linear mod-
781
              els. It recovers the causal graph by estimating a weighted adjacency matrix by optimiza-
782
              tion under the least square reconstruction loss, combined with a smooth characterization
783           for the acyclicity constraint. Thresholding is applied on the estimated weights to pro-
784           duce the inferred directed acyclic graph. Codes are available at the GitHub repository
785           https://github.com/xunzheng/notears.
786
            • NOTEARS-MLP (Zheng et al., 2020) extends NOTEARS to nonlinear cases in-
787           corporating nonlinear regression functions.  It takes use of neural networks to
788           approximate arbitrary f .     Codes are also available at the GitHub repository
789           https://github.com/xunzheng/notears.
790
            • DAG-GNN (Yu et al., 2019) is a causal discovery algorithm that formulates the
791           problem within a variational autoencoder framework, utilizing graph neural networks
792           as its encoder and decoder structure. With a modified acyclicity constraint, the
793           weighted adjacency matrix is generated by optimizations with the evidence lower
794           bound as its loss function. The Python code is available at the GitHub repository
795           https://github.com/fishmoon1234/DAG-GNN.
796         • GAE (Ng et al., 2019) performs causal discovery via a graph auto-encoding
797           framework.    The causal graph is obtained by optimizing a specially designed
798           encoder-decoder model.   The implementation is available at the GitHub reposi-
799           tory https://github.com/huawei-noah/trustworthyAI/tree/master/
800           gcastle/castle/algorithms/gradient/gae.
801         • GraN-DAG (Lachapelle et al., 2019) is a causal discovery algorithm that employs feed-
802           forward neural networks to model causal relationships. The the (i, j)-th element of the
803           weighted adjacency matrix is determined by considering the sum of all product paths be-
804           tween variables xi and xj . Utilizing the acyclicity constraint introduced by NOTEARS
805           (Zheng et al., 2018), GraN-DAG identifies a directed acyclic graph (DAG) by maximum
806           likelihood estimations, over observed data. The implementation is available at the GitHub
807
              repository https://github.com/kurowasan/GraN-DAG.
808         • PC (Spirtes et al., 2001) employs greedy search originated from the Greedy Equiv-
809           alence Search (GES) algorithm (Chickering, 2002). ICA-LiNGAM (Shimizu et al.,
              2006) is a causal discovery algorithm that assumes a linear non-Gaussian additive

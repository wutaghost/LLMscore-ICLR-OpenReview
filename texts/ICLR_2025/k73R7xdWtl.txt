009                                               A BSTRACT
010
011            Selecting a sample generation scheme from multiple text-based generative models
012            is typically addressed by choosing the model that maximizes an averaged evalua-
013            tion score. However, this score-based selection overlooks the possibility that dif-
014            ferent models achieve the best generation performance for different types of text
015            prompts. An online identification of the best generation model for various input
016
               prompts can reduce the costs associated with querying sub-optimal models. In this
               work, we explore the possibility of varying rankings of text-based generative mod-
017
               els for different text prompts and propose an online learning framework to predict
018
               the best data generation model for a given input prompt. The proposed frame-
019            work adapts the kernelized contextual bandit (CB) methodology to a CB setting
020            with shared context variables across arms, utilizing the generated data to update a
021            kernel-based function that predicts which model will achieve the highest score for
022            unseen text prompts. Additionally, we apply random Fourier features (RFF) to the
023            kernelized
                   вҲҡ        CB algorithm to accelerate the online learning process and establish a
024            O(
                e T ) regret bound for the proposed RFF-based CB algorithm over T iterations.
025            Our numerical experiments on real and simulated text-to-image and image-to-text
026            generative models show RFF-UCB performs successfully in identifying the best
027            generation model across different sample types.

      1   I NTRODUCTION
030
031   Text-based generative artificial intelligence (AI) has found numerous applications in various engi-
032   neering tasks. A prompt-based generative AI represents a conditional generative model that pro-
033   duces samples given an input text prompt. Over the past few years, several frameworks using diffu-
034   sion models and generative adversarial networks have been proposed to perform text-guided sample
035   generation tasks for various data domains including image, audio, and video (Reed et al., 2016;
036   Pan et al., 2018; Xu et al., 2018; Ding et al., 2021; Singer et al., 2022; Huang et al., 2023; Podell
037   et al., 2024). The multiplicity of developed prompt-based models has led to significant interest in
038   developing evaluation mechanisms to rank the existing models and find the best generation scheme.
039
      To address this task, several evaluation metrics have been proposed to quantify the fidelity and rel-
      evance of samples created by prompt-based generative models, such as CLIPScore (Hessel et al.,
040
      2021) and PickScore (Kirstain et al., 2023).
041
042   The existing model selection methodologies commonly aim to identify the generative model with
043   the highest relevance score, producing samples that correlate the most with input text prompts. A
044   well-known example is the CLIPScore for image generation models, measuring the expected align-
045
      ment between the input text and output image of the model using the CLIP embedding (Radford
      et al., 2021b). While the best-model identification strategy has been frequently utilized in gener-
046
      ative AI applications, this approach does not consider the possibility that the involved models can
047
      perform differently across text prompts. However, it is possible that one model outperforms another
048   model in responding to text prompts from certain categories, while that model performs worse in
049   generating samples for other text categories. Figure 1 shows one such example where two standard
050   text-to-image models exhibit different rankings on text prompts with the terms вҖқdogвҖқ and вҖқcarвҖқ. In
051   general, the different training sets and model architectures utilized to train text-based models can re-
052   sult in the modelsвҖҷ varying performance in response to different text prompts, which is an important
053   consideration in the selection of text-based generative models for various text prompts. We provide
      more examples in Figures 17 and 18 in the Appendix.

082   In this work, we aim to develop a learning algorithm to identify the best generative model for a given
083   input prompt, using observed prompt/generated samples collected from the models in the previous
084   sample generation iterations. Since the goal of such text-based model selection is to minimize
085
      the data queries from suboptimal generative models for an input text prompt, we view the model
      selection task as an online learning problem, where after each data generation the learner updates a
086
      function predicting which generative model performs the best in response to different text prompts.
087
      Here, the goal of the online learner is to utilize the previously generated samples to accurately guess
088   the generation model with the best performance for the incoming text prompt. An optimal online
089   model selection method will result in a bounded regret value, measured in comparsion to the sample
090   generation from the groundtruth-best model for the text prompts.
091
      The described online learning task can be viewed as a contextual bandit (CB) problem studied in the
092
      multi-armed bandit literature (Langford & Zhang, 2007; Li et al., 2010). In a CB task, the online
093
      learner observes the context variable (the text prompt in our setting) and guesses the best arm for the
094   current input context. Specifically, we focus on the kernelised upper confidence bound (kernel-UCB)
095   approach and adapt this methodology to propose the Shared-Context Kernel UCB (SCK-UCB) for
096   an online prompt-based selection of generative models. According to the SCK-UCB approach, the
097   learner utilizes a UCB-score from a kernel-based prediction function to choose the generative model
098   for the incoming text prompt and subsequently update the kernel-based prediction rule based on
099   theвҲҡgenerated data for the upcoming iterations. We prove that the proposed SCK-UCB achieves an
100   O( T ) regret bound over a horizon of T iterations.
       e
101
      Since the user applying the CB-based model selection approach may have limited compute power
102   and not be able to afford growing computational costs in the online learning process, we propose to
103   utilize the random Fourier features (RFF) framework (Rahimi & Recht, 2007b) to balance the com-
104   putational load between the iterations of SCK-UCB. We discuss that in the kernel-UCB methods,
105   including our proposed SCK-UCB, the computational cost per iteration will grow cubically as O(t3 )
106   with iteration t. To address the growing cost per iteration of kernel-UCB, we leverage the RFF ap-
107   proach and develop the proxy RFF-UCB algorithm which approximates the solution to SCK-UCB,
      while the computational costs grow only linearly O(t). We show that the regret bound for SCK-

133
134   (Automatic) Evaluation of conditional generative models. Evaluating the conditional genera-
135   tive models has been studied extensively in the literature. For text-to-image (T2I) generation, earlier
136
      methods primarily rely on the Inception score (Salimans et al., 2016) and FreМҒchet inception dis-
      tance (Heusel et al., 2017). More recent works propose reference-free metrics for robust automatic
137
      evaluation of T2I and image captioning, with notable examples being CLIPScore (Hessel et al.,
138
      2021) and PickScore (Kirstain et al., 2023). Kim et al. (2022) propose a mutual-information-based
139   metric, which attains consistency across benchmarks, sample parsimony, and robustness. To pro-
140   vide a holistic evaluation of T2I models, several works focus on multi-objective evaluation. Astolfi
141   et al. (2024) propose to evaluate conditional image generation in terms of prompt-sample consis-
142   tency, sample diversity, and fidelity. Kannen et al. (2024) introduce a framework to evaluate T2I
143   models regarding cultural awareness and cultural diversity. Masrourisaadat et al. (2024) examine
144   the performance of several T2I models in generating images such as human faces and groups and
145   present a social bias analysis. Another line of study explores evaluation approaches using large
146   language models (LLMs). Tan et al. (2024) develop LLM-based evaluation protocols that focus on
147
      the faithfulness and text-image alignment. Peng et al. (2024) introduce a GPT-based benchmark for
      evaluating personalized image generation. For evaluation of text-to-video (T2V) generation, Huang
148
      et al. (2024) introduce VBench as a comprehensive evaluation of T2V models in terms of quality
149
      and consistencey.
150
151   (Kernelized) Contextual bandits.         The contextual bandits (CB) is an efficient framework for
152   online decision-making with context information (Langford & Zhang, 2007; Foster et al., 2018),
153
      which is widely adopted in domains such as recommendation system and online advertisement (Li
      et al., 2010). A key to its formulation is the relationship between the context (vector) and the ex-
154
      pected reward. In linear CB, the reward is assumed to be linear to the context vector (Li et al., 2010;
155
      Chu et al., 2011). To incorporate non-linearity, Valko et al. (2013) propose kernelized CB, which
156   assumes the rewards are linear-realizable in a reproducing kernel Hilbert space (RKHS). However,
157   the proposed algorithm requires solving a kernel ridge regression per iteration, whose computation
158   and required space have polynomial dependence on the number of iterations. To address this prob-
159   lem, a line of study leverages the assumption that the kernel matrix is often approximately low-rank
160   and uses NystroМҲm approximations (Calandriello et al., 2019; 2020; Zenati et al., 2022). [Recently,
161   a line of study utilizes (contextual) bandit algorithms to improve the performance of generative
      models (Chen et al., 2024; Lin et al., 2024).]

      5.2   SCK-UCB WITH R ANDOM F OURIER F EATURES
317
318   The SCK-UCB solves a KRR for each model at an iteration to estimate the scores, which can be
319   expensive in both computation and memory for a large number of iterations. To address this problem,
320   we leverage the random Fourier features (RFF) sampling (Rahimi & Recht, 2007a) for positive
321   definite shift-invariant kernels. At a high level, RFF maps the input data, e.g., the prompt (vector)
322   in our setting, to a randomized low-dimensional feature space and then applies fast linear methods
323   to solve the regression problem. Particularly, the inner product between these projected randomized
      features is an unbiased estimation of the kernel value.

537   image-captioning tasks demonstrate the effectiveness of the proposed framework in scenarios where
538   the available generative models have varying performance rankings depending on the type of prompt.

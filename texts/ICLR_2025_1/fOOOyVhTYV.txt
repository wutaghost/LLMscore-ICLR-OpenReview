009                                               A BSTRACT
010
011            Covering numbers are central to estimating sample complexity. Alas, standard
012            techniques for bounding covering numbers fail in estimating the covering num-
013            bers of many classes of neural networks. We introduce a generalization of covers,
014            called multicovers, which are covers w.r.t. many metrics simultaneously. Con-
015            trary to standard covering numbers, multicovering numbers behave better with the
016
               layer-wise structure in neural networks. We utilize this property to recover a recent
               result of Daniely & Granot (2019) who defined a new notion called Approximate
017
               Description Length (ADL) to establish tight bounds on the sample complexity of
018
               networks with weights of bounded Frobenius norm. We also show that ADL and
019            multicovering numbers are closely related.

022   1   I NTRODUCTION
023
024   Covering numbers are one of the most basic techniques for bounding the sample complexity of
025   function classes, and can achieve state of the art bounds in various cases. Alas, it is not clear
026   how to estimate covering numbers for function classes of layered architectures, such as neural net-
027
      works. Indeed, state-of-the-art results still exhibit a polynomial gap between upper and lower sample
      complexity bounds. This is in contrast to non-layerd function classes, in which the gaps are often
028
      logarithmic or even constant.
029
030   A major flaw of covering numbers is that it is not clear how to use them inductively on the network’s
031   layers. That is, a bound on the covering number for function classes of depth i architectures, is not
032   enough to derive a tight (up to log factors) bound for function classes of depth i + 1 architectures.
033   In this paper, we present a generalization of covering called multicover, which overcomes the above
034   barrier, at least in some cases. This allows the derivation of tight bounds on various families of
035   neural networks. In a nutshell, given a set S of d × d PSD matrices, an ϵ-multicover of a set
               d
036   X
      p ⊂ R is a set X̌ that simultaneously forms an ϵ-cover w.r.t. to any metric of the form d(x, y) =
037      (x − y)⊤ R(x − y) for R ∈ S. The ϵ-multicovering number of X is the minimal size of an ϵ-
038   multicover of X . We note that if S consists of a single matrix R, multicover is just a standard cover
039   w.r.t. the norm corresponding to R. However, for other sets S we get a notion of covering that is
040   fundamentally different from standard covering w.r.t. a metric.
041   We present techniques which allow for layerwise induction. Using these techniques and for the case
042   of S being the class of PSD matrices with trace at most 1, we show the following. Given a bound
043   on the multicovering number of X , a class L of d × d matrices, and a non-linearity σ : R → R, we
044   derive bounds which are often tight on the multicovering number of
045                 LX = {Ax : A ∈ L, x ∈ X } and σ(X ) = {(σ(x1 ), . . . , σ(xd )) : x ∈ X }
046   The tools we present allow us to derive nearly tight bounds on the sample complexity of constant
047   depth networks with weights of bounded norm. For instance, assume that the activation is the ReLU-
048   like softplus activation σ(x) = log(1 + ex ) and consider the class N of networks of depth l, width
049   d, and weight matrices with spectral norm at most O(1) and Frobenius norm at most R. This and
050   similar classes have been studied intensively in recent years, because sample complexity bounds on
051   such classes can potentially be sublinear in the number of network parameters, thus shedding light
052   on a main mystery of modern neural networks. We show that if the input distribution is supported
053   in [−1, 1]d then the sample complexity of N is Õ(dR2 ) which is sublinear in number of parameters
      and is tight up to poly-log factors.

       In this section we show that multicover is closely related to the notion of approximate description
1219   length (ADL) as defined by Daniely & Granot (2019). We start with a definition that is slightly
1220   different from the definition used in Daniely & Granot (2019). We say that X has ε-ADL of n if
1221   there is a protocol between two entities, Alice and Bob with the following properties. Upon seeing
1222   x ∈ X , Alice, that is allowed to use randomness, sends a message s ∈ {0, 1}n to Bob. Upon seeing
1223   s, Bob generates a vector x̂ that is an ε-estimator to x. Formally, there is a probability space (Ω, P )
1224   (representing Alice’s randomness) and functions A : X × Ω → {0, 1}n and B : {0, 1}n → Rd,m
1225   such that for any x ∈ X the random variable ω 7→ B(A(x, ω)) is an ε-estimator of x. We denote by
1226   ADL(X , ε) the minimal k for which X has an ε-ADL of k.
1227   The following lemma shows that ADL is closely related to multicover.
1228   Lemma B.1. ADL(X , ε) = ⌊log2 (M (X , ε))⌋
1229
1230   Proof. Observe that X has ε-ADL of n if and only if there is a set X̌ such that for any x ∈ X there
1231   is a random vector X ∈ X̌ that is a ε-estimator of x. By lemma 3.1 this is valid if and only if X̌
1232   is an ε-multicover. It follows that ADL(X , ε) is the minimal k for which X has an ε-multicover of
1233   size 2k . In other words, ADL(X , ε) = ⌊log2 (M (X , ε))⌋.

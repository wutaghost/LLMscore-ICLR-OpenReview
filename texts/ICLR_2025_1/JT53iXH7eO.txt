009                                              A BSTRACT
010
011            This paper addresses an important problem of object addition for images with
012            only text guidance. It is challenging because the new object must be integrated
013            seamlessly into the image with consistent visual context, such as lighting, texture,
014            and spatial location. While existing text-guided image inpainting methods can add
015            objects, they either fail to preserve the background consistency or involve cumber-
016            some human intervention in specifying bounding boxes or user-scribbled masks.
017            To tackle this challenge, we introduce Diffree, a Text-to-Image (T2I) model that
018
               facilitates text-guided object addition with only text control. To this end, we cu-
               rate OABench, an exquisite synthetic dataset by removing objects with advanced
019
               image inpainting techniques. OABench comprises 74K real-world tuples of an
020
               original image, an inpainted image with the object removed, an object mask, and
021            object descriptions. Trained on OABench using the Stable Diffusion model with
022            an additional mask prediction module, Diffree uniquely predicts the position of the
023            new object and achieves object addition with guidance from only text. Extensive
024            experiments demonstrate that Diffree excels in adding new objects with a high
025            success rate while maintaining background consistency, spatial appropriateness,
026            and object relevance and quality.

029   1   I NTRODUCTION
030
031   With the recent remarkable success of Text-to-Image (T2I) models (e.g., Stable Diffusion (Podell
032   et al., 2023), Midjourney (Midjourney, 2022), and DALL-E (Betker et al., 2023; Ramesh et al.,
033
      2022)), creators can quickly generate high-quality images with text guidance. The rapid develop-
      ment has driven various text-guided image editing techniques (Brooks et al., 2023; Geng et al., 2023;
034
      Zhang et al., 2024; 2023; Sheynin et al., 2023). Among these techniques, text-guided object addition
035
      which inserts an object into the given image has attracted much attention due to its diverse applica-
036   tions, such as advertisement creation, visual try-on, and renovation visualization. While important,
037   object addition is challenging because the object must be integrated seamlessly into the image with
038   consistent visual context, such as illumination, texture, and spatial location.
039
      Existing techniques for object addition in images can be broadly categorized into mask-guided and
040
      text-guided approaches, as depicted in Figure 2. Mask-guided algorithms typically require the spec-
041
      ification of a region where the new object will be inserted. For example, traditional image inpainting
042   methods (Bertalmio et al., 2000; Suvorov et al., 2022; Lugmayr et al., 2022; Yu et al., 2018; Pathak
043   et al., 2016) focus on seamlessly filling user-defined masks within an image to match the surrounding
044   context. Recent advancements, such as PowerPaint (Zhuang et al., 2024), have effectively incorpo-
045   rated objects into images given their shape and textual descriptions while maintaining background
046   consistency. However, manually delineating an ideal region for all objects, considering shape, size,
047   and position, can be labor-intensive and typically requires drawing skills or professional knowledge.
048   On the other hand, text-guided object addition methods, such as InstructPix2Pix (Brooks et al.,
049   2023), attempt to add new objects using only text-based instructions. Despite this, these methods
050
      have a low success rate and often result in background inconsistencies, as demonstrated in Figure 2
      and Figure 7. Additionally, when employing text-guided methods for iterative object addition, the
051
      quality of the inpainted image tends to degrade progressively with each step, as depicted in Figure 8.
052
053   To tackle the above challenges, we introduce Diffree, a diffusion model with an additional object
      mask predictor module that can predict an ideal mask for a candidate inpainting object and achieve

083   an inpainted image, a mask image of the object, and an object description. The data curation process
084   is illustrated in Figure 5. Note that object addition can be deemed as the inverse process of object
085   removal. We build OABench by removing objects in the image using advanced image inpainting
086   algorithms such as PowerPaint (Zhuang et al., 2024). In this way, we can obtain an original image
087

088
      as the source data, which has two benefits. First, the source image captures comprehensive natu-
089
      ral scenes where the location and shape of one individual object often exhibit intrinsic alignment
090   with the overall scene. It helps guarantee the reasonability of new objects’ location. For instance, a
091   monitor is commonly situated behind computer peripherals. Second, the ground-truth mask of the
092   object already exists in the instance segmentation dataset, which can be directly utilized to remove
093   objects with background consistency preserved. By contrast, InstructPix2Pix (Brooks et al., 2023)
094   collects image pairs using proprietary T2I model (Rombach et al., 2022) under prompt pair with
095   subtle modifications. While this approach maintains new objects’ reasonability, it poses difficulties
096   in preserving background consistency.
097   With OABench, Diffree is trained to predict masks and images containing the new object given the
098   original image and object description. Thanks to the extensive coverage of objects in natural scenes
099   in OABench, Diffree can add various objects to the same image while matching the visual context
100   well as shown in Figure 3. Moreover, Diffree can iteratively insert objects into a single image while
101   preserving the background consistency using the generated mask as shown in Figures 1 and 4.
102   For evaluation, we propose a set of evaluation rules through existing metrics (Hessel et al., 2021;
103   Zhang et al., 2018; Heusel et al., 2017; Xie et al., 2023; OpenAI, 2023), including consistency of
104   background, reasonableness of object location, quality, diversity and correlation of generated ob-
105   ject, and success rate. Extensive experiments show that Diffree performs better in object addition
106   than previous mask-guided and text-guided techniques. For instance, Diffree obtains a significantly
107   higher success rate than InstructPix2Pix. For successful cases, Diffree still outperforms Instruct-
      Pix2Pix in various quantitative metrics.

131   Text-to-Image Diffusion Models Recently, text-to-image (T2I) diffusion models (Nichol et al.,
132   2022; Ramesh et al., 2022; Betker et al., 2023), have shown exceptional capability in image gen-
133   eration quality and extraordinary proficiency in accurately following text prompts, under the dual
134   support of large-scale text-image dataset (Schuhmann et al., 2022; Zhao et al., 2024) and model op-
135   timizations (Dhariwal & Nichol, 2021; Ho et al., 2020; Rombach et al., 2022). DALLE-2 (Ramesh
136
      et al., 2022) enhances text-image alignment via CLIP (Radford et al., 2021) joint feature space,
      DALLE-3 (Betker et al., 2023) further improves the prompt following abilities by training on highly
137
      descriptive generated image captions. Stable Diffusion (Rombach et al., 2022), which is well-
138
      established and widely adopted, garners significant attention and application within and beyond
139   the research community. Given that T2I models generate comprehensive images from text prompts,
140   even minor alterations in prompts can result in substantial changes to the resultant image (Brooks
141   et al., 2023). Consequently, there has been an increased focus not only on T2I generation but also
142   on image editing based on additional conditions such as text inputs, masks, et al.
143
      Text-Guided Image Editing The effectiveness of the text-guided image editing methods (Brooks
144
      et al., 2023; Zhang et al., 2024; Sheynin et al., 2023) largely depends on the composition of its
145   dataset and how it is collected. InstructPix2Pix (Brooks et al., 2023) combines two large pretrained
146   models, a large language model (Mann et al., 2020) and a T2I model (Rombach et al., 2022), to
147   generate a dataset for training a diffusion model to follow written image editing text prompts. Its
148   innovative data collection method allows InstructPix2Pix to follow instructions and shows amaz-
149   ing effects while making its consistency difficult to guarantee due to both input and output being
150   generated by the T2I model. Emu Edit (Sheynin et al., 2023) adapts its architecture for multi-task
151   learning by framing an extensive array of tasks as generative tasks, demonstrating robust and ver-
152   satile performance. MagicBrush (Zhang et al., 2024) introduces a manually annotated dataset in
153
      which the T2I model generates both input and output. The image editing performance of fine-tuning
      InstructPix2Pix on MagicBrush shows better. Unlike the previous methods, we propose a novel
154
      and easily expandable collection method, thanks to the existing instance segmentation dataset, we
155
      use real images as output and synthetic images without a specific object as input. SmartMask Singh
156   et al. (2024) predicts masks for added objects but relies on additional segmentation and mask-guided
157   inpainting models, along with detailed scene descriptions. In contrast, our approach uses a single
158   model with an object description, eliminating complexity, resource dependency, and potential limi-
159   tations stemming from external models or detailed input requirements. Our work closely relates to
160   the concurrent work PIPE (Wasserman et al., 2024), which independently explores similar concepts
161   and methodologies. Both studies involve removing objects to collect an object addition dataset and
      train a diffusion model for text-guided object addition. Our approach additionally trains an Ob-

195   Given an image and the object description, our goal is to add the object to the image while preserving
196   the background consistency. Following this, we initially introduce OABench, a synthetic dataset for
197   this task, comprising image-text pairs with corresponding object masks and images containing the
198   object. We provide an overview of our data collection pipeline in Section 3.1. We next present
199
      Diffree, an architecture amalgamating a Stable Diffusion model with an Object Mask Predictor
      (OMP) module in Section 3.2. The evaluation procedure is presented in Section 3.3.

      3.1   OAB ENCH
203
      We combine existing instance segmentation dataset (Gupta et al., 2019; Lin et al., 2014) with pow-
204   erful image inpainting method (Zhuang et al., 2024) to generate the OABench. Unlike other in-
205   structions follow methods (Brooks et al., 2023; Zhang et al., 2024), generating both data pairs using
206   existing text-to-image (T2I) models (Rombach et al., 2022; Ramesh et al., 2022) with prompt pairs
207   and filtering, we use the real image with objects to synthesize the image without the object, as de-
208   picted in Figure 5. Furthermore, an object in the real image naturally aligns with its background, i.e.,
209   it is appropriate for generating the corresponding image without the same object. The tri-phase data
210   generation process is described in the following sections, with comprehensive procedural specifics
211   detailed in Section A5 of the Appendix.
212   Collection and Filtering We gather and refine instances suitable for image inpainting by applying
213   a set of rules from the LVIS dataset (Gupta et al., 2019), a large instance segmentation dataset an-
214   notated for COCO (Lin et al., 2014) dataset. As depicted in Figure 5, in images containing multiple
215   instances, we enforce size constraints to exclude instances that are too big or too small (typically re-
      lated to object components or background elements like buttons on clothing or rivers). Subsequently,

254   For an image x and a text prompt d, Diffree predicts a binary mask m that specifies the region in
255   x and generates an image x̃. The masked region x̃ ⊙ m aligns with the text prompt d. To this end,
256

      2022)) with an object mask prediction (OMP) module as shown in Figure 6.

262
      resources while maintaining quality and flexibility. Stable Diffusion encompasses both forward and
      reverse processes. Given an image x̃, the forward process adds noise to the encoded latent z̃ = E(x̃):

      Classifier-free Guidance Classifier-free diffusion guidance (Ho & Salimans, 2022) is a method that
310   involves the joint training of a conditional diffusion model and an unconditional diffusion model. By
311   combining the output score estimates from both models, this approach achieves a balance between
312   sample quality and diversity. Training for the unconditional diffusion model is achieved by fixing the
313   conditioning value to a null variable intermittently throughout the training process. We follow the
314   approach of Brooks et al. (2023) by stochastically and independently defining our input conditions
315   x and d as null variables with a probability of 5%.

      interior design domains. See more applications in Section A6 of the Appendix.

533   5     C ONCLUSION
534
535   We propose a novel method, Diffree, that leverages a diffusion model with an object mask predic-
536   tor for text-guided object addition. Beyond the method, we build a high-quality synthetic dataset,
537   OABench, through a novel data collection method for this task. Diffree distinguishes itself by pre-
538   serving background consistency without requiring additional masks, which solves shortcomings of
539   previous text-guided and mask-guided object addition methods. The quantitative and qualitative
      results demonstrate the superiority of our method.

918
      Category filtering Initially, we manually annotate a list of categories which are typically considered
919   parts of complete instances, as detailed in Table A1. We then remove the data from these categories,
920   as they are typically challenging to remove from images and tend to produce unnatural inpainting
921   results. Surprisingly, even after excluding these categories from the dataset, our model retains the
922   capability to add such instances (e.g., shirts) to images.
923
      Size limitation We limit the size of the instance mask at the pixel level through a maxi-
924
      mum/minimum size ratio. Instances below the minimum size threshold are still highly probable
925   to represent segments of complete instances. At the same time, instances that exceeded the max-
926   imum size threshold are very likely to be background (e.g., sky or grass). In our case, we set the
927   maximum size ratio to 0.95 and the minimum size ratio to 0.01.
928
      Non-edge contact Considering that instances touching the image boundaries are predisposed to
929
      incompleteness and pose challenges in background reconstruction, we exclude this subset of the
930
      data. We directly filter based on whether the mask information exists at the image edge.
931
932   Integrity detection To ensure the completeness of instance masks, we initially applied dilation
933   and erosion operations to refine slightly separated mask segments. Subsequently, using OpenCV’s
934
      connectedComponentsWithStats (Itseez, 2015), we analyze and sort the connected regions by size.
      If the ratio of the largest region’s area to any other region’s area exceeds a predefined integrity ratio
935
      threshold (set at 18), this largest region is considered representative of the instance.
936
937   Non-hollow detection We employ contour analysis to identify any potential hollow structures within
938   the mask to ensure the non-occlusion of instance masks. Utilizing OpenCV’s findContours function,
939   we extract both the external and internal contours and designate instances with masks containing
940
      child contours as hollow. The hollow instances are considered to be partially obscured, while non-
      hollow masks are retained for further instance filtering.
941
942   Aspect ratio filtering We exclude instances with extreme aspect ratios, defined as those exceeding
943   a threshold of 10 in either the horizontal or vertical dimension, due to their propensity to pose
944   challenges for inpainting and their higher likelihood of representing partial objects.
945   Non-occlusion detection algorithm To ensure that the instances in our dataset are free from oc-
946   clusions, we implement an occlusion detection algorithm based on the spatial relationships between
947   instance masks. For each pair of instances with overlapping bounding boxes, determined by an Inter-
948   section over Union (IoU) exceeding a predefined threshold (set at 0.05), we compute the proportion
949   of the overlapping area covered by each instance’s mask. Specifically, we calculate the intersection
950   area of their bounding boxes and assess how much of this area is occupied by each mask.
951   If the maximum of these coverage ratios is below an occlusion threshold (set at 0.15), the instances
952   are considered non-occluding and are retained for further processing. However, if the minimum
953   coverage ratio exceeds an area ratio threshold (set at 0.45), indicating significant mutual occlusion,
954   both instances are discarded to prevent incomplete objects. In cases where one instance significantly
955   occludes the other (evidenced by a larger coverage ratio), we discard the instance with the smaller
956   coverage ratio. This selective filtering ensures that only non-occluded, fully visible instances are
957   included in the dataset, enhancing the quality and reliability of the OABench.
958
959   A5.2    DATA S YNTHESIS
960
961   We subsequently utilize the advanced mask-guided image inpainting method, PowerPaint (Zhuang
962   et al., 2024), to remove targeted instances filtered in the prior phase. Before performing inpainting,
963   we apply a dilation operation to the instance masks, as overly precise masks are insufficient for
964   effective inpainting, using a slightly enlarged mask typically results in superior inpainting outcomes.
965
966   A5.3    P OST-P ROCESSING
967
968   In the post-processing stage, our objective is to systematically filter out inpainting results that exhibit
969   suboptimal object removal performance during the data synthesis phase. We then calculate the clip
970   score using the instance name and the corresponding region of the inpainted image, where higher
971   scores denote suboptimal removal efficacy. A preset threshold of 0.65 filters out instances with
      higher scores, thereby optimizing data quality and maintaining a reasonable data volume.

       A8       I N - DEPTH DISCUSSION OF OMP MODULE
1046
1047
       In this section, we provide an in-depth discussion summarizing its role:
1048
1049   1. The OMP module ensures background consistency, which is crucial for iterative additions.
1050   In mask-guided methods (e.g., PowerPaint Zhuang et al. (2024)), the instance mask is a required
1051   input. Typically, a post-processing step involves mix the synthesized object into the input image’s
1052   background using the instance mask, ensuring the background remains unchanged.
1053
       In contrast, text-guided methods (e.g., InstructPix2pix Brooks et al. (2023) and our Diffree) do
1054
       not require an instance mask as input, making this mix operation unavailable. This limitation can
1055   lead to quality degradation, especially in iterative additions. To the best of our knowledge, we are
1056   the first to introduce an output mask through the OMP module in text-guided shape free methods.
1057   This enables the mix operation and allows for iterative additions. We provided an ablation study
1058   (i.e., Figure 8) to evaluate the impact of omitting the OMP module on Diffree’s iterative results.
1059   Without the OMP module, the background deteriorates rapidly after multiple steps, rendering further
1060   additions infeasible.
1061   2. The instance mask generated by the OMP module can be integrated with various existing
1062   works to develop exciting applications.
1063
1064
       As explained in the Appendix A6, the instance mask can be used in many applications that require a
       mask as input. For example, when combined with AnyDoor Chen et al. (2023), Diffree can achieve
1065
       image-prompted object addition. We highlight a few additional points:
1066
1067   (1) Combining with shadow generation methods to produce realistic shadows
1068   The task, shadow generation, aims to create plausible shadows for a composite foreground, given
1069   a composite image without foreground shadows and the foreground object mask Liu et al. (2024).
1070   However, existing mask-guided and text-guided inpainting methods pose challenges for shadow
1071   generation, especially for objects casting long shadows. These challenges arise for two reasons:
1072   first, there is a misalignment between the data masks and the actual shadows of objects (e.g., long
1073   shadows); second, for mask-guided method, it is difficult for users to draw the estimated masks of
1074   long shadow area with objects , in addition it is challenging for the model understands the respective
1075   parts of shadows and objects in the mask. Therefore, combining inpainting works with shadow
1076
       generation works can lead to better results. In mask-guided methods, the shadow generation input
       can be derived from the input mask and output image. Thanks the OMP module’s output mask,
1077
       Diffree, as a text-guided method, effectively integrates with such methods to generate coherent
1078
       objects and realistic shadows, as depicted in Figure A11.
1079
       (2) Providing a starting point for user or designer adjustments

009                                               A BSTRACT
010
011            Predicting the response of nonlinear dynamical systems subject to random, broad-
012            band excitation is important across a range of scientific disciplines, such as struc-
013            tural dynamics and neuroscience. Building data-driven models requires experi-
014            mental measurements of the system input and output, but it can be difficult to
015            determine whether inaccuracies in the model stem from modelling errors or noise.
016            Therefore there is a need to determine the maximum component of the output
017
               that could theoretically be predicted using the input, if an improved model was
               to be developed through the investment of resources. This paper presents a novel
018
               method to identify the component of the output that could potentially be mod-
019
               elled, and quantify the level of noise in the output, as a function of frequency.
020            The method uses input-output measurements and an available, but approximate,
021            model of the system. A trainable, frequency dependent parameter balances an
022            output prediction generated by the model with noisy measurements of the output
023            to predict the input to the system. This parameter is utilised to estimate the noise
024            level and then calculate a nonlinear coherence metric as a measure of causality or
025            predictability from the input. There are currently no solutions to this problem in
026            the absence of an accurate benchmark model.

      1   I NTRODUCTION
030
      Nonlinear dynamical systems are observed across a wide range of scientific disciplines (Mosekilde,
031
      1997), from structural dynamics (Nayfeh & Mook, 1979; Worden & Tomlinson, 2001) and electrical
032
      engineering (Kyamakya et al., 2013), to neuroscience (Corinto & Torcini, 2019) and economics
033   (Bischi, 2010). Modelling their behaviour offers meaningful insights, enables predictions, and
034   allows for control to generate desirable outcomes.
035
036   Accurate models generally involve experimental measurements, e.g. to carry out model up-
037   dating for physics-based models or to train data-driven models. However, it can be difficult to
038   determine whether the prediction errors are due to additional noise or inaccuracies in the model.
039   Consider a nonlinear dynamical system subject to broadband random excitation, x(t), which is
040   especially prevalent in structural dynamics. The output of the system is measured to be yn (t), where
041                             yn (t) = y(t) + ϵn (t) = M{x(τ )τ ≤t } + ϵn (t) .                       (1)
042
043
      Here M{·} represents the true causal dynamical system, ϵn (t) is additive noise and y(t) is the
      component of yn (t) that is caused by x(t). The noise is assumed to be due to uncoupled, unmeasured
044
      signal sources rather than measurement noise, hence there is no noise on the input. A forward model
045
      can be used to make an imperfect prediction yz (t), where
046
047                               yz (t) = G{x(t)} = M{x(τ )τ ≤t } + ϵz (t) .                           (2)
048   Here ϵz (t) denotes the model errors which are caused by the inability of the model, G{·}, to fully
049   capture the functional M{·}. Determining the component of yn (t) caused by x(t) is important
050   for applications which depend on the ability to predict the best possible estimate of yn (t) from x.
051   However, this is challenging because it is difficult to distinguish between ϵn (t) and ϵz (t).
052
053   The input-output causal relationship can be quantified, as a function of frequency, using a
      nonlinear coherence metric between x(t) and yn (t). Similar to correlation, it is a metric between

      0 and 1 of the component of yn (t) that can be predicted using x(t), as a function of frequency. In
055   this context, it is a measure of causality because x(t) is the signal which directly excites the system
056   to generate y. This is also equal to the linear coherence, γY2 Yn (f ) (Kay, 1993), between y(t) and
057   yn (t) which cannot be calculated directly due to the unavailability of y(t). Only a lower bound on
058   the nonlinear coherence can be estimated using a model prediction, yz (t). Determining the true
059   nonlinear coherence is challenging because it is difficult to distinguish between ϵn (t) and ϵz (t),
060   which both appear as random noise. When the input to the system can be controlled, it is possible
061   to estimate the contribution of y(t) through repeat measurements (see Appendix A). However,
062   when the input cannot be controlled, there are currently no methods that can infer the nonlinear
063   coherence, γY2 Yn (f ), without a high-accuracy model of M{·}, which can require large models
064   and lots of data to compute. Ideally, γY2 Yn (f ) could be estimated using a small quantity of data to
065   determine the feasibility of a project before investing in data collection and computational resources.
066
      Despite its wide ranging applications, this work is specifically motivated by feedforward ac-
067
      tive noise reduction (ANR), where x is used as a reference signal to reduce the target signal,
068
      yn (t). In order to reduce the unwanted noise, a model must be developed to accurately predict
069   yn (t) using x(t). The maximum noise reduction that can be achieved is quantified by the causal
070   relationship between x(t) and yn (t). For linear systems, such as those in sound cancelling
071   earbuds and headsets, this is quantified using the linear coherence metric, and the level  of
                                                                                                   2
072   noise reduction that can be achieved, in decibels, is equal to ΓXY (f ) = 10 log10 1 − γXY         .
073   However, ANR has recently begun implementation in cars, where the transmission pathway
074   from x to yn is highly nonlinear (De Brett, 2020; Oh et al., 2015). For nonlinear systems such
075   as this, the nonlinear coherence would be used to quantify the level of noise reduction that is
076   possible. This is crucial in evaluating the benefits of investing resources into developing a high
077
      fidelity model of the car to enhance ANR performance. It is not currently possible to calculate the
      nonlinear coherence without investing the necessary resources into building the best model possible.
078
079
      This work aims to estimate the nonlinear coherence between x(t) and yn (t) for a particular
080   class of nonlinear dynamical systems, with relatively small quantities of data and an incomplete
081   model. These systems can be described by the general class of second order ordinary differential
082   equation (ODE) N (y, ẏ, ÿ) = x, where N (y, ẏ, ÿ) represents an arbitrary nonlinear function. The
083   most common form of ODE of this form seen in practice is:
084

                                       ωn2      ωn

087   Providing measurements are taken across nonlinearities (which are often localised), this can be ex-
088   tended to systems with multiple degrees-of-freedom (extension to multiple degrees-of-freedom is
089
      part of ongoing work). Nonlinear dynamical systems of this type are common in many scientific
      disciplines and there has been significant work in modelling their response using data-based ma-
090
      chine learning methods (Wang et al., 2016; Liang et al., 2005; Worden et al., 2018; Massingham
091
      et al., 2024). However, it is extremely difficult using feedforward modelling techniques (e.g. con-
092   volutional neural networks) due to the exponentially growing number of nonlinear monomial terms
093   which contribute to the output. This numerical challenge is captured by the Volterra series perspec-
094   tive (Volterra, 1959). It is especially challenging for systems with long memory and for random
095   broadband inputs. In this paper, the structure of Eq. 3 is exploited to estimate the nonlinear coher-
096   ence for systems of this type, without a perfect model of the system. This enables the calculation of
097   the causal component of a system response, which can be used to guide the investment of time and
098   money into high performance model development. To the best of the author’s knowledge, there are
099   currently no other methods that achieve this. The key contributions of the work are as follows:
100
             • A new strategy is introduced for estimating the maximum possible performance of time
101
               series prediction algorithms in the presence of noise.
102
             • A novel method is presented for calculating the nonlinear coherence between input and
103
               output data for a class of nonlinear ODEs subject to random input, in the presence of noise.
104
105          • The robustness of the method is demonstrated on three simulated nonlinear dynamical sys-
106
               tems which are widely applicable across several scientific disciplines.
107          • The applicability of the method is demonstrated on an experimental dataset with a strong
               nonlinearity.

      2     R ELATED W ORK
109
110
      The presented method for inferring the maximum performance of feedforward models and calculat-
111   ing the level of noise present is novel, and there are no similar methods in the literature. However,
112   there are three broad areas of research in the literature that overlap with, and contribute to, the goals
113   of this paper.
114
115   Kalman Filter The Kalman filter (Kalman, 1960), and its extensions, aim to estimate the
116   true state of a system by optimally combining the predictions of a state space model with noisy
117   measurements. A comparison of the model error and noise level is used to balance the two signals
118   optimally. This relies on knowledge of at least one of the model error or noise variance because
119
      the true system state is not available for hyperparameter tuning. In this paper, the primary aim is to
      estimate the level of noise and model error, but a similar principle is utilised.
120
121
      Scaling Laws Estimating scaling laws for various machine learning tasks has received sig-
122   nificant attention in recent years (Hestness et al., 2017; 2019; Henighan et al., 2020), with the
123   aim of forecasting how the error of various models will improve as more data is collected and
124   larger models with more parameters are built. However, this approach is limited, as it relies on
125   using the performance of a model trained on a small quantity of data to extrapolate and predict the
126   performance of a bigger model trained using large amounts of data. Extrapolation is difficult, as it
127   assumes there is no noise; the loss will plateau at the noise level. It is therefore difficult to evaluate
128   the maximum performance unless the model is already close to the maximum, at which point
129   increasing the model size and number of training examples will not reduce the prediction error. For
130
      the method presented in this paper, even with a poor forward model, it is possible to estimate the
      maximal predictive performance of a feedforward model by identifying the level of noise.
131
132
      Learning Input Noise Levels As discussed above, if there is noise added to the output, it is
133   difficult to determine the noise level because it is difficult to differentiate between unmodelled
134   components of the system response and noise. However, if there is noise in the input, there are
135   methods for inferring the true input and hence the input noise level. The Error-in-Variable approach
136   can be used to infer the input to a machine learning algorithm and hence infer the noise level (Van
137   Gorp J, 2000; Seghouane & Fleury, 2001). Therefore, with the method presented in this paper, by
138   inverting the problem and trying to predict the input to the system using the output, the output noise
139   could be determined. The issue with applying this to time series data is it requires inferring the true
140   system response for potentially millions of time samples, and so the inference is extremely high
141
      dimensional. This paper proposes an alternative approach to this problem, which is well suited to
      time series applications.
142
143
      Other methods have been presented in the literature that aim to identify a ‘nonlinear’ coher-
144   ence. However, these methods generally rely on a perfect forward model of the system and so
145   model error and additive noise are combined in any estimation (Worden et al., 2018). Our approach
146   is distinct in that only an approximate forward model is needed.

166   A key observation of the structure of Eq. 3 is that whilst it is difficult to learn a mapping
167   from x to y in the ‘forward’ direction, it is relatively simple to map from y to x in the ‘reverse’
168   direction, even for highly nonlinear systems with long memory. This is because there is no
169   nonlinear mixing of terms in time in the reverse direction, and so there is not an infinite Volterra
170   series (Volterra, 1959) as there is in the forward direction. Therefore, even for highly nonlinear
171   systems with long memory, there is a relatively simple mapping from y back to x. This generalises
172   to systems with multiple degrees-of-freedom, providing the nonlinear degrees-of-freedom are
173
      measured and so the memory of the reverse mapping is linear and hence straightforward to learn. In
      addition, it could be applied to dynamical systems beyond those with the structure described by Eq.
174
      3, providing the memory of the reverse mapping is not long and nonlinear.

      3.3   G ENERATING THE FORWARD PREDICTION
293
294   The architecture described in Section 3.1 requires a forward prediction of y, generated using a
295   model G. This may be the best forward prediction currently available but it can be a simpler model
296   too. In this paper, the prediction of a temporal CNN trained on the available data is used. This is
297   an arbitrary choice and any forward model could be used because the presented method uses the
298   forward prediction to estimate an improved estimation of the nonlinear coherence. The better the
299   available forward model, the better the estimation of the nonlinear coherence; this is demonstrated
300   in the simulations below by applying the CNN to systems with varying strengths of nonlinearity. It
301   has been found that it is preferable to use a more complex model, such as a CNN, so that the forward
302
      prediction cannot be easily mapped back to the input and the architecture drives towards ŷ = y, as
      discussed in Section 3.4.
303
304
      3.4   ROLE OF λ
305
306   The solution for K found by the architecture described in Section 3.1 with λ set to 0 is usually
307   smaller than the true value of K across the entire frequency range. It is hypothesised that this is
308   because yz is generated directly from x, and so there is often a mapping directly from yz back to
309   x, rather than by inferring y first. The term λ was therefore included to add some bias to drive the
310   architecture towards inferring y; the resulting value of K was used to infer the nonlinear coherence.
311   Its value is determined using the validation dataset. As λ increases, initially LVx remains constant,
312   before beginning to increase rapidly. The best estimation of the nonlinear coherence is found at
313   the point when LVx begins to increase rapidly. In order to identify this point, λ was increased
314
      incrementally and the model was trained for 100 epochs at each step. When the normalised loss LVx ,
      averaged over the past 100 epochs, increased by 0.001 above the minimum value, the previous value
315
      of λ was chosen. If LVx does not increase, as is the case for low levels of noise, λ = 0.99 is used.
316
      Example plots of λ against LVx from the tests in Sections 4 and 5 can be found in Appendix D.
317

      4     S IMULATIONS
325
326
      The method was evaluated on three simulated systems, each with different functional forms of
327   nonlinearities in order to demonstrate the applicability and usefulness of the method. For each
328   system, the robustness of the method was tested for different levels of noise. The spectrum of
329   the noise was bandlimited and effectively flat around the resonant peak. The noise level can be
330   quantified by the nonlinear coherence between x and yn , which is represented equivalently here as
331   Co (Y, Yn ): values close to 1.0 indicate low noise and values close to 0 indicate high noise. In each
332   case, the nonlinearity of the test case is quantified by the normalised mean squared error between
333   the true response and the linearised response. The method presented in this paper was used to
334   predict the nonlinear coherence between x and yn , using the K(f ) obtained from the architecture
335
      defined in Section 3.1, and the derivation of γY2 Yn (f ) in Section 3.2. The predictions were made
      using just 10 frames of data.
336
337
      To benchmark the performance of the algorithm, the predictions of the nonlinear coherence
338   are compared against those of the forward model, denoted Co (Yz , Yn ), and a linearisation of the
339   system, denoted Co (X, Yn ). The forward model, described in Section 3.3, was chosen to be a 1D
340   temporal convolutional neural network with dilations, as described by van den Oord et al. (2016).
341   The prediction of this forward model was used as yz to demonstrate the improved estimation of the
342   nonlinear coherence that can be calculated using the method presented in this paper. The generated
343   Yz is therefore used both to provide a baseline and also to estimate the nonlinear coherence. It
344   is likely that better forward models could be trained both now and in the future for the examples
345   shown; the aim is to demonstrate how the presented method utilises the forward model to improve
346
      the estimation of the nonlinear coherence, and so it is not benchmarked against the forward
      prediction in the traditional sense and the selection of a forward model is somewhat arbitrary.
347
348
      In the presence of noise, even with a perfect model it is difficult to determine whether the
349   entire response has been captured. The lower bound on the nonlinear coherence is Co (Yz , Yn )
350   and the upper bound is 1. The true nonlinear coherence could lie anywhere within this range, and
351   the results in this section demonstrate that the method presented in this paper provides a relatively
352   accurate estimation that could be useful in practice.

461
462   The method was then evaluated on an experimental dataset consisting of 360 frames of length 6000.
463   This demonstrates the applicability of the method to an arbitrary nonlinearity. A cantilever was
464   excited through a nonlinear connection using a bandlimited random input. The nonlinear connection
465   consisted of magnets (the cantilever is steel) and a rubber tip which caused rattling. This nonlinear
466
      connection is complex and has not been characterised. A single mode of the cantilever was excited,
      so it effectively had one degree-of-freedom. An accelerometer was placed on the shaker as a
467
      reference signal, x, and a second accelerometer was placed along the cantilever as a target signal,
468
      yn . The set up was considered to be effectively noiseless and then three levels of additional noise
469   were introduced using post-processing to represent uncorrelated noise in the system. Noise was
470   introduced artificially because for this experimental rig it is difficult to introduce mechanical noise
471   that is uncorrelated with the input due to strong coupling between inputs. Two minutes of data (10
472   frames) was used to generate the forward prediction and then predict the nonlinear coherence. See
473   Figure 10 in Appendix C.4 for the full experimental setup.
474

476   case only 45 % of the response was captured by the linearised component of the response, and a
477
      CNN (kernel width=10, hidden layers = 5, features=5) captured just 74 % due to the complexity
      of the system response. When noise is added, the challenge is to determine the component of yn
478
      that is caused by x, as this represents the component of yn that could theoretically be reduced using
479
      an ANC system. Figure 5 shows the predicted and true nonlinear coherence for three levels of
480   noise. For all three levels of noise, the predicted nonlinear coherence was an excellent estimation of
481   the true nonlinear coherence. In the low noise case, it demonstrates that significant improvements
482   in the prediction could be achieved by collecting more resources and building larger models.
483   Conversely, in the high noise case, the potential improvements are relatively small compared with
484   the overall noise level. This demonstrates the insight provided by the presented method; in an ANR

      not in the high noise case.

499   6        L IMITATIONS
500
501   The main challenge in the work is determining the optimal value for λ. The intuition is that LVx
502   increases slowly as λ increases until the nonlinear coherence prediction is close to the true nonlinear
503
      coherence, and then it begins to increase more rapidly. This can be seen in Appendix D. However, it
      is difficult to determine this transition point and accurately identify the optimal value for λ. Across
504
      the range of examples tested, thresholding LVx worked well empirically and the method was robust
505
      to small changes in the threshold value, but an interpretable method for setting λ would improve
506   this work.
507
508   Furthermore, another limitation relates to the quality of the forward model prediction and the
509   level of additive noise. Define δ such that Ŷ = Y + δ and then consider:

525   7        C ONCLUSIONS
526
527   This paper presents a novel method for calculating the nonlinear coherence for a nonlinear dynamical
528   system in the presence of noise. A set of parameters are learnt to optimally combine an output
529   prediction, calculated using an available model, with noisy measurements of the output to predict
530   the input to the system. The nonlinear coherence is calculated using these parameters and is used
531
      as a metric of causality. Previously, the only way to estimate this was to model the system and
      assume the unmodelled component of yn to be noise. For highly nonlinear systems, this can be a
532
      poor estimation of the nonlinear coherence. The presented method improves upon this and is able to
533
      estimate the nonlinear coherence with excellent accuracy using a relatively small quantity of data.
534   This would allow for users to collect experimental input and output data for a particular nonlinear
535   dynamical system, and identify how much of the output is caused by the input across the frequency
536   range, without developing expensive models. The method works for a broad class of nonlinear
537   dynamical systems which are widely applicable across many scientific disciplines, from structural
538   dynamics to neuroscience. The main limitation of the method is the interpretability and selection
539   of the hyperparameter λ. Future work should focus on this and investigating the applicability of the
      method to a wider class of nonlinear dynamical systems.

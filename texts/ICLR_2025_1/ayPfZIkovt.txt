009                                              A BSTRACT
010
011            Low Rank Adaptation (LoRA) is a popular Parameter Efficient Fine Tuning (PEFT)
012            method that effectively adapts large pre-trained models for downstream tasks.
013            LoRA parameterizes model updates using low-rank matrices at each layer, signifi-
014            cantly reducing the number of trainable parameters and, consequently, resource
015            requirements during fine-tuning. However, the lower bound on the number of
016            trainable parameters remains high due to the use of the low-rank matrix model. In
017            this paper, we address this limitation by proposing a novel approach that employs a
018
               low rank tensor parameterization for model updates. The proposed low rank tensor
               model can significantly reduce the number of trainable parameters, while also
019
               allowing for finer-grained control over adapter size. Our experiments on Natural
020
               Language Understanding, Instruction Tuning, Preference Optimization and Protein
021            Folding benchmarks demonstrate that our method is both efficient and effective
022            for fine-tuning large language models, achieving a reduction in the number of
023            parameters while maintaining comparable performance.

026   1   I NTRODUCTION
027
028   The advent of Large Language Models (LLMs) – billion parameter scale models pre-trained on vast
029   corpora of data – has enabled unprecedented capabilities across a wide range of tasks. However,
030   as LLM sizes continue to grow exponentially, their computational and memory demands represent
031   significant challenges, particularly for those lacking access to high-performance computing infras-
032   tructure (Varoquaux et al., 2024). This has spurred interest in parameter efficient fine tuning (PEFT)
033
      techniques (Ding et al., 2023), which facilitate the adaptation of LLMs to specific applications,
      downstream tasks or user preferences, by using only a small fraction of trainable parameters. Most
034
      importantly, they reduce GPU memory requirements, primarily by shrinking optimizer states (Liao
035
      et al., 2023). Moreover, they provide greater efficiency in storage and deployment, enabling the man-
036   agement of multiple fine-tuned LLMs with reduced storage footprints and faster load times (Sheng
037   et al., 2023; Wen & Chaudhuri, 2024), which is particularly relevant for applications requiring rapid
038   model switching across numerous task- or user-specific models.
039
      Beyond computational benefits, PEFT techniques can also mitigate overfitting risks associated with
040
      fine-tuning high-capacity LLMs. By constraining model updates, PEFT methods can act as an implicit
041
      regularization mechanism, improving generalization (Fu et al., 2023; Sun et al., 2023). Parameter
042   sharing, a well-established technique in deep learning architecture design, has been shown to improve
043   generalization across various tasks such as protein folding (Jumper et al., 2021; Lin et al., 2023),
044   image segmentation (Ronneberger et al., 2015), and generative modeling (Rombach et al., 2022).
045   Incorporating parameter sharing in PEFT methods has also improved performance in specialized
046   applications with limited data, such as in medical domains (Dutt et al., 2023; Zhu et al., 2024).
047
      Low Rank Adaptation (LoRA) is a popular PEFT approach that uses a low rank parameterization
048   of weight matrix updates (Hu et al., 2021). For instance, these allow to fine tune a 175 billion
049   parameter LLM using only 5 million trainable parameters (Hu et al., 2021) without performance
050   degradation. Since the model updates can be merged with the frozen weights, LoRA incurs no
051   additional inference cost when deployed, unlike prompt (Li & Liang, 2021a; Liu et al., 2023) and
052   adapter-based (Houlsby et al., 2019; He et al., 2021; Pfeiffer et al., 2020) PEFT methods. However,
053   the lower bound on trainable parameters often remains substantial for large-scale models. Recent
      works have aimed to further reduce the number of parameters in LoRA by allocating different ranks

      across update matrices in different layers (Valipour et al., 2022; Zhang et al., 2023b), using fixed low
055   rank projections (Zhang et al., 2023a; Kopiczko et al., 2023), and parameterizing low rank matrices
056   using a random basis (Koohpayegani et al., 2024).
057
058
      In this work, we introduce Low Rank Tensor Adapters (LoRTA), which exploit redundancy in weight
      updates across different layers, heads, and attention matrices by representing updates as a unified
059
      5th-order low-rank tensor model. This holistic approach not only reduces the number of trainable
060
      parameters but also facilitates learning by exploiting shared information among various model
061   components. We show that the parameter-sharing schemes in state-of-the-art methods (Kopiczko
062   et al., 2023; Koohpayegani et al., 2024) are implicit low-rank tensor models with fixed random factors,
063   while our explicit tensor model is fully trainable and more expressive. Our higher-order tensor update
064   and Candecomp-Parafac model (Harshman & Lundy, 1994) enjoys greater parameter efficiency and
065   favorable scaling over low-rank tensor models recently proposed for vision transformers (Jie & Deng,
066   2023; Edalati et al., 2023) and LLMs (Yang et al., 2024; Bershatsky et al., 2024).
067   The two main advantages of our proposed method over existing matrix and tensor based approaches
068   can be summarized as follows:
069
070       (A1) It enables a reduction in the number of trainable parameters by using updates with lower
071            tensor rank.
072
073       (A2) The number of parameters scales better with tensor rank, number of fine-tuned matrices,
074            embedding dimension and attention heads.
075
076   As a consequence, our approach also provides finer-grained control of adapter size. We evaluate
077   our method on diverse benchmarks including Natural Language Understanding, Instruction Tuning,
078   Preference Optimization, and Protein Folding. Our experiments demonstrate that LoRTA can achieve
079   up to an order of magnitude reduction in the number of trainable parameters compared to state-of-the-
080
      art PEFT methods, with minimal performance trade-offs.

145
      introducing dynamic rank adjustment for low-rank matrices during fine-tuning. The fundamental
146
      concept involves optimally allocating the parameter resources by selectively pruning less crucial
147   components of the matrices based on an importance metric. LoRA-FA (Zhang et al., 2023a) reduces
148   the number of trainable parameters by freezing the A matrix to its random initialisation, while
149   achieving similar performance to LoRA.

      3.2   L O RTA: A MORE EFFICIENT TENSOR MODEL
217
218   In the previous section, we explored PEFT models that share parameters across layers, highlighting
219   their correspondence to tensor factorization models. Namely, VeRA and NOLA utilize fixed projection
220   matrices shared across layers. However, this strategy can result in models that are larger than necessary
221   relative to their degrees of freedom due to the inclusion of these random matrices. Although these
222   matrices can be generated on the fly by solely storing the pseudo-random number generator seed, this
223   still incurs additional resource demands during training, and increases loading time for inference.
224   To address this issue, we propose modeling the trainable adapters using a low-rank CPD structure.
225   This choice is motivated by the favorable properties of CPD: it is universal, capable of exactly
226   factorizing any tensor, yet remains concise and parsimonious, typically requiring only a small number
227   of parameters to achieve low approximation error (Sidiropoulos et al., 2017). This contrasts with
228   tensor adapters used in vision (Jie & Deng, 2023) and recently in LLM finetuning (Bershatsky et al.,
229   2024), which rely on Tucker and Tensor-Train models. In fact, for small ranks, CPD is equivalent to
230
      Tucker when the core tensor in Tucker is the identity tensor. However Tucker is always parametrized
      with a dense tensor and therefore requires a larger number of parameters for the same rank.
231
                                                                                   d
232   LoRTA represents all weight updates as a 5th-order tensor dW̃ ∈ Rd× H ×H×L×M . By integrating
233   updates of layers, heads and the Q, K, V , P matrices into a unified low-rank CPD tensor model,
234   LoRTA exploits redundancy across different modes of the tensor. This approach can thus not only
235   improve parameter efficiency but also facilitate learning by exploiting the shared information among
236   various components of the model. This contrasts with existing PEFT approaches, which tensorize
237   each weight update independently (Yang et al., 2024) or only share parameters across layers (Jie &
238
      Deng, 2023; Bershatsky et al., 2024). In order to illustrate how additional tensor modes can result in
      parameter efficiency gains, Figure 1 compares – for a single weight update – LoRA with a rank one
239
      tensor model that adds attention heads as a mode.
240
241   By utilizing a low-rank CPD model, we can express this tensor as:

246   tively, and CH ∈ RH×r , CL ∈ RL×r , CM ∈ R4×r are factor matrices for the attention heads,
247   layers, and the four matrices Q, K, V , P . Each weight matrix update can then be retrieved as:
248
249                dW̃[:, :, k, l, i] = A (Diag (CH [k, :]) Diag (CL [l, :]) Diag (CM [i, :])) B ⊤ ,

252   Note that, unlike previous implicit tensor models such as NOLA and VeRA, which rely on fixed
253   random projections or parameters and learn only scaling coefficients, our proposed model is fully
254   trainable. All factor matrices (A, B, CH , CL , CM ) are learned during training, providing greater
255
      expressiveness and forgoing the dependency on pre-defined random bases or projections.

      4.4   P ROTEIN F OLDING
487
488   Protein folding, the process by which a protein’s amino acid sequence determines its three-dimensional
489   structure, is a fundamental problem in molecular biology. Accurate prediction of protein structures
490   from their sequences has significant implications for understanding protein function and designing
491   new proteins for therapeutic purposes. ESMFold (Lin et al., 2023) is a frontier model for this task
492   trained in two stages. First, ESM-2, a BERT-based (Devlin et al., 2019) protein language model, is
493   trained with the masked-language-modeling objective on amino acid sequences. This unsupervised
494
      pretraining allows the model to capture complex patterns and relationships within protein sequences.
      Remarkably, valuable structural information emerges in the model’s features during this process (Rao
495
      et al., 2020). In the second stage, ESM-2 is frozen, and a model head predicting three-dimensional
496
      protein structures is trained on top of language model features.
497
498   We re-train ESMFold in the second stage – fine-tuning ESM-2 parameters (we use ESM-2 35M
499   instead of the ESM-2 3B model used in Lin et al. (2023) due to compute constraints) with LoRA and
500
      LoRTA instead of freezing them. We evaluate performance with the Local Distance Difference Test for
      Cα atoms (LDDT-Cα) (Mariani et al., 2013) – that measures accuracy of predicted protein structures
501
      by comparing the distance between alpha carbons in predicted and true structures. LDDT-Cα ranges
502
      from 0 (poor accuracy) to 1 (perfect match). See Appendix E.4 for experiment details.

524   5     C ONCLUSION
525
526   We have introduced LoRTA, a novel approach that employs a low-rank tensor model for LLM updates.
527   By extending low-rank adaptation to higher-order tensors, LoRTA overcomes the inherent lower
528   bounds on the number of trainable parameters while offering finer-grained control over adapter
529   size. Our experiments across various benchmarks demonstrate that LoRTA achieves comparable and
530   sometimes superior performance than baselines at a reduced parameter count.
531   Furthermore, we have shown that previous works have implicitly utilized low-rank tensor models
532   with random factors. Nothing precludes our higher-order tensor model from using randomized factors
533   for increased efficiency—a potential direction for future work that could further reduce computational
534   overhead. Lastly, developing more efficient implementations of tensor operations that result in greater
535   memory efficiency also remains a relevant future work direction which could make LoRTA even
536   more suitable for resource-constrained environments.

846
847   Model Compression While these techniques differ from PEFT in that they focus on reducing
848
      the requirements of a trained model rather than efficient adaptation, they offer valuable insights
      for developing more efficient PEFT approaches. Pruning and quantization are key techniques for
849
      compressing neural networks, that have also been extensively applied to LLMs. Pruning removes
850
      less important weights, with some methods achieving high compression rates, e.g. (Ma et al., 2023).
851   Quantization reduces weight precision, decreasing model size and also allowing more efficient
852   operations (Lin et al., 2024a). Knowledge distillation is an alternative approach that involves
853   transferring knowledge from a large “teacher” model to a smaller “student” model (Gu et al., 2024).
854
      Low Rank Training. Exploiting low rank structure to improve efficiency during both training
855
      and inference in deep models has long been studied (Sainath et al., 2013), and also combined with
856
      sparsity (Sprechmann et al., 2015). Recent advancements include Cuttlefish (Wang et al., 2023) and
857   ELRT (Sui et al., 2024).
858
859   Data efficient fine tuning. An alternative approach to reducing fine-tuning costs is to reduce the
860
      amount of data. In this direction, Few-shot and continual learning approaches have been shown to be
      effective in LLM fine-tuning tasks (Lin et al., 2024b; Wang et al., 2024).
861
862   Efficient Architectures Another relevant direction in resource usage is using more efficient model
863   architectures. Mixture of Experts (MoE) technique, implemented in models like Switch Transform-
      ers (Fedus et al., 2022) and GLaM (Du et al., 2022), has shown promise in scaling model capacity

864
      while maintaining computational efficiency by activating only relevant sub-models for given inputs.
865   There is also relevant work on non-transformer architectures, such as RWKV (Peng et al., 2023) and
866   Mamba (Gu & Dao, 2023), which combines the strengths of RNNs and Transformers to achieve
867   efficient inference and training.

      To fairly compare the parameter efficiency of LoRTA with LoRA, we adjust the tensor rank in LoRTA
      to match the effective total tensor rank in LoRA, which is r′ = r × 4L due to LoRA applying a rank
875
      r update to each of the 4L matrices individually. For a given tensor rank, LoRTA reduces the number
876

1042   G    P RACTICAL I MPLICATIONS OF A DAPTER S IZE R EDUCTION
1043
1044   The reduction in adapter size is primarily motivated by the need to improve task-switching efficiency
1045
       and minimize storage requirements in scenarios involving a large number—potentially thousands—of
       adapters. Frequent CPU-GPU transfers for loading adapters in such settings can introduce significant
1046
       overhead. By further compressing parameters, it becomes feasible for thousands of customized models
1047
       to coexist with a base LLM in GPU memory, substantially enhancing scalability and performance in
1048   multi-task environments.
1049
1050   During training, the reduction in GPU memory usage from shrinking optimizer states is marginal for
1051
       parameter reductions beyond LoRA. Memory consumption in these cases is dominated by activations
       and caches stored during forward and backpropagation. Additional memory savings could be achieved
1052
       by compressing activations or gradients, leveraging the low-rank structure of updates, or dynamically
1053
       recomputing them. While our model features fewer trainable parameters and could theoretically
1054   benefit from the efficient tensor CP structure, such as faster training and lower memory usage, these
1055   advantages are not yet realized due to the limitations of our current implementation. Future work will
1056   focus on optimizing this implementation. Future work will address these optimizations. However, the
1057   reduced parameter count already provides lower storage requirements and faster I/O.
1058
       We conducted hardware profiling to compare the performance of our LoRTA implementation against
1059
       LoRA using HuggingFace PEFT. The results demonstrate negligible differences in resource consump-
1060   tion between the two methods. The slight gap in training time for LoRTA can be addressed through
1061   further optimizations, ranging from leveraging tools like Torch Compile, to implementing our CP
1062   tensor adapter model more efficiently.

009                                                    A BSTRACT
010
011             This paper presents a method to explain the internal representation structure of
012             a neural network for image generation. Specifically, our method disentangles
013             primitive feature components from the intermediate-layer feature of the neural
014             network, which ensures that each feature component is exclusively used to gen-
015
                erate a specific set of image regions. In this way, the generation of the entire
                image can be considered as the superposition of different pre-encoded primitive
016
                regional patterns, each being generated by a feature component. We find that the
017
                feature component can be represented as an OR relationship between the demands
018             for generating different image regions, which is encoded by the neural network.
019             Therefore, we extend the Harsanyi interaction to represent such an OR interaction
020             to disentangle the feature component. Experiments show a clear correspondence
021             between each feature component and the generation of specific image regions.

      1       I NTRODUCTION
025
      The interpretability of deep neural networks (DNNs) has received increasing attention along with
026
      the fast development of deep learning. However, there is a clear technical boundary between tech-
027
      niques of explaining a single scalar output score of a DNN1 and methods of explaining the high-
028   dimensional output (e.g., an image) of a DNN. For example, for DNNs for classification, attribution
029   methods (Simonyan, 2013; Shrikumar et al., 2017; Selvaraju et al., 2017) were developed to esti-
030   mate attributions of input variables to the scalar classification confidence. Zeiler & Fergus (2014)
031   and desai & Ramaswamy (2020) visualized inference patterns encoded by a DNN, which determined
032   the scalar classification confidence. In contrast, for image generation, the generated image contains
033   much richer information than a scalar classification confidence, so it is difficult to apply previous
034   explanation techniques designed for a single scalar output of a DNN. Instead, image generation is
035   usually explained by controlling image generation results via input engineering (Härkönen et al.,
036
      2020; Voynov & Babenko, 2020).
037   Therefore, the essence of such difference in the explanation techniques lies in the two facts. (1)
038   The explanation of a single scalar output2 can be considered to explain the the structure of a DNN’s
039   inference logic. (2) In comparison, there is no solid theory developed to explain the internal rep-
040   resentation structure of the neural network for image generation. For example, as Figure 1 shows,
041   both attribution/importance scores of input variables to the classification confidence (Selvaraju et al.,
042
      2017) and interactions between input variables for inference (Ren et al., 2024a; 2023a) all reflect po-
      tential representation structure of the DNN.
043
044   However, how to explain the internal representation structure of an image-generation model has not
045   been sophisticatedly formulated. This problem can be discussed in the following two aspects.
046   First, we can empirically say that most DNNs do not encode image information at a pixel level3 .
047   Instead, image generation of the DNN is conducted somewhat like pasting a set of pre-encoded re-
048   gional patterns, rather than let each pixel be generated independently. Therefore, how to faithfully

078   Second, is it possible for the image generation result to be mathematically represented as the super-
079   position of different primitive regional patterns?
080   Therefore, in order to formulate the representation structure of a given DNN for image generation,
081   in this study, we disentangle the feature f of an intermediate layer of the DNN into different feature
082   components f = f0 + ∆f1 + ∆f2 + ... + ∆fm , each being used to generate a specific subset of
083   image regions. Specifically, as Figure 1 shows, the disentanglement of feature components should
084   satisfy the following two requirements.
085   • Each feature component ∆fi is responsible for generating a specific primitive regional pattern Si .
086
087   • The generation of the entire image can be explained as the superposition of different primitive
088
      regional patterns.
089   In this way, above two requirements ensure the faithfulness of the explanation of the image genera-
090   tion. The generation of a certain subset of image regions is exclusively determined as the superpo-
091   sition of different primitive regional patterns, each being generated by a specific feature component.
092   To this end, we prove that the feature component ∆fi represents a certain OR relationship between
093   the demands for the generation of different image regions. We extend the theory of the Harsanyi in-
094   teraction (Harsanyi, 1958) to mathematically formulate the OR relationship between different prim-
095   itive regional patterns used by a given neural network. Our theory ensures that the generated image
096   can be represented as a linear superposition of these primitive regional patterns.
097
      As Figure 2 shows, the OR interaction means that when the DNN is required to generate any one
098   region of Si , ∆fi must be added into the feature in the intermediate layer.
099
100   In sum, this paper proposes a new method to disentangle feature components from the intermediate
101
      layer of a DNN. It is theoretically guaranteed that each feature component is exclusively used to
      generate a specific set of image regions. Experiments have demonstrated the faithfulness of the
102
      proposed explanation of image generation.

134   Interaction-based DNN explanation. In the field of explainable AI, an emerging question is
135   whether the decision-making process of DNNs can be interpreted as a set of sparse symbolic con-
136
      cepts. To address this, a theoretical system based on the Harsanyi interaction has been proposed
      to explain how symbolic concepts are encoded by DNNs. Over the past three years, about 20 ar-
137
      ticles have been published in the field of explainable AI, which aimed to tackle the mathematical
138
      possibility of explaining DNN inference logic through a limited set of logical patterns. Most of
139   these studies were surveyed by Ren et al. (2024a). Specifically, Sundararajan et al. (2020); Janizek
140   et al. (2021); Tsai et al. (2023) proposed different types of interactions between input variables of
141   a DNN. Ren et al. (2023a) used the Harsanyi dividend (Harsanyi, 1958) to represent the AND in-
142   teraction in a DNN. They also experimentally discovered that DNNs usually encoded a limited set
143   of interactions between input variables, i.e., the sparsity of AND interactions. Li & Zhang (2023)
144   revealed that low-order interactions exhibited higher transferability across different input samples in
145   discriminative neural networks. Ren et al. (2024a) proved the three common conditions under which
146   the sparsity of interactions could be guaranteed. Ren et al. (2023b) proposed a method to learn op-
147
      timal masked states of input variables based on interactions and alleviated the bias of the Shapley
      value caused by the sub-optimal masked states of input variables. Furthermore, Chen et al. (2024)
148
      extracted common interactions across different neural networks, and interactions shared by different
149
      neural networks usually represented generalizable inference patterns. Cheng et al. (2024) proposed
150   to extract the interactions from the intermediate layers of neural networks, so as to illustrate how
151   DNNs gradually learned and forgot inference patterns during forward propagation.
152
153
      In addition, the interaction theory can also explain the representation power of a neural network.
      Specifically, this can be elaborated as follows. Wang et al. (2020) discovered and proved the neg-
154
      ative correlation between DNNs’ adversarial transferability and the interaction inside adversarial
155
      perturbations. Ren et al. (2021) found adversarial attacks primarily affected the high-order interac-
156   tions than low-order interactions. Similarly, Zhou et al. (2024) revealed that low-order interactions
157   tended to generalize better than high-order interactions. Liu et al. (2023) explained the intuition that
158   DNNs learned low-order interactions more easily than high-order interactions. Deng et al. (2022)
159   proved a counter-intuitive bottleneck i.e., bivariate interactions of middle orders were usually not
160   well encoded by a neural network. Ren et al. (2023c) found that Bayesian neural networks (BNNs)
161   tended to avoid encoding complex Harsanyi interactions, compared to normal neural networks. Ren
      et al. (2024b) and Zhang et al. (2024) discovered and proved the dynamics of learning interactions

649
650
651   The Harsanyi dividend was designed as a standard metric to measure interactions between input
652   variables encoded by the network. In this section, we present several desirable axioms and theorems
653   that the Harsanyi dividend interaction I(S) satisfies. This further demonstrates the trustworthiness
654   of the Harsanyi dividend interaction.
655

656

657   the main paper to let u(S) = v(xS ) − v(x∅ ).
658
659   • Efficiency axiom. The output   Pscore of a model can be decomposed into interaction effects of
660
      different patterns, i.e., u(N ) = S⊆N I(S).
661   • Linearity axiom. If we merge output scores of two models u1 and u2 as the output of model u,
662   i.e. ∀S ⊆ N, u(S) = u1 (S) + u2 (S), then their interaction effects Iu1 (S) and Iu2 (S) can also be
663   merged as ∀S ⊆ N, I(S) = Iu1 (S) + Iu2 (S).
664
      • Dummy axiom. If a variable i ∈ N is a dummy variable, i.e., ∀S ⊆ N \ {i}, u(S ∪ {i}) = u(S)
665   then it has no interaction with other variables, ∀∅ =

668

782   F     D ISCUSSION FOR THE INFLUENCE OF THE INPUT CODE
783
784   The BigGAN model uses ResNet GAN architecture, i.e., uses residual blocks, and the input code (a
785   concatenated vector of the latent vector and the class vector) will be fed into the following modules.
786   Therefore, after modifying the intermediate feature of a specific layer, the generated image might
787
      still change if the input code changed, due to the residual structure.
788   Therefore, to validate the influence of the input code is small, we randomly selected input codes,
789   and modify the intermediate feature to a same f . The results are shown in Figure 8.
790
      The experiment result showed that the influence of the input code is relatively small (the background
791   might change a little, but the main parts of the image didn’t change). Therefore, we can consider
792   that most of the information is contained in the intermediate feature.

814   H    E XPERIMENT SETTINGS
815
816   This section includes detailed experiment settings in this paper.
817   In our experiment, we use BigGAN-128 to generate the image categories of ImageNet. Therefore,
818   when calculating the baseline intermediate feature, we randomly generated images of all the 1000
819   classes in ImageNet. And the truncation of the BigGAN model was set to 0.4. After disentangling
820   feature components, we ranked those feature components by their L-2 norm. In order to find out
821   which components are salient, we removed the feature components from the intermediate feature
822   in descending order, utill the generated image is similar to the baseline image. Empirically, we
823
      found that for most samples, the top 60% feature components were salient. And in Figure 5 we
      also found that in log space, about 40% interactions had relatively low interaction strength. When
824
      calculating the distribution of action regions, we performed a weighted sum of the number of feature
825
      components contained in each region, where the weight of a single feature component in a region is
826   its L-1 norm divided by the number of its action regions. When we create heat map visualizations,
827   we apply an exponential operation based on the weights mentioned above to enhance the contrast
828   between different regions.

009                                               A BSTRACT
010
011            In this paper, we propose a generation-detection cycle consistent (G DCC) learn-
012            ing framework that jointly optimizes both layout-to-image (L2I) generation and
013            object detection (OD) tasks in an end-to-end manner. The key of G DCC lies in
014            the inherent duality between the two tasks, where L2I takes all object boxes and
015            labels as input conditions to generate images, and OD maps images back to these
016            layout conditions. Specifically, in G DCC, L2I generation is guided by a layout
017            translation cycle loss, ensuring that the layouts used to generate images align with
018
               those predicted from the synthesized images. Similarly, OD benefits from an im-
               age translation cycle loss, which enforces consistency between the synthesized
019
               images fed into the detector and those generated from predicted layouts. While
020
               current L2I and OD tasks benefit from large-scale annotated layout-image pairs,
021            our G DCC enables more efficient use of unpaired layout data, thereby further en-
022            hancing data efficiency. It is worth noting that our G DCC framework is compu-
023            tationally efficient thanks to the perturbative single-step sampling strategy and a
024            priority timestep re-sampling strategy during training, while maintaining the same
025            inference cost as the original L2I and OD models. Extensive experiments demon-
026            strate that G DCC significantly improves the controllability of diffusion models and
027            the accuracy of object detectors. Our code will be released.

030   1   I NTRODUCTION
031
032   Recent advancements in both layout-to-image (L2I) generation [36] and object detection (OD) [20]
033
      tasks have achieved remarkable success, largely driven by the availability of large-scale annotated
      datasets. Specifically, L2I generation methods incorporate image-based [36; 75; 33] or prompt-
034
      based [6; 73] conditional controls into text-to-image (T2I) diffusion models [51] to achieve more
035
      precise control over the instance placement during image synthesis. These methods train diffusion
036   models to generate realistic images from structured layouts, which include bounding boxes and
037   object class labels that define the spatial positioning and types of objects in the scene. On the other
038   hand, OD takes an image as input and identifies the objects within it by predicting their bounding
039   boxes and class labels. Current advancements have led to significant improvements in the precision
040   of instance placement for L2I generation and the prediction accuracy of OD models.
041
      Although both L2I generation and OD have been extensively studied, few have noticed the strong
042   correlation between these two tasks, i.e., they can be viewed as inverse tasks of each other, where
043   L2I maps layouts to images and OD maps images to layouts. This natural duality between these
044   two tasks has largely been overlooked in previous research. Our key finding is that such duality
045   can be effectively leveraged to improve the performance of both tasks. Specifically, if we map an
046   image to its corresponding layout using an OD model, and then map that layout back to an image
047   using an L2I model, we should ideally recover the original image. Similarly, mapping a layout to
048   an image and then mapping that image back should yield the original layout. This cycle consistency
049   not only enforces tighter alignment between the two tasks but also provides a natural regularization
050
      that enhances the learning processes of both tasks. Moreover, the cycle consistency allows for the
      use of unpaired data, opening up new possibilities for improving data efficiency.
051
052   Based on the above insight, in this paper, we are the first to propose a generation-detection cycle
053   consistent (G DCC) learning framework that jointly optimizes L2I generation and OD in an end-to-
      end manner. In G DCC, consistency is maintained in two directions through two key components:

071
      Our G DCC framework offers several key advantages. First, G DCC enables mutual enhancement be-
      tween L2I generation and OD, setting it apart from earlier approaches that focus on using one task
072
      to improve the other [33; 6; 67]. Such mutual enhancement results in more powerful L2I or OD
073
      models, as opposed to relying on pre-trained ones that are not fully optimized for improving the
074   other task and may introduce errors during the training. Second, G DCC demonstrates superior data
075   efficiency by effectively utilizing unpaired layout data, a capability not achieved by previous meth-
076   ods. Third, G DCC is computationally efficient in both training and inference. Our training process
077   is accelerated by a perturbative single-step sampling strategy and a priority timestep re-sampling
078   strategy, and our inference cost remains unchanged because the original network architectures of the
079   L2I and OD models are preserved.
080   The key contributions of this paper are as follows:
081
082       • We are the first to identify the duality between L2I generation and OD, an insight that has
083         previously been overlooked in the literature.
084       • Inspired by the task duality, we propose a generation-detection cycle consistent (G DCC) frame-
085         work that jointly optimizes both tasks in an end-to-end manner and enables mutual enhancement
086         between them.
087       • Our G DCC demonstrates both data and computational efficiency by allowing for the use of un-
088         paired data and incorporating a perturbative single-step sampling strategy along with a priority
089         timestep re-sampling strategy to accelerate training.
090
091
      Extensive experimental results confirm that G DCC establishes new benchmarks in both L2I gener-
      ation and OD. For L2I generation, it achieves up to a 2.1% FID improvement over baseline L2I
092
      methods, and shows a 2.1% increase in YOLO score, indicating superior alignment between gener-
093
      ated images and conditional layouts. For OD, G DCC achieves up to a 0.9% point gain in AP, further
094   validating the mutual enhancement between L2I generation and OD tasks. These results confirm the
095   effectiveness of our cycle-consistent framework in improving the controllability of diffusion models
096   for image synthesis and the accuracy of object detectors.

      Diffusion Models. Diffusion probabilistic models, first introduced in [56], have witnessed signifi-
101
      cant advancements both theoretically [13; 24; 31] and methodologically [25; 57; 58] in recent years.
102   Latent Diffusion Model [51] further reduces computational costs by applying the diffusion process
103   in the latent feature space rather than the pixel space. Due to their exceptional sample quality,
104   diffusion models have set new standards across various benchmarks [11; 63; 72], including image
105   editing [2; 29; 40; 44; 22], image-to-image transformation [53; 62; 32], and text-to-image (T2I)
106   generation [45; 46; 49; 48; 51; 54; 16]. Recent layout-to-image (L2I) studies seek to achieve more
107   precise control over instance placement by extending pre-trained T2I models with layout condi-
      tions such as bounding boxes and object labels. Early approaches [76; 36; 27; 74; 59; 60] relied

108
      on a closed-set vocabulary from training labels (e.g., COCO [3]) without using text prompts. With
109   the emergence of image-text models such as CLIP [47], open-vocabulary methods became feasi-
110   ble [77; 9; 73; 6; 67; 10; 69; 7]. These methods incorporate layout information as text embeddings
111   into pre-trained T2I diffusion models [51] to achieve more precise control over instance positioning.
112
      In this paper, we boost L2I generation performance from a new perspective by proposing a cycle-
113
      consistent learning framework to achieve mutual benefits with OD, which naturally performs the
114
      inverse mapping of L2I from images to layouts. Our framework is computationally efficient thanks
115   to the perturbative single-step sampling strategy and a priority timestep re-sampling strategy during
116   training, while maintaining the same inference cost as the original L2I and OD models.
117
118
      L2I Generation and OD. Several works have involved both L2I and OD tasks, but primarily use
      one to enhance the other. For example, ControlNet++ [33] uses pre-trained discriminative reward
119
      models to fine-tune controllable diffusion models. However, these reward models are constrained by
120
      their original training data and struggle to adapt to the styles of synthesized images, which hinders
121   their ability to provide more accurate feedback signals for training L2I models. On the other hand,
122   GeoDiffusion [6] demonstrates that OD can benefit from high-quality synthesized data generated by
123   L2I models. DetDiffusion [67] further exploits the synergy between L2I and perceptive models (e.g.,
124   semantic segmentation models) to enhance generation controllability, and show that the synthesized
125   images can improve the performance in downstream tasks such as OD. Despite these advances, the
126   potential of tuning L2I models to generate samples specifically designed to boost OD performance
127   remains underexplored.
128   This paper, for the first time, fully recognizes the duality between L2I and OD tasks and proposes
129   a unified framework G DCC that enables mutual enhancement between the two tasks. Furthermore,
130   in addition to leveraging large-scale paired layout-image data, our framework can effectively utilize
131   unpaired layout data, resulting in superior data efficiency.
132
      Cycle-Consistent Learning. Cycle-consistent learning is a technique that leverages cyclic trans-
133   formations to regularize the training process, ensuring that the data or tasks remain aligned when
134   converted back and forth between representations. It can be applied within a single task through
135   sample cycling, such as object tracking [65; 43; 68], temporal representation learning [14] and im-
136   age generation [78; 30; 71; 37; 8; 33]. It has also been shown to improve model performance
137   across related tasks such as question answering v.s. question generation [61; 55; 34], captioning v.s.
138   grounding [18; 66], vision-language navigation v.s. instruction generation [64], etc.
139   In this paper, we explore the uncharted potential of cycle-consistent learning between L2I generation
140   and OD tasks, wherein the correlation and inherent duality have long been overlooked. These two
141   tasks are seamlessly integrated into an end-to-end cycle-consistent learning framework, where their
142   symmetrical structures provide informative feedback signals that enhance each other. Moreover, our
143   framework allows for the usage of unpaired layout data, leading to superior data efficiency.

185   3.2.1    TASK D UALITY AND C YCLE -C ONSISTENCY
186
187   From §3.1, it becomes evident that L2I and OD can be viewed as inverse tasks of each other, where
188   the input and output of L2I generation correspond to the output and input of OD, respectively.
189   Though largely overlooked in previous research, such task duality can be effectively leveraged to
190
      improve the performance of both tasks through cycle consistency learning.
191   Specifically, if a layout is mapped to an image using an L2I generator G, and then mapped back to a
192   layout using an object detector D, the process should recover the original layout. This forces consis-
193   tency in what we term a layout translation cycle. In this cycle, D remains fixed while G is trained
194   to minimize the discrepancy between the predicted and the original input layouts, ensuring more
195   precise and realistic image generation that faithfully reflects the input layout. Similarly, mapping
196
      an image to a layout and then back again should ideally recover the original image. This ensures
      consistency in an image translation cycle. In this case, G is fixed, and D is trained to minimize
197
      the difference between the predicted and original images, thus enhancing its ability to accurately
198
      predict layouts from images. These two cycle-consistent learning processes improve both G and D
199   in an end-to-end manner similar to GAN [17], with each receiving feedback from the other. In the
200   following, we will present G DCC in both paired (§3.2.2) and unpaired (§3.2.3) data settings.
201

      Priority Timestep Re-Sampling. In the training of DMs, a random timestep t is selected from 1
284   to tmax at each training step, and the model is trained to predict the added noise at this particular
285   timestep. However, in our experiment, since tthre ≪ tmax , the traditional uniform sampling strategy
286   results in a low probability of selecting a t ∈ [1, tthre ] to trigger the layout or image translation cycle
287   loss in Eq. (9) or (11). This leads to slow convergence during training. To alleviate this issue,
288   we propose a priority timestep re-sampling strategy, which applies a re-weighting factor w > 1
289   to prioritize the selection of t ∈ [1, tthre ]. The re-weighted timestep probability density function
290   preweight (t) is defined as follows:
291
292                       \label {equ:reweight} p_{\text {reweight}}(t) = \begin {cases} w / t_{\text {thre}} & \text {if } t \leq t_{\text {thre}} \\ \big (1 - w \cdot {t_{\text {thre}}} / {t_{\text {max}}}\big ) / \big (t_{\text {max}} - t_{\text {thre}}\big ) & \text {otherwise} \end {cases}.                      (12)
293
      This strategy increases the frequency of triggering layout and image translation cycle losses during
294
      training, thus accelerating convergence. The effectiveness of this re-sampling strategy is demon-
295
      strated by the results shown in Table 6b. When combined with the perturbative single-step denoising
296   strategy introduced above, our G DCC becomes significantly more streamlined and efficient.

298   3.2.3    G DCC IN UNPAIRED DATA SETTING
299
300   In addition to leveraging large-scale annotated layout-image pairs to achieve mutual improvement
301   of the L2I generator and object detector, G DCC also facilitates more efficient use of unpaired data,
302
      thereby further enhancing data efficiency. In this section, we explore G DCC learning with layouts as
      the sole training data. To obtain more unpaired layouts, we utilize VisorGPT [70], a recent generative
303
      pre-training model to automatically sample layouts based on its learned visual priors.
304
305   In the unparied data setting, the sampled layout lsyn ∈ RN×5 or the real-world layout lreal ∈ RN×5
306   functions identically to the layout input l in the paried data setting for the L2I generation described
307   in Eq.(5). This allows for the calculation of the layout translation cycle loss LlayoutTC in Eq.(7) and
308
      image translation cycle loss LimageTC in Eq.(10). However, due to the absence of the corresponding
      image x0 , it becomes impossible to calculate Lldm and Lpred , and thus cannot apply the perturbative
309
      single-step sampling and priority timestep re-sampling strategies. To reduce the GPU memory in
310
      this situation, only a subset of the gradients is retained during the T -step samplings for L2I image
311   generation. In summary, the training loss of G reduces to Lgen = LlayoutTC and the training loss of D
312   simplifies to Ldet = LimageTC under the unpaired data setting for G DCC learning. Experiment results
313   are presented in Table 5. Related details are shown in Appendix §D.

      5   C ONCLUSION
532
      In this paper, we propose G DCC, an end-to-end framework that jointly optimizes L2I generation
533
      and OD tasks. By exploring the inherent duality between these two tasks, G DCC facilitates mutual
534
      enhancement of L2I and OD models through the layout and image translation cycle losses. Addi-
535   tionally, G DCC allows for more efficient use of unpaired layout data, thereby further enhancing data
536   efficiency. Notably, our G DCC is computationally efficient thanks to the perturbative single-step
537   sampling and priority timestep re-sampling strategies during training, while maintaining the same
538   inference cost as the original L2I and OD models. Extensive experiments confirm that G DCC signif-
539   icantly improves the controllability of diffusion-based L2I models and the accuracy of OD models.

      Limitation and Future Work. In this work, we explore the inherent duality between layout-to-
773   image (L2I) generation and object detection (OD). However, due to restrictions in computational
774   resources, this duality is not extended to a broader range of controllable T2I generation and discrim-
775   inative models, such as segmentation mask controllable models paired with segmentation models,
776   and depth map controllable models paired with depth models, etc.. In future work, we aspire to
777   expand the end-to-end joint learning framework for broader controllable T2I generation and dis-
778   criminative models. In addition, our experiments in Table2 and Table8 also suggest that our highly
779   realistic generated images aligned with synthesized layouts can benefit the training of object de-
780   tectors. Therefore, another essential future direction deserving of further investigation is the con-
781
      struction of a large-scale synthetic dataset comprising synthesized layouts and their corresponding
      images generated by advanced L2I generation models. Overall, we believe the results presented in
782
      this paper warrant further exploration.
783
784   Social Impact. This work investigates the inherent duality between the L2I generation and OD
785   and introduces G DCC learning framework that jointly optimizes both two tasks in an end-to-end
786   manner. On positive side, the approach advances both L2I generation and OD model accuracy,
787
      leading to more precise scene synthesis and object localization. Improved L2I generation model
      can generate realistic images consistent with layouts, benefiting fields such as content creation and
788
      synthesized dataset construction. Meanwhile, the enhanced OD model offers advantages in areas
789
      like autonomous driving and surveillance systems. For potential negative social impact, the ability
790   to generate highly realistic images could be misused to produce misleading or fake content, raising
791   significant ethical concerns around surveillance, privacy, and the potential for digital manipulation.

      B    DATASETS AND E VALUATION M ETRICS
795
796
      Datasets. Our experiments are conducted on two widely used datasets. COCO-Stuff [3] consists
      of bounding box annotations covering 80 object classes and 91 stuff classes. Following [28; 36; 6],
797
      objects occupying less than 2% of the total image area are ignored, and only images with 3 to 8
798
      objects are used, resulting in a dataset of 74,777 training images and 3,097 validation images. We
799   train on the COCO-Stuff training split and test on COCO 2017 following [6]. NuImages [4] offers
800   bounding box annotations across 10 categories and 6 camera views. We exclude images with more
801   than 22 objects following [6], yielding 60,209 images for training and 14,772 images for validation.
802
      Evaluation Metric. L2I generation models are evaluated using two main criteria: fidelity and train-
803
      ability. Fidelity assesses the consistency between the generated object representations and the au-
804
      thentic distribution of images. Specifically, fidelity quality is measured using the Frechet Inception
805   Distance (FID) [23] from the perceptual perspective, while YOLO score proposed by [36] is used to
806   evaluate the alignment between the generated images and conditional layouts.

922
923   Our G DCC demonstrates efficiency in both paired layout-image and unpaired layout settings. In this
924   section, we focus on the unpaired setting and provide detailed discussions.
925
      Experimental Setup. As mentioned in §3.2.3, we adopt VisorGPT [70], a recent generative pre-
926   training model to automatically sample layouts based on its learned visual priors. More specifically,
927   VisorGPT requires users to input the object names and the number of instances for each image to
928   generate layouts. we first sample synthesized layouts by inputting the class names and the number of
929   instances from each image in the COCO 2017 [38] training set into VisorGPT. This process allows
930   us to obtain corresponding ground truth layouts and synthesized layouts with the same number.
931   To investigate the impact of varying the number of synthesized layouts on performance, we also
932   experiment by randomly increasing or decreasing the number of instances in the COCO annotations,
933   as well as altering the random seed to generate new synthesized layouts. In the end, we obtained
934
      three different ratios of synthesized layouts to ground truth layouts: 1/2, 1, and 2.
935   In the following sections, we present two main experiments. The first involves fine-tuning both the
936   generation model and detection model using the end-to-end G DCC learning framework on the syn-
937   thesized and ground truth layouts, similar to Table5. The second experiment focuses on re-training
938   the detection model using the synthesized data, akin to the generation trainability experiment in
939   Table 2, to evaluate whether these synthesized layouts can further enhance the performance of the
940
      detection model.
941   Additional Results of Fine-tuning in Unpaired Setting on COCO [3]. As shown in Table7, rely-
942   ing solely on synthesized layouts yields a modest performance improvement, albeit not as substantial
943   as when using real-world layouts alone. The utilization of combined layouts results in performance
944   improvements, with the optimal outcome observed when the ratio of synthesized to real-world lay-
945   outs is balanced at 1:1. This suggests that increasing the proportion of synthesized layouts beyond
946
      this ratio does not lead to further performance improvements. Additionally, performance in the
      unpaired setting consistently lags behind that of the paired setting.

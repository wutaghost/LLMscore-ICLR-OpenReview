                                                  A BSTRACT
013
014            Unsupervised Domain Adaptation (UDA) aims to leverage labeled data from a
015            source domain to address tasks in a related but unlabeled target domain. This
016
               problem is particularly challenging when there is a significant gap between
               the source and target domains. Traditional methods have largely focused on
017
               minimizing this domain gap by learning domain-invariant feature representations
018
               using convolutional neural networks (CNNs). However, recent advances in
019            vision transformers, such as the Swin Transformer, have demonstrated superior
020            performance in various vision tasks. In this work, we propose a novel UDA
021            approach based on the Swin Transformer, introducing three key modules to
022            improve domain adaptation. First, we develop a Graph Domain Discriminator
023            that plays a crucial role in domain alignment by capturing pixel-wise correla-
024            tions through a graph convolutional layer, operating on both shallow and deep
025            features in the transformer. This module also calculates the entropy for the
026            key attention features of the attention block to better distinguish between the
027
               source and target domains. Second, we present an Adaptive Double Attention
               module that simultaneously processes Windows and Shifted Windows attention to
028
               increase long-range dependency features. An attention reweighting mechanism
029
               is employed to dynamically adjust the contributions of the attention values,
030            thereby improving feature alignment between domains. Finally, we introduce
031            Cross-Feature Transform, where random Swin Transformer blocks are selectively
032            transformed using our proposed transform module, enhancing the model’s ability
033            to generalize across domains by transferring the source to the target features.
034            Extensive experiments demonstrate that our method improves the state-of-the-
035            art on several challenging UDA benchmarks, confirming the effectiveness of
036            our approach. In particular, our model does not include a task-specific domain
037            alignment module, making it more versatile for various applications.

040   1   I NTRODUCTION
041
042   Deep neural networks (DNNs) have achieved remarkable success in various machine learning tasks,
043   particularly in computer vision (Wang et al., 2022a; Qian et al., 2021; Jiang et al., 2022; Tan et al.,
044   2019; Chen et al., 2021b; Jiang et al., 2021). However, their performance often relies on large
045   amounts of labeled training data, which can be costly and time-consuming to collect (Csurka, 2017;
046   Zhao et al., 2020; Zhang et al., 2020; Oza et al., 2021). To address this, Unsupervised Domain
047   Adaptation (UDA) has emerged as a viable alternative, transferring knowledge from a labeled source
048   domain to an unlabeled target domain and mitigating challenges posed by domain shifts (Bousmalis
049   et al., 2017; Kuroki et al., 2019; Wilson & Cook, 2020; VS et al., 2021).
050   Traditional UDA methods have used Convolutional Neural Networks (CNNs) to align source and
051   target domains by learning transferable features across varying distributions (Kang et al., 2019;
052   Zhang et al., 2019; Jiang et al., 2020; Li et al., 2021b). While these methods have made strides
053   in reducing domain discrepancies through adversarial training and feature normalization, they
      can struggle with complex domain shifts and variability in visual patterns, highlighting ongoing


                                                        1
      Under review as a conference paper at ICLR 2025


054
      challenges in domain generalization and cross-domain alignment (Morerio et al., 2020; Jiang et al.,
055   2020).
056
057   The advent of transformers has revolutionized feature learning in both natural language processing
058
      (NLP) (Vaswani et al., 2017; Devlin et al., 2018) and computer vision (Dosovitskiy et al., 2020; Han
      et al., 2020; He et al., 2021; Khan et al., 2021). The Swin Transformer (Liu et al., 2021) excels
059
      in modeling long-range dependencies and processes images in non-overlapping patches, enabling
060
      effective localized adaptation. This multiscale approach is well-suited for UDA tasks, ensuring
061   robust feature representation and precise domain alignment.
062
063   This work introduces TransAdapter, a novel framework leveraging the Swin Transformer for UDA.
064
      It addresses the limitations of traditional CNN-based methods by incorporating three key modules:
      a Graph Domain Discriminator, an Adaptive Double Attention module, and a Cross-Feature
065
      Transform module. These components enhance domain adaptation performance by facilitating
066
      better alignment and improving local and global feature consistency across domains.
067
068   Contributions of this paper are summarized as follows:
069
070
             • The Graph Domain Discriminator captures both shallow and deep features using graph con-
               volutional layers, enhancing pixel-wise correlation and domain alignment. we incorporate
071
               entropy in the key attention features, preventing the attention mechanism from focusing too
072
               narrowly on specific regions, leading to more balanced and transferable representations.
073
074
             • The Adaptive Double Attention module captures long-range dependencies by simul-
               taneously processing window and shifted window attention. This dual mechanism
075
               maintains global and local features, while an attention reweighting module enhances
076
               feature alignment and overall model performance.
077
078          • The Cross Feature Transform module adapts the Swin Transformer for UDA tasks. By
079
               randomly selecting a transformer block and applying a specialized transform module in
               each iteration, the model dynamically explores different aspects of the feature space. This
080
               enhances domain adaptation and improves performance across diverse datasets.
081
082   In summary, integrating these three modules within the Swin Transformer framework provides a
083   robust solution for UDA, effectively addressing domain shift challenges and advancing the state-of-
084   the-art in domain adaptation for computer vision tasks.

      2.1   U NSUPERVISED D OMAIN A DAPTATION (UDA) AND T RANSFER L EARNING
089
090   Unsupervised Domain Adaptation (UDA) within transfer learning aims to learn transferable
091   knowledge that is generalizable across different domains with varying data distributions. The main
092   challenge lies in addressing the domain shift the discrepancy in the probability distributions between
093   source and target domains.
094
      Early UDA methods, such as Deep Domain Confusion (DDC), focused on learning domain-
095
      invariant characteristics by minimizing the maximum mean discrepancy (MMD) between two
096   domains (Tzeng et al., 2014a). This helped align the marginal distributions of the source and target
097   domain data at a feature level. Long et al. (Long et al., 2015b) enhanced this approach by embedding
098   hidden representations in a reproducing kernel Hilbert space (RKHS) and applying a multiple-kernel
099   variant of MMD to measure the domain distance more effectively. These hidden representations
100   refer to the activations within layers of a deep neural network, where each layer captures different
101   hierarchical features of the input data.
102   To further improve alignment, Long et al. (Long et al., 2017) proposed aligning the joint distributions
103   of multiple domain-specific layers across domains using a joint maximum mean discrepancy
104   (JMMD) metric. These layers refer to the different layers in a neural network, where each layer
105   encodes various aspects of the data, from low-level features in earlier layers to high-level semantic
106   information in deeper layers. The idea is to align not only the marginal distributions of individual
107   layers but also the joint distribution of features across multiple layers, ensuring that both lower-
      level and higher-level representations are aligned between domains. Adversarial learning methods,

      2.2             UDA WITH V ISION T RANSFORMERS
135
136   While transformers have gained popularity in NLP, their application in UDA for vision tasks is
137   still in its early stages. Some recent work has integrated transformers into CNNs to improve
138   domain adaptation, focusing on critical regions of images (Xu et al., 2021a; Yang et al., 2021b).
139   For example, methods such as cross-attention have been used to blend source and target image
140   representations (Chen et al., 2021a), while others employ multibranch architectures to leverage self-
141   attention and cross-attention mechanisms for feature learning and domain alignment (Saito et al.,
142   2019). The Swin Transformer has also been explored in the context of UDA, where its ability
143   to model local and global relationships in images is harnessed for domain adaptation. However,
144
      most of these methods require additional components or specific training strategies to prevent model
      collapse in challenging tasks (Liu et al., 2022; Yang et al., 2021a).

149   This section introduces three key modules: the Graph Domain Discriminator (GDD), Adaptive
150   Double Attention (ADA), and Cross-Feature Transform (CFT). GDD models domain relationships
151   using graphs and attention, ADA enhances feature alignment (Deng et al., 2024) via double attention
152   (Zhang et al., 2022), and CFT boosts feature transfer with cross-attention and dynamic gating.
153   Together, they enable efficient domain alignment. The overall architecture is demonstrated in

      3.2   A DAPTIVE D OUBLE ATTENTION
202
      The adaptive double attention module, shown in Figure 3, processes windows attention and shifted
203   windows attention simultaneously in each transformer block, termed double attention. It employs
204   a cross-attention mechanism between windows and shifted windows features to enhance alignment
205   and interaction, improving long-range dependency capture and adaptation through dynamic attention
206   re-weighting.
207

208
      source features, utilizing a correction block as proposed in (Li et al., 2021a). This block, depicted in
209   Figure 3, modifies the target representation C l (Fxt ,xt shif t ) to align it more closely with the source
210   representation, thereby reducing the domain gap and enhancing adaptation.
211
212

      of the correction block, ∆C l (Fxt ,xt shif t ), adjusts the representation of the target feature according

288   domains. Unlike static methods, the CFT module is applied dynamically after a randomly selected

385
386   Our domain adaptive model’s objective function combines cross-entropy loss for classification, local
387   adaptation loss (strong alignment), and global adaptation loss (weak alignment) The classification
388   layer is a single fully connected layer. For the labeled source domain, the cross-entropy loss is
389
      defined as:
390

      data, and CE(·, ·) denotes the cross-entropy loss, using only the source domain features Fcls .
395

397   entropy loss for local adaptation and the focal loss for global adaptation across both source and

      target domains:

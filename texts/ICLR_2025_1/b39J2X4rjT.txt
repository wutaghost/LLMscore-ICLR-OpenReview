009                                              A BSTRACT
010
011            In this paper, we propose an efficient sparse convolution-based architecture called
012            ESS3D for 3D visual grounding. Conventional 3D visual grounding methods
013            are difficult to meet the requirements of real-time inference due to the two-stage
014            or point-based architecture. Inspired by the success of multi-level fully sparse
015            convolutional architecture in 3D object detection, we aim to build a new 3D vi-
016            sual grounding framework following this technical route. However, as in visual
017            grounding task the 3D scene representation should be deeply interacted with text
018            features, sparse convolution-based architecture is inefficient for this interaction
019
               due to the large amount of voxel features. To this end, we propose text-guided
               pruning (TGP) and completion-based addition (CBA) to deeply fuse 3D scene
020
               representation and text features in an efficient way by gradual region pruning and
021
               target completion. Specifically, TGP iteratively sparsifies the 3D scene repre-
022            sentation and thus efficiently interacts the voxel features with text features by
023            cross-attention. To mitigate the affect of pruning on delicate geometric infor-
024            mation, CBA adaptively fixes the over-pruned region by voxel completion with
025            negligible computational overhead. Compared with previous single-stage meth-
026            ods, ESS3D achieves top inference speed and surpasses previous fastest method
027            by 100% FPS. ESS3D also achieves state-of-the-art accuracy even compared with
028            two-stage methods, with +1.13 lead of Acc@0.5 on ScanRefer, and +5.4 and
029            +5.0 leads on NR3D and SR3D respectively. The code will be released soon.

032   1   I NTRODUCTION
033
034   Incorporating multi-modal information to guide 3D visual perception is a promising direction. In
035   these years, 3D visual grounding (3DVG), also known as 3D instance referencing, has been paid
036   increasing attention as a fundamental multi-modal 3D perception task. The aim of 3DVG is to locate
037   an instance in the scene using 3D bounding box, where this instance is described by a free-form
038   query language. 3DVG is challenging since it requires understanding of both 3D scene and language
039
      description. Recently, with the development of 3D scene perception and vision-language models,
      3DVG methods have shown remarkable progress (Jain et al., 2022; Luo et al., 2022). However, with
040
      3DVG being widely applied in fields like robotics and AR / VR where inference speed is the main
041
      bottleneck, how to construct efficient real-time 3DVG model remains a challenging problem.
042
043   Since the output format of 3DVG is similar with 3D object detection, early 3DVG methods (Yuan
044   et al., 2021; Yang et al., 2021) usually adopt a two-stage framework, which first conducts 3D object
045
      detection to locate all objects in a scene, and then selects the target object by incorporating text
      information. As there are many similarities between 3D object detection and 3DVG (e.g. both of
046
      them need to extract the representation of the 3D scene), there will be much redundant feature
047
      computation during the independent adoption of the two models. As a result, two-stage methods are
048   usually slow and hard to handle real-time tasks. To solve this problem, single-stage methods (Luo
049   et al., 2022) are presented, which generates the bounding box of the target object directly from point
050   clouds. This integrated design makes single-stage methods more compact and efficient. However,
051   current single-stage 3DVG methods mainly build on point-based architecture (Qi et al., 2017), where
052   the feature extraction contains time-consuming operations like furthest point sampling and kNN,
053   especially for large scenes. They also need to aggressively downsample the point features to reduce
      computational cost, which might hurt the geometric information of small and thin objects (Xu et al.,


                                                        1
      Under review as a conference paper at ICLR 2025


054
      2024). Due to these reasons, current single-stage methods are still far from real-time (< 6 FPS) and
055   their performance is inferior to two-stage methods.
056
057   In this paper, we propose a new single-stage framework for 3DVG based on sparse convolution,
058
      namely ESS3D. Inspired by state-of-the-art 3D object detection methods (Rukhovich et al., 2022; Xu
      et al., 2024) which achieves both leading accuracy and speed with multi-level sparse convolutional
059
      architecture, we build the first sparse single-stage 3DVG network. However, different from 3D
060
      object detection, in 3DVG the 3D scene representation should be deeply interacted with text features.
061   Since the amount of voxels is very large in sparse convolution-based architecture, deep multi-modal
062   interaction like cross-attention becomes infeasible due to unaffordable computational cost. To this
063   end, we propose text-guided pruning (TGP), which first utilize text information to jointly sparsify the
064   3D scene representation and enhance the voxel and text features. To mitigate the affect of pruning on
065   delicate geometric information, we further present completion-based addition (CBA) to adaptively
066   fix the over-pruned region with negligible computational overhead. Specifically, TGP prunes the
067   voxel features according to the object distribution. It gradually removes background features and
068   features of irrelevant objects through iterative pruning and feature upsampling, which generates
069
      high-resolution and text-aware voxel features around the target object for accurate bounding box
      prediction. Since pruning may mistakenly remove the representation of target object, CBA utilizes
070
      text features to query a small set of voxel features from the complete backbone features, followed by
071
      pruned-aware addition and voxel concatenation to fix the over-pruned region. We conduct extensive
072   experiments on the popular ScanRefer (Chen et al., 2020) and ReferIt3D (Achlioptas et al., 2020)
073   datasets. Compared with previous single-stage methods, ESS3D achieves top inference speed and
074   surpasses previous fastest single-stage method by 100% FPS. ESS3D also achieves state-of-the-art
075   accuracy even compared with two-stage methods, with +1.13 lead of Acc@0.5 on ScanRefer, and
076   +5.4 and +5.0 leads on NR3D and SR3D respectively.

      2.1   3D V ISUAL G ROUNDING
081
082   3D visual grounding aims to locate a target object within a 3D scene based on natural language de-
083   scriptions. Existing methods are typically categorized into two-stage and single-stage approaches.
084   Two-stage methods follow a detect-then-match paradigm. In the first stage, they independently ex-
085   tract features from the language query using pre-trained language models (Devlin, 2018; Pennington
086   et al., 2014) and predict candidate 3D objects using pre-trained 3D detectors (Qi et al., 2019; Liu
087   et al., 2021). In the second stage, they focus on aligning the vision and text features to identify the
088   target object. Techniques for feature fusion include attention mechanisms with Transformers (He
089
      et al., 2021; Zhao et al., 2021), contrastive learning (Abdelreheem et al., 2022), and graph-based
      matching (Feng et al., 2021). Single-stage methods integrate object detection and feature extrac-
090
      tion, allowing for direct identification of the target object. Methods in this category include guiding
091
      keypoint selection using textual features (Luo et al., 2022), and measuring similarity between words
092   and objects inspired by 2D image-language pre-trained models like GLIP (Li et al., 2022), as in
093   BUTD-DETR (Jain et al., 2022). And methods like EDA (Wu et al., 2023) and G3 -LQ (Wang et al.,
094   2024) advance single-stage 3D visual grounding by enhancing multimodal feature discriminability
095   through explicit text-decoupling, dense alignment, and semantic-geometric modeling.
096
      However, existing two-stage and single-stage methods generally have high computational costs,
097
      hindering real-time applications. Our work aims to address these efficiency challenges by proposing
098   an efficient single-stage method with multi-level sparse convolutional architecture.
099
100
      2.2   M ULTI -L EVEL S PARSE C ONVOLUTIONAL A RCHITECTURES
101
102   Recently, multi-level sparse convolutional architecture has achieved great success in the field of 3D
103   object detection. Built on the voxel-based representation (Wang et al., 2022) and sparse convolution
104   operation (Choy et al., 2019; Graham et al., 2018; Xu et al., 2023), this kind of methods show great
105   efficiency and accuracy when processing scene-level 3D data. GSDN (Gwak et al., 2020) first adopts
106   sparse convolution in 3D object detection by constructing multi-level architecture with generative
107   feature upsampling. FCAF3D (Rukhovich et al., 2022) simplifies the multi-level architecture with
      anchor-free design and rotation-aware object assignment strategy, which achieves leading accuracy


                                                        2
      Under review as a conference paper at ICLR 2025


108
      with even faster speed. Aimed at real-time 3D object detection, TR3D (Rukhovich et al., 2023) fur-
109   ther accelerates FCAF3D by removing unnecessary layers and introducing category-aware proposal
110   assignment method. Additionally, DSPDet3D Xu et al. (2024) introduces the multi-level architec-
111   ture to 3D small object detection and demonstrates great accuracy and efficiency, even being able to
112   process building-level 3D scenes.
113
      Our proposed method draws inspiration from these approaches, utilizing a sparse multi-level archi-
114
      tecture with sparse convolutions and an anchor-free design. This allows for efficient processing of
115   3D data, enabling real-time performance in 3D visual grounding tasks.

196   3.2       T EXT- GUIDED P RUNING
197
198   Though efficient, ESS3D-B exhibits poor performance due to the inadequate interaction between
199   3D scene representation and text features. Motivated by previous 3DVG methods (Jain et al., 2022),
200   a simple solution is to replace Concat with cross-modal attention to update voxel and text features
201
      with two intertwined transformer decoders that process them jointly via cross-attention, as shown
      in Fig. 2 (b). However, different from point-based architectures where the scene representation is
202
      usually aggressively downsampled to control the computational cost, the amount of voxels in multi-
203
      level sparse convolutional framework is very large. In practical implementation, we find that the
204   voxels expand almost exponentially with each upsampling layer, leading to a substantial computa-
205   tional burden for the self-attention and cross-attention of scene features. To address this issue, we
206   introduce text-guided pruning (TGP) to construct ESS3D, as illustrated in Fig. 2 (c). The core idea
207   of TGP is to reduce feature size by pruning redundant voxels and guide the network to gradually
208   focus on the final target based on textual features.
209
      Overall Architecture. TGP can be regarded as a modified version of cross-modal attention, which
210   reduces the amount of voxels before attention operation to reduce the computational cost. To min-
211   imize the affect of pruning on the final prediction, we propose to prune the scene representation
212   gradually. At higher level where the amount of voxels is not too large yet, TGP prunes less voxels.
213   While at lower level where the amount of voxels is significantly increased by upsampling operation,
214   TGP prunes the voxel features more aggressively. The multi-level architecture of ESS3D consists
215   of three levels and includes two feature upsampling operations. Therefore, we correspondingly con-
      figure two TGPs with different functions, which are referred as scene-level TGP (level 3 to 2) and


                                                                                     4

216
      target-level TGP (level 2 to 1) respectively. Scene-level TGP aims to distinguish between objects
217   and the background within the scene, specifically pruning the voxels on background. Target-level
218   TGP focuses on regions mentioned in the text, intending to preserve the target object and referential
219   objects while removing other regions.

      Simplification. Although the above mentioned method can effectively prune the voxel features
      based on text description to reduce the computational cost of cross-modal attention, there are some
247
      inefficient operations in the pipeline: (1) FPS is time-consuming, especially for large scenes; (2)
248
      there are two times of interactions between voxel features and text features, the first is to guide
249   pruning and the second is to enhance the representation, which is a bit redundant. We also empiri-
250   cally observe that the amount of voxels is not large in level 3. To this end, we propose a simplified
251   version of TGP, as shown in Fig. 2 (d). We remove the FPS and merge the two multi-modal in-
252   teractions into one. We also move the merged interaction operation before pruning. In this way,
253   voxel features and text features are first deeply interacted for both feature enhancement and pruning.
254   Because in level 3 the amount of voxels is small and in level 2 / 1 the voxels are already pruned, the
255   computational cost of self-attention and cross-attention is always kept at a relatively low level.
256   Effectiveness of TGP. After pruning, the voxel scale of U1 is reduced to nearly 15% of its origi-
257   nal size without TGP, while the 3DVG performance is significantly boosted. TGP serves multiple
258   functions, including: (1) facilitating the interaction of multi-modal features through cross-attention,
259   (2) reducing the feature scale (amount of voxels) through pruning, and (3) gradually guiding the
260   network to focus on the mentioned target based on text features.

264   During the pruning process, some targets may be mistakenly removed, especially small or narrow
265   objects, as shown in Fig. 3 (b). Therefore, the addition operation between the upsampled pruned
266
      features UlG and backbone features Vl described in Equation (1) play an important role to mitigate
      the affect of over-pruning.
267
268   There are two alternative addition operation: (1) Full Addition. For the intersecting regions of Vl
269   and UlG , features are directly added. For voxel features outside the intersection of UlG and Vl which
      lack corresponding features in the other map, the missing voxel features are interpolated from the

270
      other before addition. Due to pruning process, UlG is sparser than Vl . In this way, full addition can
271   fix almost all the pruned region to avoid any risk. But this operation is computationally heavy and

276   the results of the pruning process. If some important regions are over-pruned, the network might

318   for CBA, and objectness loss as well as bounding box regression loss for the head. Pruning loss,

320
      completion and classification losses are the same, which sets voxels near the target object center as
      positives while leaving others as negatives. For bounding box regression, we use the Distance-IoU
321
      (DIoU) loss. The total loss function is computetd as the sum of these individual losses:

      5     C ONCLUSION
524
525
      In this paper, we present ESS3D, an efficient sparse single-stage method for real-time 3D visual
      grounding. Different architecture from previous 3D visual grounding (3DVG) methods, ESS3D
526
      builds on multi-level sparse convolutional architecture for efficient and fine-grained scene represen-
527
      tation extraction. To enable the interaction between voxel and textual features, we propose text-
528   guided pruning (TGP), which reduces the feature scale and guides the network to progressively
529   focus on the target object. We further introduce completion-based addition (CBA) for adaptive
530   multi-level feature fusion, effectively compensating for instances of over-pruning. Extensive ex-
531   periments demonstrate the effectiveness of our proposed modules, resulting in an efficient 3DVG
532   method that achieves state-of-the-art accuracy and inference speed.
533
      Potential Limitations. Despite of the leading accuracy and inference speed, there are still some
534   limitations of ESS3D. First, the speed of ESS3D is bit slower than ESS3D-B. Although ESS3D
535   utilizes TGP to enable deep interaction between voxel and text features in an efficient way, it un-
536   avoidably introduces additional computational overhead compared with naive concatenation. In the
537   future work, we aim to work on designing new operations for multi-modal feature interaction to
538   replace the heavy cross-attention mechanism. Second, currently the input of 3DVG methods is a
539   reconstructed point clouds. We will work on extending it to online setting with streaming RGB-D
      videos as input, which can support a wider range of practical application.
